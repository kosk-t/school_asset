docs 直下の *_教材.html のファイルを探索して、Front Matter の cover_image の値が　/images/noimage.jpg　のものを対象に順に処理する。
のキーワードを使って、以下の指示通り画像を生成する。

---画像生成指示 start---
3コマ漫画を画像で出力してください。

【テーマ】:
    各教材のFront Matterの　description, keywords, cover_description
【スタイル】: カラー画像。シンプルでかわいいイラスト、コメディ、教育、日本の漫画風
【ストーリー】: 面白いオチを入れる
【レイアウト】:
    - ページ全体のアスペクト比は 1:1.4（幅:高さ）を絶対に厳守する。
    - 読み順は 上から順に縦に3コマを並べる。
    - 文字は横書き
    - 同一コマ内で会話があるときは 先に読ませたいセリフのキャラクターを左側に置く
【キャラクター】
docs\images\データの分析と確率_教材.jpg を参考に、一貫性のあるキャラクターで。
---画像生成指示 end---

生成した画像を docs/imagesに移動し
Front Matter の cover_image にパスを設定する。
