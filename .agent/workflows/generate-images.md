docs 直下の *_教材.html のファイルを探索して、Front Matter の cover_image の値が　/images/noimage.jpg　のものを対象に順に処理する。
以下の指示通り画像を生成する。
画像生成前にまずは出力するプロット（構成案）を、ファイルごとにキャラクター、状況、ビジュアル、セリフなど説明して。
レビュー必須として、承認後に画像を作成。
生成する画像ファイル名はHTMLファイルと同じにして。
指示がプランにすべて入っていることを確実に確認して。
最後にJekyllのビルド可能なことを確認して、できない場合修正して。
作業全体もプランして。
