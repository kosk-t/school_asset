# 数学教材ライブラリ

中学生向けの数学教材を作成・管理するプロジェクトです。

## ディレクトリ構成

```
docs/
├── css/
│   └── education-template.css  # 共通スタイル
├── covers/                     # 表紙画像フォルダ
├── index.html                  # 目次ページ
└── *_教材.html                 # 各教材
assets/
└── education-template.html     # テンプレート
```

## 教材の作成

```
/create-material <テーマ名>
```

## 表紙画像（4コマ漫画）

各教材の表紙として4コマ漫画を使用。画像生成AIで作成する。

### 推奨サイズ
- **幅**: 900px
- **高さ**: 400〜500px（推奨比率 16:9 または 2:1）
- **形式**: PNG または JPG
- **比率**: 自由（アスペクト比を維持して表示、切れない）

### 表紙の追加方法
1. `docs/covers/` フォルダに画像を配置
2. 教材HTMLのタイトル下にあるコメントを解除

```html
<!-- コメントを解除 -->
<div class="cover-image">
    <img src="covers/テーマ名.png" alt="テーマ名">
</div>
```

### 推奨プロンプト（テンプレート）

```
4コマ漫画を画像で出力してください。

【テーマ】: 
    [ここに入れてください]
【スタイル】: シンプルでかわいいイラスト、コメディ、教育、日本の漫画風
【ストーリー】: 面白いオチを入れる

【レイアウト】: 縦に4つを並べる。横が広く、縦は狭く。

【キャラクター】:
- 中学生の男の子または女の子
- 先生または友達（任意）
```

## 技術仕様

- **文字コード**: UTF-8
- **数式**: KaTeX（インライン `\(` `\)`、ディスプレイ `\[` `\]`）
- **図解**: SVG
