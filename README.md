# 数学教材ライブラリ

中学生向けの数学教材を作成・管理するプロジェクトです。

## ディレクトリ構成

```
docs/
├── css/
│   └── education-template.css  # 共通スタイル
├── covers/                     # 表紙画像フォルダ
├── index.html                  # 目次ページ
└── *_教材.html                 # 各教材
assets/
└── education-template.html     # テンプレート
```

## 教材の作成

```
/create-material <テーマ名>
```

## 表紙画像

### 推奨サイズ
- **幅**: 900px（必須ではない、自動で幅100%に調整される）
- **高さ**: 400〜500px（推奨比率 16:9 または 2:1）
- **形式**: PNG または JPG
- **比率**: 自由（アスペクト比を維持して表示、切れない）

### 表紙の追加方法
1. `docs/covers/` フォルダに画像を配置
2. 教材HTMLの先頭にあるコメントを解除

```html
<!-- コメントを解除 -->
<div class="cover-image">
    <img src="covers/テーマ名.png" alt="テーマ名">
</div>
```

### 表紙画像の作成ガイド
- 背景は淡い色（白や薄い青）推奨
- テーマ名を大きく配置
- 関連するイラストや図形を入れると効果的
- 余白を十分に取る

## 技術仕様

- **文字コード**: UTF-8
- **数式**: KaTeX（インライン `\(` `\)`、ディスプレイ `\[` `\]`）
- **図解**: SVG
