1. テーマの定義

地図やゲーム画面では、位置を座標で表します。座標を使えば、2点の真ん中の位置や直線の傾きを計算で求めることができます。

このページでは、座標平面上の直線について以下を学びます：

  • 中点：2点の真ん中の座標を求める
  • 傾き：直線の「右に1進むと、どれだけ上がるか」を計算する
  • 平行な直線：傾きが同じ直線の方程式を作る
中点・傾き・平行の関係を視覚化

2. 覚えるルール

【最重要】傾き ＝ yの変化 ÷ xの変化 → 例2
傾きは「yの変化量÷xの変化量」で求める
📌 中点の公式 → 例1

点 \(A(x_1, y_1)\) と点 \(B(x_2, y_2)\) の中点 \(M\) は：

\[ M\left(\frac{x_1 + x_2}{2},\ \frac{y_1 + y_2}{2}\right) \]

→ x座標どうし、y座標どうしを足して2で割る！

💡 なぜそうなる？ 中点は「2点のちょうど真ん中」なので、各座標の平均になる。平均は「足して個数で割る」から、2点なら足して2で割る。

2点の座標から中点を求める
📌 傾きの公式 → 例2

2点 \(A(x_1, y_1)\), \(B(x_2, y_2)\) を通る直線の傾き \(m\) は：

\[ m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\text{yの増加量}}{\text{xの増加量}} \]

→ 「yの変化」÷「xの変化」で傾きが出る！

💡 なぜそうなる？ 傾きとは「xが1増えるとyがいくつ増えるか」を表す数。AからBに移動するとき、xが\((x_2-x_1)\)増えてyが\((y_2-y_1)\)増えるから、1あたりに直すと\(\frac{y_2-y_1}{x_2-x_1}\)になる。

💡 どちらの点を先にしてもOK！
\(\frac{y_2 - y_1}{x_2 - x_1}\) でも \(\frac{y_1 - y_2}{x_1 - x_2}\) でも同じ結果になる。
ただし、分子と分母で引く順番を揃えること！
例：\(\frac{6-2}{5-1}\) と \(\frac{2-6}{1-5}\) は両方とも答えは1。
📌 直線の方程式 → 例5

傾き \(m\)、y切片 \(c\) の直線は：

\[ y = mx + c \]
  • \(m\)：傾き（直線の急さ）
  • \(c\)：y切片（y軸との交点のy座標）
📌 平行な直線の条件 → 例5
2直線が平行 ⇔ 傾きが等しい

直線 \(L_1\) の傾きが \(m\) のとき、\(L_1\) に平行な直線の傾きも \(m\)

💡 なぜそうなる？ 傾きは直線の「角度（傾き具合）」を表す。平行な2直線は同じ角度で進むから、傾きも同じになる。逆に傾きが違えば、いつか交わってしまうので平行ではない。

平行な2直線は傾きが等しい

3. 例題と解き方

ルールを使って実際に解いてみましょう。

問題設定：直線 L₁ は点 A(2, 3) と B(6, 7) を通る

▶ 中点を求める

例1線分ABの中点Mを求める

点 A(2, 3) と B(6, 7) を結ぶ線分ABの中点Mの座標を求めよ。

解き方：

  1. 2点の座標を確認する → A(2, 3)、B(6, 7)
  2. x座標を足して2で割る → \(\frac{2+6}{2} = \frac{8}{2} = 4\)
  3. y座標を足して2で割る → \(\frac{3+7}{2} = \frac{10}{2} = 5\)
  4. 中点の座標として答える → M(4, 5)

▶ 傾きを求める

例2直線L₁の傾きを求める

点 A(2, 3) と B(6, 7) を通る直線L₁の傾きを求めよ。

解き方：

  1. 2点の座標を確認する → A(2, 3)、B(6, 7)
  2. yの変化を計算する → \(7 - 3 = 4\)
  3. xの変化を計算する → \(6 - 2 = 4\)
  4. 割り算して約分する → \(m = \frac{4}{4} = 1\)

答え：傾き m = 1

例3傾きが負の場合

点 P(1, 5) と Q(4, 2) を通る直線の傾きを求めよ。

解き方：

  1. 2点の座標を確認する → P(1, 5)、Q(4, 2)
  2. yの変化を計算する → \(2 - 5 = -3\)
  3. xの変化を計算する → \(4 - 1 = 3\)
  4. 割り算する → \(m = \frac{-3}{3} = -1\)

答え：傾き m = −1（右下がりの直線）

傾きが負の直線は右下がり
例4分数の傾きになる場合

点 A(1, 2) と B(5, 8) を通る直線の傾きを求めよ。

解き方：

  1. 2点の座標を確認する → A(1, 2)、B(5, 8)
  2. yの変化を計算する → \(8 - 2 = 6\)
  3. xの変化を計算する → \(5 - 1 = 4\)
  4. 割り算して約分する → \(m = \frac{6}{4} = \frac{3}{2}\)

答え：傾き m = \(\frac{3}{2}\)

▶ 平行な直線の方程式を求める

例5L₁に平行で点C(0, 2)を通る直線L₂の方程式

直線L₁（傾き1）に平行で、点C(0, 2)を通る直線L₂の方程式を求めよ。

解き方：

  1. 元の直線L₁の傾きを確認する → \(m = 1\)
  2. 平行なので同じ傾きを使う → L₂の傾きも \(m = 1\)
  3. 通る点からy切片を読み取る → C(0, 2)なので \(c = 2\)
  4. \(y = mx + c\) の形で答える → \(y = 1 \cdot x + 2 = x + 2\)

答え：y = x + 2

4. 簡単にするコツ

✏️ コツ1：座標を(x, y)の順に書く
混乱しないように、必ず「横、縦」の順で書く習慣をつける。
✏️ コツ2：傾きは「分数のまま」で計算
\(\frac{3}{2}\) のような分数でも無理に小数にしない。計算が楽になる。
✏️ コツ3：平行なら傾きをコピー
「平行な直線を求めよ」と言われたら、元の直線の傾きをそのまま使う。
✏️ コツ4：y切片は問題文から読み取る
「y軸と(0, k)で交わる」と書いてあれば、それがy切片 \(c = k\) になる。

5. よくあるつまづき

⚠️ つまづき1：傾きの分子と分母を逆にする

ダメな例： \(\frac{x_2 - x_1}{y_2 - y_1}\) と計算する

対策： 「yの変化が上（分子）、xの変化が下（分母）」と覚える

⚠️ つまづき2：中点で「足す」を「引く」にする

ダメな例： \(\frac{x_2 - x_1}{2}\) と計算

対策： 中点は「足して2で割る」。傾きと混同しない！

⚠️ つまづき3：負の数の引き算ミス

ダメな例： A(2, −3), B(4, 1) で \(1 - 3 = -2\) とする

対策： \(1 - (-3) = 1 + 3 = 4\)。マイナスの前はカッコで囲む！

⚠️ つまづき4：y切片の読み間違い

ダメな例： (0, 5)を通ると書いてあるのにy切片を0にする

対策： (0, c)のcがy切片。y軸との交点のy座標を使う。

⚠️ つまづき5：約分を忘れる

ダメな例： 傾きを\(\frac{6}{4}\)のまま答える

対策： 必ず約分して\(\frac{3}{2}\)にする。分数は最も簡単な形に！

6. 仕上げチェックリスト

中点は「足して2で割る」で計算したか
傾きは「yの変化÷xの変化」で正しく計算したか
負の数の計算で符号ミスをしていないか
平行な直線は元の直線と同じ傾きを使ったか
分数は約分して最も簡単な形にしたか

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

点 A(1, 4) と B(7, 10) を結ぶ線分ABの中点Mの座標を求めよ。

答えを見る

x座標：\(\frac{1+7}{2} = \frac{8}{2} = 4\)

y座標：\(\frac{4+10}{2} = \frac{14}{2} = 7\)

答え：M(4, 7)

練習2

点 P(2, 1) と Q(8, 7) を通る直線の傾きを求めよ。

答えを見る

\(m = \frac{7-1}{8-2} = \frac{6}{6} = 1\)

答え：傾き m = 1

📌 まとめ：中点は「足して半分」、傾きは「yの変化÷xの変化」、平行は「傾き同じ」！

おまけ：実践ストーリー「謎解き宝探しゲーム」

📖 謎解き宝探しゲーム

リクとサキは公園で謎解き宝探しイベントに参加中。公園の地図には座標が書かれていて、次のヒントが与えられた：

「2本の平行な道の交差点を結べ。その中点に宝あり。」

ヒント1：道Aは点(2, 1)を通り、傾きが2の直線。道Bは点(6, 5)を通り、道Aに平行。

まず道Aと道Bの方程式を求める：

道Aは傾き2、点(2, 1)を通る。y切片を求めると：

\[ 1 = 2 \times 2 + c \quad \Rightarrow \quad c = 1 - 4 = -3 \]

よって 道A：y = 2x − 3

道Bは道Aに平行なので傾きは同じ2。点(6, 5)を通るから：

\[ 5 = 2 \times 6 + c \quad \Rightarrow \quad c = 5 - 12 = -7 \]

よって 道B：y = 2x − 7

ヒント2：道Aとx軸の交点P、道Bとx軸の交点Qを求める（y = 0を代入）

\[ \text{点P}：0 = 2x - 3 \quad \Rightarrow \quad x = 1.5 \quad \Rightarrow \quad P(1.5, 0) \]
\[ \text{点Q}：0 = 2x - 7 \quad \Rightarrow \quad x = 3.5 \quad \Rightarrow \quad Q(3.5, 0) \]

中点の公式で宝の位置を計算！

\[ M = \left(\frac{1.5 + 3.5}{2},\ \frac{0 + 0}{2}\right) = \left(\frac{5}{2},\ 0\right) = (2.5, 0) \]

「座標(2.5, 0)...つまりx軸上の2.5の位置だ！」とリクが叫んだ。二人が走っていくと、ベンチの下に宝箱が！

中身は...公園の売店で使える500円クーポンだった。「数学って役に立つじゃん！」とサキは笑い、二人はソフトクリームを買いに行くのであった。
