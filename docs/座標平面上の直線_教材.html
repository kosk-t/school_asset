1. テーマの定義

地図上で「A地点とB地点の真ん中で待ち合わせ」と言われたら、どうやって場所を決めますか？座標を使えば、正確な位置を計算で出せます。

このページでは、座標平面上の直線に関する重要な公式を学びます：

  • 中点公式：2点のちょうど真ん中の座標を求める
  • 直線の傾き：直線の傾き具合（勾配）を計算する
  • 平行な直線：傾きが同じ直線の式を作る

2. 覚えるルール

【最重要】座標は「xはx同士、yはy同士」で計算する！
📌 中点の公式 → 例1
\[ M = \left( \frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2} \right) \]

2点 \(A(x_1, y_1)\) と \(B(x_2, y_2)\) の中点 \(M\) の座標。

💡 覚え方： 「足して2で割る」＝「平均をとる」と同じです。

x座標もy座標も、足して2で割ると真ん中になる
📌 直線の傾き（勾配） → 例2
\[ m = \frac{y_2 - y_1}{x_2 - x_1} \]

💡 意味： \(x\) が1増えたとき、\(y\) がどれだけ増えるか（変化の割合）。

英語では "Rise over Run"（Run進んでRise上がる）と覚えます。

横にどれだけ進んで(Run)、縦にどれだけ上がるか(Rise)
📌 平行な直線 → 例3
\[ m_1 = m_2 \]

2つの直線が平行なら、傾きは同じです。

例：\(y = 2x + 3\) と \(y = 2x - 5\) は平行（傾きがどちらも2）。

📌 直線の方程式
\[ y - y_1 = m(x - x_1) \]

点 \((x_1, y_1)\) を通り、傾き \(m\) の直線。

💡 使い道： 傾きと通る1点がわかれば、直線の式が作れます。

3. 例題と解き方

公式を使って計算してみましょう。符号（プラス・マイナス）に注意！

▶ 中点を求める

例12点の中点を求める

2点 \(A(2, 4)\) と \(B(6, 10)\) の中点 \(M\) の座標を求めよ。

解き方：

  1. x座標同士を足して2で割る：\(\frac{2+6}{2}\)
  2. y座標同士を足して2で割る：\(\frac{4+10}{2}\)
\[ M = \left( \frac{2+6}{2}, \frac{4+10}{2} \right) = \left( \frac{8}{2}, \frac{14}{2} \right) = (4, 7) \]

▶ 傾きを求める

例22点を通る直線の傾き

2点 \((1, 3)\) と \((4, 9)\) を通る直線の傾きを求めよ。

解き方：

  1. \(x\) の増加量（Run）：\(4 - 1 = 3\)
  2. \(y\) の増加量（Rise）：\(9 - 3 = 6\)
  3. 傾き = Rise ÷ Run
\[ m = \frac{9 - 3}{4 - 1} = \frac{6}{3} = 2 \]

▶ 平行な直線の式を求める

例3平行な直線の式

点 \((2, 5)\) を通り、直線 \(y = 3x + 1\) に平行な直線の方程式を求めよ。

解き方：

  1. 平行なので、求める直線の傾きも \(3\)
  2. 傾き \(3\) で点 \((2, 5)\) を通る直線の式を作る
  3. 公式 \(y - y_1 = m(x - x_1)\) に代入
\[ y - 5 = 3(x - 2) \] \[ y - 5 = 3x - 6 \] \[ y = 3x - 1 \]

4. 簡単にするコツ

✏️ コツ1：中点は「平均」と唱える
「足して2で割る」はテストの平均点計算と同じ。xとyそれぞれで平均を出すだけ。
✏️ コツ2：傾きは「右に1進んだら、上にいくつ？」
傾き2なら「右に1、上に2」。傾き-3なら「右に1、下に3」。図を描くと間違えない。
✏️ コツ3：平行＝傾きコピー
「平行な直線」と言われたら、元の式から傾き（xの前の数字）だけコピーして使う。切片（後ろの数字）は無視。

5. よくあるつまづき

⚠️ つまづき1：引き算の順番を逆にする

ダメな例： 傾き計算で \(\frac{y_2 - y_1}{x_1 - x_2}\) としてしまう（分母と分子で順序が逆）

対策： 必ず「後 - 前」または「2 - 1」で統一する。

⚠️ つまづき2：マイナスの計算ミス

ダメな例： \(3 - (-2)\) を \(3 - 2\) と計算してしまう

対策： マイナスを引くときはプラスになる！ \(3 - (-2) = 3 + 2 = 5\)

⚠️ つまづき3：xとyを逆にする

ダメな例： 中点計算で \(\frac{x+y}{2}\) としてしまう

対策： 座標は「xはxと、yはyと」ペアにする。

6. 仕上げチェックリスト

中点の座標は、元の2点の「間」にあるか（図で確認）
傾きの分母はxの増加量、分子はyの増加量になっているか
平行な直線なら、傾きは同じ数字になっているか
マイナスの符号計算を間違えていないか

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

2点 \((-2, 1)\) と \((4, 9)\) の中点を求めよ。

答えを見る

x座標：\(\frac{-2+4}{2} = \frac{2}{2} = 1\)

y座標：\(\frac{1+9}{2} = \frac{10}{2} = 5\)

答え：(1, 5)

練習2

点 \((1, 2)\) を通り、傾きが \(4\) の直線の方程式を求めよ。

答えを見る

\(y - 2 = 4(x - 1)\)

\(y - 2 = 4x - 4\)

\(y = 4x - 2\)

答え：\(y = 4x - 2\)

📌 まとめ：中点は平均、傾きは変化の割合、平行は傾き同じ！

おまけ：デカルトの夢「天井のハエから座標が生まれた」

🦟 天井のハエを見て思いついた大発見

17世紀のフランス。哲学者で数学者のルネ・デカルトは、体が弱く、朝はベッドで寝ながら考え事をするのが日課でした。

ある日、天井を這い回るハエを見て、ふと思いました。

「あのハエの位置を、言葉だけで正確に誰かに伝えるにはどうすればいいだろう？」

彼は気づきました。「部屋の隅を基準にして、横に何メートル、縦に何メートル進んだ場所、と言えばいいじゃないか！」

これが「座標（Coordinate）」の誕生です。

それまでの幾何学（図形）と代数学（計算）は全く別の学問でした。しかし、デカルトが「点を数（座標）で表す」ことを発明したおかげで、図形を計算で解けるようになったのです。

この「座標平面」は、デカルトの名前（Descartes）にちなんで「デカルト座標系（Cartesian coordinate system）」と呼ばれています。

もしあの時、デカルトが早起きしてハエを見ていなかったら、今のスマホのGPSもCGアニメーションも存在しなかったかもしれません。二度寝もたまには役に立つのです。

※デカルトが座標を考案した逸話として有名ですが、実際にはフェルマーなども同時期に研究していました。しかし、座標幾何学を体系化した功績はデカルトにあります（ルネ・デカルト）。
