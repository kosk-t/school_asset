📐 座標平面上の直線（中点・傾き・平行）

中学生・基礎〜標準レベル

1. テーマの定義／最重要の考え方

座標平面上の直線では、点や直線を座標で表し、計算で性質を調べます。主に以下の3つを扱います：

  • 中点：2点の真ん中の座標
  • 傾き：直線の「右に1進むと、どれだけ上がるか」を表す数
  • 平行な直線：傾きが同じ直線
🎯 最重要の考え方：傾きが等しい ⇔ 2直線は平行
中点・傾き・平行の関係を視覚化

2. 覚えるルール

【最重要】平行な2直線の傾きは等しい
📌 中点の公式

点 \(A(x_1, y_1)\) と点 \(B(x_2, y_2)\) の中点 \(M\) は：

\[ M\left(\frac{x_1 + x_2}{2},\ \frac{y_1 + y_2}{2}\right) \]

→ x座標どうし、y座標どうしを足して2で割る！

📌 傾きの公式

2点 \(A(x_1, y_1)\), \(B(x_2, y_2)\) を通る直線の傾き \(m\) は：

\[ m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\text{yの増加量}}{\text{xの増加量}} \]

→ 「yの変化」÷「xの変化」で傾きが出る！

💡 どちらの点を先にしてもOK！
\(\frac{y_2 - y_1}{x_2 - x_1}\) でも \(\frac{y_1 - y_2}{x_1 - x_2}\) でも同じ結果になる。
ただし、分子と分母で引く順番を揃えること！
例：\(\frac{6-2}{5-1}\) と \(\frac{2-6}{1-5}\) は両方とも答えは1。
📌 直線の方程式

傾き \(m\)、y切片 \(c\) の直線は：

\[ y = mx + c \]
  • \(m\)：傾き（直線の急さ）
  • \(c\)：y切片（y軸との交点のy座標）
📌 平行な直線の条件
2直線が平行 ⇔ 傾きが等しい

直線 \(L_1\) の傾きが \(m\) のとき、\(L_1\) に平行な直線の傾きも \(m\)

3. 簡単にするコツ

✏️ コツ1：座標を(x, y)の順に書く
混乱しないように、必ず「横、縦」の順で書く習慣をつける。
✏️ コツ2：傾きは「分数のまま」で計算
\(\frac{3}{2}\) のような分数でも無理に小数にしない。計算が楽になる。
✏️ コツ3：平行なら傾きをコピー
「平行な直線を求めよ」と言われたら、元の直線の傾きをそのまま使う。
✏️ コツ4：y切片は問題文から読み取る
「y軸と(0, k)で交わる」と書いてあれば、それがy切片 \(c = k\) になる。

4. 代表的な計算例

問題設定：直線 L₁ は点 A(2, 3) と B(6, 7) を通る
例1線分ABの中点を求める

公式適用：x座標とy座標それぞれ足して2で割る

\[ M = \left(\frac{2+6}{2},\ \frac{3+7}{2}\right) = \left(\frac{8}{2},\ \frac{10}{2}\right) = (4, 5) \]

答え：M(4, 5)

例2直線L₁の傾きを求める

公式適用：（yの変化）÷（xの変化）

\[ m = \frac{7 - 3}{6 - 2} = \frac{4}{4} = 1 \]

答え：傾き m = 1

例3L₁に平行で点C(0, 2)を通る直線L₂の方程式

考え方：

  • 平行 → 傾きは同じで \(m = 1\)
  • C(0, 2)を通る → y切片は \(c = 2\)
\[ y = 1 \cdot x + 2 = x + 2 \]

答え：y = x + 2

例4傾きが負の場合

点 P(1, 5) と Q(4, 2) を通る直線の傾き：

\[ m = \frac{2 - 5}{4 - 1} = \frac{-3}{3} = -1 \]

答え：傾き m = −1（右下がりの直線）

例5分数の傾きになる場合

点 A(1, 2) と B(5, 8) を通る直線の傾き：

\[ m = \frac{8 - 2}{5 - 1} = \frac{6}{4} = \frac{3}{2} \]

答え：傾き m = \(\frac{3}{2}\)（約分を忘れずに！）

5. 基本的な計算手順

▶ 中点を求める手順

1
2点の座標を確認する \((x_1, y_1)\), \((x_2, y_2)\)
2
x座標どうしを足して2で割る：\(\frac{x_1 + x_2}{2}\)
3
y座標どうしを足して2で割る：\(\frac{y_1 + y_2}{2}\)
4
中点の座標として答える

▶ 傾きを求める手順

1
2点の座標を確認する \((x_1, y_1)\), \((x_2, y_2)\)
2
yの変化を計算：\(y_2 - y_1\)
3
xの変化を計算：\(x_2 - x_1\)
4
割り算して約分：\(m = \frac{y_2 - y_1}{x_2 - x_1}\)
短い例： A(3, 1), B(7, 9) の傾き

→ \(m = \frac{9-1}{7-3} = \frac{8}{4} = 2\) → 答え：2

▶ 平行な直線の方程式を求める手順

1
元の直線の傾き \(m\) を求める（または問題から読み取る）
2
平行な直線も同じ傾き \(m\) を使う
3
通る点や切片から \(c\) を決める
4
\(y = mx + c\) の形で答える
短い例： 傾き3の直線に平行で、(0, 5)を通る直線

→ 傾きは3、y切片は5 → 答え：y = 3x + 5

6. よくあるつまづき

⚠️ つまづき1：傾きの分子と分母を逆にする

ダメな例： \(\frac{x_2 - x_1}{y_2 - y_1}\) と計算する

対策： 「yの変化が上（分子）、xの変化が下（分母）」と覚える

⚠️ つまづき2：中点で「足す」を「引く」にする

ダメな例： \(\frac{x_2 - x_1}{2}\) と計算

対策： 中点は「足して2で割る」。傾きと混同しない！

⚠️ つまづき3：負の数の引き算ミス

ダメな例： A(2, −3), B(4, 1) で \(1 - 3 = -2\) とする

対策： \(1 - (-3) = 1 + 3 = 4\)。マイナスの前はカッコで囲む！

⚠️ つまづき4：y切片の読み間違い

ダメな例： (0, 5)を通ると書いてあるのにy切片を0にする

対策： (0, c)のcがy切片。y軸との交点のy座標を使う。

⚠️ つまづき5：約分を忘れる

ダメな例： 傾きを\(\frac{6}{4}\)のまま答える

対策： 必ず約分して\(\frac{3}{2}\)にする。分数は最も簡単な形に！

7. 仕上げチェックリスト

中点は「足して2で割る」で計算したか
傾きは「yの変化÷xの変化」で正しく計算したか
負の数の計算で符号ミスをしていないか
平行な直線は元の直線と同じ傾きを使ったか
分数は約分して最も簡単な形にしたか
📌 まとめ：中点は「足して半分」、傾きは「yの変化÷xの変化」、平行は「傾き同じ」！
