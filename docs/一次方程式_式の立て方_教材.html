1. テーマの定義

「りんご3個とみかん5個で350円」「兄は弟より3歳年上」…こんな文章問題、見たことあるよね？方程式は解けるのに、文章題になると手が止まる人は多い。でも大丈夫、「式の立て方」にはパターンがあるんだ。

このページでは、文章問題を方程式に変換する力を身につけます：

  • 何を x とおくか：求めたいものを文字で表す
  • 等しい量を見つける：「＝」でつなげる関係を探す
  • パターン別の立式法：代金・個数・速さ・年齢・割合

2. 覚えるルール

【最重要】立式の3ステップ：①求めたいもの → x　②等しい量を探す　③式を立てる
立式の3ステップ
📌 パターン1：代金の問題 → 例1

キーワード：「○円の品物を△個買った」「合計○円」「おつり○円」

\[ \text{（単価）} \times \text{（個数）} = \text{（代金）} \]

→ よくある等式： 買った金額の合計 ＝ 払った金額、または 払った金額 − 買った金額 ＝ おつり

📌 パターン2：個数の問題 → 例2

キーワード：「○個ずつ配る」「△個余る」「□個足りない」

\[ \text{（全体の個数）} = \text{（1人あたり）} \times \text{（人数）} \pm \text{（余り/不足）} \]

→ 注意：「余る」ならプラス、「足りない」ならマイナス

📌 パターン3：速さ・時間・距離の問題 → 例3

キーワード：「時速○km」「△時間かかった」「□km離れている」

\[ \text{（距離）} = \text{（速さ）} \times \text{（時間）} \]

→ よくある等式： 行きの距離 ＝ 帰りの距離、A地点までの距離 ＋ B地点までの距離 ＝ 全体の距離

📌 パターン4：年齢の問題 → 例4

キーワード：「○年後」「△年前」「□歳年上/年下」

→ ポイント： 年齢差は何年経っても変わらない！ 「3歳年上」は永遠に3歳年上

\[ \text{（現在の年齢）} + n = \text{（n年後の年齢）} \]
📌 パターン5：割合・%の問題 → 例5

キーワード：「○%増える」「△割引」「□倍になる」

\[ \text{（もとの量）} \times \frac{\text{割合}}{100} = \text{（割合に相当する量）} \]

→ 変換： 10%増 → 1.1倍、20%引き → 0.8倍、3割増 → 1.3倍

💡 「等しい量」の見つけ方

文章題で一番難しいのは「何と何が等しいか」を見つけること。コツは「〜は〜である」「合計で〜になる」「〜と同じ」という表現を探すこと。

例えば「りんご3個とみかん5個で350円」なら、「りんごの代金 + みかんの代金 = 350円」という等式が隠れている。

3. 例題と解き方

パターンごとに立式の方法を練習しよう。「求めたいもの → x」「等しい量 → 等式」の流れを意識！

▶ 代金の問題

例1買い物の問題

1個120円のりんごを何個か買い、1000円札を出したらおつりが280円だった。りんごは何個買ったか。

解き方：

  1. 求めたいもの：りんごの個数 → x 個とおく
  2. 等しい量を探す：「払った金額 − 買った金額 = おつり」
  3. 式を立てる：
    • りんごの代金：\(120x\) 円
    • おつり：\(1000 - 120x = 280\)
\[ 1000 - 120x = 280 \]

（解くと \(x = 6\)、答え：6個）

▶ 個数の問題

例2配る問題

ある個数のあめを、生徒に5個ずつ配ると3個余り、6個ずつ配ると4個足りない。生徒は何人いるか。

解き方：

  1. 求めたいもの：生徒の人数 → x 人とおく
  2. 等しい量を探す：あめの総数は同じ！
  3. 2通りで表す：
    • 5個ずつ配って3個余る → 総数は \(5x + 3\)
    • 6個ずつ配ると4個足りない → 総数は \(6x - 4\)
\[ 5x + 3 = 6x - 4 \]

（解くと \(x = 7\)、答え：7人）

▶ 速さの問題

例3往復の問題

家から学校まで行くのに、行きは分速60mで歩き、帰りは分速80mで歩いたら、往復で35分かかった。家から学校までの距離を求めよ。

解き方：

  1. 求めたいもの：家から学校までの距離 → x mとおく
  2. 等しい量を探す：「行きの時間 + 帰りの時間 = 35分」
  3. 式を立てる：
    • 行きの時間：\(\frac{x}{60}\) 分
    • 帰りの時間：\(\frac{x}{80}\) 分
\[ \frac{x}{60} + \frac{x}{80} = 35 \]

（解くと \(x = 1200\)、答え：1200m）

▶ 年齢の問題

例4年齢の問題

現在、父は42歳、子は12歳である。父の年齢が子の年齢の2倍になるのは何年後か。

解き方：

  1. 求めたいもの：何年後か → x 年後とおく
  2. x年後の年齢：
    • 父：\(42 + x\) 歳
    • 子：\(12 + x\) 歳
  3. 等しい量：「父の年齢 = 子の年齢の2倍」
\[ 42 + x = 2(12 + x) \]

（解くと \(x = 18\)、答え：18年後）

▶ 割合の問題

例5割引の問題

定価の20%引きで売ったら、売値は960円になった。定価はいくらか。

解き方：

  1. 求めたいもの：定価 → x 円とおく
  2. 20%引き = 80% = 0.8倍
  3. 等しい量：「定価 × 0.8 = 売値」
\[ 0.8x = 960 \]

（解くと \(x = 1200\)、答え：1200円）

4. 簡単にするコツ

✏️ コツ1：キーワードをマーキングする
文章題を読んだら、まず「求めたいもの」と「等しい」「合計」「○○は△△である」などのキーワードに下線を引こう。
✏️ コツ2：単位を揃える
「時速○km」と「距離△m」が混在していたら、どちらかに統一してから立式する。単位がバラバラだと計算ミスのもと。
✏️ コツ3：図や表を描く
特に速さの問題や年齢の問題は、図や表を描くと関係が見えやすい。「行き」「帰り」の矢印を描いたり、「現在」「x年後」の表を作ったり。
✏️ コツ4：答えを代入して確かめる
解いた後、答えを元の文章題に当てはめて確認。「りんご6個で720円、おつり280円」→ 1000円払ったから OK！

5. よくあるつまづき

⚠️ つまづき1：何を x にするか迷う

ダメな例： 「りんごの代金」「買った金額」など、関係する量をどれも x にしようとして混乱

対策： 「何を求めよ」と聞かれているものを x にする！問題文の最後に答えがある。

⚠️ つまづき2：「余る」と「足りない」を逆にする

ダメな例： 「3個足りない」を \(+3\) にしてしまう

対策： 「足りない」は「もっと必要」→ マイナスで表す。「余る」は「多い」→ プラス。

⚠️ つまづき3：割合の変換ミス

ダメな例： 「20%引き」を「0.2倍」にしてしまう

対策： 「○%引き」は「(100−○)%」= 「(1−○/100)倍」。20%引きなら 0.8倍！

⚠️ つまづき4：速さ・時間・距離の関係を忘れる

ダメな例： 時間 = 距離 × 速さ としてしまう

対策： 「は・じ・き」で覚える。距離(き) = 速さ(は) × 時間(じ)。時間 = 距離 ÷ 速さ。

6. 仕上げチェックリスト

問題文の「求めるもの」を x と正しくおけた
「等しい量」を見つけて等式を作れた
単位を揃えてから立式できた
「余る/足りない」「○%引き」を正しく変換できた
答えを元の文章に代入して確認した

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

1冊150円のノートを何冊か買い、1000円札を出したら100円のおつりがきた。ノートは何冊買ったか。

答えを見る

立式： ノートの冊数を x とすると、おつり = 払った金額 − ノートの代金 より

\[ 1000 - 150x = 100 \]

\[ 150x = 900 \]

\[ x = 6 \]

答え：6冊

練習2

兄と弟がいて、兄は弟より4歳年上である。5年後には兄の年齢が弟の年齢の1.5倍になる。現在の弟の年齢を求めよ。

答えを見る

立式： 弟の現在の年齢を x 歳とすると、兄は (x + 4) 歳

5年後：弟 (x + 5) 歳、兄 (x + 9) 歳

「兄 = 弟の1.5倍」より

\[ x + 9 = 1.5(x + 5) \]

\[ x + 9 = 1.5x + 7.5 \]

\[ 1.5 = 0.5x \]

\[ x = 3 \]

答え：3歳

📌 まとめ：文章題は「求めるもの → x」「等しい量 → 等式」の2ステップで立式できる！

おまけ：歴史ストーリー「3700年前の数学パピルス」

📖 リンド数学パピルス（紀元前1650年頃、古代エジプト）

今から約3700年前、古代エジプトのある書記官アーメス（アハメスとも呼ばれる）が、一本のパピルス（紙のようなもの）に数学の問題を書き写していた。

このパピルス、後に発見者の名前から「リンド数学パピルス」と呼ばれることになる。長さ5m以上もある巻物で、85問もの数学問題が書かれていた。

その中にこんな問題がある：

「ある量とその7分の1を足すと19になる。ある量を求めよ」

これ、現代の一次方程式で書くと...

\[ x + \frac{x}{7} = 19 \]

解くと \(x = 16\frac{5}{8}\)。古代エジプト人は、この未知の量を「アハ」（エジプト語で「量」の意味）と呼んでいた。

驚くべきことに、彼らは「試しの値を入れて調整する」という方法で、方程式を解いていた。現代の「検算」に近い発想だ！

「まず7を入れてみる → 7 + 1 = 8。でも答えは19だから、7を 19/8 倍すればいい！」という具合。

この方法、実は効率的で、古代エジプトの建築や測量に大活躍した。ピラミッドの正確な設計も、こうした計算技術に支えられていたのだ。

※リンド数学パピルスは現在、大英博物館に所蔵されています（Wikipedia: リンド数学パピルス）。
