1. テーマの定義

「面積が12㎡の長方形の花壇を作りたい」「ボールを投げ上げたとき何秒後に地面に落ちるか」——こうした問題を解くには、2乗を含む方程式を解く必要があります。

このページでは、二次方程式の解き方を学びます：

  • 二次方程式とは何か：\(ax^2 + bx + c = 0\) の形の方程式
  • 3つの解法：因数分解・平方完成・解の公式
  • 使い分けの基準：どの方法をいつ使うか

2. 覚えるルール

二次方程式とは、\(x\) の2乗を含む方程式です：

\[ ax^2 + bx + c = 0 \quad (a \neq 0) \]
🎯 最重要：どの方法でも「=0」の形にして左辺を変形する！
状況に応じて使い分けよう
📌 零積の法則（因数分解の基本）
\[ A \times B = 0 \quad \Rightarrow \quad A = 0 \ \text{または} \ B = 0 \]

積が0なら、どちらかが0。因数分解で解くときに使う！

💡 なぜそうなる？ 0でない数どうしをかけると、必ず0以外の答えになる（例：\(2 \times 3 = 6\)）。だから積が0になるには、少なくとも一方が0でなければならない。

📌 因数分解のパターン（まず共通因数をチェック！）
  • 共通因数でくくる：\(2x^2 + 4x = 2x(x + 2)\) → 例2
  • 完全平方：\(x^2 \pm 2ax + a^2 = (x \pm a)^2\)
  • 和と差の積：\(a^2 - b^2 = (a + b)(a - b)\) → 例3

共通因数がなければ、x²の係数で方法が変わる：

📌 和と積で因数分解（a = 1 のとき） → 例1

\(x^2 + bx + c\) の形なら、「足して b、かけて c」になる2数を探す

\[ x^2 + bx + c = (x+p)(x+q) \quad \text{のとき} \quad p+q=b,\ pq=c \]

💡 なぜそうなる？ \((x+p)(x+q)\) を展開すると \(x^2 + (p+q)x + pq\) になる。これが \(x^2 + bx + c\) と一致するには、\(p+q=b\) かつ \(pq=c\) であればよい。

📌 たすき掛けで因数分解（a ≠ 1 のとき） → 例4

\(ax^2 + bx + c\) で \(a \neq 1\) なら、たすき掛けで組み合わせを探す

\[ ax^2 + bx + c = (px + q)(rx + s) \quad \text{のとき} \quad pr = a,\ qs = c,\ ps + qr = b \]
たすき掛けの手順
  1. \(a\)（x²の係数）と \(c\)（定数項）をそれぞれ2数の積に分解
  2. 斜めに掛けて足す（たすき掛け）
  3. 足した結果が \(b\)（xの係数）になる組み合わせを選ぶ
  4. \((px + q)(rx + s)\) の形に書く
合わなければ別の組み合わせを試す！
📌 平方完成の公式 → 例5
\[ x^2 + ax = \left(x + \frac{a}{2}\right)^2 - \left(\frac{a}{2}\right)^2 \]

ポイント：\(x\) の係数を半分にして2乗を作り、足した分を引く

💡 なぜ係数を半分にする？ \((x+p)^2\) を展開すると \(x^2 + 2px + p^2\) になる。\(x\)の係数は\(2p\)なので、元の式の\(x\)の係数\(a\)と比べると \(a = 2p\)、つまり \(p = \frac{a}{2}\)（半分）になる。

\(x\) の係数（6）を半分（3）にして完成させる
📌 解の公式 → 例6
\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

覚え方：「マイナス b、プラスマイナス、ルートの中は b² 引く 4ac、割る 2a」

📌 判別式 \(D = b^2 - 4ac\) → 例7
  • \(D > 0\)：異なる2つの実数解
  • \(D = 0\)：1つの実数解（重解）
  • \(D < 0\)：実数解なし

💡 なぜ判別式で解の個数がわかる？ 解の公式 \(x = \frac{-b \pm \sqrt{D}}{2a}\) でルートの中身が\(D\)。\(D > 0\)ならルートが計算でき±で2つの解、\(D = 0\)なら\(\pm 0\)で解は1つ、\(D < 0\)なら負の数のルートは実数にないので解なし。

3. 例題と解き方

ルールを使って実際に解いてみましょう。

▶ 因数分解で解く

例1因数分解で解く（基本パターン）

\(x^2 - 5x + 6 = 0\) を解く

解き方：

  1. すでに \(=0\) の形なので、左辺を因数分解する
  2. 「足して -5、かけて 6」になる2数を探す → -2 と -3
  3. 因数分解して零積の法則を使う
\[ x^2 - 5x + 6 = (x - 2)(x - 3) = 0 \] \[ x - 2 = 0 \ \text{または} \ x - 3 = 0 \] \[ x = 2, \ 3 \]
例2因数分解で解く（共通因数あり）

\(x^2 + 4x = 0\) を解く

解き方：

  1. 定数項がない → 共通因数 \(x\) があるのでくくり出す
  2. 零積の法則で、各因数 = 0 とする
\[ x^2 + 4x = x(x + 4) = 0 \] \[ x = 0 \ \text{または} \ x + 4 = 0 \] \[ x = 0, \ -4 \]

注意： \(x = 0\) を忘れやすい！両辺を \(x\) で割ると \(x = 0\) が消えてしまうので、必ず因数分解で解くこと。

例3因数分解で解く（和と差の積）

\(x^2 - 16 = 0\) を解く

解き方：

  1. \(x\) の1次の項がない → \(a^2 - b^2\) の形を探す
  2. \(x^2 - 16 = x^2 - 4^2\) なので「和と差の積」\((a+b)(a-b)\) を使う
  3. 零積の法則を適用
\[ x^2 - 16 = (x + 4)(x - 4) = 0 \] \[ x + 4 = 0 \ \text{または} \ x - 4 = 0 \] \[ x = \pm 4 \]
例4たすき掛けで因数分解して解く

\(2x^2 + 5x + 3 = 0\) を解く

解き方：

  1. \(x^2\) の係数が1でない → たすき掛けで因数分解
  2. \(x^2\) の係数 2 を「2×1」、定数項 3 を「3×1」に分解
  3. 斜めに掛けて足した結果が \(x\) の係数（5）になる組み合わせを探す
  4. 零積の法則を適用
\[ (2x + 3)(x + 1) = 0 \] \[ 2x + 3 = 0 \ \text{または} \ x + 1 = 0 \] \[ x = -\frac{3}{2}, \ -1 \]

▶ 平方完成で解く

例5平方完成で解く

\(x^2 + 4x - 5 = 0\) を解く

解き方：

  1. 定数項を右辺に移動
  2. \(x\) の係数（4）を半分にして2乗 → \(2^2 = 4\) を両辺に足す
  3. 左辺を \((x + 2)^2\) の形にまとめる
  4. 両辺の平方根をとる（±を忘れずに）
  5. \(x\) について解く
\[ x^2 + 4x = 5 \quad \text{（定数項を右辺へ）} \] \[ x^2 + 4x + 4 = 5 + 4 \quad \text{（両辺に } 2^2 = 4 \text{ を足す）} \] \[ (x + 2)^2 = 9 \] \[ x + 2 = \pm 3 \quad \text{（平方根をとる）} \] \[ x = -2 + 3 = 1, \quad x = -2 - 3 = -5 \]

▶ 解の公式で解く

例6解の公式で解く

\(2x^2 + 3x - 2 = 0\) を解く

解き方：

  1. 係数を読み取る：\(a = 2, b = 3, c = -2\)
  2. 判別式 \(D = b^2 - 4ac\) を計算
  3. 解の公式 \(x = \frac{-b \pm \sqrt{D}}{2a}\) に代入
  4. 計算して約分
\[ D = 3^2 - 4 \cdot 2 \cdot (-2) = 9 + 16 = 25 \] \[ x = \frac{-3 \pm \sqrt{25}}{2 \cdot 2} = \frac{-3 \pm 5}{4} \] \[ x = \frac{-3 + 5}{4} = \frac{1}{2}, \quad x = \frac{-3 - 5}{4} = -2 \]
例7判別式で解の個数を判定

\(x^2 - 4x + 4 = 0\) の解の個数を調べ、解を求める

解き方：

  1. 係数を読み取る：\(a = 1, b = -4, c = 4\)
  2. 判別式 \(D = b^2 - 4ac\) を計算して解の個数を判定
  3. \(D = 0\) なら重解、解の公式で求める
\[ D = (-4)^2 - 4 \cdot 1 \cdot 4 = 16 - 16 = 0 \]

\(D = 0\) なので解は1つ（重解）

\[ x = \frac{-(-4)}{2 \cdot 1} = \frac{4}{2} = 2 \]

（検算：\((x-2)^2 = x^2 - 4x + 4 = 0\) ✓）

4. 簡単にするコツ

✏️ コツ1：まず共通因数をくくる
\(2x^2 + 6x = 0\) → \(2x(x + 3) = 0\) のように、最初に共通因数を出す。
✏️ コツ2：係数が小さいときは因数分解を試す
係数が小さい整数なら、因数分解が一番早い。「足して・かけて」で2数を探す。
✏️ コツ3：因数分解できないときは解の公式
無理に因数分解しようとせず、解の公式を使う。どんな問題でも解ける万能ツール。
✏️ コツ4：右辺を0にしてから解く
\(x^2 = 5x + 6\) → \(x^2 - 5x - 6 = 0\) のように、まず右辺を0にする。
✏️ コツ5：平方完成は頂点も分かる一石二鳥
グラフの頂点座標も求めたいときは平方完成。\(y = a(x-p)^2 + q\) の頂点は \((p, q)\)。

5. よくあるつまづき

⚠️ つまづき1：共通因数の \(x = 0\) を忘れる

ダメな例： \(x^2 + 3x = 0\) で \(x = -3\) だけ答える

対策： \(x(x+3) = 0\) なので \(x = 0\) と \(x = -3\) の2つ！

⚠️ つまづき2：符号のミス

ダメな例： \((x-2)(x+3) = 0\) から \(x = -2, 3\) と答える

対策： \(x - 2 = 0\) なら \(x = 2\)、\(x + 3 = 0\) なら \(x = -3\)

⚠️ つまづき3：解の公式で分母を忘れる

ダメな例： \(x = -b \pm \sqrt{b^2-4ac}\) と書いて \(2a\) で割り忘れる

対策： 分母 \(2a\) は分子全体（\(-b\) も \(\pm\sqrt{}\) も）にかかる

⚠️ つまづき4：右辺を0にしていない

ダメな例： \(x^2 = 3x + 10\) のまま因数分解しようとする

対策： まず \(x^2 - 3x - 10 = 0\) にしてから因数分解

⚠️ つまづき5：判別式で4acの符号ミス

ダメな例： \(c\) が負のとき、\(-4ac\) が正になることを忘れる

対策： \(D = b^2 - 4 \times a \times c\) で符号も込みで代入

6. 仕上げチェックリスト

右辺を0にしてから解いたか
共通因数があるときは \(x = 0\) の解を忘れていないか
符号（＋と−）は正しいか
解の公式で分母 \(2a\) を忘れていないか
答えを元の方程式に代入して検算したか

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の二次方程式を因数分解で解きなさい：\(x^2 - 7x + 12 = 0\)

答えを見る

足して -7、かけて 12 → -3 と -4

\((x - 3)(x - 4) = 0\)

答え：\(x = 3, 4\)

練習2

次の二次方程式を解の公式で解きなさい：\(x^2 + 2x - 5 = 0\)

答えを見る

\(a = 1, b = 2, c = -5\)

\(D = 4 - 4 \cdot 1 \cdot (-5) = 4 + 20 = 24\)

\(x = \frac{-2 \pm \sqrt{24}}{2} = \frac{-2 \pm 2\sqrt{6}}{2} = -1 \pm \sqrt{6}\)

答え：\(x = -1 + \sqrt{6}, -1 - \sqrt{6}\)

📌 まとめ：「=0」にして因数分解、ダメなら解の公式！

おまけ：歴史ストーリー「4000年前の数学者たち」

📖 4000年前の数学者たち（紀元前2000年バビロニア）

1930年、イラクの遺跡から発掘された古い粘土板。解読してみると...

考古学者：「は？これ...二次方程式じゃん」
数学史家：「しかも解き方まで書いてある。紀元前2000年の粘土板に」
全員：「...マジか」

バビロニア（現在のイラク）では、4000年も前に既に二次方程式を解いていたのだ！

【粘土板に書かれていた問題】

「長方形の畑がある。縦と横を足すと14メートル、面積は45平方メートル。縦と横の長さを求めよ」

現代風に書くと：

\[ x + y = 14 \] \[ xy = 45 \]

これを解くと...\(y = 14 - x\) を代入して：

\[ x(14 - x) = 45 \] \[ x^2 - 14x + 45 = 0 \]

二次方程式だ！

【バビロニア人の解き方】

彼らは解の公式を知らなかった。でも「平方完成」に似た方法で解いていた：

  1. \(x + y\) の半分（7）を計算
  2. \(7^2 = 49\) を計算
  3. \(49 - 45 = 4\) を計算
  4. \(\sqrt{4} = 2\) を計算
  5. \(7 + 2 = 9\) と \(7 - 2 = 5\) が答え

検算：\(9 + 5 = 14\) ✓、\(9 \times 5 = 45\) ✓

驚くべき事実：
• 彼らは位置記数法（桁の概念）も持っていた（エジプトより進んでいた）
• 60進法を使っていた（今の時間・角度の60分、60秒はここから！）
• \(\sqrt{2}\) の近似値を小数点以下5桁まで正確に計算していた

【でも、なぜ失われた？】

バビロニア文明が衰退すると、この知識は長い間忘れ去られた。ヨーロッパで二次方程式が再発見されるのは、なんと3000年後の中世。

もしバビロニアの知識が途切れなかったら、人類の数学はもっと早く進歩していたかもしれない。
