1. テーマの定義

因数分解のやり方は知っている。でも問題を見たとき、「どのパターンを使えばいいの？」と迷ったことはないだろうか？

このページでは、パターンを見抜く判断力を身につけます：

  • 5秒で判断：式を見た瞬間に使うパターンがわかる
  • 判断フローチャート：迷わないための決定木
  • 5つのパターン：共通因数・和と差の積・完全平方・和と積・たすき掛け
💡 なぜ「パターン判別」が重要？
因数分解は二次方程式を解く最速の武器。でも正しいパターンを選べなければ時間をロスする。パターンを瞬時に見抜ければ、テストで大きな差がつく！

2. 覚えるルール

🎯 最重要：まず「共通因数」をチェック！その後で公式パターンを探せ！
共通因数のチェックが最優先！その後で公式パターンを探す
📌 パターン①：共通因数をくくり出す【最優先】 → 例1
\[ ab + ac = a(b + c) \]

見分け方：全ての項に同じ因数が含まれている

→ だから結果は：共通因数を外に出し、残りをカッコ内に書く

✅ メリット：最優先でチェック。これをやらないと他のパターンが見えないことも多い！

💡 考え方を見る

「分配法則」の逆。\(a(b + c) = ab + ac\) を右から左に読むと、共通因数 \(a\) を外に出す操作になる。数字も文字もまとめてくくれる。

📌 パターン②：和と差の積（2項のとき） → 例2
\[ a^2 - b^2 = (a + b)(a - b) \]

見分け方：2項だけで、両方が「何かの2乗」になっている

→ だから結果は：それぞれの平方根を取って、(a+b)(a-b) の形にする

✅ メリット：一瞬で因数分解完了。2項のときは真っ先にチェック！

💡 考え方を見る

(a+b)(a-b) を展開すると \(a^2 - ab + ab - b^2 = a^2 - b^2\)。真ん中の項が消えるから「2乗 - 2乗」の形になる。

📌 パターン③：完全平方（3項で定数項が「半分の2乗」） → 例3
\[ a^2 + 2ab + b^2 = (a + b)^2 \] \[ a^2 - 2ab + b^2 = (a - b)^2 \]

見分け方：「1次の項の係数÷2」を2乗すると定数項になる

例：\(x^2 + 6x + 9\) → \((6÷2)^2 = 9\) ✓ → 完全平方！

→ だから結果は：\((x ± 3)^2\) の形（符号は1次の項と同じ）

✅ メリット：気づければ一瞬。二次方程式の重解パターンにもつながる。

💡 考え方を見る

\((a + b)^2 = a^2 + 2ab + b^2\)。1次の項 \(2ab\) の係数の半分が \(b\) で、その2乗 \(b^2\) が定数項。だから「半分の2乗」で判定できる。

📌 パターン④：和と積（x²の係数が1のとき） → 例4
\[ x^2 + (p+q)x + pq = (x + p)(x + q) \]

見分け方：\(x^2\) の係数が1で、完全平方ではない

見つけ方：「足して（1次の項の係数）、かけて（定数項）」になる2数を探す

→ だから結果は：見つけた2数 \(p, q\) を使って \((x + p)(x + q)\) の形にする

✅ メリット：最もよく使うパターン。慣れれば10秒で見つかる。

💡 考え方を見る

\((x + p)(x + q) = x^2 + (p+q)x + pq\) を展開して比較。1次の項は「和」、定数項は「積」になっている。だから「足して○、かけて△」で探せる。

📌 パターン⑤：たすき掛け（x²の係数が1でないとき） → 例5
\[ ax^2 + bx + c = (mx + p)(nx + q) \quad (mn = a, pq = c) \]

見分け方：\(x^2\) の係数が1でない

見つけ方：\(a\) と \(c\) を2数に分解し、たすき掛けで \(b\) になる組み合わせを探す

→ だから結果は：見つけた組み合わせで \((mx + p)(nx + q)\) の形にする

✅ メリット：慣れれば強力。でも見つからなければ解の公式へ切り替える。

💡 考え方を見る

\((mx + p)(nx + q) = mnx^2 + (mq + np)x + pq\)。つまり \(a = mn\)、\(c = pq\)、\(b = mq + np\)。たすき掛け図で「クロスして足して \(b\) になる」組み合わせを探す。

📌 パターン比較まとめ
パターン 式の特徴 公式 所要時間
共通因数 全項に共通因数 \(ab + ac = a(b+c)\) 5秒
和と差の積 2項で両方が2乗 \(a^2 - b^2\) 5秒
完全平方 定数項=半分の2乗 \((a \pm b)^2\) 10秒
和と積 x²の係数が1 \((x+p)(x+q)\) 15秒
たすき掛け x²の係数が1でない \((mx+p)(nx+q)\) 30秒〜

3. 例題と解き方

「どのパターンを使うか」の判断に注目して例題を解きましょう。

▶ パターン①：共通因数をくくり出す

例1共通因数パターン

\(6x^2 + 9x\) を因数分解する

判断（5秒）：

  1. 全ての項に \(3x\) が含まれている → 共通因数をくくり出す！

解き方：

  1. 共通因数 \(3x\) を外に出す
  2. 各項を \(3x\) で割った結果をカッコ内に書く
\[ 6x^2 + 9x = 3x(2x + 3) \]

所要時間：約5秒

例1b共通因数 → さらに因数分解

\(2x^2 - 8\) を因数分解する

判断（5秒）：

  1. 全ての項に \(2\) が含まれている → まずくくり出す
  2. くくった後に他のパターンが適用できるかチェック！

解き方：

  1. 共通因数 \(2\) をくくり出す
  2. カッコ内が \(x^2 - 4\) → 和と差の積！
\[ 2x^2 - 8 = 2(x^2 - 4) = 2(x + 2)(x - 2) \]

ポイント：くくった後も他のパターンをチェックすること！

▶ パターン②：和と差の積

例2和と差の積パターン

\(25x^2 - 16\) を因数分解する

判断（5秒）：

  1. 2項だけ
  2. \(25x^2 = (5x)^2\)、\(16 = 4^2\) で両方が2乗 → 和と差の積！

解き方：

  1. それぞれの平方根を取る：\(5x\) と \(4\)
  2. (和)(差) の形にする
\[ 25x^2 - 16 = (5x)^2 - 4^2 = (5x + 4)(5x - 4) \]

所要時間：約5秒

▶ パターン③：完全平方

例3完全平方パターン

\(x^2 + 10x + 25\) を因数分解する

判断（10秒）：

  1. 1次の項の係数は \(10\)
  2. \((10 \div 2)^2 = 5^2 = 25\) → 定数項と一致！ → 完全平方！

解き方：

  1. 「半分」の \(5\) を使う
  2. 1次の項が正なので \((x + 5)^2\)
\[ x^2 + 10x + 25 = (x + 5)^2 \]

所要時間：約10秒

例3b完全平方パターン（マイナス）

\(x^2 - 14x + 49\) を因数分解する

判断（10秒）：

  1. 1次の項の係数は \(-14\)
  2. \((-14 \div 2)^2 = (-7)^2 = 49\) → 定数項と一致！ → 完全平方！

解き方：

\[ x^2 - 14x + 49 = (x - 7)^2 \]

ポイント：1次の項がマイナスなら \((x - ○)^2\) になる

▶ パターン④：和と積

例4和と積パターン

\(x^2 + 5x + 6\) を因数分解する

判断（10秒）：

  1. \(x^2\) の係数が \(1\)
  2. \((5 \div 2)^2 = 6.25 \neq 6\) → 完全平方ではない
  3. → 和と積！「足して5、かけて6」の2数を探す

解き方：

  1. 6の約数ペア：(1,6), (2,3)
  2. 足して5になるのは \(2 + 3 = 5\) ✓
\[ x^2 + 5x + 6 = (x + 2)(x + 3) \]

所要時間：約15秒

例4b和と積パターン（異符号）

\(x^2 + 2x - 15\) を因数分解する

判断：

  1. 「足して2、かけて\(-15\)」の2数を探す
  2. かけてマイナス → 異符号の2数！

解き方：

  1. 15の約数ペア：(1,15), (3,5)
  2. 差が2になるのは \(5 - 3 = 2\) ✓
  3. かけて\(-15\) → \(+5\) と \(-3\)
\[ x^2 + 2x - 15 = (x + 5)(x - 3) \]

ポイント：定数項がマイナスなら「差」で考える！

▶ パターン⑤：たすき掛け

例5たすき掛けパターン

\(2x^2 + 7x + 3\) を因数分解する

判断（10秒）：

  1. \(x^2\) の係数が \(2 \neq 1\) → たすき掛け！
  2. \(2 = 1 \times 2\)、\(3 = 1 \times 3\) の分解を試す

解き方：

\[ 2x^2 + 7x + 3 = (x + 3)(2x + 1) \]

所要時間：約30秒（慣れれば15秒）

▶ 複合パターン

例6複合パターン

\(3x^2 - 12x + 12\) を因数分解する

判断（10秒）：

  1. まず共通因数チェック → 全項に \(3\) がある → くくり出す！
  2. くくった後の式をさらに因数分解

解き方：

  1. 共通因数 \(3\) をくくり出す：\(3(x^2 - 4x + 4)\)
  2. カッコ内は \((-4 \div 2)^2 = 4\) → 完全平方！
\[ 3x^2 - 12x + 12 = 3(x^2 - 4x + 4) = 3(x - 2)^2 \]

ポイント：共通因数をくくった後が本番！他のパターンを忘れずにチェック

4. 簡単にするコツ

✏️ コツ1：共通因数は「最優先」
どんな式でも、まず共通因数がないかチェック。これを飛ばすと他のパターンが見えなくなる。
✏️ コツ2：2項なら「和と差の積」を疑う
項が2つだけの式は、まず \(a^2 - b^2\) の形かチェック。両方が何かの2乗になっていれば一瞬！
✏️ コツ3：完全平方は「半分の2乗」で見抜く
\(x^2 + bx + c\) の形で、\(c = (b/2)^2\) なら完全平方。暗算できる！

例：\(x^2 - 8x + 16\) → \(16 = (-8/2)^2 = 16\) ✓

✏️ コツ4：定数項の約数を先にリストアップ
和と積で探すとき、定数項の約数ペアを書き出すと速い。

例：\(c = 12\) → (1,12), (2,6), (3,4) とその符号違い

✏️ コツ5：たすき掛けは「約数が少ない」ものから
\(x^2\) の係数や定数項の約数が少ない場合、たすき掛けは数回で見つかる。約数が多いと大変なので、見つからなければ解の公式へ。

5. よくあるつまづき

⚠️ つまづき1：共通因数をくくり忘れる

ダメな例：\(2x^2 + 6x + 4\) をいきなりたすき掛けしようとする

対策：まず \(2\) をくくり出す！ → \(2(x^2 + 3x + 2) = 2(x + 1)(x + 2)\)

⚠️ つまづき2：くくった後を因数分解し忘れる

ダメな例：\(3x^2 - 27 = 3(x^2 - 9)\) で終わり

対策：カッコ内 \(x^2 - 9\) は和と差の積！ → \(3(x + 3)(x - 3)\)

⚠️ つまづき3：完全平方を見逃す

ダメな例：\(x^2 - 6x + 9\) で「足して-6、かけて9」を探す

対策：\(9 = (-6/2)^2 = 9\) ✓ → 完全平方 \((x - 3)^2\) で一発！

⚠️ つまづき4：符号を間違える（和と積）

ダメな例：\(x^2 - 3x - 10\) で「足して-3、かけて10」と考える

対策：定数項は \(-10\) なので「足して-3、かけて\(-10\)」→ \((+2)\) と \((-5)\)

⚠️ つまづき5：たすき掛けに時間をかけすぎる

ダメな例：\(x^2 + 5x + 7\) を5分間たすき掛けで探す

対策：「足して5、かけて7」の整数ペアは存在しない → 因数分解できない式！解の公式へ。

7. 練習問題

まず「どのパターンを使うか」を判断してから因数分解しよう！

練習1：パターンを選んで因数分解

\(4x^2 - 9\) を因数分解しなさい

答えを見る

判断：2項で、\(4x^2 = (2x)^2\)、\(9 = 3^2\) → 和と差の積！

\(4x^2 - 9 = (2x)^2 - 3^2 = (2x + 3)(2x - 3)\)

答え：\((2x + 3)(2x - 3)\)

練習2：パターンを選んで因数分解

\(x^2 - 8x + 16\) を因数分解しなさい

答えを見る

判断：\((-8 \div 2)^2 = 16\) → 定数項と一致！ → 完全平方！

\(x^2 - 8x + 16 = (x - 4)^2\)

答え：\((x - 4)^2\)

練習3：パターンを選んで因数分解

\(5x^2 - 20x + 15\) を因数分解しなさい

答えを見る

判断：まず共通因数 \(5\) をくくり出す → さらに因数分解

\(5x^2 - 20x + 15 = 5(x^2 - 4x + 3)\)

カッコ内：「足して-4、かけて3」→ \(-1\) と \(-3\)

\(= 5(x - 1)(x - 3)\)

答え：\(5(x - 1)(x - 3)\)

📌 まとめ：「共通因数 → 2項なら和差 → 完全平方チェック → 和と積/たすき掛け」の順でチェック！
