1. テーマの定義

円や球は、建物のドーム、ボール、ピザなど身近なところにたくさんあります。これらの大きさや周囲の長さを計算できると、材料の量や必要なスペースがわかります。

このページでは、円と球に関する以下の内容を学びます：

  • 円の基本要素：半径・直径・円周率の関係
  • 円の計算：円周の長さと面積の求め方
  • 球の計算：表面積と体積の求め方
  • 等分の考え方：球を分けたときの1部分の面積

2. 覚えるルール

【最重要】半径と直径の関係 \(d = 2r\)（直径は半径の2倍）を常に意識！
円周：\(C = 2\pi r = \pi d\)　　円の面積：\(S = \pi r^2\)
中心O、半径r、直径d、円周Cの関係
📌 円の公式 → 例1 → 例3
  • 直径と半径の関係：\(d = 2r\) または \(r = \dfrac{d}{2}\)
  • 円周の長さ：\(C = 2\pi r = \pi d\)
  • 円の面積：\(S = \pi r^2\)
\[ \text{円周} = 2\pi r, \quad \text{円の面積} = \pi r^2 \]

💡 なぜそうなる？

  • 円周：円周率\(\pi\)は「円周÷直径」と定義される数。だから逆に、円周 = 直径 × \(\pi\) = \(2r \times \pi\)
  • 面積：円を細かいおうぎ形に切って並べ替えると、高さ\(r\)、横幅\(\pi r\)（円周の半分）の長方形に近づく。だから面積 ≒ \(r \times \pi r = \pi r^2\)
📌 球の公式 → 例4
  • 球の表面積：\(S = 4\pi r^2\)（円の面積の4倍！）
  • 球の体積：\(V = \dfrac{4}{3}\pi r^3\)
  • 大円の円周：球の中心を通る断面の円周 = \(2\pi r\)
\[ \text{球の表面積} = 4\pi r^2, \quad \text{球の体積} = \frac{4}{3}\pi r^3 \]

💡 なぜ表面積が円の面積の4倍？ 古代ギリシャの数学者アルキメデスが発見した関係。球を「円柱の側面」と比べると、同じ半径・高さの円柱の側面積（\(2\pi r \times 2r = 4\pi r^2\)）と球の表面積が等しくなる。これは偶然ではなく、球の曲がり具合が関係している。

球は中心から表面までの距離がすべて等しい立体
📌 等分の考え方 → 例6

球の表面を \(n\) 等分するとき、1部分の面積：

\[ \text{1部分の面積} = \frac{4\pi r^2}{n} \]

例：野球ボールの皮が2枚 → 1枚の面積 = \(\dfrac{4\pi r^2}{2} = 2\pi r^2\)

📌 円周率 \(\pi\) の扱い
  • 厳密には無限に続く小数（\(\pi = 3.14159265...\)）
  • 計算では \(\pi \approx 3.14\) または \(\pi = 3.1416\) を使うことが多い
  • 「\(\pi\) のままで答えよ」と指示があれば \(\pi\) を残す
  • 「小数第○位まで」の指示があれば四捨五入する

3. 例題と解き方

ルールを使って実際に解いてみましょう。

▶ 円の計算

例1円周を求める（半径から）

半径 \(r = 5\) cm の円の円周を求めよ。（\(\pi = 3.14\)）

解き方：

  1. 半径がそのまま与えられているので、公式に代入する
  2. 円周の公式 \(C = 2\pi r\) を使う
  3. \(\pi = 3.14\) を代入して計算する
\[ C = 2\pi r = 2 \times 3.14 \times 5 = 31.4 \text{ cm} \]
例2円周を求める（直径から）

直径 \(d = 14\) cm の円の円周を求めよ。（\(\pi\) のまま）

解き方：

  1. 直径が与えられているので、公式 \(C = \pi d\) を使う
  2. \(\pi\) のまま答えるので、数値だけ計算する
\[ C = \pi d = 14\pi \text{ cm} \]
例3円の面積を求める

半径 \(r = 6\) cm の円の面積を求めよ。（\(\pi = 3.14\)）

解き方：

  1. 円の面積の公式 \(S = \pi r^2\) を使う
  2. 半径を2乗する → \(6^2 = 36\)
  3. \(\pi = 3.14\) を掛ける
\[ S = \pi r^2 = 3.14 \times 6^2 = 3.14 \times 36 = 113.04 \text{ cm}^2 \]

▶ 球の計算

例4球の表面積を求める

半径 \(r = 3\) cm の球の表面積を求めよ。（\(\pi\) のまま）

解き方：

  1. 球の表面積の公式 \(S = 4\pi r^2\) を使う
  2. 半径を2乗する → \(3^2 = 9\)
  3. 4を掛けて、\(\pi\) をつける
\[ S = 4\pi r^2 = 4\pi \times 3^2 = 4\pi \times 9 = 36\pi \text{ cm}^2 \]
例5球の高さから表面積を求める

地面に置いた球の高さが 74 mm。この球の表面積を求めよ。（小数第1位まで、\(\pi = 3.14\)）

解き方：

  1. 高さ = 直径なので \(d = 74\) mm
  2. 半径を求める → \(r = \dfrac{74}{2} = 37\) mm
  3. 球の表面積の公式に代入する
  4. 小数第1位まで四捨五入する
\[ S = 4\pi r^2 = 4 \times 3.14 \times 37^2 = 4 \times 3.14 \times 1369 = 17194.64 \text{ mm}^2 \]

答え：17194.6 mm²

地面に置いた球の高さから半径を求める
例6球の表面を等分したときの1部分の面積

半径 \(r = 37\) mm の球（野球ボール）の表面を2枚の皮で覆う。1枚あたりの面積は？（\(\pi\) のまま）

解き方：

  1. まず球の表面積を求める → \(S = 4\pi \times 37^2 = 5476\pi\) mm²
  2. 2等分するので、全体を2で割る
  3. \(\pi\) のまま答える
\[ \text{1枚の面積} = \frac{4\pi r^2}{2} = 2\pi r^2 = 2\pi \times 37^2 = 2738\pi \text{ mm}^2 \]

4. 簡単にするコツ

✏️ コツ1：直径が与えられたら、まず半径を求める
公式は半径 \(r\) を使うものが多い。直径 \(d\) が与えられたら \(r = \dfrac{d}{2}\) で半径に変換してから計算。
✏️ コツ2：球の表面積 = 円の面積 × 4
同じ半径なら、球の表面積は円の面積のちょうど4倍。覚えやすい！
✏️ コツ3：「高さ」は直径と同じ
地面に置いた球の「高さ」は直径 \(d = 2r\)。問題文の「高さ」から半径を求めるときは2で割る。
✏️ コツ4：\(\pi\) は最後にまとめる
計算の途中は \(\pi\) をそのまま文字として扱い、最後に数値を代入すると計算ミスが減る。
✏️ コツ5：単位の換算に注意
面積は \(\text{cm}^2\)、\(\text{m}^2\) など、長さの単位の2乗になる。体積は3乗。

5. よくあるつまづき

⚠️ つまづき1：直径と半径を混同

ダメな例： 直径 10 cm を公式にそのまま代入して \(\pi \times 10^2\)

対策： 公式は基本的に半径 \(r\) を使う。直径なら必ず2で割って半径にしてから代入。

⚠️ つまづき2：球の表面積で「4」を忘れる

ダメな例： 球の表面積を \(\pi r^2\) で計算（これは円の面積）

対策： 球の表面積は \(4\pi r^2\)。円の面積の4倍と覚える。

⚠️ つまづき3：「高さ」の意味を間違える

ダメな例： 球の「高さ」を半径だと思って計算

対策： 地面に置いた球の高さ = 直径 = 2r。半径はその半分。

⚠️ つまづき4：単位の2乗・3乗を忘れる

ダメな例： 面積を「cm」で答える

対策： 面積は cm²、m² など（長さの2乗）。体積は cm³、m³（3乗）。

⚠️ つまづき5：四捨五入のタイミングを間違える

ダメな例： 計算途中で四捨五入してしまい、最終答えがずれる

対策： 四捨五入は最後の1回だけ。途中は \(\pi\) のまま、または多めの桁数で計算。

6. 仕上げチェックリスト

直径が与えられた場合、半径に変換したか（\(r = d \div 2\)）
円と球の公式を正しく使い分けたか（円の面積 \(\pi r^2\)、球の表面積 \(4\pi r^2\)）
単位は面積なら²、体積なら³をつけたか
\(\pi\) の扱いは問題の指示通りか（そのまま or 3.14など）
小数点以下・有効数字の指示に従って正しく丸めたか

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

直径 20 cm の円の面積を求めよ。（\(\pi\) のまま）

答えを見る

半径 \(r = \dfrac{20}{2} = 10\) cm

\(S = \pi r^2 = \pi \times 10^2 = 100\pi\) cm²

答え：\(100\pi\) cm²

練習2

地面に置いた球の高さが 12 cm のとき、この球の表面積を求めよ。（\(\pi = 3.14\)）

答えを見る

高さ = 直径なので、半径 \(r = \dfrac{12}{2} = 6\) cm

\(S = 4\pi r^2 = 4 \times 3.14 \times 6^2 = 4 \times 3.14 \times 36 = 452.16\) cm²

答え：452.16 cm²

📌 まとめ：半径 \(r\) をつかんで、円は \(\pi r^2\)、球は \(4\pi r^2\) で攻略！

おまけ：歴史ストーリー「アルキメデスの墓石」

📖 アルキメデスの墓石（紀元前3世紀・シラクサ）

紀元前3世紀、地中海に浮かぶシチリア島のシラクサ。ここに当時最高の数学者アルキメデスがいた。王冠の偽物を見破った「ユーレカ！」の逸話で有名な彼だが、本人が最も誇りに思っていた発見は別にあった。

ある日、アルキメデスは球の表面積を研究していた。球を円柱にすっぽり入れて比べてみると...なんと、球の表面積と、円柱の側面積がぴったり同じだったのだ！

「待て待て...円柱の側面積は \(2\pi r \times 2r = 4\pi r^2\) だから...」

「つまり球の表面積も \(4\pi r^2\)...これは円の面積 \(\pi r^2\) のちょうど4倍じゃないか！」

アルキメデスは叫んだ。「なんて美しいんだ！球と円柱がこんな関係で結ばれているなんて！」

彼はこの発見をあまりにも気に入り、こんな遺言を残したという。
「私の墓には、球が内接する円柱の図を刻んでくれ」

約150年後、ローマの政治家キケロがシラクサを訪れたとき、荒れ果てた墓地で「球と円柱」が刻まれた墓石を発見。それがアルキメデスの墓だった。

数学者が人生最高の発見として選んだのは、複雑な定理ではなく「球の表面積 = 円の面積 × 4」というシンプルな関係。2000年以上経った今も、私たちはこの公式を使っている。
...ちなみに彼は、戦争中に数学の問題を解いていて敵兵に殺されたらしい。最期の言葉は「私の円を踏むな！」だったとか。天才と変人は紙一重である。
