1. テーマの定義

「この2つの三角形が同じ形だと証明せよ」——テストでよく見る問題。合同条件は知っているのに、どれを使えばいいかわからないという人は多いはず。

このページでは、合同条件の選び方を学びます：

  • 3つの合同条件：SSS・SAS・ASAの使い分け
  • 直角三角形の特別条件：斜辺を使った2つの条件
  • 判断フロー：問題を見て「どれを使う？」を即決する方法

2. 覚えるルール

【最重要】合同条件は「与えられた情報」で選ぶ！
辺が多い → SSS or SAS
角が多い → ASA
直角三角形 → 斜辺を使う条件
三角形の合同条件5パターン
📌 合同条件① SSS（3辺がそれぞれ等しい） → 例1

3組の辺がそれぞれ等しいとき、2つの三角形は合同

→ 使う場面： 辺の長さが3つわかっているとき

✅ メリット： 角度を計算しなくていいので楽！

💡 なぜ3辺でOK？

3辺の長さが決まると、三角形の形は1通りに決まる。「辺だけで形が決まる」から、角は調べなくていい。

📌 合同条件② SAS（2辺とその間の角がそれぞれ等しい） → 例2

2組の辺とその間の角がそれぞれ等しいとき、2つの三角形は合同

→ 使う場面： 辺2つと、その「間の」角がわかっているとき

⚠️ 注意： 角は必ず2辺の「間」！間でない角はダメ！

💡 なぜ「間の角」限定？

2辺と「間でない角」がわかっても、三角形の形が1通りに決まらないことがある。間の角なら確実に1通りに決まる。

📌 合同条件③ ASA（1辺とその両端の角がそれぞれ等しい） → 例3

1組の辺とその両端の角がそれぞれ等しいとき、2つの三角形は合同

→ 使う場面： 辺1つと、その辺の「両端の」角がわかっているとき

✅ メリット： 角が多くわかっているときに使いやすい！

📌 直角三角形の合同条件①（斜辺と1つの鋭角） → 例4

直角三角形で、斜辺と1つの鋭角がそれぞれ等しいとき合同

→ 使う場面： 直角三角形＋斜辺＋鋭角がわかっているとき

📌 直角三角形の合同条件②（斜辺と他の1辺） → 例5

直角三角形で、斜辺と他の1辺がそれぞれ等しいとき合同

→ 使う場面： 直角三角形＋斜辺＋もう1辺がわかっているとき

💡 なぜ斜辺＋1辺でOK？

直角があるので、斜辺と他の1辺が決まれば残りの1辺も三平方の定理で決まる。つまり実質的にSSS（3辺）と同じ。

3. 例題と解き方

問題を見て、どの合同条件を使えばいいか判断する練習をしましょう。

▶ 一般の三角形

例1SSS（3辺）を使う問題

△ABCと△DEFにおいて、AB=DE、BC=EF、CA=FDのとき、△ABC≡△DEFを証明せよ。

条件を選ぶ判断：

  1. 直角三角形ではない → 普通の三角形ルート
  2. わかっている辺は？ → AB、BC、CA の3辺
  3. 3辺がわかっている → SSSを使う！

証明：

△ABCと△DEFにおいて
　AB = DE …①
　BC = EF …②
　CA = FD …③
①②③より、3組の辺がそれぞれ等しいから
△ABC ≡ △DEF（SSS）
例2SAS（2辺＋間の角）を使う問題

△ABCと△DEFにおいて、AB=DE、∠ABC=∠DEF、BC=EFのとき、△ABC≡△DEFを証明せよ。

条件を選ぶ判断：

  1. 直角三角形ではない → 普通の三角形ルート
  2. わかっている辺は？ → AB、BC の2辺
  3. 角∠ABCは辺ABと辺BCの「間の角」か？ → Yes！
  4. 2辺と間の角 → SASを使う！

証明：

△ABCと△DEFにおいて
　AB = DE …①
　∠ABC = ∠DEF …②
　BC = EF …③
①②③より、2組の辺とその間の角がそれぞれ等しいから
△ABC ≡ △DEF（SAS）
例3ASA（1辺＋両端の角）を使う問題

△ABCと△DEFにおいて、∠BAC=∠EDF、AC=DF、∠ACB=∠DFEのとき、△ABC≡△DEFを証明せよ。

条件を選ぶ判断：

  1. 直角三角形ではない → 普通の三角形ルート
  2. わかっている辺は？ → ACの1辺
  3. 角∠BACは辺ACの端の角か？ → Yes！（頂点Aにある）
  4. 角∠ACBも辺ACの端の角か？ → Yes！（頂点Cにある）
  5. 1辺と両端の角 → ASAを使う！

証明：

△ABCと△DEFにおいて
　∠BAC = ∠EDF …①
　AC = DF …②
　∠ACB = ∠DFE …③
①②③より、1組の辺とその両端の角がそれぞれ等しいから
△ABC ≡ △DEF（ASA）

▶ 直角三角形

例4斜辺と1つの鋭角を使う問題

直角三角形ABCと直角三角形DEFにおいて、∠ACB=∠DFE=90°、AB=DE（斜辺）、∠ABC=∠DEFのとき、△ABC≡△DEFを証明せよ。

条件を選ぶ判断：

  1. 直角三角形である → 直角三角形ルート！
  2. 斜辺ABがわかっている → 斜辺の条件が使える
  3. 角∠ABCは鋭角 → 斜辺＋1つの鋭角を使う！

証明：

△ABCと△DEFにおいて
　∠ACB = ∠DFE = 90° …①
　AB = DE（斜辺） …②
　∠ABC = ∠DEF …③
①②③より、直角三角形で斜辺と1つの鋭角がそれぞれ等しいから
△ABC ≡ △DEF
例5斜辺と他の1辺を使う問題

直角三角形ABCと直角三角形DEFにおいて、∠ACB=∠DFE=90°、AB=DE（斜辺）、BC=EFのとき、△ABC≡△DEFを証明せよ。

条件を選ぶ判断：

  1. 直角三角形である → 直角三角形ルート！
  2. 斜辺ABがわかっている → 斜辺の条件が使える
  3. 辺BCがわかっている（鋭角ではなく辺） → 斜辺＋他の1辺を使う！

証明：

△ABCと△DEFにおいて
　∠ACB = ∠DFE = 90° …①
　AB = DE（斜辺） …②
　BC = EF …③
①②③より、直角三角形で斜辺と他の1辺がそれぞれ等しいから
△ABC ≡ △DEF

▶ 図から条件を見つける問題

例6条件を自分で見つける問題

四角形ABCDで、対角線ACとBDが点Oで交わり、AO=CO、BO=DOのとき、△ABO≡△CDOを証明せよ。

対角線が点Oで交わる四角形

条件を探す：

  1. AO = CO（問題文にある）… 辺1つ目
  2. BO = DO（問題文にある）… 辺2つ目
  3. ∠AOB と ∠COD は対頂角だから等しい … 角1つ

条件を選ぶ判断：

  1. 直角三角形ではない → 普通の三角形ルート
  2. わかった辺は AO、BO の2辺
  3. ∠AOBは辺AOと辺BOの「間の角」か？ → Yes！
  4. 2辺と間の角 → SASを使う！

証明：

△ABOと△CDOにおいて
　AO = CO …①（仮定）
　BO = DO …②（仮定）
　∠AOB = ∠COD …③（対頂角）
①②③より、2組の辺とその間の角がそれぞれ等しいから
△ABO ≡ △CDO（SAS）

4. 簡単にするコツ

✏️ コツ1：まず「辺の数」を数える
条件を選ぶ第一歩は「わかっている辺がいくつあるか」。
3辺→SSS、2辺→SAS（間の角があれば）、1辺→ASA（両端の角があれば）
✏️ コツ2：直角三角形なら特別条件を優先
直角三角形で斜辺がわかっているなら、特別条件（斜辺＋鋭角 or 斜辺＋1辺）を使おう。
普通の条件（SSS、SAS、ASA）でも証明できるが、特別条件のほうが楽なことが多い。
✏️ コツ3：対頂角・平行線を見落とすな
図から条件を見つける問題では、問題文に書かれていない条件が隠れている。
よく使う隠れ条件：
・対頂角は等しい
・平行線の錯角・同位角は等しい
・共通の辺・角は等しい（同じ辺や角を2つの三角形が共有している）
✏️ コツ4：証明の書き方のテンプレ
証明の書き方は決まったパターンがある：
①「△ABCと△DEFにおいて」で始める
②条件を「〜 = 〜 …①」「〜 = 〜 …②」と番号付きで列挙
③「①②③より、〜だから」と結ぶ
④「△ABC ≡ △DEF（合同条件名）」で終わる

5. よくあるつまづき

⚠️ つまづき1：SASで「間の角」を間違える

ダメな例： 「AB=DE、BC=EF、∠CAB=∠FDE だからSAS」

→ ∠CABは辺ABと辺BCの「間の角」ではない！（辺CAと辺ABの間の角）

対策： 角の頂点が、2辺の両方に含まれているか確認。∠ABCなら、頂点Bが辺ABにも辺BCにも含まれている→間の角OK。

⚠️ つまづき2：ASAで「両端の角」を間違える

ダメな例： 「AC=DF、∠ABC=∠DEF、∠BAC=∠EDF だからASA」

→ ∠ABCは辺ACの「端の角」ではない！（頂点Bは辺ACに含まれない）

対策： 角の頂点が、わかっている辺の端点かを確認。辺ACなら、角は頂点Aか頂点Cにあるべき。

⚠️ つまづき3：条件を書く順番がバラバラ

ダメな例： 「∠ABC=∠DEF、AB=DE、BC=EF だからSAS」

→ 順番がバラバラだと「間の角」がどこかわかりにくい

対策： 辺→角→辺の順番で書く。こうすると「真ん中の角」が間の角だとわかりやすい。

⚠️ つまづき4：斜辺を見落とす

ダメな例： 直角三角形なのにSSSで証明しようとして、3辺目がわからず詰まる

対策： 直角三角形なら、まず「斜辺」を探す。斜辺は直角の向かい側にある最も長い辺。斜辺がわかれば、あと1つの情報（鋭角 or 他の1辺）で合同が示せる。

6. 仕上げチェックリスト

直角三角形かどうかを最初に確認した
わかっている辺の数を数えた（3辺/2辺/1辺）
SASなら「間の角」、ASAなら「両端の角」を確認した
対頂角や共通の辺など、隠れた条件を見つけた
証明の最後に合同条件名（SSS/SAS/ASAなど）を書いた

7. 練習問題

学んだことを使って、どの合同条件を使うか判断してみよう。

練習1

△ABCと△DEFにおいて、AB=DE、AC=DF、BC=EFのとき、△ABC≡△DEFを証明せよ。どの合同条件を使う？

答えを見る

使う条件：SSS（3組の辺がそれぞれ等しい）

判断理由：AB、AC、BCの3辺がわかっている→SSS

証明：
△ABCと△DEFにおいて
　AB = DE …①
　AC = DF …②
　BC = EF …③
①②③より、3組の辺がそれぞれ等しいから
△ABC ≡ △DEF（SSS）

練習2

右の図で、AB//CD、AB=CDのとき、△ABO≡△CDOを証明せよ。どの合同条件を使う？

答えを見る

使う条件：ASA（1組の辺とその両端の角がそれぞれ等しい）

見つける条件：
・AB = CD（仮定）… 辺
・∠OAB = ∠OCD（AB//CDの錯角）… ABの端の角
・∠OBA = ∠ODC（AB//CDの錯角）… ABの端の角

証明：
△ABOと△CDOにおいて
　AB = CD …①（仮定）
　∠OAB = ∠OCD …②（錯角）
　∠OBA = ∠ODC …③（錯角）
①②③より、1組の辺とその両端の角がそれぞれ等しいから
△ABO ≡ △CDO（ASA）

📌 まとめ：まず辺の数を数え、間の角・両端の角をチェック。直角三角形なら斜辺を探せ！

おまけ：歴史ストーリー「幾何学に王道なし」

📖 ユークリッドと王様（紀元前300年頃・エジプト）

紀元前300年頃、エジプトのアレクサンドリア。ここに世界最高の図書館があった。その図書館で働く数学者ユークリッドは、ある日とんでもない本を書き上げた。

その名も『原論（ストイケイア）』——幾何学のすべてをまとめた13巻の大著だ。

この本、ちょっと変わっていた。最初に「点」「線」「角」などの定義を決め、誰もが納得する基本ルール（公理）を5つだけ置く。そこから全ての定理を論理だけで証明していく。三角形の合同条件も、この本で初めて「なぜ3辺が等しければ合同なのか」が証明された。

さて、この本の噂を聞いたエジプト王プトレマイオス1世。ユークリッドを宮殿に呼び出してこう言った。

「おい、この本、難しすぎるぞ。もっと簡単に幾何学をマスターできる近道はないのか？」

王様である。国を治める仕事で忙しい。効率的に学びたいのは当然だ。

しかしユークリッドは、こう答えた。

「幾何学に王道なし（There is no royal road to geometry）」

つまり、「王様だからって特別な近道はありませんよ」ということ。王様もへったくれもなく、みんな同じように一歩一歩学ぶしかない、と。

王様の反応は記録に残っていない。たぶん苦笑いしたんじゃないだろうか。

ちなみにこの『原論』、その後2300年以上読み継がれ、聖書の次に多く印刷された本とも言われる。リンカーン大統領も愛読者だったとか。

「王道はない」とは言ったが、実はユークリッドはちゃんと「効率的な学び方」を本に書いている。それが今回学んだ合同条件だ。

3辺を測る代わりに「2辺と間の角」で済ませたり、「1辺と両端の角」だけで三角形の形を決めたり。これらは「無駄な情報を省いて効率よく判断する方法」そのものだ。

近道はないが、効率的なルートはある。
それを2300年前に整理してくれたユークリッドに感謝しよう。

※ユークリッドの『原論』は、その後のすべての数学教科書の原型となりました（ユークリッド原論 - Wikipedia）。
