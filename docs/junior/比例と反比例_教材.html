1. テーマの定義

「速度が2倍になれば、進む距離も2倍」「人数が2倍になれば、1人あたりの量は半分」。世の中には、一方が増えると他方も連動して変わる量がたくさんある。この「変化の仕方」を数式で表すのが比例と反比例です。

このページでは、比例と反比例の基本を学びます：

  • 座標の意味：点の位置を数で表す
  • 比例：\(y = ax\) の形と原点を通る直線
  • 反比例：\(y = \frac{a}{x}\) の形と双曲線
  • 利用：比例・反比例を使って問題を解く

2. 覚えるルール

【最重要】比例と反比例の見分け方：
比例 → \(y = ax\)（xが2倍ならyも2倍）
反比例 → \(y = \frac{a}{x}\)（xが2倍ならyは半分）
座標平面と比例のグラフ
📌 座標の基本
  • 座標平面：横軸（x軸）と縦軸（y軸）で点の位置を表す平面
  • 原点 O：x軸とy軸の交点、座標は \((0, 0)\)
  • 座標：点の位置を \((x, y)\) で表す（横の値, 縦の値）
💡 考え方を見る

座標は「住所」のようなもの。「東に3km、北に2km」のように、2つの数で場所を特定する。

📌 比例の式 → 例1
\[ y = ax \quad (a \neq 0) \]
  • \(a\)：比例定数（\(y \div x\) で求まる）
  • xが2倍、3倍になると、yも2倍、3倍になる
  • グラフは原点を通る直線
  • \(a > 0\)：右上がり、\(a < 0\)：右下がり

✅ メリット： 比例の式がわかれば、どんなxの値でもyを計算できる

📌 反比例の式 → 例3
\[ y = \frac{a}{x} \quad (a \neq 0) \]
  • \(a\)：比例定数（\(x \times y\) で求まる）
  • xが2倍、3倍になると、yは \(\frac{1}{2}\) 倍、\(\frac{1}{3}\) 倍になる
  • グラフは双曲線（なめらかな曲線）
  • \(a > 0\)：第1象限と第3象限、\(a < 0\)：第2象限と第4象限
反比例のグラフ（双曲線）
📌 比例定数の求め方 → 例5
種類 式 比例定数の求め方
比例 \(y = ax\) \(a = \frac{y}{x}\)
反比例 \(y = \frac{a}{x}\) \(a = xy\)
📌 グラフから式を求める
  1. グラフ上の点の座標 \((x, y)\) を読み取る
  2. 比例定数 \(a\) を計算する
  3. 式 \(y = ax\) または \(y = \frac{a}{x}\) を完成させる

3. 例題と解き方

比例・反比例の式を作り、グラフを読み取る練習をしましょう。

▶ 比例

例1比例の式を求める

\(y\) は \(x\) に比例し、\(x = 3\) のとき \(y = 12\) である。\(y\) を \(x\) の式で表せ。

解き方：

  1. 比例の式 \(y = ax\) に代入
  2. \(12 = a \times 3\) より \(a = 4\)
  3. 式は \(y = 4x\)
\[ y = 4x \]
例2比例のグラフをかく

\(y = -2x\) のグラフをかけ。

解き方：

  1. 原点 \((0, 0)\) を通る
  2. もう1点を求める：\(x = 1\) のとき \(y = -2\) → 点 \((1, -2)\)
  3. 2点を通る直線をかく（右下がり）

ポイント： 比例のグラフは原点ともう1点でOK！

▶ 反比例

例3反比例の式を求める

\(y\) は \(x\) に反比例し、\(x = 4\) のとき \(y = 3\) である。\(y\) を \(x\) の式で表せ。

解き方：

  1. 反比例の比例定数 \(a = xy\) を求める
  2. \(a = 4 \times 3 = 12\)
  3. 式は \(y = \frac{12}{x}\)
\[ y = \frac{12}{x} \]
例4反比例のグラフをかく

\(y = \frac{6}{x}\) のグラフをかけ。

解き方：

  1. いくつかの点を求める
    • \(x = 1\) → \(y = 6\) → 点 \((1, 6)\)
    • \(x = 2\) → \(y = 3\) → 点 \((2, 3)\)
    • \(x = 3\) → \(y = 2\) → 点 \((3, 2)\)
    • \(x = 6\) → \(y = 1\) → 点 \((6, 1)\)
  2. 点をなめらかな曲線で結ぶ（第1象限）
  3. 第3象限にも対称な曲線をかく

ポイント： 反比例は軸に近づくが、軸には触れない！

▶ 比例定数を求める

例5グラフから式を求める

点 \((2, 8)\) を通る比例のグラフの式を求めよ。

解き方：

  1. 比例の式 \(y = ax\) に \((2, 8)\) を代入
  2. \(8 = a \times 2\) より \(a = 4\)
\[ y = 4x \]
例6反比例のグラフから式を求める

点 \((-3, 4)\) を通る反比例のグラフの式を求めよ。

解き方：

  1. 反比例の比例定数 \(a = xy\)
  2. \(a = (-3) \times 4 = -12\)
\[ y = \frac{-12}{x} = -\frac{12}{x} \]

▶ 利用問題

例7比例の利用

時速4kmで歩くとき、歩いた時間を \(x\) 時間、進んだ距離を \(y\) kmとする。\(y\) を \(x\) の式で表し、3時間で進む距離を求めよ。

解き方：

  1. 距離 = 速さ × 時間 より \(y = 4x\)
  2. \(x = 3\) を代入：\(y = 4 \times 3 = 12\)
\[ y = 4x, \quad x = 3 \text{ のとき } y = 12 \text{km} \]
例8反比例の利用

面積が24cm²の長方形の横を \(x\) cm、縦を \(y\) cmとする。\(y\) を \(x\) の式で表し、横が8cmのときの縦を求めよ。

解き方：

  1. 面積 = 横 × 縦 より \(xy = 24\)、つまり \(y = \frac{24}{x}\)
  2. \(x = 8\) を代入：\(y = \frac{24}{8} = 3\)
\[ y = \frac{24}{x}, \quad x = 8 \text{ のとき } y = 3 \text{cm} \]

4. 簡単にするコツ

✏️ コツ1：比例か反比例かを見分ける
  • 比例：一方が増えると他方も増える（距離と時間、代金と個数）
  • 反比例：一方が増えると他方が減る（速さと時間、人数と1人分の量）
✏️ コツ2：比例定数の求め方を使い分ける
  • 比例：\(a = \frac{y}{x}\)（yをxで割る）
  • 反比例：\(a = xy\)（xとyを掛ける）
✏️ コツ3：グラフは「原点」を確認
  • 原点を通る → 比例 \(y = ax\)
  • 原点を通らない直線 → 一次関数（中2で学ぶ）
  • 曲線（双曲線）→ 反比例 \(y = \frac{a}{x}\)
✏️ コツ4：グラフの傾きで正負がわかる
  • 比例で右上がり → \(a > 0\)
  • 比例で右下がり → \(a < 0\)
  • 反比例で第1・3象限 → \(a > 0\)
  • 反比例で第2・4象限 → \(a < 0\)

5. よくあるつまづき

⚠️ つまづき1：比例と反比例の式を混同

ダメな例： 反比例なのに \(y = ax\) と書いてしまう

対策：

  • 比例：\(y = ax\)（xは分子にある）
  • 反比例：\(y = \frac{a}{x}\)（xは分母にある）
⚠️ つまづき2：比例定数の求め方を間違える

ダメな例： 反比例で \(a = \frac{y}{x}\) と計算

対策：

  • 比例：\(a = \frac{y}{x}\)（割り算）
  • 反比例：\(a = xy\)（掛け算）
⚠️ つまづき3：座標の順序を間違える

ダメな例： 点 \((3, 2)\) を「y座標が3、x座標が2」と読む

対策： 座標は必ず \((x, y)\) の順。「横(x)、縦(y)」と覚える。

⚠️ つまづき4：反比例のグラフで軸と交わる

ダメな例： 反比例のグラフをx軸やy軸と交わるように描く

対策： 反比例のグラフは軸に限りなく近づくが、決して交わらない。\(x = 0\) や \(y = 0\) にはならない。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(y\) は \(x\) に比例し、\(x = 2\) のとき \(y = -6\) である。

(1) \(y\) を \(x\) の式で表せ。

(2) \(x = -5\) のとき、\(y\) の値を求めよ。

答えを見る

(1) \(a = \frac{-6}{2} = -3\) より \(y = -3x\)

(2) \(y = -3 \times (-5) = 15\)

練習2

\(y\) は \(x\) に反比例し、\(x = 6\) のとき \(y = -2\) である。

(1) \(y\) を \(x\) の式で表せ。

(2) \(y = 4\) のとき、\(x\) の値を求めよ。

答えを見る

(1) \(a = 6 \times (-2) = -12\) より \(y = -\frac{12}{x}\)

(2) \(4 = -\frac{12}{x}\) より \(x = -\frac{12}{4} = -3\)

練習3

水槽に毎分5Lの割合で水を入れる。入れ始めてから \(x\) 分後の水の量を \(y\) Lとする。

(1) これは比例か反比例か答えよ。

(2) \(y\) を \(x\) の式で表せ。

(3) 水が40L入るのは何分後か求めよ。

答えを見る

(1) 比例（時間が増えると水の量も増える）

(2) \(y = 5x\)

(3) \(40 = 5x\) より \(x = 8\)　答え：8分後

📌 まとめ：比例は \(y = ax\)（原点を通る直線）、反比例は \(y = \frac{a}{x}\)（双曲線）！

おまけ：歴史ストーリー「ガリレオの振り子」

📖 シャンデリアの揺れから始まった科学革命

1583年、イタリアのピサ大聖堂。19歳の医学生ガリレオ・ガリレイは、退屈なミサの最中、天井のシャンデリアが揺れているのをぼんやり眺めていた。

風で大きく揺れたり、小さく揺れたり...。ふと、ガリレオは気づいた。

「あれ？揺れの大きさが変わっても、1往復する時間は同じじゃないか？」

彼は自分の脈拍で時間を測り始めた。大きく揺れても、小さく揺れても、確かに周期（1往復の時間）は同じだった。

後に研究を重ねたガリレオは、振り子の法則を発見した。

\[ T = 2\pi\sqrt{\frac{L}{g}} \]

周期 \(T\) は紐の長さ \(L\) の平方根に比例する。重さは関係ない！

つまり、紐を4倍の長さにすると、周期は2倍になる。

紐の長さ周期の倍率
1倍1倍
4倍2倍
9倍3倍

この発見は、後に振り子時計の発明につながった。おかげで人類は初めて「正確な時間」を手に入れることができたのだ。

結論：退屈なミサ中の「ぼーっとした観察」が、時計の歴史を変えた
