1. テーマの定義

四角形にはいろいろな種類があります。平行四辺形、長方形、ひし形、正方形...これらは見た目が違うだけでなく、それぞれ固有の性質を持っています。この関係を理解すると、「どの性質が使えるか」がわかり、証明問題がスッキリ解けます。

このページでは、特別な四角形の性質を学びます：

  • 平行四辺形：向かい合う辺が平行な四角形
  • 長方形：4つの角がすべて直角の四角形
  • ひし形：4つの辺がすべて等しい四角形
  • 正方形：長方形でありひし形でもある四角形

2. 覚えるルール

【最重要】平行四辺形の5つの性質と5つの条件を押さえよう！

▶ 四角形の分類

📌 ルール1：四角形の包含関係
四角形の包含関係

ポイント：

  • 正方形は長方形でありひし形でもある
  • 長方形もひし形も平行四辺形である
  • 平行四辺形は台形である

▶ 平行四辺形

📌 ルール2：平行四辺形の定義

平行四辺形：2組の向かい合う辺がそれぞれ平行な四角形

AB // DC、AD // BC
📌 ルール3：平行四辺形の性質（5つ） → 例1
# 性質 記号
① 2組の対辺がそれぞれ等しい AB = DC, AD = BC
② 2組の対角がそれぞれ等しい ∠A = ∠C, ∠B = ∠D
③ 対角線がそれぞれの中点で交わる AO = CO, BO = DO
④ 隣り合う角の和は180° ∠A + ∠B = 180°
⑤ 2組の対辺がそれぞれ平行（定義） AB // DC, AD // BC
📌 ルール4：平行四辺形になる条件（5つ） → 例2

以下のどれか1つを満たせば平行四辺形：

# 条件
① 2組の対辺がそれぞれ平行（定義）
② 2組の対辺がそれぞれ等しい
③ 2組の対角がそれぞれ等しい
④ 対角線がそれぞれの中点で交わる
⑤ 1組の対辺が平行で等しい

⑤がポイント！ 「平行」と「等しい」を1組だけ示せばOK。

▶ 長方形・ひし形・正方形

📌 ルール5：長方形の性質 → 例3

長方形：4つの角がすべて直角の四角形

長方形の対角線は等しい

長方形の性質：

  • 平行四辺形のすべての性質を持つ
  • 対角線の長さが等しい

長方形になる条件： 平行四辺形 + 1つの角が直角 or 対角線が等しい

📌 ルール6：ひし形の性質 → 例4

ひし形：4つの辺がすべて等しい四角形

ひし形の対角線は垂直に交わる

ひし形の性質：

  • 平行四辺形のすべての性質を持つ
  • 対角線が垂直に交わる
  • 対角線が角を二等分する

ひし形になる条件： 平行四辺形 + 隣り合う2辺が等しい or 対角線が垂直

📌 ルール7：正方形の性質

正方形：長方形でありひし形でもある四角形（4辺等しく4角直角）

正方形の性質：

  • 長方形のすべての性質（対角線が等しい）
  • ひし形のすべての性質（対角線が垂直）
  • 対角線が等しくて垂直に交わる

正方形になる条件： 長方形 + 隣り合う2辺が等しい、または ひし形 + 1角が直角

3. 例題と解き方

四角形の性質を使って問題を解いてみましょう。

▶ 平行四辺形

例1平行四辺形の角を求める

平行四辺形ABCDで、∠A = 70° のとき、∠B、∠C、∠D を求めなさい。

解き方：

  1. ∠C = ∠A = 70°（対角は等しい）
  2. ∠A + ∠B = 180°（隣り合う角の和）
  3. ∠B = 180° - 70° = 110°
  4. ∠D = ∠B = 110°（対角は等しい）
\[ \angle B = 110°, \quad \angle C = 70°, \quad \angle D = 110° \]
例2平行四辺形であることを証明する

四角形ABCDで、AB // DC、AB = DC のとき、ABCDが平行四辺形であることを証明しなさい。

【証明】

AB // DC、AB = DC より、

1組の対辺が平行で等しいので、

四角形ABCDは平行四辺形である。

→ 条件⑤を使うと、証明が1行で終わる！

▶ 長方形

例3長方形の対角線

長方形ABCDの対角線AC、BDの交点をOとする。AC = 10cm のとき、OB の長さを求めなさい。

解き方：

  1. 長方形の対角線は等しいので、BD = AC = 10cm
  2. 平行四辺形の性質より、対角線は中点で交わるので、OB = BD ÷ 2
\[ OB = \frac{10}{2} = 5 \text{ cm} \]

▶ ひし形

例4ひし形の角

ひし形ABCDで、∠ABD = 35° のとき、∠ABC を求めなさい。

解き方：

  1. ひし形の対角線は角を二等分する
  2. よって、∠ABC = 2 × ∠ABD
\[ \angle ABC = 2 \times 35° = 70° \]
例5四角形の分類

次の条件を満たす四角形ABCDは何という四角形か答えなさい。

（1）AB // DC、AD // BC、∠A = 90°

（2）AB = BC = CD = DA、AC ⊥ BD

解答：

（1）2組の対辺が平行 → 平行四辺形。さらに1角が直角 → 長方形

（2）4辺が等しい → ひし形。対角線が垂直はひし形の性質なので追加情報なし。よってひし形

例6証明：対角線を使う

平行四辺形ABCDの対角線AC、BD の交点をOとする。OA = OC、OB = OD を証明しなさい。

【証明】

△OABと△OCDにおいて、

  1. AB = CD（平行四辺形の対辺）
  2. ∠OAB = ∠OCD（AB // CD より錯角）
  3. ∠OBA = ∠ODC（AB // CD より錯角）

①②③より、1辺とその両端の角がそれぞれ等しいので、

△OAB ≡ △OCD

合同な三角形の対応する辺は等しいので、

\[ OA = OC, \quad OB = OD \]

4. 簡単にするコツ

✏️ コツ1：「平行四辺形になる条件⑤」をフル活用
1組の対辺が「平行」で「等しい」→ 平行四辺形。これが最も使いやすい条件！
✏️ コツ2：包含関係を意識する
正方形 → 長方形の性質も使える。ひし形 → 平行四辺形の性質も使える。
✏️ コツ3：対角線の交点は中点
平行四辺形（長方形・ひし形・正方形含む）では、対角線は互いの中点で交わる。
✏️ コツ4：対角線の性質で判別
対角線が「等しい」→ 長方形系。「垂直」→ ひし形系。「両方」→ 正方形。
✏️ コツ5：錯角・同位角をどんどん使う
平行四辺形には平行線が2組ある。錯角・同位角で等しい角がたくさん作れる。

5. よくあるつまづき

⚠️ つまづき1：「性質」と「条件」を混同する

ダメな例： 「平行四辺形だから対辺が平行」を証明で使う（これは定義なので当然）

対策：

  • 性質：平行四辺形 → 対角線が中点で交わる
  • 条件：対角線が中点で交わる → 平行四辺形
⚠️ つまづき2：正方形を長方形・ひし形と認識しない

ダメな例： 「正方形の対角線が等しい」の証明で、長方形の性質を使わない

対策： 正方形は長方形でありひし形。両方の性質が使える！

⚠️ つまづき3：台形と平行四辺形の区別

ダメな例： 「1組の対辺が平行」だけで平行四辺形と言う

対策： 1組平行だけなら台形。平行四辺形は「2組」平行。

⚠️ つまづき4：条件⑤を忘れる

ダメな例： AB // DC、AB = DC があるのに、もう1組の平行や等しい辺を探す

対策： 「1組の対辺が平行で等しい」だけで平行四辺形確定！

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

平行四辺形ABCDで、∠B = 115° のとき、∠A と ∠D を求めなさい。

答えを見る

∠A + ∠B = 180°（隣り合う角）より、∠A = 180° - 115° = 65°

∠D = ∠B = 115°（対角）

練習2

次の四角形はそれぞれ何という四角形か答えなさい。

（1）4辺がすべて等しく、4角がすべて直角

（2）対角線が等しく、中点で交わる

（3）対角線が垂直に交わり、中点で交わる

答えを見る

（1）正方形（長方形でありひし形）

（2）長方形（対角線が等しい平行四辺形）

（3）ひし形（対角線が垂直な平行四辺形）

練習3

四角形ABCDで、AB = DC = 6cm、AB // DC のとき、四角形ABCDが平行四辺形であることを示しなさい。

答えを見る

AB // DC、AB = DC より、

1組の対辺が平行で等しいので、

四角形ABCDは平行四辺形である。

📌 まとめ：平行四辺形の性質・条件は5つずつ！長方形は「対角線等しい」、ひし形は「対角線垂直」、正方形は「両方」！

おまけ：数学雑学「マンホールはなぜ丸い？」

📖 マンホールはなぜ丸い？

街を歩くと、道路にマンホールの蓋がたくさんあります。よく見ると、ほとんどが円形。なぜ四角形じゃないのでしょうか？

実は、これは四角形の性質が関係しています。

正方形のマンホールだと何が起こる？

正方形のマンホール蓋を想像してください。一辺が 60cm の正方形です。

この蓋を穴の上で斜めに持ち上げると...なんと穴の中に落ちてしまうのです！

正方形の蓋は斜めにすると穴より小さくなる

なぜ落ちる？

正方形の対角線の長さを計算すると：

\[ \text{対角線} = 60\sqrt{2} \approx 84.9 \text{ cm} \]

一方、穴の1辺は 60cm。蓋を斜めに傾けると、対角線の長さ（約85cm）が穴の幅（60cm）より長いので、蓋は穴の中に落ちてしまいます！

円形なら大丈夫？

円形のマンホールなら、どの方向から見ても直径は同じ。どう傾けても穴より大きいままなので、絶対に落ちません。

円形はどの方向でも幅が同じ

数学の教訓

この話から学べることは2つ：

  • 正方形の対角線は1辺の √2 倍（約1.41倍）
  • 円は「どこから見ても同じ幅」という特別な性質を持つ

実は、マンホールが円形なのは他にも理由があります。製造が簡単、転がして運べる、どの角度でも設置できる、など。でも「落ちない」という安全性が一番大事なんです！

ちなみに、三角形（正三角形のルーローの三角形）も「どこから見ても同じ幅」という性質を持っています。でも円が一番シンプルで安全なので、世界中で円形マンホールが使われているのです。
