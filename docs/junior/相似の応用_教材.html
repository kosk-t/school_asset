1. テーマの定義

地図の縮尺、模型の設計、写真の拡大——相似な図形では、辺の比だけでなく面積や体積の比にも規則があります。これを知ると、複雑な計算をせずに答えが出せます。

このページでは、相似の応用を学びます：

  • 平行線と線分の比：平行線が作る線分の比の性質
  • 中点連結定理：三角形の中点を結ぶ線分の性質
  • 面積比・体積比：相似比と面積比・体積比の関係

2. 覚えるルール

🎯 最重要：相似比が \(m:n\) なら、面積比は \(m^2:n^2\)、体積比は \(m^3:n^3\)
平行線と線分の比：DE // BC なら AD:DB = AE:EC
📌 ルール1：平行線と線分の比 → 例1

\(\triangle ABC\) で \(DE // BC\) のとき：

\[ AD : DB = AE : EC \] \[ AD : AB = AE : AC = DE : BC \]

→ だから結果は： 平行線があれば、対応する線分の比が等しくなる

💡 考え方を見る

DE // BC なら \(\triangle ADE \sim \triangle ABC\)（AA相似）になるため、対応する辺の比が等しくなる。

📌 ルール2：平行線と線分の比の逆 → 例2
\[ AD : DB = AE : EC \quad \Rightarrow \quad DE // BC \]

線分の比が等しければ、その2点を結ぶ線分は底辺に平行。

✅ メリット： 比から平行を証明できる

中点連結定理：中点を結ぶ線分は底辺に平行で、長さは半分
📌 ルール3：中点連結定理 → 例3

\(\triangle ABC\) で M, N がそれぞれ AB, AC の中点のとき：

\[ \text{① } MN // BC \] \[ \text{② } MN = \frac{1}{2} BC \]

→ だから結果は： 中点を結べば「平行」と「半分の長さ」がわかる

✅ メリット： 四角形の対角線の中点を結ぶと平行四辺形になる、などの応用に使える

相似比が 1:2 なら、面積比は 1:4、体積比は 1:8
📌 ルール4：面積比 → 例4
\[ \text{相似比 } m : n \quad \Rightarrow \quad \text{面積比 } m^2 : n^2 \]

→ だから結果は： 相似比を2乗すれば面積比になる

💡 考え方を見る

面積は「縦×横」で計算する。相似比が \(m:n\) なら、縦も横も \(m:n\) 倍になるので、面積は \(m \times m : n \times n = m^2 : n^2\) 倍になる。

📌 ルール5：体積比 → 例5
\[ \text{相似比 } m : n \quad \Rightarrow \quad \text{体積比 } m^3 : n^3 \]

→ だから結果は： 相似比を3乗すれば体積比になる

💡 考え方を見る

体積は「縦×横×高さ」で計算する。相似比が \(m:n\) なら、3方向すべて \(m:n\) 倍になるので、体積は \(m^3 : n^3\) 倍になる。

相似比 面積比（2乗） 体積比（3乗）
\(1 : 2\) \(1 : 4\) \(1 : 8\)
\(1 : 3\) \(1 : 9\) \(1 : 27\)
\(2 : 3\) \(4 : 9\) \(8 : 27\)
\(3 : 5\) \(9 : 25\) \(27 : 125\)

3. 例題と解き方

平行線と比、中点連結定理、面積比・体積比を使った問題を解いてみましょう。

▶ 平行線と線分の比

例1平行線を使って線分の長さを求める

\(\triangle ABC\) で \(DE // BC\)、\(AD = 4\)、\(DB = 6\)、\(AE = 5\) のとき、\(EC\) を求めよ。

解き方：

  1. \(DE // BC\) より、\(AD : DB = AE : EC\)
  2. \(4 : 6 = 5 : EC\)
  3. 内項の積 = 外項の積 より、\(4 \times EC = 6 \times 5\)
  4. \(EC = \frac{30}{4} = 7.5\)
\[ 4 : 6 = 5 : EC \] \[ EC = 5 \times \frac{6}{4} = \boxed{7.5} \]
例2比から平行を判定する

\(\triangle ABC\) で、\(AD = 3\)、\(DB = 6\)、\(AE = 2\)、\(EC = 4\) のとき、\(DE // BC\) かどうか判定せよ。

解き方：

  1. \(AD : DB = 3 : 6 = 1 : 2\)
  2. \(AE : EC = 2 : 4 = 1 : 2\)
  3. \(AD : DB = AE : EC\) が成り立つ
  4. よって、\(DE // BC\)
\[ AD : DB = 1 : 2, \quad AE : EC = 1 : 2 \] \[ \therefore \boxed{DE // BC} \]

▶ 中点連結定理

例3中点連結定理で長さを求める

\(\triangle ABC\) で M, N がそれぞれ AB, AC の中点。\(BC = 12\) cm のとき、\(MN\) の長さを求めよ。

解き方：

  1. M, N が中点なので、中点連結定理が使える
  2. \(MN = \frac{1}{2} BC\)
  3. \(MN = \frac{1}{2} \times 12 = 6\)
\[ MN = \frac{1}{2} BC = \frac{1}{2} \times 12 = \boxed{6 \text{ cm}} \]
例4中点連結定理の逆を使う

\(\triangle ABC\) で M は AB の中点、MN // BC、\(MN = 5\) cm のとき、BC の長さを求めよ。

解き方：

  1. M が中点で MN // BC なら、N も AC の中点
  2. 中点連結定理より \(MN = \frac{1}{2} BC\)
  3. \(5 = \frac{1}{2} BC\) より、\(BC = 10\)
\[ BC = 2 \times MN = 2 \times 5 = \boxed{10 \text{ cm}} \]

▶ 面積比

例5相似比から面積比を求める

\(\triangle ABC \sim \triangle DEF\) で相似比が \(2:3\)。\(\triangle ABC\) の面積が 20 cm² のとき、\(\triangle DEF\) の面積を求めよ。

解き方：

  1. 相似比 \(2:3\) → 面積比 \(2^2 : 3^2 = 4 : 9\)
  2. \(\triangle ABC : \triangle DEF = 4 : 9\)
  3. \(20 : \triangle DEF = 4 : 9\)
  4. \(\triangle DEF = 20 \times \frac{9}{4} = 45\)
\[ \text{面積比} = 4 : 9 \] \[ \triangle DEF = 20 \times \frac{9}{4} = \boxed{45 \text{ cm}^2} \]
例6面積比から相似比を求める

2つの相似な三角形の面積比が \(16:25\)。相似比を求めよ。

解き方：

  1. 面積比 = 相似比の2乗
  2. 相似比 = 面積比の平方根
  3. \(\sqrt{16} : \sqrt{25} = 4 : 5\)
\[ \text{相似比} = \sqrt{16} : \sqrt{25} = \boxed{4 : 5} \]

▶ 体積比

例7相似比から体積比を求める

相似な2つの立体の相似比が \(2:5\)。小さい方の体積が 24 cm³ のとき、大きい方の体積を求めよ。

解き方：

  1. 相似比 \(2:5\) → 体積比 \(2^3 : 5^3 = 8 : 125\)
  2. 小 : 大 = \(8 : 125\)
  3. \(24 : V = 8 : 125\)
  4. \(V = 24 \times \frac{125}{8} = 375\)
\[ \text{体積比} = 8 : 125 \] \[ V = 24 \times \frac{125}{8} = \boxed{375 \text{ cm}^3} \]

4. 簡単にするコツ

✏️ コツ1：「平行」を見たら比を書く
平行線があれば、すぐに線分の比を式にする。\(DE // BC\) なら \(AD:DB = AE:EC\) が自動的に成り立つ。
✏️ コツ2：中点を見たら中点連結定理
「中点」というキーワードが出たら、中点連結定理を疑う。\(MN = \frac{1}{2}BC\) と \(MN // BC\) の2つがセットで使える。
✏️ コツ3：面積なら2乗、体積なら3乗
  • 面積は「平面」→ 2次元 → 2乗
  • 体積は「空間」→ 3次元 → 3乗
迷ったら「次元の数」を思い出す。
✏️ コツ4：よく出る相似比を覚える

5. よくあるつまづき

⚠️ つまづき1：面積比と相似比を混同

ダメな例： 相似比が 2:3 だから面積比も 2:3 と答える

対策： 面積比は相似比の2乗。2:3 なら 4:9 になる。

⚠️ つまづき2：中点連結定理の「半分」を忘れる

ダメな例： \(MN = BC\) としてしまう

対策： 中点連結定理は「半分」がポイント。\(MN = \frac{1}{2}BC\) を忘れない。

⚠️ つまづき3：比の計算ミス

ダメな例： \(4:6 = 5:x\) で \(x = \frac{4 \times 5}{6}\) としてしまう

対策： 比の計算は「内項の積 = 外項の積」または「\(x = 5 \times \frac{6}{4}\)」。どちらの辺が分母・分子かを確認。

⚠️ つまづき4：体積比を2乗してしまう

ダメな例： 立体の相似比が 2:3 で、体積比を 4:9 と答える

対策： 体積は3次元なので3乗。2:3 → 8:27 になる。

6. 仕上げチェックリスト

平行線があれば線分の比を正しく書いたか
中点連結定理で「半分」を忘れていないか
面積比は相似比の2乗、体積比は3乗になっているか
比の計算で分母・分子を間違えていないか
単位（cm², cm³）を正しく書いたか

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(\triangle ABC\) で \(DE // BC\)、\(AD = 6\)、\(AB = 10\)、\(DE = 9\) のとき、\(BC\) の長さを求めよ。

答えを見る

\(DE // BC\) より \(AD : AB = DE : BC\)

\(6 : 10 = 9 : BC\)

\(BC = 9 \times \frac{10}{6} = 9 \times \frac{5}{3} = \boxed{15}\)

練習2

相似な2つの円錐の相似比が \(3:4\)。大きい方の体積が 256π cm³ のとき、小さい方の体積を求めよ。

答えを見る

体積比 = 相似比の3乗 = \(3^3 : 4^3 = 27 : 64\)

小 : 大 = 27 : 64

小 : 256π = 27 : 64

小 = 256π × \(\frac{27}{64}\) = \(\boxed{108\pi \text{ cm}^3}\)

📌 まとめ：平行なら比が等しい、中点なら半分、面積は2乗・体積は3乗！

おまけ：数学雑学「ピザのサイズの罠」

🍕 Mサイズ2枚 vs Lサイズ1枚、どっちがお得？

ピザ屋でこんな会話をしたことはないだろうか？

「Mサイズ（直径25cm）を2枚頼もうか」
「いや、Lサイズ（直径35cm）1枚の方がお得じゃない？」

直感的には「M2枚 = L1枚くらい」と思うかもしれない。でも、相似比と面積比を知っていれば、驚きの事実がわかる。

計算してみよう：

サイズ 直径 面積
M 25 cm \(\pi \times 12.5^2 \approx 491 \text{ cm}^2\)
L 35 cm \(\pi \times 17.5^2 \approx 962 \text{ cm}^2\)

Mサイズ2枚の合計面積 = \(491 \times 2 = 982 \text{ cm}^2\)

Lサイズ1枚の面積 = \(962 \text{ cm}^2\)

あれ？ M2枚の方がちょっと多い！

でも待って。MサイズとLサイズの価格差を考えてみよう。もしMが1000円、Lが1800円だとしたら？

  • M2枚：2000円で982 cm² → 1cm²あたり約2.04円
  • L1枚：1800円で962 cm² → 1cm²あたり約1.87円

Lサイズの方が単価が安い！

これが「面積は半径の2乗に比例する」という相似の法則。直径が1.4倍（25→35cm）になると、面積は約2倍（\(1.4^2 \approx 1.96\)）になる。

次にピザを注文するときは、「相似比の2乗」を思い出して、1cm²あたりの値段を計算してみよう。数学ができると、お得に食べられる！
