1. テーマの定義

一次関数は \(y = ax + b\) という式で表されるシンプルな関数。でもテストでは「式を求めよ」「交点を求めよ」「変域を求めよ」など、いろんな聞き方をされる。問題を見た瞬間に「これはこのタイプだ！」とわかれば、迷わず解ける。

このページでは、一次関数の問題を5つのタイプに分類し、それぞれの攻略法を学びます：

  • タイプ1：式を求める：傾き・切片・通る点から式を決定
  • タイプ2：グラフを描く/読む：式からグラフ、グラフから式
  • タイプ3：交点を求める：2直線の連立方程式
  • タイプ4：変域を求める：xの範囲からyの範囲
  • タイプ5：利用問題：実生活の状況を一次関数で表す

2. 覚えるルール

【最重要】問題文のキーワードで解法を判断せよ！
問題タイプの判断フロー
📌 一次関数の基本式
\[ y = ax + b \]

a：傾き（変化の割合）= \(\frac{\text{yの増加量}}{\text{xの増加量}}\)　b：切片（x=0のときのy）

✅ メリット： aとbさえわかれば、どんな問題も解ける！

📌 タイプ1：式を求める → 例1
条件 解法
傾きaと切片bがわかる そのまま \(y = ax + b\) に代入
傾きaと1点(p, q)がわかる \(q = ap + b\) を解いてbを求める
2点(x₁, y₁)と(x₂, y₂)がわかる ① \(a = \frac{y_2 - y_1}{x_2 - x_1}\) でaを求める
② 1点を代入してbを求める
📌 タイプ3：交点を求める → 例3

2つの直線 \(y = a_1x + b_1\) と \(y = a_2x + b_2\) の交点は、連立方程式を解けばOK！

\[ a_1x + b_1 = a_2x + b_2 \]

これを解いてxを求め、どちらかの式に代入してyを求める。

📌 タイプ4：変域を求める → 例4

ポイント：傾きaの符号でyの増減が決まる！

  • \(a > 0\)（右上がり）：xが増えるとyも増える → yの最小はx最小のとき、yの最大はx最大のとき
  • \(a < 0\)（右下がり）：xが増えるとyは減る → yの最大はx最小のとき、yの最小はx最大のとき
📌 タイプ5：利用問題 → 例5

立式のコツ：「変わらないもの」を傾きa、「最初の値」を切片bにする

  • 速さの問題：\(y = (\text{速さ}) \times x + (\text{初期位置})\)
  • 料金の問題：\(y = (\text{単価}) \times x + (\text{基本料金})\)
  • 温度の問題：\(y = (\text{変化率}) \times x + (\text{初期温度})\)

3. 例題と解き方

5つのタイプを実際の問題で練習しよう。「このタイプだからこう解く」という流れを意識！

▶ タイプ1：式を求める

例12点を通る直線の式

2点 (1, 3) と (4, 9) を通る直線の式を求めよ。

解き方：

  1. 傾きaを求める：\(a = \frac{9 - 3}{4 - 1} = \frac{6}{3} = 2\)
  2. 式に1点を代入：\(y = 2x + b\) に (1, 3) を代入
  3. \(3 = 2 \times 1 + b\) より \(b = 1\)
\[ y = 2x + 1 \]

▶ タイプ2：グラフを読み取る

例2グラフから式を読み取る

下のグラフが表す一次関数の式を求めよ。（グラフは点(0, 2)を通り、傾きが-1の直線）

解き方：

  1. 切片を読み取る：y軸との交点は (0, 2) → b = 2
  2. 傾きを読み取る：xが1増えるとyが1減る → a = -1
\[ y = -x + 2 \]

▶ タイプ3：交点を求める

例32直線の交点

\(y = 2x - 1\) と \(y = -x + 5\) の交点の座標を求めよ。

解き方：

  1. 2つの式を等しくおく：交点では両方のyが等しい
  2. \(2x - 1 = -x + 5\)
  3. \(3x = 6\) より \(x = 2\)
  4. \(y = 2 \times 2 - 1 = 3\)
\[ \text{交点} = (2, 3) \]

▶ タイプ4：変域を求める

例4yの変域

\(y = -2x + 5\) で、xの変域が \(-1 \leq x \leq 3\) のとき、yの変域を求めよ。

解き方：

  1. 傾きを確認：\(a = -2 < 0\) → 右下がり（xが増えるとyは減る）
  2. x = -1 のとき：\(y = -2 \times (-1) + 5 = 7\)（yの最大値）
  3. x = 3 のとき：\(y = -2 \times 3 + 5 = -1\)（yの最小値）
\[ -1 \leq y \leq 7 \]

▶ タイプ5：利用問題

例5携帯電話の料金

ある携帯プランは基本料金1000円で、通話1分あたり20円かかる。x分通話したときの料金y円を一次関数で表せ。また、40分通話したときの料金を求めよ。

解き方：

  1. 傾きa：1分あたり20円 → a = 20
  2. 切片b：基本料金1000円 → b = 1000
  3. x = 40 を代入：\(y = 20 \times 40 + 1000 = 1800\)
\[ y = 20x + 1000 \]

40分通話したときの料金は1800円

4. 簡単にするコツ

✏️ コツ1：まずキーワードでタイプを判別
「式を求めよ」→ タイプ1、「交点」→ タイプ3、「変域」→ タイプ4。問題文のキーワードで解法が決まる！
✏️ コツ2：2点から傾きは「たて÷よこ」
傾き = (yの増加量) ÷ (xの増加量) = 「たて÷よこ」と覚える。グラフを見て「右に3、上に6」なら傾きは 6÷3 = 2。
✏️ コツ3：変域は端点を代入
yの変域を求めるときは、xの変域の両端を代入するだけ。傾きの符号で最大・最小が入れ替わることだけ注意。
✏️ コツ4：利用問題は「単価」と「初期値」を探す
「1分あたり○円」「1個あたり○g」→ これが傾き。「最初に○円」「出発時に○m」→ これが切片。

5. よくあるつまづき

⚠️ つまづき1：傾きの符号を間違える

ダメな例： (1, 5) と (3, 1) の傾きを \(\frac{5-1}{3-1} = 2\) としてしまう

対策： 引く順番を揃える！ \(\frac{1-5}{3-1} = \frac{-4}{2} = -2\) が正解。

⚠️ つまづき2：変域の最大・最小を逆にする

ダメな例： 傾きがマイナスなのに「xが大きいときyも大きい」と考える

対策： \(a < 0\) なら右下がり！ グラフをざっくり描いて確認しよう。

⚠️ つまづき3：交点でyを求め忘れる

ダメな例： xだけ求めて「交点は (2, ?)」で終わる

対策： 交点の座標は (x, y) のペア。xを求めたら必ずyも求める！

⚠️ つまづき4：切片をx軸との交点と勘違い

ダメな例： 「切片が3」を「(3, 0)を通る」と解釈

対策： 切片はy軸との交点のy座標。「b = 3」は「(0, 3)を通る」という意味。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1（タイプ1）

傾きが3で、点(2, 5)を通る直線の式を求めよ。

答えを見る

\(y = 3x + b\) に (2, 5) を代入

\(5 = 3 \times 2 + b\)

\(b = -1\)

答え：\(y = 3x - 1\)

練習2（タイプ3 + タイプ4）

\(y = x + 2\) と \(y = -2x + 8\) の交点を求め、さらに \(y = x + 2\) でxの変域が \(0 \leq x \leq 4\) のときのyの変域を求めよ。

答えを見る

交点：

\(x + 2 = -2x + 8\) より \(3x = 6\)、\(x = 2\)

\(y = 2 + 2 = 4\)

交点：(2, 4)

変域：

\(y = x + 2\) は傾き \(a = 1 > 0\)（右上がり）

x = 0 のとき y = 2、x = 4 のとき y = 6

yの変域：\(2 \leq y \leq 6\)

おまけ：数学雑学「−40度の奇跡」

🌡️ 華氏と摂氏が一致する温度

日本では温度を「摂氏（せっし）」で表す。水が0°Cで凍り、100°Cで沸騰。

一方、アメリカでは「華氏（かし）」を使う。水が32°Fで凍り、212°Fで沸騰。

この2つの温度、実は一次関数で変換できる：

\[ F = \frac{9}{5}C + 32 \]

（Fは華氏、Cは摂氏）

ここで問題。「華氏と摂氏が同じ数字になる温度はあるか？」

直感的には「ないでしょ、だって単位が違うし...」と思うかもしれない。

でも、F = C として方程式を解いてみると...

\[ C = \frac{9}{5}C + 32 \]

\(C - \frac{9}{5}C = 32\) より \(-\frac{4}{5}C = 32\)、つまり \(C = -40\)

−40°C = −40°F

なんと、マイナス40度のとき、摂氏と華氏はぴったり同じ数字になる！

カナダやアラスカの真冬、「今日は−40度だよ」と言えば、摂氏でも華氏でも同じ。変換不要だ。...まあ、どっちにしても極寒だけど。

一次関数のグラフで考えると、\(y = \frac{9}{5}x + 32\) と \(y = x\) の交点が (−40, −40) というわけ。2直線の交点を求める問題そのものだ！
