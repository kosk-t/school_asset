1. テーマの定義

「\(3x^2y\)」と「\(2x^2y\)」を足すとどうなる？「\(S = \frac{1}{2}ah\)」を \(h\) について解くとどうなる？中2の式の計算では、中1で学んだ文字式をさらに発展させ、より複雑な式を扱います。

このページでは、式の計算の基本を学びます：

  • 単項式と多項式：項・次数・係数の理解
  • 多項式の加減：同類項をまとめる
  • 単項式の乗除：指数法則を使う
  • 等式の変形：ある文字について解く

2. 覚えるルール

【最重要】同類項のまとめ方：
文字の部分が同じ項だけをまとめる。係数を足し引きする。
単項式と多項式の用語
📌 単項式と多項式
  • 単項式：数や文字の積だけでできた式（例：\(5x\)、\(-3ab\)、\(7\)）
  • 多項式：単項式の和でできた式（例：\(2x + 3\)、\(x^2 - 5x + 6\)）
  • 項：多項式を構成する各単項式
  • 係数：文字にかかっている数
📌 次数
  • 単項式の次数：かけ合わされている文字の個数（\(3x^2y\) は次数3）
  • 多項式の次数：各項の次数のうち最大のもの

例： \(5x^3 + 2x^2 - x + 4\) の次数は3（最高次の項 \(5x^3\) が3次）

📌 多項式の加法・減法 → 例1
  1. かっこを外す（減法の場合は符号を変える）
  2. 同類項をまとめる
\[ (3x^2 + 2x) + (x^2 - 5x) = 3x^2 + 2x + x^2 - 5x = 4x^2 - 3x \]
📌 単項式の乗法 → 例3
  1. 係数同士をかける
  2. 同じ文字は指数を足す（\(x^a \times x^b = x^{a+b}\)）
\[ 3x^2 \times 2x^3 = 6x^{2+3} = 6x^5 \]
📌 単項式の除法 → 例4
  1. 係数同士を割る
  2. 同じ文字は指数を引く（\(x^a \div x^b = x^{a-b}\)）
\[ 12x^5 \div 4x^2 = 3x^{5-2} = 3x^3 \]
📌 等式の変形 → 例5

等式をある文字について解く（その文字 = の形にする）

  1. 目的の文字を含む項を一方に集める
  2. 目的の文字の係数で両辺を割る
\[ 2x + 3y = 6 \text{ を } x \text{ について解く} \] \[ 2x = 6 - 3y \] \[ x = \frac{6 - 3y}{2} = 3 - \frac{3y}{2} \]

✅ メリット： 公式を変形できれば、求めたい量に合わせて使い分けられる

3. 例題と解き方

式の計算の基本操作を練習しましょう。

▶ 多項式の加法・減法

例1多項式の加法

\((5x^2 + 3x - 2) + (2x^2 - 4x + 7)\) を計算せよ。

解き方：

  1. かっこを外す：\(5x^2 + 3x - 2 + 2x^2 - 4x + 7\)
  2. \(x^2\) の項：\(5x^2 + 2x^2 = 7x^2\)
  3. \(x\) の項：\(3x - 4x = -x\)
  4. 定数項：\(-2 + 7 = 5\)
\[ (5x^2 + 3x - 2) + (2x^2 - 4x + 7) = 7x^2 - x + 5 \]
例2多項式の減法

\((4x^2 + 2x - 3) - (x^2 - 5x + 1)\) を計算せよ。

解き方：

  1. 引くかっこ内の符号を反転：\(4x^2 + 2x - 3 - x^2 + 5x - 1\)
  2. \(x^2\) の項：\(4x^2 - x^2 = 3x^2\)
  3. \(x\) の項：\(2x + 5x = 7x\)
  4. 定数項：\(-3 - 1 = -4\)
\[ (4x^2 + 2x - 3) - (x^2 - 5x + 1) = 3x^2 + 7x - 4 \]

▶ 単項式の乗法・除法

例3単項式の乗法

\(4x^2y \times 3xy^2\) を計算せよ。

解き方：

  1. 係数：\(4 \times 3 = 12\)
  2. \(x\) の指数：\(2 + 1 = 3\)
  3. \(y\) の指数：\(1 + 2 = 3\)
\[ 4x^2y \times 3xy^2 = 12x^3y^3 \]
例4単項式の除法

\(18a^3b^2 \div 6ab\) を計算せよ。

解き方：

  1. 係数：\(18 \div 6 = 3\)
  2. \(a\) の指数：\(3 - 1 = 2\)
  3. \(b\) の指数：\(2 - 1 = 1\)
\[ 18a^3b^2 \div 6ab = 3a^2b \]

▶ 等式の変形

例5等式を変形する（基本）

\(y = 3x - 6\) を \(x\) について解け。

解き方：

  1. \(x\) を含む項を左辺に移動：\(y + 6 = 3x\)
  2. 両辺を3で割る：\(x = \frac{y + 6}{3}\)
\[ x = \frac{y + 6}{3} \]
例6公式の変形

三角形の面積の公式 \(S = \frac{1}{2}ah\) を \(h\) について解け。

解き方：

  1. 両辺に2をかける：\(2S = ah\)
  2. 両辺を \(a\) で割る：\(h = \frac{2S}{a}\)
\[ h = \frac{2S}{a} \]
例7速さの公式の変形

速さの公式 \(v = \frac{d}{t}\) を \(t\) について解け。

解き方：

  1. 両辺に \(t\) をかける：\(vt = d\)
  2. 両辺を \(v\) で割る：\(t = \frac{d}{v}\)
\[ t = \frac{d}{v} \]

確認： 「時間 = 距離 ÷ 速さ」で意味も合っている！

▶ 式の値

例8式の値を求める

\(a = 2, b = -3\) のとき、\(a^2 - 2ab + b^2\) の値を求めよ。

解き方：

  1. \(a^2 = 2^2 = 4\)
  2. \(-2ab = -2 \times 2 \times (-3) = 12\)
  3. \(b^2 = (-3)^2 = 9\)
  4. 合計：\(4 + 12 + 9 = 25\)
\[ a^2 - 2ab + b^2 = 4 + 12 + 9 = 25 \]

別解： \(a^2 - 2ab + b^2 = (a - b)^2 = (2 - (-3))^2 = 5^2 = 25\)

4. 簡単にするコツ

✏️ コツ1：同類項は縦に揃えて計算
複雑な多項式は、同じ次数の項を縦に揃えて書くと間違いにくい。 
  3x² + 2x - 5
+ x² - 4x + 3
─────────────
  4x² - 2x - 2
✏️ コツ2：指数法則を暗記
  • 乗法：\(x^a \times x^b = x^{a+b}\)（指数を足す）
  • 除法：\(x^a \div x^b = x^{a-b}\)（指数を引く）
  • 累乗：\((x^a)^b = x^{ab}\)（指数をかける）
✏️ コツ3：等式変形は「逆算」で考える
「\(y = 3x + 5\) を \(x\) について解く」なら：
「\(x\) に何をしたら \(y\) になるか」を逆にたどる。
  1. \(x\) に3をかけた → 3で割る
  2. 5を足した → 5を引く
✏️ コツ4：式の値は「先に簡単化」できないか確認
\(a^2 - 2ab + b^2\) は \((a-b)^2\) と変形できる。
先に因数分解や公式を使うと、計算が楽になることがある。

5. よくあるつまづき

⚠️ つまづき1：減法で符号変換を忘れる

ダメな例： \((3x + 2) - (x - 5) = 3x + 2 - x - 5 = 2x - 3\)

対策： 引くかっこ内はすべての符号を変える！

\[ (3x + 2) - (x - 5) = 3x + 2 - x + 5 = 2x + 7 \]
⚠️ つまづき2：指数の計算ミス

ダメな例： \(x^2 \times x^3 = x^6\)（指数をかけてしまう）

対策：

  • 乗法：指数を足す（\(x^2 \times x^3 = x^5\)）
  • 累乗の累乗：指数をかける（\((x^2)^3 = x^6\)）
⚠️ つまづき3：異なる文字をまとめる

ダメな例： \(3xy + 2x^2 = 5x^3y\)

対策： \(xy\) と \(x^2\) は同類項ではない。まとめられない！

⚠️ つまづき4：等式変形で分母を忘れる

ダメな例： \(ax = b\) を \(x\) について解いて \(x = b - a\)

対策： かけ算を外すには割り算！

\[ ax = b \Rightarrow x = \frac{b}{a} \]

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の計算をせよ。

(1) \((3x^2 - 4x + 5) + (2x^2 + x - 3)\)

(2) \((5a - 2b) - (3a - 4b)\)

答えを見る

(1) \(5x^2 - 3x + 2\)

(2) \(5a - 2b - 3a + 4b = 2a + 2b\)（符号変換に注意）

練習2

次の計算をせよ。

(1) \(5x^2y \times 2xy^3\)

(2) \(24a^4b^3 \div 8a^2b\)

答えを見る

(1) \(10x^3y^4\)　（係数：5×2=10、x：2+1=3、y：1+3=4）

(2) \(3a^2b^2\)　（係数：24÷8=3、a：4-2=2、b：3-1=2）

練習3

次の等式を [ ] 内の文字について解け。

(1) \(y = 2x - 8\) [\(x\)]

(2) \(S = \pi r^2\) [\(r^2\)]

答えを見る

(1) \(x = \frac{y + 8}{2}\)

(2) \(r^2 = \frac{S}{\pi}\)

📌 まとめ：同類項をまとめ、指数法則を使い、等式は「逆算」で変形！

おまけ：数学雑学「次元の話」

📏 なぜ「次数」は大事なのか？

「次数」という言葉、なんだか難しそうに聞こえるけど、実は物理の世界と深くつながっている。

【長さ × 長さ = 面積】

縦3m、横5mの長方形の面積は？

\[ 3\text{m} \times 5\text{m} = 15\text{m}^2 \]

答えは「15平方メートル」。単位に「2乗」がついた！

【長さ × 長さ × 長さ = 体積】

一辺2mの立方体の体積は？

\[ 2\text{m} \times 2\text{m} \times 2\text{m} = 8\text{m}^3 \]

答えは「8立方メートル」。単位に「3乗」がついた！

このように、次数は「次元」を表している。

次数物理的意味単位の例
1次長さm
2次面積m²
3次体積m³

だから、「\(x^2\) と \(x^3\) は足せない」というルールは、「面積と体積は足せない」と同じ意味なのだ。

結論：次数を合わせるルールは、物理世界の「次元」を反映している！
