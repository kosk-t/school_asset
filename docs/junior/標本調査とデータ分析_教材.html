1. テーマの定義

テレビの視聴率調査や選挙の出口調査では、全員にアンケートを取るのではなく、一部の人だけ調べて全体の傾向を予測します。このように「一部から全体を推測する」のが標本調査です。

このページでは、データ分析の実践的な手法を学びます：

  • 母集団と標本：調べたい全体と、実際に調べる一部の関係
  • 全数調査と標本調査：すべて調べる方法と一部だけ調べる方法の使い分け
  • 四分位数と箱ひげ図：データのばらつきを視覚的に把握する方法

2. 覚えるルール

【最重要】母集団（ぼしゅうだん）：調査の対象となる全体。標本（ひょうほん）：実際に調査する一部。標本から母集団の性質を推測するのが標本調査。
母集団から標本を抽出する様子。標本調査では、母集団全体の性質を標本から推測する
📌 全数調査と標本調査の違い → 例1

全数調査（ぜんすうちょうさ）：母集団のすべてを調べる方法。

例：国勢調査（全国民を調べる）、学校のクラス全員のテスト結果

→ メリット： 正確なデータが得られる

→ デメリット： 時間・費用・労力がかかる

標本調査（ひょうほんちょうさ）：母集団の一部だけを調べて全体を推測する方法。

例：視聴率調査、出口調査、製品の抜き取り検査

→ メリット： 時間・費用・労力が少なくて済む

→ デメリット： 推測なので誤差が生じる可能性がある

✅ 使い分け：

  • 全数調査：正確さが最重要な場合（国勢調査、選挙の開票など）
  • 標本調査：迅速さやコストが重要な場合、または調査で破壊される場合（電球の寿命検査など）
📌 無作為抽出（むさくいちゅうしゅつ） → 例2

標本を選ぶとき、どの個体も同じ確率で選ばれるようにする方法。

→ だから結果は： 標本が母集団を公平に代表し、偏り（かたより）のない推測ができる

✅ メリット： 特定の集団に偏らず、母集団の性質を正しく推測できる

💡 考え方を見る

例えば、中学生の身長調査で「背の高い生徒ばかり選ぶ」と平均身長が実際より高くなってしまいます。無作為抽出では、くじ引きや乱数表を使ってランダムに選ぶことで、このような偏りを防ぎます。

📌 四分位数（しぶんいすう） → 例3

データを小さい順に並べたとき、4等分する3つの値。

\[ \begin{aligned} &\text{第1四分位数 } Q_1: \text{下位25%の位置（データを4等分して1番目）} \\ &\text{第2四分位数 } Q_2: \text{中央値（データを4等分して2番目 = データの真ん中）} \\ &\text{第3四分位数 } Q_3: \text{上位25%の位置（データを4等分して3番目）} \end{aligned} \]

→ だから結果は： データの散らばり具合（ばらつき）がわかる

四分位範囲（しぶんいはんい）： \(Q_3 - Q_1\)。データの真ん中50%の幅で、ばらつきの大きさを表す

📌 箱ひげ図（はこひげず） → 例4

四分位数・最小値・最大値の5つの値を使って、データの分布を1つの図にまとめたもの。

箱ひげ図の構造。箱の長さが四分位範囲（データの真ん中50%の幅）を表す

→ だから結果は： 複数のデータ分布を一目で比較できる

✅ メリット： 平均値だけではわからない「データの散らばり」や「偏り」を視覚的に把握できる

3. 例題と解き方

標本調査の考え方と、四分位数・箱ひげ図の計算を実際にやってみましょう。

▶ 母集団と標本の理解

例1全数調査か標本調査か？

次の調査は全数調査と標本調査のどちらが適切か答えなさい。

(1) ある工場で生産された電池1万個の寿命を調べる

(2) クラス全員の数学のテスト結果を調べる

(3) 全国の中学生のスマートフォン所持率を調べる

解き方：

  1. 調査の目的と制約を考える
  2. すべて調べる必要があるか、一部で推測できるかを判断
  3. 破壊検査（調べると使えなくなる）かどうかも考慮

答え：

(1) 標本調査 → 電池の寿命を調べると電池が使えなくなる（破壊検査）ので、全部調べると製品が残らない

(2) 全数調査 → クラス全員なら時間も費用もかからないので、全員調べた方が正確

(3) 標本調査 → 全国の中学生（約300万人）全員を調べるのは時間・費用がかかりすぎるので、一部から推測

例2無作為抽出の重要性

ある市の中学生1万人の平均身長を調べるため、300人の標本を抽出する。次の方法のうち、無作為抽出といえるものはどれか。

(a) バスケットボール部の生徒300人を選ぶ

(b) 全生徒の名簿から、くじ引きで300人を選ぶ

(c) 身長順に並べて、上から300人を選ぶ

解き方：

  1. どの生徒も同じ確率で選ばれるかをチェック
  2. 特定の集団に偏っていないかを確認

答え：(b)

(a) はバスケ部員に偏る → 背の高い生徒が多く選ばれてしまう

(b) はくじ引きなのでどの生徒も同じ確率 → 無作為抽出

(c) は背の高い順に選んでいる → 平均身長が実際より高くなる

▶ 四分位数の計算

例3四分位数を求める

次のデータの第1四分位数\(Q_1\)、第2四分位数\(Q_2\)、第3四分位数\(Q_3\)を求めなさい。

データ：3, 5, 7, 8, 10, 12, 15, 18, 20（9個）

解き方：

  1. データを小さい順に並べる（すでに並んでいる）
  2. \(Q_2\)（中央値）を求める → データ数が9個（奇数）なので、真ん中は5番目 → \(Q_2 = 10\)
  3. \(Q_1\)を求める → \(Q_2\)より小さいデータ（3, 5, 7, 8）の中央値 → 2番目と3番目の平均 → \((5 + 7) ÷ 2 = 6\) → \(Q_1 = 6\)
  4. \(Q_3\)を求める → \(Q_2\)より大きいデータ（12, 15, 18, 20）の中央値 → 2番目と3番目の平均 → \((15 + 18) ÷ 2 = 16.5\) → \(Q_3 = 16.5\)
\[ Q_1 = 6, \quad Q_2 = 10, \quad Q_3 = 16.5 \]

四分位範囲： \(Q_3 - Q_1 = 16.5 - 6 = 10.5\)

▶ 箱ひげ図の作成と読み取り

例4箱ひげ図を描く

例3のデータから箱ひげ図を描きなさい。

データ：3, 5, 7, 8, 10, 12, 15, 18, 20（最小値=3、最大値=20、\(Q_1=6\)、\(Q_2=10\)、\(Q_3=16.5\)）

解き方：

  1. 数直線を描く（最小値〜最大値が入る範囲）
  2. 5つの値（最小値、\(Q_1\)、\(Q_2\)、\(Q_3\)、最大値）を数直線上にマーク
  3. \(Q_1\)から\(Q_3\)まで箱を描く
  4. \(Q_2\)（中央値）の位置に縦線を引く
  5. 最小値から\(Q_1\)、\(Q_3\)から最大値まで「ひげ」を伸ばす
例3のデータの箱ひげ図
例52つの箱ひげ図を比較する

A組とB組の数学のテスト結果（100点満点）を箱ひげ図で表した。どちらのクラスの方が成績のばらつきが大きいか答えなさい。

A組とB組の箱ひげ図の比較

解き方：

  1. 四分位範囲（箱の長さ）を比較する
  2. A組：\(Q_3 - Q_1 = 75 - 45 = 30\)
  3. B組：\(Q_3 - Q_1 = 70 - 55 = 15\)

答え：A組の方がばらつきが大きい

理由：A組の四分位範囲（30点）の方が、B組（15点）より大きいため。また、A組の方が最大値と最小値の差（範囲）も大きい。

4. 簡単にするコツ

✏️ コツ1：全数調査か標本調査かは「3つの質問」で判断
  1. 母集団の数は多い？ → 多ければ標本調査が有利
  2. 調べると壊れる？（破壊検査） → 壊れるなら標本調査しかできない
  3. 時間・費用は限られている？ → 限られていれば標本調査
これらのどれか1つでも当てはまれば標本調査、すべて当てはまらず正確さが最重要なら全数調査。
✏️ コツ2：四分位数は「データを4等分する3つの仕切り」と覚える
データを小さい順に並べて4つのブロックに分けたとき、3つの仕切りが\(Q_1\)、\(Q_2\)、\(Q_3\)。
  • \(Q_2\)（中央値）から先に求める
  • \(Q_1\)は「\(Q_2\)より小さいデータ」の中央値
  • \(Q_3\)は「\(Q_2\)より大きいデータ」の中央値
✏️ コツ3：箱ひげ図は「5つの値」さえわかれば描ける
最小値、\(Q_1\)、\(Q_2\)、\(Q_3\)、最大値の5つを数直線上にマークすれば、あとは箱とひげを繋ぐだけ。
  • 箱の左端 = \(Q_1\)
  • 箱の真ん中の線 = \(Q_2\)
  • 箱の右端 = \(Q_3\)
  • 左のひげ = 最小値から\(Q_1\)
  • 右のひげ = \(Q_3\)から最大値
✏️ コツ4：ばらつきを比べるには「箱の長さ」を見る
箱ひげ図で複数のデータを比較するとき、箱の長さ（四分位範囲）がばらつきの指標。
  • 箱が長い → ばらつきが大きい
  • 箱が短い → ばらつきが小さい（データが集中している）
✏️ コツ5：標本調査の精度は「標本の大きさ」と「無作為性」で決まる
  • 標本を大きくする → 推測の誤差が小さくなる
  • 無作為抽出する → 偏りのない推測ができる
どちらか一方だけでは不十分。無作為に選ばれた大きな標本が理想。

5. よくあるつまづき

⚠️ つまづき1：母集団と標本を逆に考える

ダメな例： 「ある市の中学生300人にアンケート」で、「300人が母集団」と考える

対策： 母集団 = 知りたい全体（この場合、市の全中学生）、標本 = 実際に調べた一部（300人）。「調べたい全体は何か？」を最初に考える。

⚠️ つまづき2：四分位数を求めるとき\(Q_2\)を2回数える

ダメな例： データが9個のとき、\(Q_1\)を求めるために「1番目から5番目（\(Q_2\)を含む）」の中央値を求める

対策： データ数が奇数のとき、\(Q_2\)（真ん中の値）は\(Q_1\)や\(Q_3\)を求める計算には含めない。\(Q_2\)より小さいデータだけ、\(Q_2\)より大きいデータだけを使う。

⚠️ つまづき3：箱ひげ図で「箱の真ん中の線」を平均値と勘違い

ダメな例： 箱の真ん中の線を「平均値」だと思い込む

対策： 箱の真ん中の線は中央値（\(Q_2\)）であって平均値ではない。中央値はデータを並べた真ん中、平均値はすべての値の合計÷個数。

⚠️ つまづき4：無作為抽出を「適当に選ぶ」と誤解する

ダメな例： 「目についた人を選ぶ」「手前から順に選ぶ」を無作為抽出だと思う

対策： 無作為抽出はどの個体も同じ確率で選ばれるように、くじ引きや乱数表を使って機械的に選ぶこと。人間の判断で「適当に」選ぶと偏りが生じる。

⚠️ つまづき5：箱ひげ図で「ひげの長さ」だけ見てばらつきを判断

ダメな例： 最大値と最小値の差（範囲）だけでばらつきを比較する

対策： 最大値・最小値は「外れ値（極端な値）」の影響を受けやすい。ばらつきを比べるには四分位範囲（箱の長さ）を見る。箱の長さが長いほど、データの真ん中50%が広く散らばっている。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次のデータの四分位数（\(Q_1\)、\(Q_2\)、\(Q_3\)）と四分位範囲を求めなさい。

データ：4, 6, 8, 9, 11, 13, 14, 16（8個）

答えを見る

解答：

  1. データ数8個（偶数）なので、\(Q_2\)は4番目と5番目の平均 → \((9 + 11) ÷ 2 = 10\) → \(Q_2 = 10\)
  2. \(Q_1\)：\(Q_2\)より小さいデータ（4, 6, 8, 9）の中央値 → 2番目と3番目の平均 → \((6 + 8) ÷ 2 = 7\) → \(Q_1 = 7\)
  3. \(Q_3\)：\(Q_2\)より大きいデータ（11, 13, 14, 16）の中央値 → 2番目と3番目の平均 → \((13 + 14) ÷ 2 = 13.5\) → \(Q_3 = 13.5\)
  4. 四分位範囲：\(Q_3 - Q_1 = 13.5 - 7 = 6.5\)

答え：\(Q_1 = 7\)、\(Q_2 = 10\)、\(Q_3 = 13.5\)、四分位範囲 = 6.5

練習2

ある学校で、全校生徒600人のうち300人に「朝食を食べるか」をアンケート調査したところ、240人が「はい」と答えた。全校生徒のうち、朝食を食べる生徒は何人と推測されるか。

答えを見る

解答：

  1. 標本（300人）のうち、朝食を食べる割合 = \(240 ÷ 300 = 0.8\)（80%）
  2. 母集団（600人）にこの割合を適用 → \(600 × 0.8 = 480\)人

答え：480人

おまけ：歴史ストーリー「選挙速報の秘密」

📖 選挙速報の秘密

選挙の日、テレビをつけると「当選確実」の速報が流れる。投票が締め切られてまだ数分しか経っていないのに、なぜもう結果がわかるのだろうか？

実は、開票作業はとても時間がかかる。数万票、数十万票を人の手で数えるには何時間もかかる。でも、テレビ局は「出口調査」という標本調査を使っている。投票所から出てきた有権者に「誰に投票しましたか？」と聞くのだ。

なぜ数百人に聞くだけで結果がわかるのか？

例えば、ある選挙区で10万人が投票し、出口調査で500人に聞いたとする。この500人（標本）のうち、A候補に投票した人が300人（60%）なら、10万人（母集団）のうち約6万人がA候補に投票したと推測できる。

\[ \text{推定得票数} = 100{,}000 \times 0.6 = 60{,}000 \text{票} \]

もちろん、500人だけの調査では誤差がある。でも、無作為抽出で偏りなく選び、標本を大きくすれば（例：1000人、2000人）、誤差は小さくなる。

統計学の理論によれば、標本の大きさが\(n\)のとき、誤差は約\(\frac{1}{\sqrt{n}}\)に比例する。つまり、1000人に聞けば誤差は約3%程度に収まり、「当選確実」と判断できるのだ。

世界初の科学的世論調査

このアイデアを確立したのは、アメリカの統計学者ジョージ・ギャラップ（1901-1984）。1936年のアメリカ大統領選挙で、ギャラップは5万人への調査で結果を的中させた。一方、当時の有名な雑誌『リテラリー・ダイジェスト』は200万人にアンケートを送ったのに、結果を大外ししてしまった。

なぜか？ 雑誌は電話帳や自動車登録名簿から回答者を選んだため、裕福な層に偏ってしまったのだ。一方、ギャラップは人数は少ないが無作為抽出で偏りなく選んだため、正確な予測ができた。

教訓：大きな標本より、偏りのない標本

この事件は「標本調査の精度は、標本の大きさだけでなく、無作為性（偏りのなさ）で決まる」ことを証明した。数学の力で、未来を予測できる――それが標本調査の魅力だ！
