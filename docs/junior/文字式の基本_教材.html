1. テーマの定義

「りんごを3個買って、みかんを何個か買った」とき、合計は何個？みかんの数がわからないままでも、\(3 + x\) と書けば「りんご3個＋みかんx個」を表せる。文字を使えば、未知の数や変化する値を扱えるようになります。

このページでは、文字式の基本を学びます：

  • 文字式の表し方：積・商の省略ルール
  • 式の値：文字に数を代入して計算
  • 一次式の計算：同類項をまとめて加減

2. 覚えるルール

【最重要】文字式の省略ルール：
× は省略、÷ は分数で書く、数字は文字の前
文字式の省略ルール
📌 積（掛け算）の表し方 → 例1
ルール 普通の書き方 文字式での書き方
×を省略 \(a \times b\) \(ab\)
数字は前 \(x \times 3\) \(3x\)
1を省略 \(1 \times x\) \(x\)
-1は−だけ \((-1) \times x\) \(-x\)
同じ文字は累乗 \(a \times a \times a\) \(a^3\)
📌 商（割り算）の表し方 → 例2

割り算は分数で表す。÷ 記号は使わない。

\[ a \div b = \frac{a}{b}, \quad x \div 3 = \frac{x}{3}, \quad (a + b) \div 2 = \frac{a + b}{2} \]
📌 式の値（代入） → 例3

文字に具体的な数を入れることを代入という。代入して計算した結果を式の値という。

\[ x = 3 \text{ のとき、} 2x + 5 = 2 \times 3 + 5 = 11 \]

⚠️ 注意： 代入するときは×を復活させる！（\(2x\) → \(2 \times 3\)）

💡 考え方を見る

「代入」は「置き換える」という意味。\(x\) という箱に \(3\) を入れるイメージ。

📌 項と係数
  • 項：＋や−で区切られた部分（\(3x - 5\) なら項は \(3x\) と \(-5\)）
  • 係数：文字にかかっている数（\(3x\) の係数は \(3\)）
  • 定数項：文字を含まない項（\(-5\) は定数項）
\[ \underbrace{3x}_{\text{係数3}} + \underbrace{(-5)}_{\text{定数項}} \]
📌 同類項 → 例5

同類項とは、文字の部分が同じ項のこと。

  • \(3x\) と \(5x\) は同類項（文字部分が \(x\) で同じ）
  • \(3x\) と \(3y\) は同類項ではない（文字が違う）
  • \(x^2\) と \(x\) は同類項ではない（次数が違う）

→ だから結果は： 同類項は係数同士を足し引きしてまとめられる

📌 一次式の加法・減法 → 例6
  1. かっこを外す（減法の場合は符号を変える）
  2. 同類項をまとめる
\[ (3x + 2) + (5x - 4) = 3x + 2 + 5x - 4 = 8x - 2 \]
\[ (3x + 2) - (5x - 4) = 3x + 2 - 5x + 4 = -2x + 6 \]

⚠️ 減法のコツ： 引くかっこの中は、すべての符号が反転する！

3. 例題と解き方

文字式のルールを使って、実際に式を書いたり計算したりしてみましょう。

▶ 文字式の表し方

例1積を文字式で表す

次の式を文字式で表せ。

(1) \(a \times 4\)　(2) \(x \times x \times y\)　(3) \((-1) \times a \times b\)

解き方：

  1. (1) 数字は前、×を省略 → \(4a\)
  2. (2) 同じ文字は累乗、アルファベット順 → \(x^2 y\)
  3. (3) -1は−だけ、アルファベット順 → \(-ab\)
例2商を文字式で表す

次の式を文字式で表せ。

(1) \(a \div 5\)　(2) \((x + y) \div 3\)

解き方：

  1. (1) 分数で表す → \(\displaystyle \frac{a}{5}\)
  2. (2) かっこごと分数に → \(\displaystyle \frac{x + y}{3}\)

▶ 式の値

例3式の値を求める

\(x = -2\) のとき、\(3x^2 - 4x + 1\) の値を求めよ。

解き方：

  1. \(x\) に \(-2\) を代入（×を復活、かっこをつける）
  2. \(3 \times (-2)^2 - 4 \times (-2) + 1\)
  3. 累乗から計算：\(3 \times 4 + 8 + 1\)
  4. \(= 12 + 8 + 1 = 21\)
\[ 3x^2 - 4x + 1 = 3 \times (-2)^2 - 4 \times (-2) + 1 = 12 + 8 + 1 = 21 \]
例42つの文字を代入

\(a = 3, b = -2\) のとき、\(2a - 3b\) の値を求めよ。

解き方：

  1. \(a\) に \(3\)、\(b\) に \(-2\) を代入
  2. \(2 \times 3 - 3 \times (-2)\)
  3. \(= 6 + 6 = 12\)
\[ 2a - 3b = 2 \times 3 - 3 \times (-2) = 6 + 6 = 12 \]

▶ 一次式の計算

例5同類項をまとめる

\(5x + 3 - 2x + 7\) を簡単にせよ。

解き方：

  1. \(x\) の項を集める：\(5x - 2x = 3x\)
  2. 定数項を集める：\(3 + 7 = 10\)
\[ 5x + 3 - 2x + 7 = (5x - 2x) + (3 + 7) = 3x + 10 \]
例6一次式の加法

\((4x - 3) + (2x + 5)\) を計算せよ。

解き方：

  1. かっこを外す：\(4x - 3 + 2x + 5\)
  2. 同類項をまとめる：\((4x + 2x) + (-3 + 5)\)
  3. \(= 6x + 2\)
\[ (4x - 3) + (2x + 5) = 6x + 2 \]
例7一次式の減法

\((4x - 3) - (2x + 5)\) を計算せよ。

解き方：

  1. 引くかっこの中の符号をすべて反転：\(4x - 3 \textcolor{red}{- 2x - 5}\)
  2. 同類項をまとめる：\((4x - 2x) + (-3 - 5)\)
  3. \(= 2x - 8\)
\[ (4x - 3) - (2x + 5) = 4x - 3 - 2x - 5 = 2x - 8 \]
例8分数係数の計算

\(\displaystyle \frac{2x - 1}{3} + \frac{x + 2}{2}\) を計算せよ。

解き方：

  1. 通分する（分母の最小公倍数は6）
  2. \(\displaystyle \frac{2(2x - 1)}{6} + \frac{3(x + 2)}{6}\)
  3. 分子を展開：\(\displaystyle \frac{4x - 2 + 3x + 6}{6}\)
  4. 同類項をまとめる：\(\displaystyle \frac{7x + 4}{6}\)
\[ \frac{2x - 1}{3} + \frac{x + 2}{2} = \frac{2(2x-1) + 3(x+2)}{6} = \frac{7x + 4}{6} \]

4. 簡単にするコツ

✏️ コツ1：代入するときは「かっこ」をつける
負の数を代入するときは必ずかっこをつける。忘れると符号ミスの原因に。
\[ x = -3 \text{ のとき } 2x = 2 \times (-3) = -6 \]
✏️ コツ2：減法は「＋(−)」に変換
減法で迷ったら、引く式全体に \(-1\) をかけていると考える。
\[ A - B = A + (-1) \times B \]
✏️ コツ3：同類項は縦に並べる
複雑な式は、同類項を縦に並べて書くとまとめやすい。 
  5x + 3
+ 2x - 7
---------
  7x - 4
✏️ コツ4：分数係数は最後に約分
分数の計算は通分してから同類項をまとめ、最後に約分する。途中で約分すると計算が複雑になることも。

5. よくあるつまづき

⚠️ つまづき1：\(2a\) と \(a^2\) の混同

ダメな例： \(a + a = a^2\) と計算

対策：

  • \(a + a = 2a\)（足し算 → 係数が増える）
  • \(a \times a = a^2\)（掛け算 → 指数が増える）
⚠️ つまづき2：減法で符号を変え忘れる

ダメな例： \((3x + 2) - (x + 5) = 3x + 2 - x + 5 = 2x + 7\)

対策： 引くかっこの中はすべての項の符号が変わる！

\[ (3x + 2) - (x + 5) = 3x + 2 \textcolor{red}{- x - 5} = 2x - 3 \]
⚠️ つまづき3：異なる文字をまとめる

ダメな例： \(3x + 2y = 5xy\)

対策： 文字が違う項はまとめられない。\(3x + 2y\) のままが答え。

⚠️ つまづき4：代入で×を忘れる

ダメな例： \(x = 2\) のとき \(3x = 32\)

対策： 代入するときは×を復活させる！

\[ 3x = 3 \times 2 = 6 \]

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の式を文字式で表せ。

(1) \(x \times (-5) \times y\)

(2) \((a - b) \div 4\)

(3) \(a \times a \times a \times b \times b\)

答えを見る

(1) \(-5xy\)　（数字は前、アルファベット順）

(2) \(\displaystyle \frac{a - b}{4}\)　（割り算は分数）

(3) \(a^3 b^2\)　（同じ文字は累乗）

練習2

\(x = -2, y = 3\) のとき、次の式の値を求めよ。

(1) \(4x - 2y\)

(2) \(x^2 + xy\)

答えを見る

(1) \(4 \times (-2) - 2 \times 3 = -8 - 6 = -14\)

(2) \((-2)^2 + (-2) \times 3 = 4 - 6 = -2\)

練習3

次の計算をせよ。

(1) \((5x - 3) + (2x + 7)\)

(2) \((4a + 6) - (3a - 2)\)

答えを見る

(1) \(5x - 3 + 2x + 7 = 7x + 4\)

(2) \(4a + 6 - 3a + 2 = a + 8\)　（引くかっこ内の符号反転に注意）

📌 まとめ：文字式は「×省略・÷は分数・数字は前」のルールを守り、同類項だけをまとめる！

おまけ：歴史ストーリー「なぜ未知数はxなのか」

📖 アラビア語から生まれた「x」

数学で未知数といえば \(x\)。なぜ \(a\) でも \(n\) でもなく \(x\) なのか？その答えは、翻訳ミスにあった...かもしれない。

9世紀のバグダッド。数学者アル=フワーリズミーは、未知数を「シャイ」（アラビア語で「もの」）と呼んでいた。

この知識がスペインに伝わったとき、翻訳者たちは困った。アラビア語の「シャイ」の音は、スペイン語にない発音だったのだ。

そこで彼らは、ギリシャ文字の「χ」（カイ）で代用。やがてこれがラテン語圏で「x」に変わった...という説がある。

ただし、別の説もある。17世紀の数学者デカルトが、印刷の都合で「x」を選んだというものだ。

当時の印刷所には、アルファベットの活字がたくさんあった。でも「x」はあまり使わない文字だったので、余っていた。

「未知数には余ってる活字を使おう。a, b, c は既知数、x, y, z は未知数だ」

こうして、印刷所の都合で \(x\) が数学の主役になった...かもしれない。

結論：「x」が選ばれた理由は、翻訳の都合か、印刷の都合か、今も謎のまま
