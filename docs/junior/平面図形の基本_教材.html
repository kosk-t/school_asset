1. テーマの定義

図形はすべて「点」と「線」からできています。三角形も四角形も円も、基本となるパーツを組み合わせたもの。ここで図形の基本をマスターすれば、どんな図形問題もこわくない！

このページでは、平面図形の基本を学びます：

  • 直線と角：直線・半直線・線分の違い、角の種類と性質
  • 垂直と平行：直角で交わる・交わらない2直線の関係
  • 3つの移動：図形を動かす平行移動・回転移動・対称移動

2. 覚えるルール

【最重要】対頂角は等しい

2本の直線が交わると4つの角ができます。向かい合う角を対頂角といい、必ず等しくなります。

対頂角は向かい合う角で、常に等しい
📌 直線・半直線・線分 → 例1
名前 意味 長さ
直線AB A、Bを通り両方向に無限に伸びる 測れない（無限）
半直線AB Aを端点としてB方向に無限に伸びる 測れない（無限）
線分AB AからBまでの部分 測れる（有限）
📌 垂直と平行 → 例2

垂直：2直線が直角（90°）で交わる → 記号 \(\perp\)

平行：2直線がどこまで伸ばしても交わらない → 記号 \(\parallel\)

📌 角の種類 → 例3
名前 角度の範囲 特徴
鋭角 \(0° \lt \theta \lt 90°\) 直角より小さい
直角 \(\theta = 90°\) ぴったり90°
鈍角 \(90° \lt \theta \lt 180°\) 直角より大きく平角より小さい
平角 \(\theta = 180°\) 一直線になる角
📌 3つの移動 → 例4〜6
移動 やり方 変わるもの 変わらないもの
平行移動 一定の方向に一定の距離ずらす 位置 形・大きさ・向き
回転移動 1点を中心に一定の角度回す 位置・向き 形・大きさ
対称移動 直線を軸として折り返す 位置・向き（裏返し） 形・大きさ

→ 3つの移動すべてで、形と大きさは変わらない！

📌 点対称と線対称

点対称：ある点を中心に180°回転させると元の図形と重なる

線対称：ある直線を軸に折り返すと元の図形と重なる

3. 例題と解き方

ルールを使って問題を解いてみましょう。

▶ 直線・線分・半直線

例1直線・半直線・線分の区別

右の図で、次のものを表す記号を書きなさい。

（1）点Aと点Bを通る直線

（2）点Aを端点として点Bの方向に伸びる半直線

（3）点Aから点Bまでの線分

解き方：

  1. 直線は「両方向に無限に伸びる」→ 直線AB
  2. 半直線は「端点から一方向に伸びる」→ 半直線AB（Aが端点）
  3. 線分は「両端が決まっている」→ 線分AB

ポイント：半直線ABと半直線BAは違う！端点がどちらかで区別する。

▶ 垂直と平行

例2垂直と平行の記号

右の図で、直線ℓと直線mは直角に交わり、直線ℓと直線nは交わりません。これを記号で表しなさい。

解き方：

  1. 「直角で交わる」→ 垂直の記号 \(\perp\) を使う
  2. 「交わらない」→ 平行の記号 \(\parallel\) を使う
\[ \ell \perp m, \quad \ell \parallel n \]

▶ 角の計算

例3対頂角を使って角度を求める

2直線が交わってできた4つの角のうち、1つが65°のとき、他の3つの角を求めなさい。

解き方：

  1. 対頂角は等しい → 65°の向かいも65°
  2. となりの角と足すと180°（直線だから）→ \(180° - 65° = 115°\)
  3. 115°の向かいも対頂角で115°
\[ 65°, 115°, 65°, 115° \]

▶ 3つの移動

例4平行移動

三角形ABCを右に3cm、上に2cm平行移動した三角形を描きなさい。

解き方：

  1. 各頂点A、B、Cを同じ方向に同じ距離だけ動かす
  2. A → A'（右3cm、上2cm）
  3. B → B'（右3cm、上2cm）
  4. C → C'（右3cm、上2cm）
  5. A'、B'、C'を結んで三角形を描く

ポイント：すべての点を同じだけ動かすのがコツ！

例5回転移動

三角形ABCを点Oを中心に90°回転移動させた三角形を描きなさい。

解き方：

  1. 各頂点と点Oを結ぶ
  2. OA、OB、OCの長さを変えずに、90°回す
  3. A → A'、B → B'、C → C'
  4. A'、B'、C'を結んで三角形を描く

ポイント：中心からの距離は変わらない！角度だけ変わる。

例6対称移動

三角形ABCを直線ℓを軸として対称移動させた三角形を描きなさい。

解き方：

  1. 各頂点から軸ℓに垂線を下ろす
  2. 垂線の長さを測り、反対側に同じ距離の点をとる
  3. A → A'、B → B'、C → C'
  4. A'、B'、C'を結んで三角形を描く

ポイント：軸からの距離は同じ！向きが逆になる。

4. 簡単にするコツ

✏️ コツ1：対頂角は「X」の形で見つける
2本の直線が交わると「X」の形ができる。その向かい合う角が対頂角！これを探す練習をしよう。
✏️ コツ2：移動は「何が変わるか」で区別
  • 平行移動 → 位置だけ変わる（向きはそのまま）
  • 回転移動 → 位置と向きが変わる
  • 対称移動 → 位置と向きが変わる（裏返し）
✏️ コツ3：対称移動は「折り紙」でイメージ
対称移動は紙を折って重ねるイメージ。軸ℓで折ると元の図形と移動後の図形がピッタリ重なる。
✏️ コツ4：回転移動はコンパスを使う
回転の中心に針を刺して、半径を変えずにグルッと回すイメージ。中心からの距離は変わらない。

5. よくあるつまづき

⚠️ つまづき1：半直線ABと半直線BAを混同

ダメな例： 半直線ABと半直線BAは同じ

対策： 半直線は最初の文字が端点。半直線ABはAが端点、半直線BAはBが端点で全く別物！

⚠️ つまづき2：対頂角と隣り合う角を間違える

ダメな例： となりの角を「対頂角」と呼ぶ

対策： 対頂角は「向かい合う」角。となりの角は対頂角ではなく、足して180°になる関係。

⚠️ つまづき3：移動で図形の向きを無視

ダメな例： 対称移動なのに向きが同じまま

対策： 対称移動は「折り返し」なので必ず裏返しになる。元の図形と向きが逆になっているかチェック！

⚠️ つまづき4：平行と垂直の記号を逆にする

ダメな例： 平行を \(\perp\)、垂直を \(\parallel\) と書く

対策： 平行（\(\parallel\)）は「2本の線が並んでいる」形、垂直（\(\perp\)）は「T字型」と覚える。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

2本の直線が交わり、4つの角ができました。1つの角が40°のとき、他の3つの角を求めなさい。

答えを見る

答え：40°、140°、140°

  1. 対頂角は等しい → 40°の向かいも40°
  2. となりの角と足すと180° → \(180° - 40° = 140°\)
  3. 140°の対頂角も140°
練習2

次の移動について、正しいものを選びなさい。

（1）図形を5cm右にずらす移動は？

（2）図形を点Oを中心に45°回す移動は？

（3）図形を直線ℓで折り返す移動は？

答えを見る

答え：

（1）平行移動（一定方向に一定距離ずらす）

（2）回転移動（1点を中心に回す）

（3）対称移動（直線を軸に折り返す）

📌 まとめ：対頂角は等しい、3つの移動は「ずらす・回す・折り返す」で覚えよう！

おまけ：歴史ストーリー「幾何学の父タレス」

📖 ピラミッドの高さを測った男（紀元前600年ごろ・エジプト）

紀元前600年ごろ、ギリシャの哲学者タレスがエジプトを訪れた。当時、大ピラミッドは「王の墓」として3000年以上もそびえ立っていたが、誰もその高さを正確に測ることができなかった。

なぜか？ピラミッドは巨大すぎて、壁を登って測ることも不可能。エジプト人たちは「神の力で作られたから人間には測れない」と信じていた。

そこにやってきたタレス。彼は何と言ったか？

「棒1本と太陽があれば測れるよ」

エジプト人は笑った。「は？棒で？アホなこと言うな」と。

タレスは1本の棒を地面に立て、太陽の動きを観察した。そして棒の影の長さが棒の高さと等しくなる瞬間を待った。

その瞬間、ピラミッドの影も測った。

\[ \text{棒の高さ} = \text{棒の影の長さ} \] \[ \Rightarrow \text{ピラミッドの高さ} = \text{ピラミッドの影の長さ} \]

結果、ピラミッドの高さは約147m。これは現代の測量結果とほぼ一致！

エジプト人たちは驚いた。3000年間の謎を、よそ者が棒1本で解いてしまったのだ。

タレスは帰国後「幾何学の父」と呼ばれるようになった。今でも「タレスの定理」という名前が数学に残っている。

...ところでタレスさん、あなた哲学者なのに数学やってたんですか？昔は哲学と数学、区別なかったんですね。
