1. テーマの定義

時計の針、観覧車、円形の競技場——円の中には角度に関する美しい法則が隠れています。この法則を使うと、見えない角度を計算だけで求められます。

このページでは、円周角の定理を学びます：

  • 円周角と中心角：円周上の点から見た角と、中心から見た角の関係
  • 円周角の定理：同じ弧に対する円周角はすべて等しい
  • 定理の逆：4点が同じ円周上にあることの証明
  • 円に内接する四角形：対角の和が180°になる性質

2. 覚えるルール

🎯 最重要：円周角は中心角の半分！ 同じ弧に対する円周角はすべて等しい！
円周角は中心角の半分：∠APB = ½ ∠AOB
📌 ルール1：円周角と中心角の関係 → 例1
\[ \text{円周角} = \frac{1}{2} \times \text{中心角} \]

円周角：円周上の点Pから弧の両端A, Bを見た角 ∠APB

中心角：中心Oから弧の両端A, Bを見た角 ∠AOB

→ だから結果は： 中心角がわかれば、円周角は半分！

💡 考え方を見る

円周角が中心角の半分になる理由は、三角形OAPとOBPが二等辺三角形になることから証明できる。どの位置にPがあっても、この関係は変わらない。

円周角の定理：どこにPを取っても、同じ弧に対する円周角は等しい
📌 ルール2：円周角の定理 → 例2
\[ \text{同じ弧に対する円周角はすべて等しい} \]

円周上のどこにPを取っても、同じ弧ABに対する円周角 ∠APB は同じ値になる。

→ だから結果は： 弧が同じなら、円周角の大きさも同じ

✅ メリット： 離れた場所にある角度が等しいことを証明できる

📌 ルール3：半円の弧に対する円周角は90° → 例3
\[ \text{直径ABの円周角} = 90° \]

直径ABに対する円周角は常に90°（直角）になる。

→ だから結果は： 直径を見たら、円周上のどこから見ても90°

💡 考え方を見る

直径に対する中心角は180°。円周角はその半分なので、\(\frac{180°}{2} = 90°\) になる。

円に内接する四角形：向かい合う角の和は180°
📌 ルール4：円に内接する四角形の対角の和 → 例4
\[ \angle A + \angle C = 180° \] \[ \angle B + \angle D = 180° \]

円に内接する四角形では、向かい合う角の和が180°になる。

→ だから結果は： 対角の一方がわかれば、もう一方は180°から引くだけ

📌 ルール5：円周角の定理の逆 → 例5
\[ \angle APB = \angle AQB \quad \Rightarrow \quad \text{4点A, P, Q, Bは同じ円周上} \]

2点A, Bを見込む角が等しい点P, Qは、A, Bと同じ円周上にある。

→ だから結果は： 角度が等しければ「同一円周上」と言える

定理 内容 使いどころ
円周角と中心角 円周角 = 中心角 ÷ 2 中心角 → 円周角、または逆
円周角の定理 同じ弧に対する円周角は等しい 離れた角度が等しいことの証明
半円の円周角 直径に対する円周角 = 90° 直角を見つける
内接四角形 対角の和 = 180° 四角形の角度計算

3. 例題と解き方

円周角の定理を使って、さまざまな角度の問題を解いてみましょう。

▶ 円周角と中心角

例1中心角から円周角を求める

中心角 ∠AOB = 80° のとき、同じ弧に対する円周角 ∠APB を求めよ。

解き方：

  1. 円周角 = 中心角 ÷ 2
  2. ∠APB = 80° ÷ 2 = 40°
\[ \angle APB = \frac{1}{2} \times 80° = \boxed{40°} \]
例2円周角から中心角を求める

円周角 ∠APB = 35° のとき、同じ弧に対する中心角 ∠AOB を求めよ。

解き方：

  1. 中心角 = 円周角 × 2
  2. ∠AOB = 35° × 2 = 70°
\[ \angle AOB = 2 \times 35° = \boxed{70°} \]

▶ 円周角の定理

例3同じ弧に対する円周角

円周上に4点A, B, P, Qがあり、∠APB = 50° のとき、同じ弧ABに対する ∠AQB を求めよ。

解き方：

  1. P, Qは同じ弧ABに対する円周角
  2. 円周角の定理より、∠APB = ∠AQB
\[ \angle AQB = \angle APB = \boxed{50°} \]

▶ 半円の弧に対する円周角

例4直径に対する円周角

ABが円の直径のとき、円周上の点Pにおける ∠APB を求めよ。

解き方：

  1. 直径に対する中心角は 180°
  2. 円周角 = 中心角 ÷ 2 = 180° ÷ 2 = 90°
\[ \angle APB = \frac{180°}{2} = \boxed{90°} \]

（直径を見込む円周角は常に直角！）

▶ 円に内接する四角形

例5内接四角形の角度

四角形ABCDが円に内接し、∠A = 70° のとき、∠C を求めよ。

解き方：

  1. 円に内接する四角形では、対角の和 = 180°
  2. ∠A + ∠C = 180°
  3. ∠C = 180° - 70° = 110°
\[ \angle C = 180° - 70° = \boxed{110°} \]

▶ 定理の逆

例64点が同一円周上にあることを証明

∠APB = ∠AQB = 45° のとき、4点A, P, Q, Bが同一円周上にあることを証明せよ。

解き方：

  1. A, Bを2点とする
  2. ∠APB = ∠AQB = 45° より、P, Qから見たABの角度が等しい
  3. 円周角の定理の逆より、4点A, P, Q, Bは同一円周上にある
\[ \angle APB = \angle AQB \quad \therefore \boxed{\text{4点は同一円周上}} \]

▶ 複合問題

例7複数の定理を組み合わせる

円Oで、中心角 ∠AOB = 100°。四角形APBQが円に内接するとき、∠Q を求めよ。

解き方：

  1. 円周角 ∠APB = 中心角 ÷ 2 = 100° ÷ 2 = 50°
  2. 四角形APBQが内接するので、∠P + ∠Q = 180°
  3. ∠Q = 180° - 50° = 130°
\[ \angle APB = \frac{100°}{2} = 50° \] \[ \angle Q = 180° - 50° = \boxed{130°} \]

4. 簡単にするコツ

✏️ コツ1：「弧」に注目する
円周角は「どの弧を見ているか」で決まる。問題を解くときは、まず弧を色分けしたりマークしたりして、どの弧に対する角かを確認しよう。
✏️ コツ2：直径を見たら90°
「直径」「半円」というキーワードが出たら、円周角は90°。これは暗記しておくと便利。逆に、90°の円周角があれば、それは直径を見ていることになる。
✏️ コツ3：内接四角形は「対角で180°」
四角形が円に内接していたら、向かい合う角の和が180°。片方の角がわかれば、もう片方は引き算だけ。
✏️ コツ4：中心があるかないか確認
  • 中心Oが見えていれば → 中心角を使う（円周角 = 中心角 ÷ 2）
  • 中心Oがなければ → 円周角の定理（同じ弧なら等しい）

5. よくあるつまづき

⚠️ つまづき1：中心角と円周角を逆にする

ダメな例： 円周角が中心角の2倍と答える

対策： 「円周角は中心角の半分」。中心は全体を見ているから大きい、円周は遠くから見ているから小さい、と覚える。

⚠️ つまづき2：違う弧の円周角を等しいとする

ダメな例： 違う弧に対する円周角を等しいと勘違い

対策： 円周角の定理は「同じ弧に対する」が条件。まず「どの弧か」を確認してから、等しいかどうかを判断。

⚠️ つまづき3：内接四角形の「隣の角」を180°にする

ダメな例： ∠A + ∠B = 180° と答える

対策： 内接四角形で180°になるのは「対角」（向かい合う角）。隣り合う角ではない！

⚠️ つまづき4：直径でないのに90°とする

ダメな例： 弦ABに対する円周角を90°と答える

対策： 90°になるのは「直径に対する円周角」だけ。普通の弦では90°にならない。

6. 仕上げチェックリスト

どの弧に対する角かを確認したか
中心角と円周角を正しく区別したか（円周角は半分）
直径に対する円周角は90°になっているか
内接四角形では「対角」の和が180°になっているか
角度の単位「°」を書いたか

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

円Oで中心角 ∠AOB = 140° のとき、弧ABに対する円周角を求めよ。

答えを見る

円周角 = 中心角 ÷ 2

= 140° ÷ 2 = \(\boxed{70°}\)

練習2

四角形ABCDが円に内接し、∠A = 65°、∠B = 80° のとき、∠C と ∠D を求めよ。

答えを見る

対角の和 = 180° より

∠C = 180° - ∠A = 180° - 65° = \(\boxed{115°}\)

∠D = 180° - ∠B = 180° - 80° = \(\boxed{100°}\)

📌 まとめ：円周角は中心角の半分、同じ弧なら円周角は等しい、直径を見たら90°、内接四角形は対角で180°！

おまけ：数学雑学「タレスの定理」

📖 なぜ半円の上は直角になるのか？（紀元前6世紀）

「直径に対する円周角は90°」——この事実はタレスの定理とも呼ばれる。相似の教材でピラミッドを測った、あのタレスだ。

古代ギリシャの人々はこの事実に驚いた。「なぜ、円周上のどこに点を取っても、直径を見れば必ず直角になるのか？」

証明は意外とシンプル。直径の両端をA、B、円の中心をO、円周上の点をPとする。

OA = OB = OP = r（すべて半径）
  1. OA = OP = r（どちらも半径）→ △OAPは二等辺三角形
  2. OB = OP = r（どちらも半径）→ △OBPも二等辺三角形
  3. 二等辺三角形の底角をそれぞれα、βとすると...
  4. ∠APB = α + β
  5. △APBの内角の和より、2α + 2β = 180°
  6. よって α + β = 90°

つまり、直径の両端と円周上の点で作る三角形は、必ず直角三角形になる！

この発見にタレスは感動し、牛を神に捧げたという伝説がある。紀元前の人々にとって、「どこに点を取っても必ず90°」という規則性は、まるで神の法則のように思えたのだろう。

※タレスの定理については、古代の歴史家ディオゲネス・ラエルティオスが記録を残しています。
