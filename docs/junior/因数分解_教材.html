1. テーマの定義

「展開」は式を広げる操作でしたが、因数分解はその逆。式を「積の形」に戻す操作です。二次方程式を解くときや、計算を楽にするときに大活躍します。

このページでは、以下の内容を学びます：

  • 共通因数でくくる：全ての項に共通する因数を外に出す
  • 乗法公式の逆：展開公式を逆に使って積の形に戻す
  • たすき掛け：\(ax^2 + bx + c\) の形を因数分解する

2. 覚えるルール

【最重要】因数分解 = 式を「積の形」に変形すること。まず共通因数を探し、次に公式が使えるか確認！
因数分解の5パターン：この順番でチェック！
📌 パターン①：共通因数でくくる → 例1
\[ ab + ac = a(b + c) \]

全ての項に共通する因数（数や文字）を外に出す。まずこれを最初にチェック！

→ だから結果は：共通因数 × (残りの式) の形になる。

💡 考え方を見る

分配法則 \(a(b + c) = ab + ac\) の逆です。「全部に共通してかかっているもの」を探して外に出します。

📌 パターン②：和と差の積（差の2乗） → 例2
\[ a^2 - b^2 = (a + b)(a - b) \]

「2乗 − 2乗」の形を見たら即座にこの公式！

✅ メリット：\(x^2 - 9\) のような式が一瞬で因数分解できる。

📌 パターン③：完全平方式 → 例3
\[ a^2 + 2ab + b^2 = (a + b)^2 \] \[ a^2 - 2ab + b^2 = (a - b)^2 \]

「真ん中の項が \(2ab\) になっているか？」をチェック。

→ だから結果は：同じ式の2乗 \((a \pm b)^2\) になる。

📌 パターン④：和と積のパターン → 例4
\[ x^2 + (a + b)x + ab = (x + a)(x + b) \]

\(x^2\) の係数が1のとき使える。「足して真ん中、かけて後ろ」になる2数を探す。

✅ メリット：\(x^2 + 5x + 6 = (x+2)(x+3)\) のように、足して5、かけて6になる2と3を見つければOK。

📌 パターン⑤：たすき掛け → 例5
\[ ax^2 + bx + c = (px + q)(rx + s) \]

\(x^2\) の係数が1でないとき（\(2x^2\) や \(3x^2\) など）に使う。

→ だから結果は：たすき掛けで \(x\) の係数を確認しながら因数を見つける。

たすき掛け：クロスして足すと真ん中の係数になる組み合わせを探す

3. 例題と解き方

5つのパターンを順番に練習しましょう。

▶ 共通因数でくくる

例1共通因数を見つける

\(3x^2 + 6x\) を因数分解しなさい。

解き方：

  1. 各項を確認：\(3x^2 = 3 \cdot x \cdot x\)、\(6x = 2 \cdot 3 \cdot x\)
  2. 共通因数を探す：\(3x\) が共通
  3. くくり出す：\(3x^2 + 6x = 3x(x + 2)\)
\[ 3x^2 + 6x = 3x(x + 2) \]

▶ 和と差の積

例22乗 − 2乗の形

\(x^2 - 16\) を因数分解しなさい。

解き方：

  1. \(x^2 - 16 = x^2 - 4^2\) と見る（16 = 4²）
  2. 公式 \(a^2 - b^2 = (a+b)(a-b)\) を適用
  3. \(a = x\)、\(b = 4\) なので \((x+4)(x-4)\)
\[ x^2 - 16 = (x + 4)(x - 4) \]

▶ 完全平方式

例3完全平方の形

\(x^2 + 6x + 9\) を因数分解しなさい。

解き方：

  1. 最初と最後の項を確認：\(x^2\) と \(9 = 3^2\)
  2. 真ん中の項をチェック：\(6x = 2 \cdot x \cdot 3\) → OK！
  3. 公式 \(a^2 + 2ab + b^2 = (a+b)^2\) を適用
\[ x^2 + 6x + 9 = (x + 3)^2 \]

▶ 和と積のパターン

例4足して○、かけて△

\(x^2 + 7x + 12\) を因数分解しなさい。

解き方：

  1. 「足して7、かけて12」になる2数を探す
  2. 12の約数ペア：(1,12), (2,6), (3,4)
  3. 足して7になるのは 3と4
  4. \((x+3)(x+4)\)
\[ x^2 + 7x + 12 = (x + 3)(x + 4) \]
例4'負の数が入る場合

\(x^2 - 2x - 15\) を因数分解しなさい。

解き方：

  1. 「足して-2、かけて-15」になる2数を探す
  2. かけて負 → 符号が異なる2数
  3. -15 = (-5)×3 で、-5+3 = -2 → -5と3
  4. \((x-5)(x+3)\)
\[ x^2 - 2x - 15 = (x - 5)(x + 3) \]

▶ たすき掛け

例5x²の係数が1でない場合

\(2x^2 + 5x + 3\) を因数分解しなさい。

解き方：

  1. \(x^2\) の係数が2 → たすき掛けを使う
  2. 2 = 2×1、3 = 1×3 または 3×1
  3. たすき掛けで確認：
    • 2と3、1と1 → 2×1 + 1×3 = 5 ✓
  4. \((2x + 3)(x + 1)\)
\[ 2x^2 + 5x + 3 = (2x + 3)(x + 1) \]

▶ 複合パターン

例6共通因数 + 公式

\(2x^2 - 8\) を因数分解しなさい。

解き方：

  1. まず共通因数：2が共通 → \(2(x^2 - 4)\)
  2. 次に公式：\(x^2 - 4 = x^2 - 2^2 = (x+2)(x-2)\)
  3. 合わせて \(2(x+2)(x-2)\)
\[ 2x^2 - 8 = 2(x + 2)(x - 2) \]

※ 共通因数を先に出すと、残りの因数分解が楽になる！

4. 簡単にするコツ

✏️ コツ1：共通因数を必ず最初にチェック！
共通因数を先に出すと、残りの式が簡単になることが多い。
  • \(3x^2 - 12 = 3(x^2 - 4) = 3(x+2)(x-2)\)
✏️ コツ2：「2乗の形」を見つける目を養う
数2乗
1, 2, 3, 4, 51, 4, 9, 16, 25
6, 7, 8, 9, 1036, 49, 64, 81, 100
これらを暗記しておくと、「\(x^2 - 49\)」を見た瞬間に \((x+7)(x-7)\) と分かる。
✏️ コツ3：展開して検算する
答えが正しいか不安なら、因数分解した結果を展開して元に戻るか確認。
  • \((x+3)(x+4) = x^2 + 4x + 3x + 12 = x^2 + 7x + 12\) ✓
✏️ コツ4：符号のパターンを覚える
\(x^2 + bx + c\) の形で：
  • \(c > 0\) かつ \(b > 0\) → 両方プラス：\((x+?)(x+?)\)
  • \(c > 0\) かつ \(b < 0\) → 両方マイナス：\((x-?)(x-?)\)
  • \(c < 0\) → 符号が異なる：\((x+?)(x-?)\)

5. よくあるつまづき

⚠️ つまづき1：共通因数を見落とす

ダメな例：\(4x^2 - 16\) を \((2x+4)(2x-4)\) で終わる

対策：さらに因数分解できる！ \(= 4(x+2)(x-2)\) または最初に4でくくる。

⚠️ つまづき2：符号を間違える

ダメな例：\(x^2 - 5x + 6 = (x-2)(x+3)\)（足して5にならない）

対策：足して-5、かけて+6 → 両方マイナス → \((x-2)(x-3)\)

⚠️ つまづき3：たすき掛けの組み合わせミス

ダメな例：たすき掛けで最初の組み合わせで諦める

対策：複数の組み合わせを試す。符号も含めて確認する。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の式を因数分解しなさい。

(1) \(x^2 - 9\)

(2) \(x^2 + 8x + 16\)

(3) \(x^2 - x - 20\)

答えを見る

(1) \(x^2 - 9 = x^2 - 3^2 = (x+3)(x-3)\)（和と差の積）

(2) \(x^2 + 8x + 16 = (x+4)^2\)（完全平方：\(2 \times x \times 4 = 8x\) ✓）

(3) 足して-1、かけて-20 → -5と4 → \((x-5)(x+4)\)

練習2

次の式を因数分解しなさい。

(1) \(3x^2 + 12x\)

(2) \(2x^2 + 7x + 3\)

答えを見る

(1) 共通因数3xでくくる：\(3x^2 + 12x = 3x(x + 4)\)

(2) たすき掛け：2 = 2×1、3 = 1×3
2×3 + 1×1 = 7 ✓ → \((2x + 1)(x + 3)\)

📌 まとめ：因数分解は「共通因数 → 公式 → たすき掛け」の順でチェック。展開して検算すれば完璧！

おまけ：数学雑学「完全数の謎」

🔢 神が愛した数「完全数」

古代ギリシャの数学者たちは、ある特別な数に夢中だった。それが「完全数」だ。

完全数とは、「自分自身を除く約数の和が、自分自身と等しくなる数」のこと。

たとえば6。6の約数（6以外）は 1, 2, 3。

\[ 1 + 2 + 3 = 6 \]

なんと、足すとピッタリ6になる！古代ギリシャ人はこれを「神が創った完璧な数」と呼んだ。

次の完全数は28。約数は 1, 2, 4, 7, 14。

\[ 1 + 2 + 4 + 7 + 14 = 28 \]

因数分解で約数を見つける力があれば、完全数を発見できる。実際、古代の数学者たちは因数分解（約数の分解）を駆使して完全数を探した。

ちなみに、次の完全数は 496、その次は 8128。どんどん見つけにくくなる。

そして現代、驚くべき謎が残っている。

見つかっている完全数は51個（2024年時点）。しかも全て偶数だ。「奇数の完全数」は、2000年以上探し続けて1つも見つかっていない。

存在しないのか、まだ見つかっていないだけなのか——誰も証明できていない。

因数分解という「式を分解する技術」は、こんな数学のミステリーにもつながっているのだ。

※完全数の研究は紀元前300年頃のユークリッドに始まり、現在も続いています（Wikipedia: 完全数）。
