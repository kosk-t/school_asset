1. テーマの定義

入試では、三平方の定理を「ただ使う」だけでなく、どこに直角三角形を見つけるかが問われます。空間図形を平面に展開したり、補助線を引いたりする「発想力」が必要です。

このページでは、三平方の定理の総合問題を攻略します：

  • 空間図形の辺・対角線：四角錐、円錐、角柱など
  • 展開図を使った最短距離：立体を開いて直線にする
  • 座標と図形の融合問題：座標平面上の図形

2. 問題の攻略パターン

🎯 総合問題の鉄則：「見えない直角三角形を見つける」

総合問題では、直角三角形がすぐには見えません。以下のパターンを使って直角三角形を作り出すことが大切です。

📌 パターン1：展開図で最短距離 → 例1

立体の表面上の最短距離 = 展開図上の直線

  • 立体を展開して平面にする
  • 2点を直線で結ぶ
  • 三平方の定理で距離を計算

✅ ポイント： 曲がっているように見える経路も、展開すれば直線になる

立体の表面上の最短距離：展開すると直線になる
📌 パターン2：空間に補助線を引く → 例2

四角錐や三角錐では、頂点から底面に垂線を下ろすと直角三角形が作れる。

  • 正四角錐 → 頂点から底面の対角線の交点に垂線
  • 三角錐 → 頂点から底面に垂線
📌 パターン3：座標に置いて計算 → 例3

複雑な図形も、座標を設定すれば計算しやすくなる。

  • 原点を決める（頂点や中心に置くと楽）
  • x軸、y軸を決める
  • 各点の座標を求める
  • 2点間の距離の公式を使う
📌 パターン4：相似と三平方の組み合わせ → 例4

相似な三角形を見つけ、辺の比を使ってから三平方の定理を使う。

  • 平行線があれば相似を疑う
  • 辺の比から長さを求める
  • その後、三平方の定理で残りを計算

3. 例題と解き方

入試レベルの問題を解いてみましょう。

▶ 展開図を使った最短距離

例1円柱の表面上の最短距離

底面の半径が 3cm、高さが 8cm の円柱がある。底面の円周上の点Aから、上面の円周上で反対側にある点Bまで、側面を通る最短距離を求めよ。

解き方：

  1. 円柱の側面を展開すると、横 = 円周 = \(2\pi \times 3 = 6\pi\) cm、縦 = 8cm の長方形
  2. 点Aと点Bは「反対側」なので、横方向に円周の半分 = \(3\pi\) cm 離れている
  3. 展開図上でAからBまで直線で結ぶ
  4. \(AB = \sqrt{(3\pi)^2 + 8^2} = \sqrt{9\pi^2 + 64}\)
\[ AB = \sqrt{(3\pi)^2 + 8^2} = \boxed{\sqrt{9\pi^2 + 64} \text{ cm}} \]

※ \(\pi \approx 3.14\) とすると約 \(\sqrt{88.8 + 64} \approx 12.4\) cm

例2円錐の表面上の最短距離

母線の長さが 6cm、底面の半径が 2cm の円錐がある。底面の円周上の点Aから、側面を1周して点Aに戻る最短距離を求めよ。

解き方：

  1. 円錐の側面を展開するとおうぎ形になる
  2. おうぎ形の半径 = 母線 = 6cm
  3. おうぎ形の弧 = 底面の円周 = \(2\pi \times 2 = 4\pi\) cm
  4. おうぎ形の中心角 = \(\frac{4\pi}{2\pi \times 6} \times 360° = \frac{4\pi}{12\pi} \times 360° = 120°\)
  5. 点Aから頂点Oを挟んで反対側のA'まで直線を引く
  6. 三角形OAA'は二等辺三角形（OA = OA' = 6）、頂角120°
  7. 頂角が120°のとき、底辺 = 斜辺と同じ長さ（正三角形の関係から）
\[ AA' = \boxed{6 \text{ cm}} \]

※中心角120°の二等辺三角形では、底辺 = 斜辺（証明：30°-60°-90°の比を使う）

▶ 空間図形の辺・高さ

例3正四角錐の高さ

底面が1辺 6cm の正方形、すべての辺が 6cm の正四角錐の高さを求めよ。

解き方：

  1. 底面の対角線の長さ = \(6\sqrt{2}\) cm
  2. 対角線の交点をMとすると、OM = \(\frac{6\sqrt{2}}{2} = 3\sqrt{2}\) cm
  3. 頂点をPとすると、PMが高さ（PMは底面に垂直）
  4. 三角形POMで、PO = 6（辺）、OM = \(3\sqrt{2}\)
  5. \(PM = \sqrt{6^2 - (3\sqrt{2})^2} = \sqrt{36 - 18} = \sqrt{18} = 3\sqrt{2}\)
\[ h = \sqrt{6^2 - (3\sqrt{2})^2} = \sqrt{36 - 18} = \boxed{3\sqrt{2} \text{ cm}} \]

▶ 座標と図形の融合

例4座標平面上の三角形

3点 A(0, 0)、B(8, 0)、C(3, 4) を頂点とする三角形ABCについて、辺BCの長さを求めよ。

解き方：

  1. B(8, 0) と C(3, 4) の2点間の距離を求める
  2. x方向の差：\(8 - 3 = 5\)
  3. y方向の差：\(0 - 4 = -4\)
  4. \(BC = \sqrt{5^2 + (-4)^2} = \sqrt{25 + 16} = \sqrt{41}\)
\[ BC = \sqrt{(8-3)^2 + (0-4)^2} = \sqrt{25 + 16} = \boxed{\sqrt{41}} \]

▶ 相似と三平方の組み合わせ

例5直角三角形と相似

直角三角形ABCで、∠C = 90°、AC = 6cm、BC = 8cm である。頂点Cから斜辺ABに垂線CHを引くとき、CHの長さを求めよ。

解き方：

  1. まず斜辺AB = \(\sqrt{6^2 + 8^2} = \sqrt{100} = 10\) cm
  2. 三角形ABCの面積 = \(\frac{1}{2} \times 6 \times 8 = 24\) cm²
  3. 三角形ABCの面積 = \(\frac{1}{2} \times AB \times CH = \frac{1}{2} \times 10 \times CH\)
  4. 24 = 5 × CH より、CH = \(\frac{24}{5}\)
\[ CH = \frac{24}{5} = \boxed{4.8 \text{ cm}} \]

※面積を2通りで表す方法が使える

4. 解くときの戦略

✏️ 戦略1：まず図を描く
問題を読んだら、必ず図を描く。空間図形は立体っぽく、展開図は平面で描く。図を描くと「どこに直角があるか」が見えてくる。
✏️ 戦略2：「最短」と聞いたら展開を考える
立体の表面上の最短距離は、展開すると直線になる。円柱は長方形、円錐はおうぎ形に展開。
✏️ 戦略3：求めたいものを含む直角三角形を探す
「高さを求めよ」なら、高さを1辺に持つ直角三角形を探す。見つからなければ補助線を引いて作る。
✏️ 戦略4：計算は最後にまとめてルートを取る
途中で何度も計算せず、\(\sqrt{a^2 + b^2 + c^2}\) のように2乗のまま足し算してから最後にルート。計算ミスが減る。

5. よくあるつまづき

⚠️ つまづき1：展開の仕方を間違える

ダメな例： 円柱の側面を三角形に展開してしまう

対策： 円柱 → 長方形（横 = 円周、縦 = 高さ）、円錐 → おうぎ形（半径 = 母線）

⚠️ つまづき2：「反対側」の意味を取り違える

ダメな例： 「反対側」を円周の全周分と考える

対策： 「反対側」= 円周の半分（直径の両端）。全周ではない。

⚠️ つまづき3：四角錐で底面の中心を間違える

ダメな例： 底面の1辺の中点を使う

対策： 正四角錐では対角線の交点が中心。1辺の中点ではない。

⚠️ つまづき4：垂線と斜辺を混同

ダメな例： 四角錐の辺を高さと間違える

対策： 高さは底面に垂直。辺は斜めに傾いている。

6. 仕上げチェックリスト

図を描いて状況を把握したか
直角三角形を正しく見つけたか
展開図は正しい形か（円柱→長方形、円錐→おうぎ形）
補助線を引いた場合、その長さは正しいか
単位を書いたか

7. 練習問題

入試レベルの問題に挑戦してみよう。

練習1

底面が1辺 4cm の正方形、高さが 6cm の正四角柱がある。底面の頂点Aから、上面の対角にある頂点Gまでの最短距離を、側面を通って求めよ。

答えを見る

側面を展開すると、Aから斜め上にGがある。

展開図上で、横方向の距離 = 4 + 4 = 8cm（2面分）

縦方向の距離 = 6cm

\(AG = \sqrt{8^2 + 6^2} = \sqrt{64 + 36} = \sqrt{100} = 10\)

答え：\(\boxed{10 \text{ cm}}\)

練習2

1辺が 4cm の正四面体（すべての面が正三角形）の高さを求めよ。

答えを見る

底面は正三角形（1辺 4cm）。

底面の重心から頂点までの距離 = \(\frac{2}{3} \times \frac{\sqrt{3}}{2} \times 4 = \frac{4\sqrt{3}}{3}\) cm

頂点から重心までの垂線が高さ。

\(h = \sqrt{4^2 - \left(\frac{4\sqrt{3}}{3}\right)^2} = \sqrt{16 - \frac{16}{3}} = \sqrt{\frac{32}{3}} = \frac{4\sqrt{2}}{\sqrt{3}} = \frac{4\sqrt{6}}{3}\)

答え：\(\boxed{\frac{4\sqrt{6}}{3} \text{ cm}}\)（約 3.27cm）

📌 まとめ：展開して直線に、補助線で直角三角形を作る、図を描いて考える

おまけ：数学雑学「蟻の最短経路」

🐜 蟻と砂糖の問題

こんな問題を考えてみよう。

「直方体の部屋の隅に蟻がいる。反対側の隅に砂糖がある。蟻が壁や天井を這って砂糖にたどり着く最短経路はどれくらい？」

直感的には「まっすぐ床を歩いて、壁を登って...」と考えがちだ。でも、これは最短ではない。

正解は「展開図を描いて直線を引く」だ。

たとえば、縦3m、横4m、高さ2mの部屋で考えよう。蟻は床の隅A、砂糖は天井の反対側の隅Bにある。

直感的な経路（床→壁→天井）だと、\(4 + 2 + 3 = 9\) m かかる。

でも、部屋を展開して考えると...

展開パターン 計算 距離
床+壁1 \(\sqrt{(3+2)^2 + 4^2}\) \(\sqrt{41} \approx 6.4\) m
壁2+天井 \(\sqrt{3^2 + (4+2)^2}\) \(\sqrt{45} \approx 6.7\) m

展開の仕方によって距離が変わる！一番短いのは約6.4mで、直感的な9mより2.6mも短い。

この問題、ヘンリー・デュードニーという数学パズルの大家が1903年に発表したもの。「蟻とミツ」問題として有名になった。

...直感を信じちゃいけない。展開図を描けば、曲がり角のない一直線が最短だとわかる。蟻も数学を知っていたら、もっと早く砂糖にありつけたかもしれない。

※この問題は「Spider and Fly problem（蜘蛛とハエ問題）」としても知られています（Wikipedia）。
