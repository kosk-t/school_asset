1. テーマの定義

「この2つの三角形は完全に同じ形だ」と言いたいとき、すべての辺と角を測る必要はありません。実は、特定の3つの条件さえ確認すれば十分なのです。これが「合同条件」で、図形の証明問題を解く鍵になります。

このページでは、三角形の合同条件を学びます：

  • 合同の意味：ぴったり重なる図形の関係
  • 3つの合同条件：三角形が合同であることを示す3つの方法
  • 証明の書き方：合同を使った証明問題の解き方

2. 覚えるルール

【最重要】三角形の合同条件は3つ！「3辺」「2辺とその間の角」「1辺と両端の角」
📌 ルール1：合同とは

2つの図形がぴったり重なるとき、その2つの図形は合同であるという。

記号：\(\triangle ABC \equiv \triangle DEF\)（「≡」は合同を表す）

合同な三角形は裏返しても重なる

→ 対応する辺と角がすべて等しい！

📌 ルール2：合同条件①「3辺がそれぞれ等しい」（SSS） → 例1

3組の辺がそれぞれ等しいとき、2つの三角形は合同。

3辺がそれぞれ等しい（SSS: Side-Side-Side）

覚え方： 辺・辺・辺 → 「へん・へん・へん」

📌 ルール3：合同条件②「2辺とその間の角がそれぞれ等しい」（SAS） → 例2

2組の辺と、その間にはさまれた角がそれぞれ等しいとき、2つの三角形は合同。

2辺とその間の角（SAS: Side-Angle-Side）

覚え方： 辺・角・辺 → 「はさみうち」の角！

⚠️ 注意： 「間の角」でないとダメ！

📌 ルール4：合同条件③「1辺とその両端の角がそれぞれ等しい」（ASA） → 例3

1組の辺と、その両端の角がそれぞれ等しいとき、2つの三角形は合同。

1辺とその両端の角（ASA: Angle-Side-Angle）

覚え方： 角・辺・角 → 辺の「両端」の角！

📌 ルール5：証明の書き方 → 例4

合同の証明は以下の形式で書く：

【証明】

△ABC と △DEF において、

① ○○ = △△ （理由）

② □□ = ◇◇ （理由）

③ ●● = ★★ （理由）

①②③より、【合同条件】が成り立つから、

△ABC ≡ △DEF

ポイント：

  • 等しいものを3つ見つける
  • なぜ等しいか理由を書く
  • どの合同条件を使ったか明記する
📌 合同条件のまとめ
条件 略称 内容 覚え方
条件① SSS 3辺がそれぞれ等しい へん・へん・へん
条件② SAS 2辺とその間の角 はさみうち
条件③ ASA 1辺とその両端の角 両端の角

3. 例題と解き方

合同条件を使って証明問題を解いてみましょう。

▶ 合同条件の確認

例13辺がそれぞれ等しい（SSS）

△ABCと△DEFで、AB = DE = 5cm、BC = EF = 7cm、CA = FD = 6cm のとき、2つの三角形は合同か。

解き方：

  1. AB = DE（5cm）
  2. BC = EF（7cm）
  3. CA = FD（6cm）

3辺がそれぞれ等しいので、△ABC ≡ △DEF

例22辺とその間の角（SAS）

△ABCと△DEFで、AB = DE = 4cm、AC = DF = 6cm、∠A = ∠D = 60° のとき、2つの三角形は合同か。

解き方：

  1. AB = DE（4cm）
  2. AC = DF（6cm）
  3. ∠A = ∠D（60°）← これが2辺の間の角

2辺とその間の角がそれぞれ等しいので、△ABC ≡ △DEF

例31辺とその両端の角（ASA）

△ABCと△DEFで、BC = EF = 8cm、∠B = ∠E = 50°、∠C = ∠F = 70° のとき、2つの三角形は合同か。

解き方：

  1. BC = EF（8cm）
  2. ∠B = ∠E（50°）← 辺BCの端
  3. ∠C = ∠F（70°）← 辺BCの端

1辺とその両端の角がそれぞれ等しいので、△ABC ≡ △DEF

▶ 証明問題

例4基本の証明

下の図で、AB = CB、∠ABD = ∠CBD のとき、△ABD ≡ △CBD を証明しなさい。

【証明】

△ABDと△CBDにおいて、

  1. AB = CB （仮定）
  2. ∠ABD = ∠CBD （仮定）
  3. BD = BD （共通）

①②③より、2辺とその間の角がそれぞれ等しいので、

\[ \triangle ABD \equiv \triangle CBD \]
例5平行線を使った証明

下の図で、AB // DC、AB = DC のとき、△ABO ≡ △CDO を証明しなさい。

【証明】

△ABOと△CDOにおいて、

  1. AB = DC （仮定）
  2. ∠ABO = ∠DCO （AB // DC より錯角）
  3. ∠BAO = ∠CDO （AB // DC より錯角）

①②③より、1辺とその両端の角がそれぞれ等しいので、

\[ \triangle ABO \equiv \triangle CDO \]
例6合同を使って辺や角が等しいことを示す

例5の条件で、AO = CO を証明しなさい。

解き方：

  1. 例5より △ABO ≡ △CDO
  2. 合同な三角形の対応する辺は等しいので、AO = CO

→ 合同を示した後、対応する辺・角が等しいと言える！

4. 簡単にするコツ

✏️ コツ1：「共通な辺・角」を探す
2つの三角形が重なっている部分は「共通」。同じ辺や角を共有していることが多い。
✏️ コツ2：対頂角・同位角・錯角を使う
等しい角を見つけるとき、これらの性質が使えることが多い。平行線があれば特に注目！
✏️ コツ3：仮定を図に書き込む
問題文の条件（等しい辺・角）を図に印で書き込むと、どの条件が使えるか見えやすい。
✏️ コツ4：結論から逆算する
「何を示したいか」を確認。AO = CO を示したいなら、まずAOとCOを含む三角形の合同を考える。
✏️ コツ5：条件は3つ！数えながら確認
合同を示すには、等しいものが3組必要。2つしか見つからなければ、もう1つ探す。

5. よくあるつまづき

⚠️ つまづき1：「間の角」でない角をSASに使う

ダメな例： AB = DE、AC = DF、∠B = ∠E で「2辺とその間の角」と言う

対策： ∠Aが間の角！∠Bは間の角ではない。2辺に「はさまれた」角かチェック。

⚠️ つまづき2：対応する頂点の順番を間違える

ダメな例： △ABC ≡ △FDE と書く（対応がずれている）

対策： A↔D、B↔E、C↔F のように対応する頂点を同じ順で書く。

⚠️ つまづき3：理由を書かない

ダメな例： 「AB = CB」だけで理由がない

対策： なぜ等しいか（仮定、共通、対頂角、錯角など）を必ず書く。

⚠️ つまづき4：使う合同条件を明記しない

ダメな例： 3つの等式を書いて「よって合同」で終わる

対策： 「3辺がそれぞれ等しい」など、どの条件を使ったか明記する。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の条件で、2つの三角形が合同といえるか。合同なら使う条件を答えなさい。

（1）3辺がそれぞれ等しい

（2）2辺と1つの角がそれぞれ等しい

（3）1辺とその両端の角がそれぞれ等しい

答えを見る

（1）合同（SSS）

（2）「間の角」かどうかによる。間の角なら合同（SAS）、そうでなければ合同とは限らない

（3）合同（ASA）

練習2

下の図で、M は AB の中点で、CM ⊥ AB のとき、△ACM ≡ △BCM を証明しなさい。

答えを見る

【証明】

△ACMと△BCMにおいて、

① AM = BM （Mは中点）

② ∠AMC = ∠BMC = 90° （CM ⊥ AB）

③ CM = CM （共通）

①②③より、2辺とその間の角がそれぞれ等しいので、

△ACM ≡ △BCM

📌 まとめ：合同条件は3つ！「3辺（SSS）」「2辺と間の角（SAS）」「1辺と両端の角（ASA）」

おまけ：歴史ストーリー「ユークリッドと『原論』」

📖 2300年読み継がれた教科書

紀元前300年頃、エジプトのアレクサンドリア。プトレマイオス1世が建てた図書館で、一人の数学者が歴史的な仕事をしていました。

その名はユークリッド。

彼は、それまでバラバラに知られていた幾何学の定理を、「5つの公理」から論理的に導くという壮大なプロジェクトを始めました。

その成果が『原論（ストイケイア）』全13巻。この本には、三角形の合同条件もしっかり載っています。

実は、ユークリッドは合同条件を「定理」として証明しました。「2辺とその間の角が等しければ合同」は、第1巻命題4として登場します。

プトレマイオス王が「もっと簡単に幾何学を学べないか」と尋ねたとき、ユークリッドはこう答えたと言われています。

「幾何学に王道なし」（There is no royal road to geometry.）

つまり、「近道はない。一つ一つ積み上げていくしかない」と。

『原論』は2300年以上読み継がれ、聖書に次いで世界で2番目に多く出版された本とも言われます。今みなさんが学んでいる合同条件も、ユークリッドの時代から変わっていないのです。

※「幾何学に王道なし」の逸話は、プロクロスの『ユークリッド原論注釈』に記録されています。
