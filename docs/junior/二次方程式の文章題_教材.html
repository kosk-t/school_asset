1. テーマの定義

「長方形の面積が48cm²になるのは...」「連続する3つの整数の積が...」「点Pが動くと面積が...」──これらの問題は、すべて二次方程式で解けます。文章を読んで正しく式を立てる力が、入試で差がつくポイントです。

このページでは、二次方程式の文章題を学びます：

  • 面積の問題：長方形・正方形の面積から辺を求める
  • 数の問題：連続する整数、2乗の関係など
  • 動点の問題：点が動いたときの面積変化

2. 覚えるルール

【最重要】「何をxとおくか」を決めたら、条件を式に！最後に「解の吟味」を忘れずに！

▶ 文章題の解き方フロー

📌 ルール1：文章題の解き方5ステップ
  1. 文をよく読む：何を求めるか、条件は何かを整理
  2. xをおく：求めるもの（または関連する量）をxとする
  3. 式を立てる：条件を等式で表す
  4. 方程式を解く：因数分解・平方完成・解の公式
  5. 解の吟味：問題の条件に合うか確認して答える
📌 ルール2：解の吟味（とても重要！）

二次方程式は解が2つ出ることが多い。どちらも答えになるとは限らない！

条件 確認すること
長さ・面積 正の数か？（負は不適）
人数・個数 正の整数か？
時間 正の数か？範囲内か？
動点の位置 図形の範囲内か？

▶ 問題タイプ別

📌 ルール3：面積の問題 → 例1, 2

長方形や正方形の面積から辺を求める

  • 縦を x cm、横を (x + ○) cm などとおく
  • 面積 = 縦 × 横 で式を立てる

例： 縦が横より3cm短く、面積が40cm²

横を x とすると、縦は (x - 3)

\[ x(x - 3) = 40 \]
📌 ルール4：数の問題 → 例3, 4

整数や数の関係を求める

表現 xのおき方
連続する2つの整数 x, x+1
連続する3つの整数 x-1, x, x+1（真ん中をx）
連続する2つの偶数 2n, 2n+2
2桁の自然数 10a + b（十の位a、一の位b）
📌 ルール5：動点の問題 → 例5

点Pが動くとき、面積がどう変わるか

  1. 点Pの位置を t cm（または x cm）とおく
  2. 底辺と高さを t で表す
  3. 面積 = (1/2) × 底辺 × 高さ で式を立てる
  4. t の範囲を確認して吟味

3. 例題と解き方

各タイプの文章題を解いてみましょう。

▶ 面積の問題

例1長方形の面積

長方形の横の長さは縦の長さより5cm長く、面積は36cm²である。縦の長さを求めなさい。

解き方：

  1. xをおく： 縦を x cm とする（x > 0）
  2. 横を表す： 横 = (x + 5) cm
  3. 式を立てる： x(x + 5) = 36
  4. 解く：
\[ x^2 + 5x - 36 = 0 \] \[ (x + 9)(x - 4) = 0 \] \[ x = -9, 4 \]
  1. 吟味： x > 0 より x = -9 は不適

答え：縦 4 cm

例2正方形の面積

正方形の各辺を2cmずつ長くすると、面積が32cm²増えた。もとの正方形の一辺の長さを求めなさい。

解き方：

  1. xをおく： もとの一辺を x cm とする（x > 0）
  2. 面積の関係：

もとの面積 = x²、新しい面積 = (x + 2)²

増えた分 = (x + 2)² - x² = 32

\[ x^2 + 4x + 4 - x^2 = 32 \] \[ 4x + 4 = 32 \] \[ 4x = 28 \] \[ x = 7 \]

吟味： x = 7 > 0 で適する

答え：一辺 7 cm

※ この問題は一次方程式になった！

▶ 数の問題

例3連続する整数

連続する2つの正の整数があり、それぞれを2乗した和は85である。この2つの整数を求めなさい。

解き方：

  1. xをおく： 小さい方を x とする（x は正の整数）
  2. 大きい方： x + 1
  3. 式を立てる： x² + (x + 1)² = 85
\[ x^2 + x^2 + 2x + 1 = 85 \] \[ 2x^2 + 2x - 84 = 0 \] \[ x^2 + x - 42 = 0 \] \[ (x + 7)(x - 6) = 0 \] \[ x = -7, 6 \]

吟味： x は正の整数なので x = 6

答え：6 と 7

例4数の平方

ある正の数の2乗から、その数の3倍を引くと10になる。この数を求めなさい。

解き方：

  1. xをおく： ある正の数を x とする（x > 0）
  2. 式を立てる： x² - 3x = 10
\[ x^2 - 3x - 10 = 0 \] \[ (x - 5)(x + 2) = 0 \] \[ x = 5, -2 \]

吟味： x > 0 より x = 5

答え：5

▶ 動点の問題

例5動点と面積

AB = 6cm、BC = 8cm の長方形ABCDがある。点PがAを出発してB、Cを通ってDまで辺上を移動する。APの長さを x cm とするとき、△APDの面積が20cm²になるのは x がいくつのときか。

解き方：

【場合1】PがAB上にあるとき（0 ≤ x ≤ 6）

  • AP = x、AD = 8（高さ）
  • △APD = (1/2) × x × 8 = 4x
  • 4x = 20 より x = 5
  • 0 ≤ 5 ≤ 6 なので適する

【場合2】PがBC上にあるとき（6 ≤ x ≤ 14）

  • △APDの面積は一定（= 24）
  • 24 ≠ 20 なのでこの範囲に答えはない

【場合3】PがCD上にあるとき（14 ≤ x ≤ 20）

  • DP = 20 - x
  • △APD = (1/2) × (20 - x) × 8 = 4(20 - x)
  • 4(20 - x) = 20 より x = 15
  • 14 ≤ 15 ≤ 20 なので適する

答え：x = 5 または x = 15

4. 簡単にするコツ

✏️ コツ1：図を描く
面積や動点の問題は必ず図を描く。x の位置を明確にする。
✏️ コツ2：求めるものをそのまま x におく
「縦を求めよ」なら縦を x に。迷ったらこれが一番シンプル。
✏️ コツ3：連続する整数は「真ん中」をxに
連続する3つの整数なら x-1, x, x+1 とおくと、和や積の計算が楽。
✏️ コツ4：吟味は「問題文に戻る」
x > 0、整数、範囲内か。問題文の条件を再確認。
✏️ コツ5：動点は「場合分け」
点Pがどの辺にいるかで式が変わる。それぞれの範囲で考える。

5. よくあるつまづき

⚠️ つまづき1：解の吟味を忘れる

ダメな例： x = 5, -3 を両方答えにする

対策： 長さは正！整数か？範囲内か？必ずチェック。

⚠️ つまづき2：何をxにおくか決めずに立式

ダメな例： 「xとyで...」と複雑になる

対策： まず「何をxとおくか」を宣言。1つの文字で表す。

⚠️ つまづき3：展開ミス

ダメな例： (x+2)² = x² + 4 とする

対策： (x+2)² = x² + 4x + 4。中央の項を忘れない！

⚠️ つまづき4：動点の範囲を無視

ダメな例： x = 15 が出ても、範囲外かチェックしない

対策： 点Pがどの辺にいる場合か、xの範囲を明確に。

6. 仕上げチェックリスト

「何をxとおく」を決めてから立式しているか？
面積の問題で、縦・横をxで正しく表せるか？
数の問題で、連続する整数を正しくおけるか？
解の吟味（正か？整数か？範囲内か？）をしているか？
動点の問題で、場合分けができるか？

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

長方形の縦の長さは横の長さより3cm短く、面積が54cm²である。横の長さを求めなさい。

答えを見る

横を x cm とすると、縦は (x - 3) cm

x(x - 3) = 54 → x² - 3x - 54 = 0

(x - 9)(x + 6) = 0 → x = 9, -6

x > 3 より x = 9 cm

練習2

ある正の数の2乗は、その数の5倍より6大きい。この数を求めなさい。

答えを見る

x² = 5x + 6 → x² - 5x - 6 = 0

(x - 6)(x + 1) = 0 → x = 6, -1

x > 0 より x = 6

練習3

連続する2つの正の偶数があり、積が168である。この2つの偶数を求めなさい。

答えを見る

小さい方を x とすると x(x + 2) = 168

x² + 2x - 168 = 0

(x + 14)(x - 12) = 0 → x = -14, 12

x > 0 より 12 と 14

📌 まとめ：「xをおく→条件を式に→解く→吟味」の4ステップ！解の吟味を絶対忘れない！

おまけ：入試頻出「動点問題の攻略」

📖 動点問題が難しく感じる理由

入試で「動点の問題」が出ると、苦手に感じる人が多いです。でも実は、動点問題には決まったパターンがあります。

動点問題の3ステップ

Step 1：場合分けする

点Pがどの辺にいるかで、三角形や四角形の形が変わります。まず「Pが AB上」「PがBC上」などと場合分け。

Step 2：それぞれの式を立てる

各場合で、底辺と高さを x で表して面積の式を作ります。

Step 3：範囲を確認する

解が出ても、その範囲の中にあるか確認。範囲外なら「その場合には答えなし」。

よくある出題パターン

  • 「面積が○○になるとき」：方程式を解いて x を求める
  • 「面積の最大値」：二次関数のグラフで考える
  • 「面積が一定」：式を立てると変数が消えることを確認

動点問題は図を描いて点Pを動かしてみることが大切。何が変わって何が変わらないかを把握すれば、式が自然に立てられます。
