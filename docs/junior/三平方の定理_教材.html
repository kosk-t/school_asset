1. テーマの定義

「5メートルのはしごを壁に立てかけると、どのくらいの高さまで届く？」「テレビの画面サイズ55インチって、実際どのくらい？」——こうした問題を解くには、直角三角形の辺の長さを求める必要があります。

このページでは、三平方の定理を学びます：

  • 三平方の定理：直角三角形の3辺の関係式 \(a^2 + b^2 = c^2\)
  • 定理の逆：3辺がわかれば直角三角形かどうか判定できる
  • 特別な直角三角形の比：45°-45°-90°と30°-60°-90°のパターン

2. 覚えるルール

🎯 最重要：直角三角形では \(a^2 + b^2 = c^2\)（cは斜辺）
三平方の定理：直角三角形の2辺の2乗の和は、斜辺の2乗に等しい
📌 ルール1：斜辺を求める → 例1
\[ c = \sqrt{a^2 + b^2} \]

直角をはさむ2辺 \(a\), \(b\) がわかっているとき、斜辺 \(c\) を求める。

→ だから結果は： 2辺の2乗を足して、ルートを取れば斜辺が出る

💡 考え方を見る

\(a^2 + b^2 = c^2\) の両辺のルートを取ると \(c = \sqrt{a^2 + b^2}\) になる。

📌 ルール2：他の辺を求める → 例2
\[ a = \sqrt{c^2 - b^2} \]

斜辺 \(c\) ともう1辺 \(b\) がわかっているとき、残りの辺 \(a\) を求める。

→ だから結果は： 斜辺の2乗から、わかっている辺の2乗を引いて、ルートを取る

✅ メリット： はしごの問題など、斜辺（はしごの長さ）がわかっているときに使える

📌 ルール3：定理の逆（直角三角形の判定） → 例3

3辺の長さが \(a\), \(b\), \(c\)（\(c\) が最大）のとき：

\[ a^2 + b^2 = c^2 \text{ なら直角三角形} \]

→ だから結果は： 2つの短い辺の2乗の和が、最も長い辺の2乗に等しければ直角三角形

💡 考え方を見る

三平方の定理の「逆」も成り立つ。\(a^2 + b^2 = c^2\) が成り立てば、その三角形は必ず直角三角形になる。

📌 ルール4：特別な直角三角形の辺の比 → 例4・例5

よく出てくる2つのパターンは暗記必須：

特別な直角三角形の辺の比：45°-45°-90°は1:1:√2、30°-60°-90°は1:√3:2
三角形の種類 角度 辺の比 覚え方
直角二等辺三角形 45°-45°-90° \(1 : 1 : \sqrt{2}\) 「いちいちルートに」
正三角形の半分 30°-60°-90° \(1 : \sqrt{3} : 2\) 「いちルートさんに」

3. 例題と解き方

三平方の定理を使って、いろいろなパターンの問題を解いてみましょう。

▶ 斜辺を求める

例1直角をはさむ2辺から斜辺を求める

直角三角形で、直角をはさむ2辺が 3cm と 4cm のとき、斜辺の長さを求めよ。

解き方：

  1. \(a = 3\), \(b = 4\) を三平方の定理に代入
  2. \(c^2 = 3^2 + 4^2 = 9 + 16 = 25\)
  3. \(c = \sqrt{25} = 5\)
\[ c = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = \boxed{5 \text{ cm}} \]
例2ルートが残る場合

直角三角形で、直角をはさむ2辺が 2cm と 3cm のとき、斜辺の長さを求めよ。

解き方：

  1. \(c^2 = 2^2 + 3^2 = 4 + 9 = 13\)
  2. \(c = \sqrt{13}\)（これ以上簡単にできない）
\[ c = \sqrt{2^2 + 3^2} = \sqrt{4 + 9} = \boxed{\sqrt{13} \text{ cm}} \]

▶ 他の辺を求める

例3斜辺と1辺から残りの辺を求める

直角三角形で、斜辺が 13cm、1辺が 5cm のとき、残りの辺の長さを求めよ。

解き方：

  1. 斜辺が \(c = 13\)、わかっている辺が \(a = 5\)
  2. \(b^2 = c^2 - a^2 = 13^2 - 5^2 = 169 - 25 = 144\)
  3. \(b = \sqrt{144} = 12\)
\[ b = \sqrt{13^2 - 5^2} = \sqrt{169 - 25} = \sqrt{144} = \boxed{12 \text{ cm}} \]

▶ 直角三角形かどうか判定する

例43辺から直角三角形を判定

3辺の長さが 5cm, 12cm, 13cm の三角形は直角三角形か。

解き方：

  1. 最も長い辺を \(c\) とする → \(c = 13\)
  2. 残り2辺の2乗の和を計算 → \(5^2 + 12^2 = 25 + 144 = 169\)
  3. \(c^2 = 13^2 = 169\) と比較
  4. \(5^2 + 12^2 = 13^2\) なので直角三角形
\[ 5^2 + 12^2 = 25 + 144 = 169 = 13^2 \quad \therefore \boxed{\text{直角三角形}} \]

▶ 特別な直角三角形の比を使う

例545°-45°-90°の三角形

直角二等辺三角形の直角をはさむ2辺がそれぞれ 6cm のとき、斜辺の長さを求めよ。

解き方：

  1. 45°-45°-90°の比は \(1 : 1 : \sqrt{2}\)
  2. 直角をはさむ辺が 6 なので、斜辺は \(6 \times \sqrt{2}\)
\[ \text{斜辺} = 6 \times \sqrt{2} = \boxed{6\sqrt{2} \text{ cm}} \]
例630°-60°-90°の三角形

30°-60°-90°の直角三角形で、斜辺が 10cm のとき、他の2辺の長さを求めよ。

解き方：

  1. 30°-60°-90°の比は \(1 : \sqrt{3} : 2\)（30°の向かいが1、60°の向かいが√3、90°の向かいが2）
  2. 斜辺（比の2の部分）が 10 なので、比の1は \(10 \div 2 = 5\)
  3. 30°の向かいの辺 = \(5 \times 1 = 5\) cm
  4. 60°の向かいの辺 = \(5 \times \sqrt{3} = 5\sqrt{3}\) cm
\[ \text{30°の向かい} = \boxed{5 \text{ cm}}, \quad \text{60°の向かい} = \boxed{5\sqrt{3} \text{ cm}} \]

4. 簡単にするコツ

✏️ コツ1：斜辺を見つける
斜辺は直角の向かい側にある辺で、最も長い辺。図に直角マークがあれば、その向かいの辺が斜辺。斜辺を間違えると全部間違えるので、最初に確認しよう。
✏️ コツ2：有名なピタゴラス数を覚える
よく出る「きれいな数の組」を覚えておくと計算が楽：
  • 3-4-5（最も基本）
  • 5-12-13
  • 8-15-17
  • 7-24-25
これらの倍数（6-8-10、10-24-26など）もピタゴラス数になる。
✏️ コツ3：特別な比は「まず比を書く」
45°や30°が見えたら、すぐに比を書き出す：
  • 45°-45°-90° → \(1 : 1 : \sqrt{2}\)
  • 30°-60°-90° → \(1 : \sqrt{3} : 2\)
わかっている辺と比を対応させて、他の辺を求める。
✏️ コツ4：ルートの中を簡単にする
\(\sqrt{50}\) のような数は、素因数分解して簡単にする：
\(\sqrt{50} = \sqrt{25 \times 2} = 5\sqrt{2}\)

5. よくあるつまづき

⚠️ つまづき1：斜辺を間違える

ダメな例： 適当に一番左の辺を斜辺にしてしまう

対策： 直角の向かい側を探す。または最も長い辺を探す。図がなければ、3辺のうち最大のものが斜辺。

⚠️ つまづき2：足すか引くかを間違える

ダメな例： 他の辺を求めるとき \(a^2 + c^2\) としてしまう

対策： 「斜辺を求めるなら足す」「他の辺を求めるなら引く」。斜辺の2乗が一番大きいから、斜辺から引くと覚える。

⚠️ つまづき3：ルートを取り忘れる

ダメな例： \(c^2 = 25\) なので \(c = 25\) と答える

対策： 2乗を計算したら、最後に必ずルートを取る。\(c^2 = 25\) なら \(c = \sqrt{25} = 5\)。

⚠️ つまづき4：特別な比の向きを間違える

ダメな例： 30°-60°-90°で、30°の向かいに\(\sqrt{3}\)を入れてしまう

対策： 小さい角度の向かいには小さい辺。30° → 1、60° → √3、90° → 2。「小さい角度には小さい数」と覚える。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

直角三角形で、直角をはさむ2辺が 6cm と 8cm のとき、斜辺の長さを求めよ。

答えを見る

\(c = \sqrt{6^2 + 8^2} = \sqrt{36 + 64} = \sqrt{100} = \boxed{10 \text{ cm}}\)

（3-4-5の2倍である6-8-10のパターン）

練習2

直角二等辺三角形の斜辺が \(8\sqrt{2}\) cm のとき、直角をはさむ2辺の長さをそれぞれ求めよ。

答えを見る

45°-45°-90°の比は \(1 : 1 : \sqrt{2}\)

斜辺が \(8\sqrt{2}\) なので、比の \(\sqrt{2}\) の部分が \(8\sqrt{2}\)

比の1の部分 = \(8\sqrt{2} \div \sqrt{2} = 8\)

直角をはさむ2辺はどちらも \(\boxed{8 \text{ cm}}\)

📌 まとめ：直角三角形なら \(a^2 + b^2 = c^2\)、特別な比は「いちいちルートに」「いちルートさんに」

おまけ：歴史ストーリー「ピタゴラスより1000年早かった」

📖 バビロニアの粘土板の謎

「三平方の定理」は別名「ピタゴラスの定理」とも呼ばれる。ギリシャの数学者ピタゴラス（紀元前570年頃〜紀元前495年頃）が発見したから...と思いきや、実はそうではなかった。

1930年代、イラクの遺跡から古い粘土板が発掘された。解読してみると、なんと紀元前1800年頃のもの。ピタゴラスが生まれる1000年以上も前だ。

この粘土板（プリンプトン322と呼ばれる）には、こんな数字の組が並んでいた：

a b c
119 120 169
3367 3456 4825
65 72 97

これらはすべて \(a^2 + b^2 = c^2\) を満たす「ピタゴラス数」だった。つまり、古代バビロニア人は4000年近く前から三平方の定理を知っていたのだ！

しかもこの数字、3-4-5のような簡単な組ではなく、3367-3456-4825といった複雑な組も含まれている。明らかに「たまたま見つけた」レベルではない。

研究者たちは驚いた。「ピタゴラスが発見した」という常識は、見事に覆されたのだ。

...とはいえ、「証明」したのはピタゴラスが最初かもしれない。バビロニア人は「使い方」を知っていたが、「なぜ成り立つのか」を考えたかどうかは謎のままだ。それにしても、名前は「バビロニアの定理」でもよかったのでは？
