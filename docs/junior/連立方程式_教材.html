1. テーマの定義

買い物でりんごとみかんを合わせて10個買って、合計金額が1000円だったとき、「それぞれ何個買ったか？」を知りたいことがあります。このように2つの未知数を同時に求めるのが連立方程式です。

このページでは、以下の内容を学びます：

  • 二元一次方程式：2つの文字（未知数）を含む一次方程式
  • 加減法：式を足したり引いたりして1つの文字を消す方法
  • 代入法：1つの式を他の式に代入して解く方法

2. 覚えるルール

【最重要】連立方程式を解く = 2つの式から1つの文字を消して、一次方程式に帰着させる
加減法と代入法：どちらも「1つの文字を消す」のが目標
📌 加減法（たしざん・ひきざん法） → 例1

2つの式を足したり引いたりして、一方の文字を消す方法。

→ だから結果は：消えた文字がなくなり、残った文字だけの一次方程式になる。

✅ メリット：係数をそろえれば機械的に計算できる。計算ミスが少ない。

💡 考え方を見る

なぜ式を足し引きしていいの？

等式の両辺に同じものを足しても引いても、等式は成り立ちます。連立方程式では「2つの式が同時に成り立つ」ので、一方の式の両辺を他方に足し引きできるのです。

📌 代入法 → 例3

一方の式を「\(y = \)〜」や「\(x = \)〜」の形に変形し、もう一方の式に代入する方法。

→ だから結果は：1つの文字だけの一次方程式になる。

✅ メリット：すでに「\(y = \)〜」の形があれば、そのまま代入するだけでラク。

📌 加減法の手順
  1. 係数をそろえる：消したい文字の係数の絶対値を同じにする
  2. 足すか引くか決める：
    • 符号が同じ（両方+、両方−）→ 引く
    • 符号が逆（+と−）→ 足す
  3. 一次方程式を解く
  4. もう一方の文字を求める：求めた値を元の式に代入
📌 どちらの方法を使う？
こんなとき 使う方法
係数がそろっている or そろえやすい 加減法
\(y = \)〜 や \(x = \)〜 の形がある 代入法
係数が1の文字がある どちらでもOK（代入法が楽なことも）

3. 例題と解き方

加減法と代入法の両方を練習しましょう。

▶ 加減法

例1係数がそろっているパターン

次の連立方程式を解きなさい。

\[ \begin{cases} x + y = 7 \\ x - y = 3 \end{cases} \]

解き方：

  1. \(y\) の係数を見ると、上の式は \(+1\)、下の式は \(-1\) → 符号が逆なので足す
  2. 2つの式を足す：\((x + y) + (x - y) = 7 + 3\)
  3. \(2x = 10\) より \(x = 5\)
  4. \(x = 5\) を \(x + y = 7\) に代入：\(5 + y = 7\) より \(y = 2\)
\[ \therefore x = 5, \quad y = 2 \]
例2係数をそろえるパターン

次の連立方程式を解きなさい。

\[ \begin{cases} 2x + 3y = 12 \\ 4x - y = 2 \end{cases} \]

解き方：

  1. \(x\) を消すために、上の式を2倍して \(x\) の係数をそろえる
  2. 上の式 \(\times 2\)：\(4x + 6y = 24\)
  3. 下の式と引く：\((4x + 6y) - (4x - y) = 24 - 2\)
  4. \(7y = 22\) より \(y = \dfrac{22}{7}\)...計算が複雑に！
  5. → \(y\) を消す方法に変更：下の式を3倍して \(y\) の係数をそろえる
  6. 下の式 \(\times 3\)：\(12x - 3y = 6\)
  7. 上の式と足す：\((2x + 3y) + (12x - 3y) = 12 + 6\)
  8. \(14x = 18\) より \(x = \dfrac{18}{14} = \dfrac{9}{7}\)...これも複雑

別解：この問題は計算しやすい消し方を選ぶのがポイント。

  1. 下の式 \(\times 3\)：\(12x - 3y = 6\)
  2. 上の式と足す：\(2x + 3y + 12x - 3y = 12 + 6\) → \(14x = 18\)
  3. \(x = \dfrac{9}{7}\)、これを代入して \(y = \dfrac{22}{7}\)
\[ \therefore x = \frac{9}{7}, \quad y = \frac{22}{7} \]

※ 分数が出てきても慌てずに計算しよう。

▶ 代入法

例3y = の形があるパターン

次の連立方程式を解きなさい。

\[ \begin{cases} y = 2x - 1 \\ 3x + 2y = 12 \end{cases} \]

解き方：

  1. 上の式が \(y = 2x - 1\) の形 → そのまま代入
  2. 下の式の \(y\) に代入：\(3x + 2(2x - 1) = 12\)
  3. 展開：\(3x + 4x - 2 = 12\)
  4. \(7x = 14\) より \(x = 2\)
  5. \(x = 2\) を \(y = 2x - 1\) に代入：\(y = 2 \times 2 - 1 = 3\)
\[ \therefore x = 2, \quad y = 3 \]
例4x = の形に変形するパターン

次の連立方程式を解きなさい。

\[ \begin{cases} x + 2y = 5 \\ 3x - y = 1 \end{cases} \]

解き方：

  1. 上の式を変形：\(x = 5 - 2y\)
  2. 下の式に代入：\(3(5 - 2y) - y = 1\)
  3. 展開：\(15 - 6y - y = 1\)
  4. \(-7y = -14\) より \(y = 2\)
  5. \(y = 2\) を \(x = 5 - 2y\) に代入：\(x = 5 - 4 = 1\)
\[ \therefore x = 1, \quad y = 2 \]

▶ 特殊なケース

例5分数・小数を含む連立方程式

次の連立方程式を解きなさい。

\[ \begin{cases} 0.2x + 0.3y = 1.3 \\ \dfrac{x}{2} + \dfrac{y}{3} = 2 \end{cases} \]

解き方：

  1. 小数は10倍して整数に：\(2x + 3y = 13\) ...①
  2. 分数は分母の最小公倍数（6）をかけて整数に：\(3x + 2y = 12\) ...②
  3. ①\(\times 2\) − ②\(\times 3\)で \(y\) を消す
  4. \(4x + 6y - 9x - 6y = 26 - 36\) → \(-5x = -10\) → \(x = 2\)
  5. \(x = 2\) を①に代入：\(4 + 3y = 13\) → \(y = 3\)
\[ \therefore x = 2, \quad y = 3 \]

4. 簡単にするコツ

✏️ コツ1：係数が1の文字を狙え！
\(x + 2y = 5\) のように係数が1の文字があれば、その式を変形して代入法が楽。 \(x = 5 - 2y\) とすぐに変形できる。
✏️ コツ2：小数・分数は最初に消す
  • 小数 → 10倍や100倍して整数に
  • 分数 → 分母の最小公倍数をかけて整数に
整数だけの式にしてから解く方が、計算ミスが減る。
✏️ コツ3：検算は両方の式で！
答えが \(x = 2, y = 3\) なら、2つの式の両方に代入して成り立つか確認。 1つの式だけで確認すると、その式の計算ミスに気づけない。
✏️ コツ4：符号を意識して「足すか引くか」を決める
  • 消したい文字の係数が +3 と +3 → 引く
  • 消したい文字の係数が +3 と −3 → 足す
「同符号なら引く、異符号なら足す」と覚えよう。

5. よくあるつまづき

⚠️ つまづき1：引き算で符号を間違える

ダメな例：\((2x + 3y) - (4x + 3y) = 2x + 3y - 4x + 3y\)（\(-3y\) が \(+3y\) に）

対策：引く式全体にカッコをつけて、カッコ内の全ての符号を変える。
\((2x + 3y) - (4x + 3y) = 2x + 3y - 4x - 3y = -2x\)

⚠️ つまづき2：代入するときカッコを忘れる

ダメな例：\(y = 2x - 1\) を代入して \(3x + 2 \cdot 2x - 1 = 12\)

対策：代入するときは必ずカッコで囲む。
\(3x + 2(2x - 1) = 12\) → \(3x + 4x - 2 = 12\)

⚠️ つまづき3：係数をそろえるとき片方だけ変える

ダメな例：\(2x + 3y = 12\) の両辺を2倍して \(4x + 3y = 24\)（\(y\) の係数も2倍が必要）

対策：式を何倍かするときは、全ての項を同じ数でかける。
\(2x + 3y = 12\) を2倍 → \(4x + 6y = 24\)

6. 仕上げチェックリスト

加減法で「同符号は引く、異符号は足す」を使えるか？
代入するときカッコをつけているか？
小数・分数を整数に直してから解けるか？
式を何倍かするとき、全ての項に同じ数をかけているか？
答えを両方の式に代入して検算したか？

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の連立方程式を解きなさい。

\[ \begin{cases} 3x + 2y = 16 \\ x - 2y = 0 \end{cases} \]
答えを見る

解き方（加減法）：

  1. \(y\) の係数が \(+2\) と \(-2\)（異符号）なので、足す
  2. \((3x + 2y) + (x - 2y) = 16 + 0\)
  3. \(4x = 16\) より \(x = 4\)
  4. \(x = 4\) を下の式に代入：\(4 - 2y = 0\) → \(y = 2\)

答え：\(x = 4, \quad y = 2\)

練習2

次の連立方程式を解きなさい。

\[ \begin{cases} y = 3x + 1 \\ 2x + y = 11 \end{cases} \]
答えを見る

解き方（代入法）：

  1. 上の式が \(y = 3x + 1\) の形なので、そのまま代入
  2. 下の式に代入：\(2x + (3x + 1) = 11\)
  3. \(5x + 1 = 11\) → \(5x = 10\) → \(x = 2\)
  4. \(x = 2\) を上の式に代入：\(y = 3 \times 2 + 1 = 7\)

答え：\(x = 2, \quad y = 7\)

📌 まとめ：連立方程式は「1つの文字を消して一次方程式に帰着」。加減法は係数をそろえて足し引き、代入法は式を代入！

おまけ：歴史ストーリー「方程式の語源」

📖 「方程」の誕生（紀元前2世紀・中国）

今から2200年以上前、中国の漢の時代。この頃の数学者たちは、すでに連立方程式をスラスラ解いていた。

その方法は「方程」と呼ばれていた。現代の「方程式」という言葉の語源だ。

彼らは算木（さんぎ）という棒を並べて計算した。今でいう行列のような表を作り、縦に並んだ数を消していく。これは19世紀のドイツ人数学者ガウスが「発明した」とされる消去法と、まったく同じ方法だった。

つまり、中国人はガウスより2000年も早く消去法を使っていた！

この方法は『九章算術（きゅうしょうさんじゅつ）』という本に記されている。第8章のタイトルが「方程」。連立方程式を解く章だ。

\[ \begin{cases} 3x + 2y = 39 \\ 2x + 3y = 34 \end{cases} \]

こんな問題を、紀元前の人々が棒を並べ替えながら解いていた。算木を「方形（四角形）」に並べて「程（はかる）」から「方程」という名前になったとも言われている。

今日、数学の授業で連立方程式を解くたび、2000年以上前の中国人たちと同じことをしているのだ。

※『九章算術』は中国の古代数学書で、紀元前1世紀頃に現在の形にまとめられたとされています（Wikipedia: 九章算術）。
