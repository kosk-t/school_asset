1. テーマの定義

入試の大問でよく出る「融合問題」。座標平面上に図形を描いて、関数の式を使って面積や特殊な点を求める問題です。図形と関数の両方の知識を組み合わせる力が試されます。

このページでは、図形と関数の融合問題を学びます：

  • 座標平面上の面積：三角形・四角形の面積計算
  • 等積変形：面積を変えずに頂点を移動
  • 特殊な点を求める：面積二等分・直角など

2. 覚えるルール

【最重要】座標平面の三角形の面積は「y軸を底辺」にするのが基本！

▶ 座標平面上の面積計算

📌 ルール1：y軸を底辺にする方法 → 例1

直線とx軸・y軸で囲まれた三角形は、y軸を底辺にするとラク

y軸を底辺に、x座標を高さに
\[ \triangle OAB = \frac{1}{2} \times b \times c \]

底辺 = y切片の絶対値、高さ = x切片の絶対値

📌 ルール2：三角形を分割する方法 → 例2

原点を通らない三角形は、y軸で分割して計算

y軸との交点P, Qを使って分割

△ABC = △APQ + △BPQ + △CPQ（または引き算）

📌 ルール3：座標による面積公式

3点 A(x₁, y₁), B(x₂, y₂), C(x₃, y₃) の三角形の面積：

\[ S = \frac{1}{2}|x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2)| \]

複雑な形でも、この公式を使えば座標から直接計算できる

▶ 等積変形

📌 ルール4：等積変形の原理 → 例3

底辺が共通なら、頂点を底辺と平行な直線上で動かしても面積は変わらない

△ABC = △ABC'（高さが同じ）

使い方： 「y軸上に頂点がある三角形に変形」すると面積計算が楽に！

▶ 特殊な点を求める

📌 ルール5：面積を二等分する点 → 例4

三角形の面積を二等分する直線は、頂点と対辺の中点を通る

CMはABの中点を通る
📌 ルール6：直角になる点 → 例5

「∠APB = 90°となる点P」→ 直径ABの円周上にある

または、傾きの積 = -1 を使う

\[ \text{傾き}_1 \times \text{傾き}_2 = -1 \Leftrightarrow \text{直交} \]

3. 例題と解き方

融合問題を実際に解いてみましょう。

例1三角形の面積（基本）

直線 y = -x + 4 と x軸、y軸で囲まれた三角形の面積を求めなさい。

解き方：

  1. y切片：x = 0 を代入 → y = 4、点(0, 4)
  2. x切片：y = 0 を代入 → x = 4、点(4, 0)
  3. y軸を底辺（長さ4）、高さ = x切片（4）
\[ \text{面積} = \frac{1}{2} \times 4 \times 4 = 8 \]
例2三角形の面積（分割）

3点 A(-2, 1), B(4, 4), C(2, -2) を頂点とする三角形の面積を求めなさい。

解き方：

座標による面積公式を使用：

\[ S = \frac{1}{2}|x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2)| \]

= \(\frac{1}{2}|(-2)(4-(-2)) + 4((-2)-1) + 2(1-4)|\)

= \(\frac{1}{2}|(-2)(6) + 4(-3) + 2(-3)|\)

= \(\frac{1}{2}|-12 - 12 - 6| = \frac{1}{2} \times 30 = 15\)

\[ \text{面積} = 15 \]
例3等積変形

放物線 y = x² 上の点A(-1, 1), B(2, 4) と原点Oで作る△OABの面積を、等積変形を使って求めなさい。

解き方：

  1. 直線ABの式を求める：傾き = (4-1)/(2-(-1)) = 1
  2. y = x + 2（y切片はC(0, 2)）
  3. △OABを△OACと△OBCに分割
  4. 底辺OC = 2（y軸上）
  5. △OAC = (1/2) × 2 × 1 = 1（Aのx座標の絶対値が高さ）
  6. △OBC = (1/2) × 2 × 2 = 2（Bのx座標が高さ）
\[ \triangle OAB = 1 + 2 = 3 \]
例4面積を二等分する点

△OABで O(0,0), A(6,0), B(2,4) とする。原点Oを通り△OABの面積を二等分する直線の式を求めなさい。

解き方：

  1. Oを通り面積を二等分 → 対辺ABの中点Mを通る
  2. Mの座標 = ((6+2)/2, (0+4)/2) = (4, 2)
  3. 直線OMの傾き = 2/4 = 1/2
  4. 原点を通るので y = (1/2)x
\[ y = \frac{1}{2}x \]
例5直角になる点

A(0, 4), B(6, 0) とする。x軸上の点Pで∠APB = 90°となる点の座標を求めなさい。

解き方：

  1. P(t, 0) とおく（x軸上）
  2. PAの傾き = (4-0)/(0-t) = -4/t
  3. PBの傾き = (0-0)/(6-t) = 0

これだとPBは水平線なので傾きの積では解けない。別解で考える。

別解：三平方の定理

  1. PA² + PB² = AB² の関係（∠APB = 90°）
  2. PA² = t² + 16
  3. PB² = (6-t)²
  4. AB² = 36 + 16 = 52
  5. t² + 16 + (6-t)² = 52
  6. t² + 16 + 36 - 12t + t² = 52
  7. 2t² - 12t = 0
  8. 2t(t - 6) = 0
  9. t = 0 または t = 6

t = 0 は点A、t = 6 は点Bなので、P(0, 0) または P(6, 0) は不適

→ x軸上には∠APB = 90°となる点はない（AB自体がx軸上にあるため）

※ 問題設定の確認が必要な例

4. 簡単にするコツ

✏️ コツ1：まず図を描く
座標軸と点を描いて、三角形や四角形の形を把握する。
✏️ コツ2：y軸を底辺にする
直線のy切片を使えば、底辺がy軸上になる。計算が楽になる。
✏️ コツ3：座標公式を覚えておく
複雑な三角形は座標公式で直接計算。分割よりも確実。
✏️ コツ4：等積変形で頂点を動かす
「底辺と平行な直線上に頂点を移動」で、y軸上に頂点を持っていく。
✏️ コツ5：中点は座標の平均
中点 = ((x₁+x₂)/2, (y₁+y₂)/2)。面積二等分で必ず使う。

5. よくあるつまづき

⚠️ つまづき1：底辺と高さを間違える

ダメな例： 斜めの辺を底辺にして、高さがわからなくなる

対策： 底辺は軸に平行な辺を選ぶ。y軸やx軸上の線分がベスト。

⚠️ つまづき2：符号ミス

ダメな例： x = -2 のとき「高さ -2」と計算

対策： 長さは絶対値！常に正の数。

⚠️ つまづき3：y切片を求め忘れる

ダメな例： 直線の式があるのにy切片を使わない

対策： まずy切片を求めて、y軸との交点を出す。

⚠️ つまづき4：分割を複雑にしすぎる

ダメな例： 4つ以上に分割して混乱

対策： 座標公式を使うか、2つの三角形の和・差で求める。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

直線 y = 2x - 6 とx軸、y軸で囲まれた三角形の面積を求めなさい。

答えを見る

y切片：(0, -6)、x切片：(3, 0)

底辺 = 6、高さ = 3

面積 = (1/2) × 6 × 3 = 9

練習2

3点 A(0, 0), B(4, 0), C(2, 6) を頂点とする三角形の面積を求めなさい。

答えを見る

底辺AB = 4（x軸上）

高さ = Cのy座標 = 6

面積 = (1/2) × 4 × 6 = 12

練習3

△OABで O(0,0), A(8,0), B(4,6) とする。原点Oを通り△OABの面積を二等分する直線の式を求めなさい。

答えを見る

対辺ABの中点 M = ((8+4)/2, (0+6)/2) = (6, 3)

直線OMの傾き = 3/6 = 1/2

y = (1/2)x

📌 まとめ：座標平面の面積は「y軸を底辺」が基本！等積変形で頂点をy軸上に動かすと楽に計算できる！

おまけ：入試必勝「融合問題の読み解き方」

📖 融合問題を得点源にする

「図形と関数の融合問題」は、入試の最後の大問でよく出ます。配点が高い分、ここで点を取れると大きい！

問題を読むときのポイント

1. 図を描く

問題文を読みながら、座標平面に点と直線・放物線を描く。視覚化すると関係が見えやすい。

2. 小問のつながりを意識

(1)で求めた式や座標は、(2)(3)で必ず使う。前の問題の答えが次のヒント。

3. キーワードで解法を判断

  • 「面積」→ y軸を底辺、または座標公式
  • 「二等分」→ 中点を通る直線
  • 「等しい」→ 方程式を立てる
  • 「〜となる点」→ 未知数を置いて条件を式に

典型的な出題パターン

パターンA：(1)直線の式 → (2)交点 → (3)面積

パターンB：(1)交点 → (2)面積 → (3)面積が○○になる点

パターンC：(1)式を求める → (2)二等分する直線

どのパターンでも、「まず座標を全部求める」のが基本。座標がわかれば、あとは計算で解ける！
