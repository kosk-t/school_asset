1. テーマの定義

携帯電話の料金プラン「基本料金1000円 + 1分あたり20円」は一次関数です。時間が増えると料金も一定の割合で増える。この「一定の割合で変化する」関係を式とグラフで表すのが一次関数です。

このページでは、一次関数の基本を学びます：

  • 一次関数とは：\(y = ax + b\) の形で表される関数
  • 傾きと切片：グラフの傾き（a）とy軸との交点（b）
  • グラフの描き方：切片と傾きを使って直線を描く方法

2. 覚えるルール

【最重要】一次関数の式は \(y = ax + b\)（aは傾き、bは切片）

一次関数のグラフは必ず直線になります。aとbの値でグラフの形が決まります。

📌 ルール1：一次関数とは → 例1
\[ y = ax + b \quad (a \neq 0) \]
記号 名前 意味
\(a\) 傾き（変化の割合） xが1増えたときのyの増加量
\(b\) 切片（y切片） グラフがy軸と交わる点のy座標

→ \(a = 0\) だと \(y = b\)（定数）になり、一次関数ではない！

💡 比例との関係

比例 \(y = ax\) は、\(b = 0\) の一次関数です。つまり比例は一次関数の特別な場合。切片が0なので原点を通ります。

📌 ルール2：傾きの意味 → 例2
\[ \text{傾き} = a = \frac{\text{yの増加量}}{\text{xの増加量}} = \frac{y_2 - y_1}{x_2 - x_1} \]
傾き = yの増加量 ÷ xの増加量
  • \(a > 0\) → 右上がり（xが増えるとyも増える）
  • \(a < 0\) → 右下がり（xが増えるとyは減る）
  • \(|a|\) が大きい → 急な直線
📌 ルール3：切片の意味 → 例3

切片 \(b\) は、グラフがy軸と交わる点の座標 \((0, b)\)。

切片はy軸との交点

→ \(x = 0\) を代入すると \(y = b\) になる！

📌 ルール4：グラフの描き方 → 例4

方法1：切片と傾きから描く

  1. 切片 \((0, b)\) をプロット
  2. 傾き \(a\) を使って次の点を決める（x方向に1、y方向にa進む）
  3. 2点を結んで直線を引く

方法2：2点から描く

  1. x = 0 を代入 → y切片を求める
  2. y = 0 を代入 → x切片を求める
  3. 2点を結んで直線を引く
📌 ルール5：傾きによるグラフの形
傾きの符号と大きさによる違い

3. 例題と解き方

一次関数の問題を解いてみましょう。

▶ 一次関数の判別

例1一次関数かどうか判別する

次の式が一次関数かどうか答えなさい。

（1）\(y = 3x + 5\)　（2）\(y = 4x\)　（3）\(y = x^2\)　（4）\(y = 7\)

解き方：

  1. （1）\(y = 3x + 5\) → \(a = 3, b = 5\) の形 → 一次関数 ○
  2. （2）\(y = 4x\) → \(a = 4, b = 0\) の形（比例）→ 一次関数 ○
  3. （3）\(y = x^2\) → xの2乗がある → 一次関数ではない ×
  4. （4）\(y = 7\) → \(a = 0\)（xの項がない）→ 一次関数ではない ×

▶ 傾きと切片

例2傾きを求める

一次関数 \(y = -2x + 3\) について、xが1から4まで増加したとき、yの増加量を求めなさい。

解き方：

  1. 傾き \(a = -2\)
  2. xの増加量 = \(4 - 1 = 3\)
  3. yの増加量 = 傾き × xの増加量 = \(-2 \times 3 = -6\)
\[ \text{yの増加量} = -6 \]

確認： \(x = 1\) → \(y = 1\)、\(x = 4\) → \(y = -5\)、差は \(-5 - 1 = -6\) ✓

例3切片を読み取る

一次関数 \(y = \frac{1}{2}x - 3\) のグラフがy軸と交わる点の座標を求めなさい。

解き方：

  1. y軸との交点は \(x = 0\) のとき
  2. \(y = \frac{1}{2} \times 0 - 3 = -3\)
  3. 交点の座標は \((0, -3)\)

→ 切片 \(b = -3\) なので、座標は \((0, -3)\)

▶ グラフを描く

例4グラフを描く

\(y = 2x - 1\) のグラフを描きなさい。

解き方：

  1. 切片は \(-1\) → 点 \((0, -1)\) をプロット
  2. 傾きは \(2\) → x方向に1、y方向に2進む
  3. 次の点は \((1, 1)\)
  4. 2点を結んで直線を引く
y = 2x - 1 のグラフ
例5グラフから式を読み取る

グラフが点 \((0, 2)\) を通り、傾きが \(-3\) の一次関数の式を求めなさい。

解き方：

  1. 傾き \(a = -3\)
  2. 切片 \(b = 2\)（y軸との交点が \((0, 2)\) だから）
  3. 式に代入
\[ y = -3x + 2 \]

4. 簡単にするコツ

✏️ コツ1：傾きは「yの増加量 ÷ xの増加量」
分数で覚えると間違えにくい。\(a = \frac{\Delta y}{\Delta x}\)（デルタはギリシャ文字で「増加量」を表す）
✏️ コツ2：切片は「x = 0 を代入」
y切片を求めたいときは、式に \(x = 0\) を代入するだけ。\(y = a \times 0 + b = b\)
✏️ コツ3：グラフは「切片から出発」
まず切片をプロット、そこから傾きの分だけ進む。傾きが \(\frac{2}{3}\) なら「右に3、上に2」。
✏️ コツ4：マイナスの傾きは「右下がり」
傾きがマイナスなら、右に進むほど下がる。\(y = -2x + 5\) なら「右に1、下に2」。

5. よくあるつまづき

⚠️ つまづき1：傾きと切片を逆に読む

ダメな例： \(y = 3x + 2\) で「傾き2、切片3」と答える

対策： \(y = ax + b\) の形で、xの係数が傾き、定数項が切片！

⚠️ つまづき2：比例を一次関数と認識しない

ダメな例： \(y = 5x\) は一次関数ではないと思う

対策： 比例 \(y = ax\) は \(b = 0\) の一次関数！原点を通る一次関数。

⚠️ つまづき3：傾きの符号で上下を間違える

ダメな例： 傾きが負なのに右上がりのグラフを描く

対策： 傾き正 → 右上がり、傾き負 → 右下がり。常にチェック！

⚠️ つまづき4：分数の傾きでグラフが描けない

ダメな例： 傾き \(\frac{2}{3}\) のとき、どう点を取るかわからない

対策： 分母がxの変化、分子がyの変化。「右に3、上に2」と読む。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の一次関数について、傾きと切片を答えなさい。

（1）\(y = 4x - 7\)　（2）\(y = -\frac{1}{2}x + 3\)　（3）\(y = 6x\)

答えを見る

（1）傾き 4、切片 -7

（2）傾き \(-\frac{1}{2}\)、切片 3

（3）傾き 6、切片 0

練習2

一次関数 \(y = 3x + 1\) について、xが2から5まで増加したとき、yの増加量を求めなさい。

答えを見る

傾き = 3、xの増加量 = 5 - 2 = 3

yの増加量 = 3 × 3 = 9

📌 まとめ：一次関数は \(y = ax + b\)！aが傾き（変化の割合）、bが切片（y軸との交点）。

おまけ：数学雑学「−40°の奇跡」

🌡️ 摂氏と華氏が一致する唯一の温度

「今日は90°F（華氏）だから暑い」とアメリカ人が言います。日本人は「え、90度？水が沸騰しそう...」と混乱します。

これは温度の単位が違うから。日本は摂氏（℃）、アメリカは華氏（℉）を使います。

この2つの単位の変換は、実は一次関数で表せます：

\[ F = \frac{9}{5}C + 32 \]

Cが摂氏、Fが華氏。傾きは \(\frac{9}{5}\)、切片は 32 です。

さて、ここで問題。摂氏と華氏が同じ値になる温度は何度？

つまり、\(C = F\) になる温度を求めます。

\[ C = \frac{9}{5}C + 32 \] \[ C - \frac{9}{5}C = 32 \] \[ -\frac{4}{5}C = 32 \] \[ C = -40 \]

答えは−40°！

つまり、−40℃ と −40℉ は同じ温度なのです。地球上でこれより寒い場所（南極や北極の冬）では、「摂氏でも華氏でも同じだから、どっちでもいいや」となります。

ちなみに、この −40° という温度は「日常で体験できる限界」に近い。カナダやロシアの極寒地域では実際に到達することがあります。

...というわけで、一次関数は温度変換にも使われているのです。次に天気予報を見るとき、「傾き 9/5、切片 32」を思い出してくださいね。
