1. テーマの定義

道路が交差するとき、向かい合う角度は等しくなる。これが「対頂角」です。平行な2本の道路を横切る道では、特別な角度の関係が生まれます。これらの性質を使えば、一部の角度がわかるだけで他の角度も計算できます。

このページでは、角と平行線の関係を学びます：

  • 対頂角：2直線が交わるときにできる向かい合う角
  • 同位角・錯角：平行線と直線が交わるときにできる特別な角
  • 多角形の角：三角形・四角形・n角形の内角と外角の和

2. 覚えるルール

【最重要】平行線の同位角・錯角は等しい！逆に、同位角または錯角が等しければ平行！
📌 ルール1：対頂角は等しい → 例1

2つの直線が交わるとき、向かい合う角（対頂角）は等しい。

向かい合う角が対頂角
💡 考え方を見る

隣り合う2つの角を足すと180°（直線は180°）。\(a + b = 180°\) かつ \(b + a' = 180°\) なら、\(a = a'\) になる。

📌 ルール2：平行線の同位角は等しい → 例2

平行な2直線に1本の直線が交わるとき、同じ位置にある角（同位角）は等しい。

同位角：同じ位置にある角

→ 平行線ならば同位角は等しい！同位角が等しければ平行！

📌 ルール3：平行線の錯角は等しい → 例3

平行な2直線に1本の直線が交わるとき、互い違いの位置にある角（錯角）は等しい。

錯角：互い違いの位置にある角（Zの形）

→「Z」の形を探す！錯角は等しければ平行！

💡 考え方を見る

同位角が等しいことと、対頂角が等しいことから導ける。錯角 = 対頂角の対頂角 = 同位角の対頂角。

📌 ルール4：三角形の内角の和は180° → 例4
\[ \angle A + \angle B + \angle C = 180° \]
三角形の内角の和
📌 ルール5：三角形の外角の性質 → 例5

三角形の1つの外角は、隣り合わない2つの内角の和に等しい。

外角 = 隣り合わない2つの内角の和
📌 ルール6：n角形の内角の和と外角の和 → 例6
種類 公式 覚え方
内角の和 \(180° \times (n - 2)\) 三角形に分割すると \((n-2)\) 個
外角の和 \(360°\) いつでも360°！
n角形は(n-2)個の三角形に分割できる

3. 例題と解き方

角と平行線の問題を解いてみましょう。

▶ 対頂角

例1対頂角を使って角度を求める

下の図で、\(\angle x\) の大きさを求めなさい。

解き方：

  1. 対頂角は等しい
  2. \(\angle x = 65°\)
\[ \angle x = 65° \]

▶ 同位角と錯角

例2同位角を使って角度を求める

下の図で、ℓ // m のとき、\(\angle x\) の大きさを求めなさい。

解き方：

  1. ℓ // m なので同位角は等しい
  2. \(\angle x = 70°\)
\[ \angle x = 70° \]
例3錯角を使って角度を求める

下の図で、ℓ // m のとき、\(\angle x\) の大きさを求めなさい。

解き方：

  1. ℓ // m なので錯角は等しい
  2. \(\angle x = 55°\)
\[ \angle x = 55° \]

▶ 三角形の角

例4三角形の内角の和

三角形ABCで、\(\angle A = 50°\)、\(\angle B = 65°\) のとき、\(\angle C\) を求めなさい。

解き方：

  1. 三角形の内角の和は 180°
  2. \(\angle C = 180° - 50° - 65°\)
\[ \angle C = 65° \]
例5外角の性質を使う

三角形ABCで、\(\angle A = 40°\)、\(\angle B = 70°\) のとき、頂点Cにおける外角を求めなさい。

解き方：

  1. 外角 = 隣り合わない2つの内角の和
  2. 外角 = \(\angle A + \angle B = 40° + 70°\)
\[ \text{外角} = 110° \]

▶ 多角形の角

例6正多角形の1つの内角

正八角形の1つの内角の大きさを求めなさい。

解き方：

  1. 八角形の内角の和 = \(180° \times (8 - 2) = 180° \times 6 = 1080°\)
  2. 正八角形は8つの角が等しいので、1つの内角 = \(1080° \div 8\)
\[ 1つの内角 = 135° \]
例7正多角形の1つの外角

正十角形の1つの外角の大きさを求めなさい。

解き方：

  1. 外角の和はいつでも 360°
  2. 正十角形は10個の外角が等しいので、1つの外角 = \(360° \div 10\)
\[ 1つの外角 = 36° \]
例8複合問題（平行線と三角形）

下の図で、AB // CD のとき、\(\angle x\) の大きさを求めなさい。

解き方：

  1. AB // CD なので、錯角を使って角度を求める
  2. ABと左の斜め線がなす角 40° と、CDと右の斜め線がなす角は錯角の関係で考える
  3. 点Pで三角形を考えると、外角の性質より \(\angle x = 40° + 55° = 95°\)
\[ \angle x = 95° \]

4. 簡単にするコツ

✏️ コツ1：「F」と「Z」を探す
同位角は「F」の形、錯角は「Z」の形。図を見たらこの形を探そう。
✏️ コツ2：外角は「残り2つの和」
三角形の外角を求めるとき、内角の和 180° を使わなくても、残り2つの内角を足せばOK。
✏️ コツ3：正多角形の外角から攻める
「正n角形の1つの内角」を求めるとき、内角の和を計算するより外角 = \(360° \div n\) → 内角 = \(180° -\) 外角 が早い。
✏️ コツ4：補助線を引く
平行線がないように見えても、平行な補助線を引くと同位角・錯角が使えることがある。

5. よくあるつまづき

⚠️ つまづき1：同位角と錯角を混同する

ダメな例： 錯角なのに同位角と答える

対策： 同位角は「F」の形（同じ側）、錯角は「Z」の形（反対側）。図を回転させて確認。

⚠️ つまづき2：平行でないのに同位角・錯角を使う

ダメな例： 平行かどうか確認せずに「錯角だから等しい」と言う

対策： 同位角・錯角が等しいのは平行線のときだけ！問題文をチェック。

⚠️ つまづき3：外角と内角を混同する

ダメな例： 三角形の外角の和を 180° と答える

対策： 内角の和 = 180°、外角の和 = 360°。外角は延長線との間の角。

⚠️ つまづき4：n角形の公式で n-2 を忘れる

ダメな例： 五角形の内角の和を \(180° \times 5\) で計算

対策： 内角の和 = \(180° \times (n - 2)\)。三角形に分けると \((n-2)\) 個。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

2直線が交わってできる4つの角のうち、1つが 125° のとき、残りの3つの角を求めなさい。

答えを見る

対頂角は等しいので、もう1つは 125°

隣り合う角は 180° - 125° = 55° が2つ

答え：55°、125°、55°

練習2

正六角形の1つの内角の大きさを求めなさい。

答えを見る

方法1：内角の和 = \(180° \times (6-2) = 720°\)、1つの内角 = \(720° \div 6 = 120°\)

方法2：1つの外角 = \(360° \div 6 = 60°\)、1つの内角 = \(180° - 60° = 120°\)

答え：120°

練習3

三角形ABCの頂点Aにおける外角が 130° で、\(\angle B = 50°\) のとき、\(\angle C\) を求めなさい。

答えを見る

外角 = 隣り合わない2つの内角の和 より

\(130° = \angle B + \angle C = 50° + \angle C\)

\(\angle C = 130° - 50° = 80°\)

答え：80°

📌 まとめ：対頂角は等しい！平行線の同位角・錯角も等しい！三角形の内角の和は180°、外角の和は360°！

おまけ：数学雑学「エラトステネスと地球の大きさ」

🌍 同位角で地球を測った男

紀元前240年頃、エジプト。図書館の館長エラトステネスは、ある情報を耳にしました。

「シエネ（現在のアスワン）では、夏至の日の正午に太陽が真上に来て、井戸の底まで日光が届く」

エラトステネス「ん？でもアレクサンドリア（自分のいる街）では、同じ時刻でも太陽は真上じゃないぞ」

彼は考えました。太陽光は平行線として地球に届く。2つの都市で太陽の角度が違うのは、地球が丸いからだ！

アレクサンドリアで棒を立てて影の角度を測ると、約 7.2°。これは円周の \(\frac{7.2}{360} = \frac{1}{50}\) に相当します。

そして、2都市間の距離は約 5000 スタディア（約 925 km）。

\[ \text{地球の周囲} = 5000 \times 50 = 250000 \text{ スタディア} \]

これは約 46,000 km。現代の測定値（約 40,000 km）と比べて、誤差わずか15%という驚異的な精度！

彼が使ったのは、まさに「平行線の同位角」の考え方。太陽光という平行線と、地球の中心を結ぶ直線が作る同位角が等しいことを利用したのです。

2200年以上前に、棒と影と歩測だけで地球の大きさを計算した。エラトステネス、恐るべし。

※エラトステネスの測定については、ストラボンの『地理誌』やクレオメデスの著作に記録が残っています。
