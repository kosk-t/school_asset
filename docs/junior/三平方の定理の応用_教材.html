1. テーマの定義

「部屋の対角線に家具が収まるか？」「坂道の斜面の長さは？」「地図上の2地点間の距離は？」——三平方の定理は、日常のあらゆる場面で活躍します。

このページでは、三平方の定理の応用を学びます：

  • 平面図形への応用：長方形の対角線、正三角形の高さ
  • 空間図形への応用：直方体の対角線、円錐の母線
  • 座標平面への応用：2点間の距離

2. 覚えるルール

🎯 最重要：「直角三角形を見つける」→「三平方の定理を使う」

応用問題のコツは、図形の中に直角三角形を見つけること。直角三角形さえ見つかれば、三平方の定理 \(a^2 + b^2 = c^2\) が使えます。

📌 ルール1：長方形の対角線 → 例1
\[ d = \sqrt{a^2 + b^2} \]

縦 \(a\)、横 \(b\) の長方形の対角線 \(d\) は、直角三角形の斜辺と同じ。

長方形の対角線：縦と横から斜辺（対角線）を求める
📌 ルール2：正三角形の高さ → 例2
\[ h = \frac{\sqrt{3}}{2} a \]

1辺が \(a\) の正三角形の高さ \(h\)。30°-60°-90°の比 \(1:\sqrt{3}:2\) から導出。

💡 考え方を見る

正三角形に高さを引くと、30°-60°-90°の直角三角形ができる。底辺は \(\frac{a}{2}\)、斜辺は \(a\)。比 \(1:\sqrt{3}:2\) より、高さは \(\frac{a}{2} \times \sqrt{3} = \frac{\sqrt{3}}{2}a\)。

📌 ルール3：直方体の対角線 → 例3
\[ d = \sqrt{a^2 + b^2 + c^2} \]

縦 \(a\)、横 \(b\)、高さ \(c\) の直方体の対角線 \(d\)。三平方の定理を2回使う。

💡 考え方を見る

まず底面の対角線 \(\sqrt{a^2 + b^2}\) を求め、次にこれを底辺として高さ \(c\) との三平方の定理を使う。

直方体の対角線：3辺の2乗の和のルート
📌 ルール4：2点間の距離（座標平面） → 例4
\[ AB = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2} \]

点 \(A(x_1, y_1)\) と点 \(B(x_2, y_2)\) の距離。x方向の差とy方向の差を使う。

2点間の距離：x方向とy方向の差から斜辺を求める
📌 ルール5：円錐の母線 → 例5
\[ l = \sqrt{r^2 + h^2} \]

底面の半径 \(r\)、高さ \(h\) の円錐の母線 \(l\)。円錐を切ると直角三角形が現れる。

3. 例題と解き方

三平方の定理を使って、さまざまな図形の長さを求めてみましょう。

▶ 平面図形への応用

例1長方形の対角線

縦 3cm、横 4cm の長方形の対角線の長さを求めよ。

解き方：

  1. 対角線は直角三角形の斜辺
  2. \(d = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5\)
\[ d = \sqrt{3^2 + 4^2} = \sqrt{25} = \boxed{5 \text{ cm}} \]
例2正三角形の高さ

1辺が 6cm の正三角形の高さを求めよ。

解き方：

  1. 高さを引くと、底辺は \(6 \div 2 = 3\) cm
  2. 斜辺は 6cm（正三角形の1辺）
  3. \(h = \sqrt{6^2 - 3^2} = \sqrt{36 - 9} = \sqrt{27} = 3\sqrt{3}\)
\[ h = \sqrt{6^2 - 3^2} = \sqrt{27} = \boxed{3\sqrt{3} \text{ cm}} \]

※公式 \(h = \frac{\sqrt{3}}{2} \times 6 = 3\sqrt{3}\) でも同じ結果

▶ 空間図形への応用

例3直方体の対角線

縦 3cm、横 4cm、高さ 12cm の直方体の対角線の長さを求めよ。

解き方：

  1. まず底面の対角線：\(\sqrt{3^2 + 4^2} = \sqrt{25} = 5\)
  2. 次に立体の対角線：\(\sqrt{5^2 + 12^2} = \sqrt{25 + 144} = \sqrt{169} = 13\)
\[ d = \sqrt{3^2 + 4^2 + 12^2} = \sqrt{9 + 16 + 144} = \sqrt{169} = \boxed{13 \text{ cm}} \]

※3-4-5 と 5-12-13 という有名なピタゴラス数の組み合わせ

例4円錐の母線

底面の半径が 3cm、高さが 4cm の円錐の母線の長さを求めよ。

解き方：

  1. 円錐を縦に切ると直角三角形が見える
  2. 底辺 = 半径 = 3、高さ = 4、斜辺 = 母線
  3. \(l = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5\)
\[ l = \sqrt{3^2 + 4^2} = \sqrt{25} = \boxed{5 \text{ cm}} \]

▶ 座標平面への応用

例52点間の距離

点 \(A(1, 2)\) と点 \(B(4, 6)\) の距離を求めよ。

解き方：

  1. x方向の差：\(4 - 1 = 3\)
  2. y方向の差：\(6 - 2 = 4\)
  3. \(AB = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5\)
\[ AB = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \boxed{5} \]
例6原点からの距離

点 \(P(3, -4)\) と原点 \(O(0, 0)\) の距離を求めよ。

解き方：

  1. x方向の差：\(3 - 0 = 3\)
  2. y方向の差：\(-4 - 0 = -4\)（2乗するので符号は関係なし）
  3. \(OP = \sqrt{3^2 + (-4)^2} = \sqrt{9 + 16} = 5\)
\[ OP = \sqrt{3^2 + (-4)^2} = \sqrt{25} = \boxed{5} \]

4. 簡単にするコツ

✏️ コツ1：補助線で直角三角形を作る
図形に直角三角形が見えないときは、補助線（高さ、対角線など）を引く。すると直角三角形が現れることが多い。
✏️ コツ2：空間図形は「2回三平方」
直方体の対角線などは、まず平面で1回、次に立体で1回、計2回三平方の定理を使う。公式 \(\sqrt{a^2 + b^2 + c^2}\) を覚えておくと速い。
✏️ コツ3：座標の符号は気にしない
2点間の距離で差が負になっても、2乗するので正になる。\((−3)^2 = 9\) なので、符号を気にせず計算してOK。
✏️ コツ4：有名な組を見つける
3-4-5、5-12-13、8-15-17 などのピタゴラス数が出てきたら、計算せずに答えがわかる。倍数（6-8-10など）にも注意。

5. よくあるつまづき

⚠️ つまづき1：直角三角形を見つけられない

ダメな例： 三角形がないからと諦める

対策： 補助線を引く。対角線、高さ、垂線などを引くと直角三角形が現れる。

⚠️ つまづき2：底面の対角線を忘れる

ダメな例： 直方体で縦・横・高さを直接使う

対策： 直方体の対角線は2段階。まず底面で \(\sqrt{a^2 + b^2}\) → 次に高さと合わせる。

⚠️ つまづき3：座標の差の順番を間違える

ダメな例： \(\sqrt{x_1^2 + y_1^2 + x_2^2 + y_2^2}\) と計算

対策： 必ず「差」を取ってから2乗。\((x_2 - x_1)^2\) と \((y_2 - y_1)^2\) を足す。

⚠️ つまづき4：円錐で直径を使ってしまう

ダメな例： 底面の直径を使って計算

対策： 円錐の母線を求めるときは半径を使う。直径が与えられたら2で割る。

6. 仕上げチェックリスト

直角三角形を正しく見つけたか
斜辺がどれか確認したか
空間図形で2回三平方を使ったか
座標で「差」を取ってから2乗したか
単位を書いたか

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

1辺が 10cm の正方形の対角線の長さを求めよ。

答えを見る

\(d = \sqrt{10^2 + 10^2} = \sqrt{200} = \sqrt{100 \times 2} = 10\sqrt{2}\)

答え：\(\boxed{10\sqrt{2} \text{ cm}}\)

（45°-45°-90°の比 \(1:1:\sqrt{2}\) を使えば、\(10 \times \sqrt{2} = 10\sqrt{2}\) でも求まる）

練習2

点 \(A(-2, 3)\) と点 \(B(4, -5)\) の距離を求めよ。

答えを見る

x方向の差：\(4 - (-2) = 6\)

y方向の差：\(-5 - 3 = -8\)

\(AB = \sqrt{6^2 + (-8)^2} = \sqrt{36 + 64} = \sqrt{100} = 10\)

答え：\(\boxed{10}\)

📌 まとめ：「直角三角形を見つける」→「三平方の定理を使う」、空間は2回使う

おまけ：歴史ストーリー「4000年前のピラミッド建設現場」

📖 古代エジプトの「縄張り師」

紀元前2500年頃の古代エジプト。巨大なピラミッドを建設するには、正確な直角が必要だった。土台が歪んでいたら、ピラミッドは傾いてしまう。

当時、直角を作る専門家がいた。彼らは「縄張り師（ハルペドナプタイ）」と呼ばれていた。

彼らの道具はシンプル。12個の等間隔の結び目がある縄だけだ。

この縄を使って、3つの結び目・4つの結び目・5つの結び目で三角形を作る。すると——

\[ 3^2 + 4^2 = 9 + 16 = 25 = 5^2 \]

三平方の定理により、これは直角三角形になる！

ピタゴラスが生まれる2000年以上前から、エジプト人はこの「3-4-5の法則」を知っていた。理論は知らなくても、実用的に使いこなしていたのだ。

ピラミッドの土台は驚くほど正確で、230メートルの辺の誤差はわずか数センチ。現代の測量技術と比べても遜色ない精度だという。

...縄1本で世界遺産を作った古代人。「シンプルな道具でも、数学の知識があれば何でもできる」という好例かもしれない。

※「縄張り師」についてはギリシャの歴史家による記録が残っています（ギザの大ピラミッド - Wikipedia）。
