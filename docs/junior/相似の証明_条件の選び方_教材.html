1. テーマの定義

「合同」は図形が完全に同じ形・同じ大きさ。「相似」は同じ形だけど大きさが違う。地図は実際の土地の相似図形だし、写真の拡大コピーも相似。この「相似」を証明するための3つの条件、どれを使えばいいか迷っていないかな？

このページでは、相似条件の選び方を学びます：

  • 3つの相似条件：AA（2角）・SSS（3辺の比）・SAS（2辺の比+間の角）
  • 判断のポイント：与えられた情報から最速で条件を選ぶ
  • 証明の書き方：「どの条件で相似か」を明示する

2. 覚えるルール

【最重要】相似の証明は、まず「角」の情報を探せ！ → 2つの角が見つかればAA条件でゴール
三角形の相似条件と判断フロー
📌 相似条件1：AA（2組の角がそれぞれ等しい） → 例1

2つの角が等しければ、残りの角も自動的に等しい（三角形の内角の和は180°）。

✅ メリット： 最も使いやすい！ 平行線の錯角・同位角、対頂角、共通な角など、角の等しさは見つけやすい。

→ こんなとき： 平行線がある、対頂角がある、共通の角がある

📌 相似条件2：SSS（3組の辺の比がすべて等しい） → 例2
\[ \frac{AB}{A'B'} = \frac{BC}{B'C'} = \frac{CA}{C'A'} \]

→ こんなとき： 3辺すべての長さがわかっている（珍しいケース）

注意： 実際の証明問題ではあまり使わない。3辺の比を計算するのは手間がかかる。

📌 相似条件3：SAS（2組の辺の比が等しく、その間の角が等しい） → 例3
\[ \frac{AB}{A'B'} = \frac{AC}{A'C'} \text{　かつ　} \angle A = \angle A' \]

→ こんなとき： 2辺の比が等しいことがわかり、その間の角も等しい

注意： 「間の角」でないとダメ！ 辺ABとBCの比がわかるなら、間の角は∠B。

💡 合同条件との比較

合同条件は「SSS・SAS・ASA」だったね。相似条件は「SSS・SAS・AA」。

違いは「ASA → AA」に変わったこと。相似は大きさが違ってもOKなので、辺の長さは「比」で考える。角が2つわかれば、辺の情報なしでOK（AA）。

3. 例題と解き方

相似条件ごとに証明の書き方を練習しよう。「どの条件を使うか」を最初に見極めるのがポイント！

▶ AA条件を使う証明

例1平行線と相似

下の図で、DE // BC のとき、△ADE ∽ △ABC を証明せよ。

解き方：

  1. 共通な角を見つける：∠DAE = ∠BAC（共通）
  2. 平行線の同位角：DE // BC より ∠ADE = ∠ABC
  3. AA条件で結論：2組の角がそれぞれ等しいので、△ADE ∽ △ABC
【証明】
△ADEと△ABCにおいて
∠DAE = ∠BAC（共通）…①
DE // BC より、同位角は等しいから
∠ADE = ∠ABC…②
①、②より、2組の角がそれぞれ等しいから
△ADE ∽ △ABC（AA）

▶ SSS条件を使う証明

例2辺の比が与えられた場合

△ABCで AB = 6, BC = 8, CA = 10、△DEFで DE = 3, EF = 4, FD = 5 のとき、△ABC ∽ △DEF を証明せよ。

解き方：

  1. 対応する辺の比を計算：
    • AB : DE = 6 : 3 = 2 : 1
    • BC : EF = 8 : 4 = 2 : 1
    • CA : FD = 10 : 5 = 2 : 1
  2. 3組の比がすべて等しい：SSS条件で相似
【証明】
△ABCと△DEFにおいて
AB : DE = 6 : 3 = 2 : 1
BC : EF = 8 : 4 = 2 : 1
CA : FD = 10 : 5 = 2 : 1
3組の辺の比がすべて等しいから
△ABC ∽ △DEF（SSS）

▶ SAS条件を使う証明

例32辺の比と間の角

△ABCで、辺AB上に点D、辺AC上に点Eがあり、AD = 4, AB = 8, AE = 3, AC = 6 のとき、△ADE ∽ △ABC を証明せよ。

解き方：

  1. 辺の比を計算：
    • AD : AB = 4 : 8 = 1 : 2
    • AE : AC = 3 : 6 = 1 : 2
  2. 間の角を確認：∠DAE = ∠BAC（共通）
  3. SAS条件で結論：
【証明】
△ADEと△ABCにおいて
AD : AB = 4 : 8 = 1 : 2…①
AE : AC = 3 : 6 = 1 : 2…②
∠DAE = ∠BAC（共通）…③
①、②、③より、2組の辺の比が等しく、その間の角が等しいから
△ADE ∽ △ABC（SAS）

4. 簡単にするコツ

✏️ コツ1：まず「角」を探す（AA優先）
平行線があれば同位角・錯角、頂点が共通なら対頂角、同じ頂点を共有するなら共通角。2つの角が見つかればAAでOK！
✏️ コツ2：対応する頂点の順番を揃える
△ABC ∽ △DEF と書いたら、A↔D、B↔E、C↔F が対応。順番を間違えると証明が成り立たない。
✏️ コツ3：「間の角」かどうかを確認
SAS条件は「間の角」が必須。AB, AC の比がわかるなら、間の角は ∠A。ABとBCなら ∠B。
✏️ コツ4：図に印をつける
等しい角には同じ印（●や〇）、等しい辺の比には同じ記号。視覚的に整理すると、使うべき条件が見えてくる。

5. よくあるつまづき

⚠️ つまづき1：頂点の対応を間違える

ダメな例： △ABC ∽ △EDF と書いてしまう（対応がずれる）

対策： 図を見て、同じ位置にある頂点を順番に対応させる。

⚠️ つまづき2：「間の角」でない角を使う

ダメな例： AB:DE = AC:DF で、∠B = ∠E を使ってSASと主張

対策： 2辺 AB, AC の間の角は ∠A。∠Bは間の角ではない！

⚠️ つまづき3：AAなのに3つの角を証明しようとする

ダメな例： ∠A = ∠D, ∠B = ∠E, ∠C = ∠F を全部証明

対策： AA条件は2組の角でOK。3つ目は自動的に決まる。

⚠️ つまづき4：相似と合同を混同

ダメな例： 相似の証明なのに「3辺が等しい」と書く

対策： 相似は「比」、合同は「等しい」。SSS相似は「3組の辺の比が等しい」。

6. 仕上げチェックリスト

与えられた情報から、使う相似条件を正しく選べた
頂点の対応を正しく書けた（△ABC ∽ △DEF など）
AA条件のとき、2つの角の等しさを示せた
SAS条件のとき、「間の角」を正しく指定できた
最後に「〜条件より相似」と結論を書けた

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

△ABCの辺BCの中点をM、辺ACの中点をNとする。△AMN ∽ △ABC を証明せよ。

答えを見る

【証明】

△AMNと△ABCにおいて

中点連結定理より MN // BC

∠ANM = ∠ACB（同位角）…①

∠MAN = ∠BAC（共通）…②

①、②より、2組の角がそれぞれ等しいから

△AMN ∽ △ABC（AA）

※または、AM:AB = AN:AC = 1:2、∠A共通 で SAS でもOK

練習2

下の図で、∠BAC = 90° で、点Hは頂点Aから辺BCに下ろした垂線の足である。△ABH ∽ △CBA を証明せよ。

答えを見る

【証明】

△ABHと△CBAにおいて

∠AHB = 90°（AHは垂線）

∠BAC = 90°（仮定）

よって ∠AHB = ∠BAC = 90°…①

∠ABH = ∠CBA（共通）…②

①、②より、2組の角がそれぞれ等しいから

△ABH ∽ △CBA（AA）

📌 まとめ：相似条件は「まず角を探す → 2つ見つかればAA、なければSASかSSS」の順で判断！

おまけ：歴史ストーリー「ピラミッドの高さを測った男」

📖 タレスとピラミッド（紀元前6世紀、エジプト）

紀元前6世紀のギリシャに、タレスという哲学者がいた。「万物の根源は水である」と唱えた人物で、「ギリシャ七賢人」の一人にも数えられる超有名人だ。

ある日、タレスはエジプトを旅行中、ファラオにこう聞かれた。

「あの大ピラミッドの高さを測れるか？」

当時、ピラミッドに登ることは禁じられていた。巻き尺を垂らすこともできない。さて、どうする？

タレスの答えはこうだった：

「自分の身長と自分の影の長さが同じになる瞬間を待つ。そのとき、ピラミッドの影の長さを測れば、それがピラミッドの高さだ」

相似を使った測量

これ、相似の考え方だ！

タレス（身長）: タレスの影 = ピラミッドの高さ : ピラミッドの影

影の長さが身長と同じになる瞬間（太陽が45°の角度のとき）なら、

\[ \text{タレスの身長} : \text{タレスの影} = 1 : 1 \]

つまり、ピラミッドの高さ : ピラミッドの影も 1 : 1。影を測れば高さがわかる！

ファラオは大いに感心したという。...ただし、この話の真偽は定かではない。「哲学者の伝説」として語り継がれているエピソードだ。

でも確かなことは、相似の考え方は紀元前から実用されていたということ。測量、建築、地図作成...「相似」は古代から人類の知恵を支えてきたのだ。

※このエピソードは古代ギリシャの歴史家たちによって伝えられています（Wikipedia: タレス）。
