1. テーマの定義

連立方程式を解くとき「加減法と代入法、どっちを使えばいいの？」と迷ったことはありませんか？実は、問題を見た瞬間に「こっちが楽！」と判断できるコツがあります。

このページでは、以下の内容を学びます：

  • 判断フロー：どちらの解法が楽かを一瞬で見分ける
  • 加減法向きのパターン：係数に注目する方法
  • 代入法向きのパターン：式の形に注目する方法
  • 選択ミスのリカバリー：途中で切り替える判断

2. 覚えるルール

【最重要】連立方程式を見たら、まず「y = 〜」「x = 〜」の形があるか確認！
あれば代入法、なければ係数を見て加減法。
解法選択フローチャート：まず式の形、次に係数をチェック
📌 代入法を選ぶ3つのサイン → 例1〜2
  1. 「y = 〜」「x = 〜」の形がすでにある → 即代入！
  2. 係数が1の文字がある（例：\(x + 2y = 5\)）→ 簡単に「x = 〜」に変形できる
  3. 分数係数で加減法が面倒 → 代入法の方がスムーズなことがある

✅ メリット：式を変形する手間が省ける。カッコ計算に注意すれば確実。

📌 加減法を選ぶ3つのサイン → 例3〜4
  1. 同じ文字の係数が「+と−」で逆符号（例：\(+2y\) と \(-2y\)）→ 足すだけで消える！
  2. 同じ文字の係数が同じ（例：\(+3y\) と \(+3y\)）→ 引くだけで消える！
  3. 係数を簡単な数でそろえられる（例：2と4 → 2倍で揃う）

✅ メリット：機械的に計算できる。符号ミスに注意すればスピーディ。

📌 迷ったときの最終判断
チェック項目 結果 選ぶべき解法
「y = 〜」「x = 〜」がある？ ある 代入法
係数が逆符号でそろってる？ そろってる 加減法（足す）
係数が同符号でそろってる？ そろってる 加減法（引く）
どれも当てはまらない — 係数をそろえて加減法
💡 考え方を見る

「y = 〜」の形が最優先なのはなぜ？

代入法は「式をそのまま使える」のが強み。変形なしで代入できれば、計算ミスが減り、時間も節約できます。一方、加減法は係数をそろえる作業が必要なことが多いので、すでにそろっている場合に有利です。

3. 例題と解き方

同じ問題を両方の方法で解いて、どちらが楽か比較してみましょう。

▶ 代入法が楽なパターン

例1y = の形がある（代入法が圧勝）

次の連立方程式を解きなさい。

\[ \begin{cases} y = 3x - 2 \\ 2x + y = 8 \end{cases} \]

🔍 判断：「y = 3x - 2」がある → 代入法一択！

解き方（代入法）：

  1. 上の式をそのまま下の式に代入：\(2x + (3x - 2) = 8\)
  2. 展開：\(5x - 2 = 8\) → \(5x = 10\) → \(x = 2\)
  3. \(x = 2\) を \(y = 3x - 2\) に代入：\(y = 6 - 2 = 4\)
\[ \therefore x = 2, \quad y = 4 \]
❌ もし加減法でやると…

上の式を変形：\(y - 3x = -2\) → \(-3x + y = -2\)

係数をそろえるために下の式 × 1、上の式と比較…

→ 最初から「y = 〜」があるのに、わざわざ変形するのは 時間のムダ！

例2係数1がある（代入法が楽）

次の連立方程式を解きなさい。

\[ \begin{cases} x + 3y = 7 \\ 5x - 2y = 4 \end{cases} \]

🔍 判断：上の式の \(x\) の係数が1 → 「x = 7 - 3y」とすぐ変形できる → 代入法がスムーズ

解き方（代入法）：

  1. 上の式を変形：\(x = 7 - 3y\)
  2. 下の式に代入：\(5(7 - 3y) - 2y = 4\)
  3. 展開：\(35 - 15y - 2y = 4\) → \(-17y = -31\) → \(y = \dfrac{31}{17}\)
  4. \(y\) を代入：\(x = 7 - 3 \times \dfrac{31}{17} = 7 - \dfrac{93}{17} = \dfrac{119 - 93}{17} = \dfrac{26}{17}\)
\[ \therefore x = \frac{26}{17}, \quad y = \frac{31}{17} \]

※ 分数になっても、係数1から始めれば計算は追いやすい。

▶ 加減法が楽なパターン

例3係数が逆符号でそろっている（加減法が圧勝）

次の連立方程式を解きなさい。

\[ \begin{cases} 3x + 2y = 16 \\ x - 2y = 0 \end{cases} \]

🔍 判断：\(y\) の係数が \(+2\) と \(-2\)（逆符号） → 足すだけで消える！ → 加減法一択

解き方（加減法）：

  1. 2つの式を足す：\((3x + 2y) + (x - 2y) = 16 + 0\)
  2. \(4x = 16\) → \(x = 4\)
  3. \(x = 4\) を下の式に代入：\(4 - 2y = 0\) → \(y = 2\)
\[ \therefore x = 4, \quad y = 2 \]
❌ もし代入法でやると…

下の式を変形：\(x = 2y\)

上の式に代入：\(3(2y) + 2y = 16\) → \(8y = 16\) → \(y = 2\)

→ これでも解けるが、足すだけで済む加減法の方が速い。

例4係数が同符号でそろっている（加減法が楽）

次の連立方程式を解きなさい。

\[ \begin{cases} 4x + 3y = 22 \\ 4x - y = 6 \end{cases} \]

🔍 判断：\(x\) の係数が両方 \(+4\)（同符号） → 引くだけで消える！ → 加減法が楽

解き方（加減法）：

  1. 上の式から下の式を引く：\((4x + 3y) - (4x - y) = 22 - 6\)
  2. \(4x + 3y - 4x + y = 16\) → \(4y = 16\) → \(y = 4\)
  3. \(y = 4\) を下の式に代入：\(4x - 4 = 6\) → \(4x = 10\) → \(x = \dfrac{5}{2}\)
\[ \therefore x = \frac{5}{2}, \quad y = 4 \]

▶ 係数をそろえる必要があるパターン

例5どちらでも同じくらい（係数をそろえて加減法）

次の連立方程式を解きなさい。

\[ \begin{cases} 2x + 3y = 13 \\ 3x + 2y = 12 \end{cases} \]

🔍 判断：「y = 〜」の形なし、係数もそろっていない → 係数をそろえて加減法

解き方：

  1. どの文字を消すか決める：\(x\) を消すなら上×3、下×2でそろう
  2. 上の式 \(\times 3\)：\(6x + 9y = 39\)
  3. 下の式 \(\times 2\)：\(6x + 4y = 24\)
  4. 引く：\((6x + 9y) - (6x + 4y) = 39 - 24\) → \(5y = 15\) → \(y = 3\)
  5. \(y = 3\) を元の式に代入：\(2x + 9 = 13\) → \(x = 2\)
\[ \therefore x = 2, \quad y = 3 \]

※ 係数が小さい方を何倍かするのがコツ。2と3なら「×3」と「×2」で6に。

▶ 判断ミスからのリカバリー

例6途中で「あれ、こっちの方が楽だったかも」となったとき

次の連立方程式を解きなさい。

\[ \begin{cases} x - 4y = -3 \\ 5x + 2y = 9 \end{cases} \]

🔍 判断：上の式の \(x\) の係数が1 → 代入法でいけそう？

代入法を試みる：

  1. 上の式を変形：\(x = 4y - 3\)
  2. 下の式に代入：\(5(4y - 3) + 2y = 9\)
  3. 展開：\(20y - 15 + 2y = 9\) → \(22y = 24\) → \(y = \dfrac{24}{22} = \dfrac{12}{11}\)
  4. あれ、分数になった…計算が面倒になってきた

リカバリー：最初から加減法だったらどうだった？

  1. 上の式 \(\times 5\)：\(5x - 20y = -15\)
  2. 下の式から引く：\((5x + 2y) - (5x - 20y) = 9 - (-15)\)
  3. \(22y = 24\) → 結局同じ計算量…

→ この問題はどちらも同じくらい。分数が出ても慌てず計算を続けよう！

\[ \therefore x = \frac{15}{11}, \quad y = \frac{12}{11} \]

4. 簡単にするコツ

✏️ コツ1：3秒ルールで判断
問題を見て3秒以内に「y = 〜」「x = 〜」を探す。あれば即代入法。なければ係数チェック。
迷う時間が長いと、その分計算時間が減ってしまう。
✏️ コツ2：係数が「逆符号」なら足す、「同符号」なら引く
  • \(+2y\) と \(-2y\) → 足すと \(0y\) で消える
  • \(+3x\) と \(+3x\) → 引くと \(0x\) で消える
「逆は足す、同は引く」と覚えよう。
✏️ コツ3：係数1は代入法のチャンス
\(x + 2y = 5\) のように係数が1の文字があれば、\(x = 5 - 2y\) とすぐに変形できる。
加減法より代入法の方が楽なことが多い。
✏️ コツ4：「簡単な整数」で終わりそうか予想する
定数項や係数が小さな整数だけなら、答えも整数になりやすい。
計算の途中で分数が出てきたら、見直しより先に計算を続けて答えを出そう。
（中2レベルでは分数の答えも普通にある）

5. よくあるつまづき

⚠️ つまづき1：いつも加減法しか使わない

ダメな例：\(y = 2x + 1\) があるのに、わざわざ \(-2x + y = 1\) に変形して加減法

対策：「y = 〜」がすでにあるなら、そのまま代入が最速。変形は時間のムダ。

⚠️ つまづき2：加減法で符号を間違える

ダメな例：\((2x + 3y) - (4x + 3y) = 2x + 3y - 4x + 3y\)（\(-3y\) が \(+3y\) に）

対策：引く式はカッコで囲んで、全部の符号を変える。
\((2x + 3y) - (4x + 3y) = 2x + 3y - 4x - 3y = -2x\)

⚠️ つまづき3：代入のときカッコを忘れる

ダメな例：\(y = 2x - 1\) を代入して \(3y = 3 \times 2x - 1\)（全体に3がかかっていない）

対策：代入するときは必ずカッコで囲む。
\(3y = 3(2x - 1) = 6x - 3\)

⚠️ つまづき4：途中で解法を変えて混乱

ダメな例：代入法で始めて、途中で「やっぱり加減法」と変更して計算がごちゃごちゃに

対策：一度選んだら最後までやり切る。どちらでも答えは同じ。
途中で変えると、計算ミスの原因になる。

7. 練習問題

まず「どちらの解法が楽か」を判断してから解こう。

練習1
\[ \begin{cases} y = 4x + 3 \\ 3x - y = 2 \end{cases} \]
答えを見る

🔍 判断：「y = 4x + 3」がある → 代入法

解き方：

  1. 下の式に代入：\(3x - (4x + 3) = 2\)
  2. \(3x - 4x - 3 = 2\) → \(-x = 5\) → \(x = -5\)
  3. \(y = 4(-5) + 3 = -17\)

答え：\(x = -5, \quad y = -17\)

練習2
\[ \begin{cases} 5x + 3y = 21 \\ 2x - 3y = 0 \end{cases} \]
答えを見る

🔍 判断：\(y\) の係数が \(+3\) と \(-3\)（逆符号） → 加減法（足す）

解き方：

  1. 2つの式を足す：\((5x + 3y) + (2x - 3y) = 21 + 0\)
  2. \(7x = 21\) → \(x = 3\)
  3. 下の式に代入：\(6 - 3y = 0\) → \(y = 2\)

答え：\(x = 3, \quad y = 2\)

練習3（判断が難しい）
\[ \begin{cases} 2x + 5y = 17 \\ 3x + 4y = 18 \end{cases} \]
答えを見る

🔍 判断：「y = 〜」なし、係数もそろっていない → 係数をそろえて加減法

\(x\) を消すなら上×3、下×2。\(y\) を消すなら上×4、下×5。どちらでもOK。

解き方（x を消す）：

  1. 上の式 \(\times 3\)：\(6x + 15y = 51\)
  2. 下の式 \(\times 2\)：\(6x + 8y = 36\)
  3. 引く：\(7y = 15\) → \(y = \dfrac{15}{7}\)
  4. \(y\) を元の式に代入：\(2x + \dfrac{75}{7} = 17\) → \(2x = \dfrac{44}{7}\) → \(x = \dfrac{22}{7}\)

答え：\(x = \dfrac{22}{7}, \quad y = \dfrac{15}{7}\)

📌 まとめ：「y = 〜」があれば代入法、係数がそろっていれば加減法。迷ったら係数をそろえて加減法！

おまけ：数学雑学「鶴亀算の秘密」

🐢 「足の数」で動物を数えた江戸時代

「鶴と亀が合わせて8匹いて、足の数は合計26本。鶴と亀はそれぞれ何匹？」

これは鶴亀算（つるかめざん）と呼ばれる、江戸時代から伝わる日本の伝統的な算術問題だ。

江戸時代の人々は、連立方程式を知らなかった。でも、この問題をちゃんと解いていた。どうやって？

江戸時代の解き方：

「全部が鶴だったら？」と考える。8匹全部が鶴なら、足は \(8 \times 2 = 16\) 本。

実際は26本だから、\(26 - 16 = 10\) 本足りない。

鶴1匹を亀1匹に置き換えると、足は \(4 - 2 = 2\) 本増える。

だから亀は \(10 \div 2 = 5\) 匹。鶴は \(8 - 5 = 3\) 匹。

現代の解き方（連立方程式）：

鶴を \(x\) 匹、亀を \(y\) 匹とすると：

\[ \begin{cases} x + y = 8 \\ 2x + 4y = 26 \end{cases} \]

上の式から \(x = 8 - y\) を下の式に代入して…

\(2(8 - y) + 4y = 26\) → \(16 - 2y + 4y = 26\) → \(2y = 10\) → \(y = 5\)

よって鶴は3匹、亀は5匹。

驚きポイント：

江戸時代の方法は「代入法」の考え方そのものだった！「全部が鶴だったら」という仮定は、一方の変数を固定して考えることと同じ。

実は、連立方程式が日本に入ってきたのは明治時代になってから。それまで数百年間、日本人は「和算」という独自の数学で、このような問題を解き続けていた。

ちなみに、鶴亀算という名前だが、実際の問題では「大人と子供」「りんごとみかん」「10円玉と50円玉」など、何でもいい。共通するのは「2種類のものがあって、個数と合計が分かっている」という構造だ。

江戸の人も、令和の人も、やってることは同じ。「どうやったら楽に解けるか」を工夫するのが数学の醍醐味なのだ。

※ 鶴亀算は『塵劫記（じんこうき）』（1627年）などの和算書に登場します（Wikipedia: 鶴亀算）。
