1. テーマの定義

三角形には「特別な形」があります。二等辺三角形、正三角形、直角三角形。これらは見た目だけでなく、特有の性質を持っています。この性質を知っていると、複雑な図形問題がスッキリ解けるようになります。

このページでは、特別な三角形の性質を学びます：

  • 二等辺三角形：2辺が等しい三角形の性質
  • 正三角形：3辺がすべて等しい三角形の性質
  • 直角三角形：直角を持つ三角形の合同条件

2. 覚えるルール

【最重要】二等辺三角形では「2辺が等しい ⟺ 2角が等しい」が成り立つ！

▶ 二等辺三角形

📌 ルール1：二等辺三角形の定義と名称

二等辺三角形：2つの辺が等しい三角形

二等辺三角形の各部の名前
  • 等しい2辺の間の角 → 頂角
  • 頂角に向かい合う辺 → 底辺
  • 底辺の両端の角 → 底角
📌 ルール2：二等辺三角形の性質（定理） → 例1

二等辺三角形の底角は等しい

\[ AB = AC \Rightarrow \angle B = \angle C \]

→ 2辺が等しければ、その向かいの2角も等しい！

💡 なぜそうなる？

頂点Aから底辺BCに垂線を下ろすと、△ABMと△ACMは合同（SAS）。よって∠B = ∠Cとなる。

📌 ルール3：二等辺三角形になる条件（定理の逆） → 例2

2つの角が等しい三角形は二等辺三角形

\[ \angle B = \angle C \Rightarrow AB = AC \]

→ 2角が等しければ、その向かいの2辺も等しい！

▶ 正三角形

📌 ルール4：正三角形の性質 → 例3

正三角形：3辺がすべて等しい三角形

正三角形：3辺等しい → 3角とも60°

正三角形の性質：

  • 3つの辺がすべて等しい
  • 3つの角がすべて60°

正三角形になる条件：

  • 3辺が等しい
  • 3角が等しい（すべて60°）
  • 2角が60°（残りも60°になる）

▶ 直角三角形

📌 ルール5：直角三角形の合同条件 → 例4

通常の3条件に加えて、直角三角形には特別な合同条件がある：

斜辺と他の1辺が等しい直角三角形は合同
条件 内容
条件① 斜辺と1つの鋭角がそれぞれ等しい
条件② 斜辺と他の1辺がそれぞれ等しい

ポイント： 直角三角形では、直角（90°）がすでに等しいので、残り1つの条件で合同が言える！

📌 まとめ表
三角形 定義 性質
二等辺三角形 2辺が等しい 底角が等しい
正三角形 3辺が等しい 3角とも60°
直角三角形 1角が90° 特別な合同条件あり

3. 例題と解き方

特別な三角形の性質を使って問題を解いてみましょう。

▶ 二等辺三角形

例1底角を求める

AB = AC の二等辺三角形ABCで、∠A = 40° のとき、∠B を求めなさい。

解き方：

  1. 二等辺三角形の底角は等しいので、∠B = ∠C
  2. 三角形の内角の和は180°
  3. ∠A + ∠B + ∠C = 180°
  4. 40° + ∠B + ∠B = 180°
  5. 2∠B = 140°
\[ \angle B = 70° \]
例2二等辺三角形であることを示す

△ABCで、∠B = ∠C = 65° のとき、この三角形は二等辺三角形であることを示しなさい。

解き方：

  1. ∠B = ∠C（仮定）
  2. 2つの角が等しい三角形は二等辺三角形（定理の逆）
  3. よって、AB = AC の二等辺三角形

→ 2角が等しい → 二等辺三角形！

▶ 正三角形

例3正三角形の角

正三角形ABCで、辺BCの中点をMとする。∠AMB を求めなさい。

解き方：

  1. 正三角形の各角は60°
  2. 正三角形では、頂点から底辺の中点への線は底辺に垂直
  3. よって、AM ⊥ BC
\[ \angle AMB = 90° \]

▶ 直角三角形の合同

例4直角三角形の合同を証明する

下の図で、∠ADB = ∠AEB = 90°、AD = AE のとき、△ABD ≡ △ACE を証明しなさい。

【証明】

△ABDと△ACEにおいて、

  1. ∠ADB = ∠AEC = 90° （仮定）
  2. AB = AC （二等辺三角形の等しい辺）
  3. AD = AE （仮定）

①②③より、直角三角形で斜辺と他の1辺がそれぞれ等しいので、

\[ \triangle ABD \equiv \triangle ACE \]
例5二等辺三角形の性質を使った証明

AB = AC の二等辺三角形で、Aから底辺BCに垂線AMを下ろすとき、BM = CM を証明しなさい。

【証明】

△ABMと△ACMにおいて、

  1. ∠AMB = ∠AMC = 90° （AMは垂線）
  2. AB = AC （仮定）
  3. AM = AM （共通）

①②③より、直角三角形で斜辺と他の1辺がそれぞれ等しいので、

△ABM ≡ △ACM

合同な三角形の対応する辺は等しいので、

\[ BM = CM \]

→ 二等辺三角形の頂角の二等分線は底辺を垂直に2等分する！

4. 簡単にするコツ

✏️ コツ1：「等しい辺」を見つけたら底角をチェック
2辺が等しい → その向かいの2角も等しい。これで角度の情報が増える！
✏️ コツ2：「等しい角」を見つけたら辺をチェック
2角が等しい → その向かいの2辺も等しい。二等辺三角形だとわかる！
✏️ コツ3：正三角形の3つの60°を活用
正三角形なら各角は60°。外角は120°。これらの角度はよく出てくる。
✏️ コツ4：直角三角形を見つけたら特別な合同条件
直角があれば「斜辺と他の1辺」で合同が言える。3つではなく2つの条件で済む！
✏️ コツ5：補助線で二等辺三角形を作る
頂角の二等分線や底辺の中点への線を引くと、合同な直角三角形ができる。

5. よくあるつまづき

⚠️ つまづき1：頂角と底角を逆にする

ダメな例： AB = AC で「∠Aが底角」と言う

対策： 等しい2辺がはさむ角が頂角。底辺の両端が底角。

⚠️ つまづき2：定理と逆を混同する

ダメな例： 「底角が等しいから二等辺三角形」を「二等辺三角形の性質」と言う

対策：

  • 性質：2辺等しい → 2角等しい
  • 逆：2角等しい → 2辺等しい
⚠️ つまづき3：直角三角形の合同条件を一般の三角形に使う

ダメな例： 直角がない三角形で「斜辺と他の1辺」を使う

対策： 「斜辺」は直角三角形だけにある。まず直角があることを確認！

⚠️ つまづき4：正三角形を見落とす

ダメな例： 「3辺等しい」と書いてあるのに60°を使わない

対策： 3辺等しい = 正三角形 = 全角60°。見つけたらすぐ60°を書き込む。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

AB = AC の二等辺三角形ABCで、∠B = 72° のとき、∠A を求めなさい。

答えを見る

底角が等しいので ∠C = ∠B = 72°

∠A + ∠B + ∠C = 180°

∠A + 72° + 72° = 180°

∠A = 36°

練習2

△ABCで、∠A = 60°、∠B = 60° のとき、この三角形は何三角形か答えなさい。

答えを見る

∠C = 180° - 60° - 60° = 60°

3つの角がすべて60°なので、正三角形

練習3

直角三角形ABCと直角三角形DEFで、∠C = ∠F = 90°、AB = DE = 10cm、BC = EF = 6cm のとき、2つの三角形は合同か。合同なら理由を答えなさい。

答えを見る

合同

直角三角形で、斜辺（AB = DE）と他の1辺（BC = EF）がそれぞれ等しいので合同。

📌 まとめ：二等辺三角形は「2辺等しい ⟺ 2角等しい」、正三角形は「3辺等しい ⟺ 3角とも60°」、直角三角形は特別な合同条件あり！

おまけ：実践ストーリー「建築と三角形」

📖 なぜ屋根は三角形？

日本の伝統的な家屋を見てみると、屋根はたいてい三角形の形をしています。なぜでしょうか？

実は、三角形は「変形しにくい」という特別な性質を持っています。四角形の枠は力を加えると平行四辺形に歪んでしまいますが、三角形の枠は歪みません。

さらに、二等辺三角形の屋根には大きなメリットがあります。

左右対称なので、雨が左右均等に流れ落ちます。雪の重さも左右に分散されます。しかも、頂点から降ろした垂線は底辺を垂直に2等分するので、柱を真ん中に1本立てれば安定します。

二等辺三角形の性質「頂角の二等分線は底辺を垂直に2等分する」は、まさに建築に役立っているのです。

東京スカイツリーの展望台を支える構造にも、三角形がたくさん使われています。「トラス構造」と呼ばれるこの技術は、三角形の強さを活かしたものです。

数学で学ぶ三角形の性質は、私たちの身の回りの建物を支えているのですね。
