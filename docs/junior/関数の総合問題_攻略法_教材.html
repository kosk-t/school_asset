1. テーマの定義

入試では、直線と放物線を組み合わせた「総合問題」がよく出ます。これらの問題は、実はいくつかのパターンに分類できます。パターンを見抜けば、解き方が自然と決まります。

このページでは、関数の総合問題の攻略法を学びます：

  • 問題タイプの判別：何を求める問題か見抜く
  • 解法フロー：タイプごとの解き方
  • 面積問題の攻略：三角形・台形の面積計算

2. 覚えるルール

【最重要】問題を見たら「①交点 ②変域 ③面積 ④条件を満たす点」のどれか判別しよう！

▶ 問題タイプ判別フロー

📌 ルール1：5つの問題タイプ
タイプ キーワード 解法
① 交点 「交点」「共有点」 連立方程式を解く
② 式を求める 「直線の式」「放物線の式」 2点の座標を代入
③ 変域 「yの変域」「最大・最小」 xの範囲でグラフを追う
④ 面積 「面積」「三角形」 底辺×高さ÷2
⑤ 条件を満たす点 「面積が○○のとき」「等しいとき」 条件を式にして方程式

▶ タイプ別解法

📌 ルール2：タイプ① 交点を求める → 例1

2つのグラフの交点 = 連立方程式の解

  1. 2つの式を連立する
  2. 代入法または加減法でxを求める
  3. xを代入してyを求める

例： \(y = x^2\) と \(y = 2x + 3\) の交点

\[ x^2 = 2x + 3 \Rightarrow x^2 - 2x - 3 = 0 \Rightarrow (x-3)(x+1) = 0 \]
📌 ルール3：タイプ② 式を求める → 例2

直線の式： 2点を通る直線 → 傾きを求めて \(y = ax + b\)

\[ \text{傾き} = \frac{y_2 - y_1}{x_2 - x_1} \]

放物線の式： \(y = ax^2\) の形なら、1点を代入してaを求める

📌 ルール4：タイプ③ 変域を求める → 例3

xの変域が与えられたとき、yの変域を求める

  1. 一次関数：端点のy座標を計算
  2. 二次関数（\(y = ax^2\)）：
    • xの変域が原点（頂点）を含むか確認
    • 含む → 最小値は0（a > 0のとき）
    • 含まない → 端点で最大・最小
📌 ルール5：タイプ④ 面積を求める → 例4

座標平面上の三角形の面積：

  1. 底辺を軸に平行にとる（x軸またはy軸）
  2. 高さ = 底辺から頂点までの距離
底辺をx軸上に、高さを縦にとる

y軸を底辺にする場合： △OAB でAとBがy軸上にあるとき

\[ \text{面積} = \frac{1}{2} \times |y_A - y_B| \times |x_P| \]
📌 ルール6：タイプ⑤ 条件を満たす点 → 例5

「面積が○○になる点P」「三角形が二等分される点」など

  1. 求める点を \((t, at^2)\) などと置く
  2. 条件を式で表す（面積 = 12 など）
  3. 方程式を解いてtを求める
📌 面積公式まとめ
図形 公式
三角形 \(\frac{1}{2} \times \text{底辺} \times \text{高さ}\)
台形 \(\frac{1}{2} \times (\text{上底} + \text{下底}) \times \text{高さ}\)
y軸上に2点 \(\frac{1}{2} \times |y_1 - y_2| \times |x|\)

3. 例題と解き方

各タイプの問題を実際に解いてみましょう。

例1タイプ① 交点を求める

放物線 \(y = x^2\) と直線 \(y = x + 2\) の交点の座標を求めなさい。

解き方：

  1. 連立して \(x^2 = x + 2\)
  2. \(x^2 - x - 2 = 0\)
  3. \((x-2)(x+1) = 0\) より \(x = 2, -1\)
  4. y座標：\(x=2\) のとき \(y=4\)、\(x=-1\) のとき \(y=1\)
\[ \text{交点：} (2, 4), (-1, 1) \]
例2タイプ② 式を求める

点A(-1, 1)と点B(2, 4)を通る直線の式を求めなさい。

解き方：

  1. 傾き = \(\frac{4-1}{2-(-1)} = \frac{3}{3} = 1\)
  2. \(y = x + b\) に点A(-1, 1)を代入
  3. \(1 = -1 + b\) より \(b = 2\)
\[ y = x + 2 \]
例3タイプ③ 変域を求める

\(y = x^2\) で、xの変域が \(-1 \leq x \leq 3\) のとき、yの変域を求めなさい。

解き方：

  1. xの変域が0を含むか確認 → -1から3なので含む
  2. x = 0のとき y = 0（最小値）
  3. x = -1のとき y = 1、x = 3のとき y = 9
  4. 0から遠い方（x = 3）で最大値
\[ 0 \leq y \leq 9 \]
例4タイプ④ 面積を求める

放物線 \(y = x^2\) と直線 \(y = x + 2\) について、2つの交点と原点Oで作られる三角形の面積を求めなさい。

解き方：

  1. 交点はA(-1, 1)、B(2, 4)（例1より）
  2. 直線のy切片はC(0, 2)
  3. △OABを△OACと△OBCに分割
  4. 底辺をy軸上のOCにとる（長さ2）
  5. △OAC = \(\frac{1}{2} \times 2 \times 1 = 1\)
  6. △OBC = \(\frac{1}{2} \times 2 \times 2 = 2\)
\[ \triangle OAB = 1 + 2 = 3 \]
例5タイプ⑤ 条件を満たす点

放物線 \(y = x^2\) 上に点Pをとり、△OAPの面積が6になるようにする。ただしA(-1, 1)、Pのx座標は正とする。Pの座標を求めなさい。

解き方：

  1. P(t, t²) と置く（t > 0）
  2. 座標による面積公式を使用

面積 = \(\frac{1}{2}|x_O(y_A - y_P) + x_A(y_P - y_O) + x_P(y_O - y_A)|\)

= \(\frac{1}{2}|0 \cdot (1-t^2) + (-1)(t^2 - 0) + t(0 - 1)|\)

= \(\frac{1}{2}|-t^2 - t| = \frac{1}{2}(t^2 + t)\)（t > 0より）

\(\frac{1}{2}(t^2 + t) = 6\) より \(t^2 + t - 12 = 0\)

\((t+4)(t-3) = 0\)、t > 0より t = 3

\[ P(3, 9) \]

4. 簡単にするコツ

✏️ コツ1：まず問題タイプを判別する
問題文のキーワード（交点/変域/面積/〜のとき）から、どのタイプか見抜く。
✏️ コツ2：グラフを描く
概略でいいのでグラフを描く。三角形や交点の位置関係が見えやすくなる。
✏️ コツ3：面積はy軸を底辺にすると楽
直線のy切片を使って、y軸上の線分を底辺にすると計算が簡単。
✏️ コツ4：二次関数の変域は「0を含むか」がカギ
\(y = ax^2\) でxの変域が0を含むなら、y = 0が端点になる（a > 0のとき最小）。
✏️ コツ5：交点を先に求める
多くの問題は「まず交点を求める」ことから始まる。最初にやっておく。

5. よくあるつまづき

⚠️ つまづき1：変域で0を含むか確認しない

ダメな例： \(y = x^2\) で \(-2 \leq x \leq 1\) のとき、端点だけ計算して「1 ≤ y ≤ 4」とする

対策： 0を含むので y = 0 が最小。正解は「0 ≤ y ≤ 4」

⚠️ つまづき2：交点を求めずに面積計算

ダメな例： 交点の座標がわからないまま面積を求めようとする

対策： まず交点の座標を求める。それが基本。

⚠️ つまづき3：面積の分割を忘れる

ダメな例： 三角形の底辺と高さが軸に平行でないのに、無理に計算

対策： 軸に平行な線で分割して、小さい三角形に分ける。

⚠️ つまづき4：符号ミス

ダメな例： 座標が負のとき、長さも負で計算してしまう

対策： 長さは常に正。|x₁ - x₂| のように絶対値を使う。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

放物線 \(y = 2x^2\) と直線 \(y = 4x + 6\) の交点の座標を求めなさい。

答えを見る

\(2x^2 = 4x + 6\) → \(x^2 - 2x - 3 = 0\) → \((x-3)(x+1) = 0\)

x = 3のとき y = 18、x = -1のとき y = 2

交点：(3, 18), (-1, 2)

練習2

\(y = \frac{1}{2}x^2\) で、xの変域が \(-4 \leq x \leq 2\) のとき、yの変域を求めなさい。

答えを見る

0を含む → y = 0が最小。x = -4で最大値8

0 ≤ y ≤ 8

練習3

点O(0, 0)、A(4, 0)、B(2, 6) で作られる三角形OABの面積を求めなさい。

答えを見る

底辺OA = 4、高さ = 6

面積：12

📌 まとめ：問題タイプ（交点/式/変域/面積/条件）を判別 → それぞれの解法を使う！面積はy軸を底辺に！

おまけ：入試攻略「関数問題の読み解き方」

📖 入試問題の「型」を見抜く

高校入試の関数問題は、一見すると「難しそう」に見えます。でも実は、ほとんどが決まった型の組み合わせです。

入試問題の典型パターン

パターン1：「式を求める→交点→面積」

最も多いパターン。(1)で直線の式、(2)で交点、(3)で面積という流れ。

パターン2：「変域→最大最小」

二次関数の変域問題。0を含むかどうかがポイント。

パターン3：「面積を二等分する点」

条件を満たす点Pを求める。方程式を立てて解く。

問題文の読み方

問題文にはヒントが隠れています：

  • 「交点」と書いてあれば → 連立方程式
  • 「面積」と書いてあれば → 底辺×高さ÷2
  • 「〜のとき」と書いてあれば → 方程式を立てる
  • 「変域」と書いてあれば → グラフで端点チェック

解答の「流れ」を意識する

入試問題は通常、(1)(2)(3)と続きます。前の問いの答えが次で使えることがほとんど。

もし(2)で詰まったら、(1)の答えをヒントに。(1)で求めた式や座標が、(2)以降で必ず使われます。
