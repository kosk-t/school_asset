1. テーマの定義

「2点を通る直線の式は？」「2つのグラフはどこで交わる？」これらを解決するのが一次関数の応用です。料金プランの比較や、2つのものが同じになる時点を求めるなど、実生活でも活躍します。

このページでは、一次関数の応用を学びます：

  • 式の求め方：2点から、または傾きと1点から式を求める
  • グラフの交点：2つの直線が交わる点の座標を求める
  • 変域：xやyの取りうる範囲を求める
  • 一次関数の利用：速さ・料金・水量などの問題

2. 覚えるルール

【最重要】2点 \((x_1, y_1)\), \((x_2, y_2)\) を通る直線の式は、傾き \(a = \frac{y_2 - y_1}{x_2 - x_1}\) を求めてから \(y = ax + b\) に代入
2点から傾きを求める
📌 ルール1：2点から式を求める手順 → 例1
  1. 傾きを求める：\(a = \frac{y_2 - y_1}{x_2 - x_1}\)
  2. 切片を求める：\(y = ax + b\) にどちらかの点を代入して \(b\) を求める
  3. 式を書く：\(y = ax + b\)

✅ メリット： どんな2点からでも直線の式が求まる！

📌 ルール2：傾きと1点から式を求める → 例2

傾き \(a\) と点 \((p, q)\) がわかっているとき：

  1. \(y = ax + b\) に傾き \(a\) を入れる
  2. 点 \((p, q)\) を代入して \(b\) を求める

→ \(q = ap + b\) から \(b = q - ap\)

📌 ルール3：2直線の交点の求め方 → 例3

2つの一次関数 \(y = a_1x + b_1\) と \(y = a_2x + b_2\) の交点は：

\[ a_1x + b_1 = a_2x + b_2 \]

を解いて \(x\) を求め、どちらかの式に代入して \(y\) を求める。

2直線の交点は連立方程式で求まる

→ グラフの交点 ＝ 連立方程式の解！

💡 考え方を見る

交点では2つの直線上に同じ点がある、つまり同じ \(x\) に対して \(y\) が等しい。だから2つの式を = で結んで解ける。

📌 ルール4：変域の求め方 → 例4

\(x\) の変域が \(a \leq x \leq b\) のとき、\(y\) の変域は：

  • 傾きが正：\(x = a\) で最小、\(x = b\) で最大
  • 傾きが負：\(x = a\) で最大、\(x = b\) で最小
傾きの符号と最大・最小の関係
📌 ルール5：一次関数の利用（文章題） → 例5

よく出るパターン：

パターン 傾き 切片
速さ・距離 速さ（分速、時速） 出発時の位置
料金プラン 単価（1個あたり、1分あたり） 基本料金
水量 1分あたりの増減量 最初の水量

3. 例題と解き方

一次関数の応用問題を解いてみましょう。

▶ 式の求め方

例12点から式を求める

2点 \((1, 3)\), \((4, 9)\) を通る直線の式を求めなさい。

解き方：

  1. 傾きを求める \[ a = \frac{9 - 3}{4 - 1} = \frac{6}{3} = 2 \]
  2. 切片を求める：\(y = 2x + b\) に点 \((1, 3)\) を代入 \[ 3 = 2 \times 1 + b \] \[ b = 3 - 2 = 1 \]
  3. 式を書く
\[ y = 2x + 1 \]

確認： \((4, 9)\) を代入すると \(y = 2 \times 4 + 1 = 9\) ✓

例2傾きと1点から式を求める

傾きが \(-3\) で、点 \((2, 1)\) を通る直線の式を求めなさい。

解き方：

  1. \(y = -3x + b\) に点 \((2, 1)\) を代入
  2. \(1 = -3 \times 2 + b\)
  3. \(1 = -6 + b\) より \(b = 7\)
\[ y = -3x + 7 \]

▶ 交点を求める

例32直線の交点

\(y = 2x - 1\) と \(y = -x + 5\) の交点の座標を求めなさい。

解き方：

  1. 連立方程式を立てる：交点では \(y\) が等しいので \[ 2x - 1 = -x + 5 \]
  2. \(x\) を求める \[ 2x + x = 5 + 1 \] \[ 3x = 6 \] \[ x = 2 \]
  3. \(y\) を求める：\(y = 2 \times 2 - 1 = 3\)
\[ \text{交点の座標：} (2, 3) \]

▶ 変域

例4yの変域を求める

\(y = -2x + 5\) で、\(x\) の変域が \(1 \leq x \leq 4\) のとき、\(y\) の変域を求めなさい。

解き方：

  1. 傾きを確認：\(a = -2 < 0\)（右下がり）
  2. \(x = 1\) のとき：\(y = -2 \times 1 + 5 = 3\)（最大）
  3. \(x = 4\) のとき：\(y = -2 \times 4 + 5 = -3\)（最小）
\[ -3 \leq y \leq 3 \]

→ 傾きが負なので、xが増えるとyは減る！

▶ 一次関数の利用

例5追いつき問題

A君は家から毎分80mで歩き始めた。その5分後に、B君が家から毎分200mの自転車で追いかけた。B君がA君に追いつくのは、B君が出発してから何分後か。

解き方：

  1. 変数を決める：B君出発からx分後、家からの距離をymとする
  2. A君の式：5分先に出発しているので \[ y = 80(x + 5) = 80x + 400 \]
  3. B君の式： \[ y = 200x \]
  4. 追いつく条件：距離が等しい \[ 200x = 80x + 400 \] \[ 120x = 400 \] \[ x = \frac{400}{120} = \frac{10}{3} \]
\[ \frac{10}{3} \text{分後} = 3\text{分}20\text{秒後} \]
例6料金プランの比較

プランA：基本料金500円、1分10円。プランB：基本料金200円、1分20円。何分以上使うとプランAがお得か。

解き方：

  1. 式を立てる（x分使用したときの料金y円）
    • プランA：\(y = 10x + 500\)
    • プランB：\(y = 20x + 200\)
  2. 料金が等しくなる点 \[ 10x + 500 = 20x + 200 \] \[ 500 - 200 = 20x - 10x \] \[ 300 = 10x \] \[ x = 30 \]
  3. 判断：30分を超えるとプランAの傾きが小さい（ゆるやか）ので料金が低い
\[ 30\text{分より多く使うとプランAがお得} \]
例7グラフから式を読み取る

グラフが点 \((0, 4)\) と点 \((6, -2)\) を通る一次関数の式を求めなさい。

解き方：

  1. 切片は \(4\)（y軸との交点が \((0, 4)\) だから）
  2. 傾きを求める \[ a = \frac{-2 - 4}{6 - 0} = \frac{-6}{6} = -1 \]
\[ y = -x + 4 \]

4. 簡単にするコツ

✏️ コツ1：一方の点が \((0, b)\) なら、そこが切片
片方の点のx座標が0なら、そのy座標がそのまま切片。傾きだけ計算すればOK。
✏️ コツ2：傾きの計算は「yの差 ÷ xの差」
順番さえ合わせれば、どちらの点を先にしてもOK。\(\frac{y_1 - y_2}{x_1 - x_2}\) でも同じ答え。
✏️ コツ3：交点を求めたら検算
求めた \((x, y)\) を両方の式に代入して、両方成り立てばOK。
✏️ コツ4：変域は端点だけチェック
一次関数は直線だから、途中で最大・最小になることはない。端点の2つのyを計算するだけ。
✏️ コツ5：文章題は「何がxで何がy」を最初に決める
時間なのか距離なのか、料金なのか個数なのか。軸を決めてから式を立てる。

5. よくあるつまづき

⚠️ つまづき1：傾きの計算でx, yを逆にする

ダメな例： \((1, 3)\) と \((4, 9)\) で傾き = \(\frac{4-1}{9-3} = \frac{3}{6} = \frac{1}{2}\)

対策： 傾き = yの差 ÷ xの差！「y」が上、「x」が下。

⚠️ つまづき2：変域で最大・最小を逆にする

ダメな例： 傾きが負なのに「xが大きいとyも大きい」と思う

対策： 傾きが負なら右下がり！xが増えるとyは減る。

⚠️ つまづき3：追いつき問題で時間のズレを忘れる

ダメな例： 「5分後に出発」なのに両方とも \(y = (\text{速さ}) \times x\) にする

対策： 先に出た方は「\(x + (\text{先に出た時間})\)」分経過している。

⚠️ つまづき4：交点の計算で代入ミス

ダメな例： \(x\) を求めた後、関係ない式に代入する

対策： どちらの式に代入してもOK。計算しやすい方を選ぶ。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

2点 \((2, 5)\), \((5, 11)\) を通る直線の式を求めなさい。

答えを見る

傾き：\(a = \frac{11 - 5}{5 - 2} = \frac{6}{3} = 2\)

切片：\(5 = 2 \times 2 + b\) より \(b = 1\)

答え：\(y = 2x + 1\)

練習2

\(y = 3x - 2\) と \(y = -x + 6\) の交点の座標を求めなさい。

答えを見る

\(3x - 2 = -x + 6\)

\(4x = 8\) より \(x = 2\)

\(y = 3 \times 2 - 2 = 4\)

答え：\((2, 4)\)

練習3

\(y = -\frac{1}{2}x + 4\) で、\(x\) の変域が \(-2 \leq x \leq 6\) のとき、\(y\) の変域を求めなさい。

答えを見る

傾きが負なので、\(x = -2\) で最大、\(x = 6\) で最小

\(x = -2\)：\(y = -\frac{1}{2} \times (-2) + 4 = 5\)

\(x = 6\)：\(y = -\frac{1}{2} \times 6 + 4 = 1\)

答え：\(1 \leq y \leq 5\)

おまけ：数学雑学「ガリレオと落下の法則」

🗼 ピサの斜塔から始まった「速さ」の物語

1589年、イタリア。若き数学教授ガリレオ・ガリレイは、ある実験をしようとしていました。

当時の常識は「重いものほど速く落ちる」。アリストテレス（古代ギリシャの大哲学者）がそう言ったから、約2000年間誰も疑わなかったのです。

ガリレオ「いや、それおかしくない？」

彼はピサの斜塔に登り、重さの違う2つの球を同時に落としました。

結果は......

同時に着地！

重さに関係なく、同じ時間で落ちたのです。

さらにガリレオは、斜面を転がる球の実験で「落下の法則」を発見しました。

物体が落ち始めてから \(t\) 秒後の速さ \(v\) は：

\[ v = gt \]

（\(g\) は重力加速度、約9.8 m/s²）

これは傾き \(g\)、切片 \(0\) の一次関数です！

つまり、速さは時間に比例して一定の割合で増える。1秒ごとに約9.8 m/s ずつ速くなる。

ちなみに、ピサの斜塔の実験は「伝説」とも言われています。本当にやったかどうかは定かではありません。

でも確実なのは、ガリレオが「実験で確かめる」という科学の基本を確立したこと。2000年間信じられていた常識を、たった一つの実験で覆したのです。

...というわけで、一次関数は物理学の基礎。ガリレオは「近代科学の父」と呼ばれ、彼の発見は後にニュートンの運動法則へとつながります。

※ピサの斜塔の実験については、ガリレオの弟子ヴィヴィアーニの伝記に記載がありますが、実際に行われたかは議論があります。
