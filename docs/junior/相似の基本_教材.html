1. テーマの定義

地図やコピー機の「拡大・縮小」、写真の引き伸ばし——これらは形を変えずにサイズだけを変えています。数学ではこのような関係を相似と呼びます。

このページでは、相似の基本を学びます：

  • 相似の意味：形は同じでサイズが違う図形の関係
  • 相似比：対応する辺の比（何倍になっているか）
  • 三角形の相似条件：相似を証明するための3つの条件

2. 覚えるルール

🎯 最重要：相似な図形では「対応する辺の比がすべて等しい」かつ「対応する角がすべて等しい」
相似な図形：形は同じで、対応する辺の比がすべて等しい
📌 ルール1：相似の定義と記号 → 例1

相似とは、形は同じでサイズが違う図形の関係。

  • 対応する角がすべて等しい
  • 対応する辺の比がすべて等しい

記号 \(\triangle ABC \sim \triangle DEF\)（「AかつDが対応、BかつEが対応...」の意味）

→ だから結果は： 相似な図形では、対応する部分を見つければ角度や辺の比がわかる

💡 考え方を見る

「合同」は「形もサイズも同じ」だったが、「相似」は「形だけ同じ」。拡大コピーや縮小コピーのイメージ。

📌 ルール2：相似比 → 例1
\[ \text{相似比} = m : n \quad \Rightarrow \quad \text{対応する辺の比がすべて } m : n \]

相似比とは、対応する辺の長さの比。相似比が \(1 : 2\) なら、すべての辺が2倍。

→ だから結果は： 1辺の比がわかれば、他の辺の長さも計算できる

✅ メリット： 直接測れない長さも、相似比から計算できる（地図、縮尺模型など）

三角形の相似条件：AA・SSS・SASの3つを覚えよう
📌 ルール3：AA相似（2組の角が等しい） → 例2
\[ \angle A = \angle D, \quad \angle B = \angle E \quad \Rightarrow \quad \triangle ABC \sim \triangle DEF \]

2つの角がそれぞれ等しければ、残り1つの角も自動的に等しくなる（三角形の内角の和は180°）。

→ だから結果は： 2角だけ調べれば相似を証明できる

✅ メリット： 最もよく使う条件。角度の情報だけで相似が言える

📌 ルール4：SSS相似（3辺の比が等しい） → 例3
\[ \frac{AB}{DE} = \frac{BC}{EF} = \frac{CA}{FD} \quad \Rightarrow \quad \triangle ABC \sim \triangle DEF \]

3辺の比がすべて等しければ相似。

→ だから結果は： 3辺の長さがわかっていれば相似を判定できる

📌 ルール5：SAS相似（2辺の比とその間の角が等しい） → 例4
\[ \frac{AB}{DE} = \frac{AC}{DF}, \quad \angle A = \angle D \quad \Rightarrow \quad \triangle ABC \sim \triangle DEF \]

2辺の比が等しく、その間の角（はさむ角）が等しければ相似。

→ だから結果は： 「間の角」を忘れないこと！

条件 何が等しい？ 合同との違い
AA相似 2組の角 合同にはない（相似専用）
SSS相似 3辺の「比」 合同は「長さが等しい」、相似は「比が等しい」
SAS相似 2辺の「比」と間の角 合同は「長さが等しい」、相似は「比が等しい」

3. 例題と解き方

相似比の計算と、3つの相似条件を使った証明を練習しましょう。

▶ 相似比と辺の長さ

例1相似比から辺の長さを求める

\(\triangle ABC \sim \triangle DEF\) で相似比が \(2:3\)。\(AB = 6\) cm のとき、\(DE\) の長さを求めよ。

解き方：

  1. 対応する辺を確認：\(AB\) と \(DE\) が対応
  2. 相似比 \(2:3\) を使う → \(AB : DE = 2 : 3\)
  3. \(6 : DE = 2 : 3\) を解く → \(DE = 6 \times \frac{3}{2} = 9\)
\[ AB : DE = 2 : 3 \] \[ 6 : DE = 2 : 3 \] \[ DE = 6 \times \frac{3}{2} = \boxed{9 \text{ cm}} \]
例2辺の長さから相似比を求める

\(\triangle ABC \sim \triangle DEF\) で \(BC = 4\) cm, \(EF = 10\) cm。相似比を求めよ。

解き方：

  1. 対応する辺の比を求める：\(BC : EF = 4 : 10\)
  2. 最も簡単な整数比にする：\(4 : 10 = 2 : 5\)
\[ BC : EF = 4 : 10 = \boxed{2 : 5} \]

▶ AA相似で証明する

例32組の角が等しいことを示す

下図で、\(\angle ACB = \angle DCE\)（対頂角）のとき、\(\triangle ABC \sim \triangle DEC\) を証明せよ。

対頂角と平行線による相似

証明：

  1. \(\angle ACB = \angle DCE\)（対頂角）...①
  2. \(AB // DE\) より、\(\angle ABC = \angle DEC\)（錯角）...②
  3. ①②より、2組の角がそれぞれ等しいので
    \(\triangle ABC \sim \triangle DEC\)（AA相似）

▶ SSS相似で判定する

例43辺の比で相似を判定

\(\triangle ABC\) で \(AB = 3\), \(BC = 4\), \(CA = 5\)。\(\triangle DEF\) で \(DE = 6\), \(EF = 8\), \(FD = 10\)。この2つは相似か。

解き方：

  1. 対応する辺の比を計算：
    • \(AB : DE = 3 : 6 = 1 : 2\)
    • \(BC : EF = 4 : 8 = 1 : 2\)
    • \(CA : FD = 5 : 10 = 1 : 2\)
  2. 3辺の比がすべて \(1 : 2\) で等しい
  3. よって、SSS相似より \(\triangle ABC \sim \triangle DEF\)
\[ \frac{AB}{DE} = \frac{BC}{EF} = \frac{CA}{FD} = \frac{1}{2} \quad \therefore \boxed{\text{相似}} \]

▶ SAS相似で証明する

例52辺の比と間の角で証明

\(\triangle ABC\) で \(AB = 4\), \(AC = 6\), \(\angle A = 50°\)。\(\triangle DEF\) で \(DE = 6\), \(DF = 9\), \(\angle D = 50°\)。相似であることを証明せよ。

解き方：

  1. \(\angle A = \angle D = 50°\)（間の角が等しい）...①
  2. \(AB : DE = 4 : 6 = 2 : 3\)
  3. \(AC : DF = 6 : 9 = 2 : 3\)
  4. \(AB : DE = AC : DF = 2 : 3\)（2辺の比が等しい）...②
  5. ①②より、\(\triangle ABC \sim \triangle DEF\)（SAS相似）

▶ 相似を使って長さを求める

例6相似を使って未知の辺を求める

下図で \(\triangle ABC \sim \triangle ADE\)、\(AB = 6\), \(AD = 9\), \(BC = 4\) のとき、\(DE\) の長さを求めよ。

相似な三角形で辺の長さを求める

解き方：

  1. 相似比を求める：\(AB : AD = 6 : 9 = 2 : 3\)
  2. \(BC\) と \(DE\) は対応する辺なので、\(BC : DE = 2 : 3\)
  3. \(4 : DE = 2 : 3\) より、\(DE = 4 \times \frac{3}{2} = 6\)
\[ BC : DE = 2 : 3 \] \[ 4 : DE = 2 : 3 \] \[ DE = \boxed{6} \]

4. 簡単にするコツ

✏️ コツ1：対応する頂点の順番に注意
\(\triangle ABC \sim \triangle DEF\) と書いたら、「AとD、BとE、CとF」が対応。証明で書く順番を間違えると減点されることも。図に対応を書き込んでから解こう。
✏️ コツ2：AA相似を最優先で考える
相似条件の中で最もよく使うのがAA相似。角度の情報（平行線、対頂角、共通な角など）があれば、まずAA相似を疑おう。
✏️ コツ3：比は約分して揃える
\(4:6\) と \(6:9\) が等しいか？ → どちらも約分して \(2:3\) にすれば比較しやすい。
✏️ コツ4：「間の角」を忘れない（SAS相似）
SAS相似では、2辺の比が等しいだけではダメ。その間の角が等しいことが必要。「間の角」を見落とすミスが多い。

5. よくあるつまづき

⚠️ つまづき1：対応を間違える

ダメな例： \(\triangle ABC \sim \triangle DEF\) で、\(AB\) と \(EF\) が対応すると思ってしまう

対策： 記号の順番で対応が決まる。\(\triangle ABC \sim \triangle DEF\) なら、A↔D、B↔E、C↔F。だから \(AB\) と \(DE\) が対応。

⚠️ つまづき2：合同条件と相似条件を混同

ダメな例： 「3辺が等しいからSSS相似」と書いてしまう

対策： 相似は「比が等しい」、合同は「長さが等しい」。SSS相似は「3辺の比が等しい」こと。

⚠️ つまづき3：SAS相似で「間の角」を忘れる

ダメな例： 2辺の比が等しいだけでSAS相似と判断

対策： SAS相似は「S（辺）A（角）S（辺）」の順。「間の角」（はさまれた角）が等しいことを必ず確認。

⚠️ つまづき4：AA相似なのにAAA相似と書く

ダメな例： 「3つの角がすべて等しいからAAA相似」と書く

対策： 2組の角が等しければ、残り1組は自動的に等しい。正式名称は「AA相似」（2組の角）。

6. 仕上げチェックリスト

対応する頂点の順番は正しいか（A↔D、B↔E、C↔F）
相似条件（AA・SSS・SAS）を正しく選んだか
SAS相似なら「間の角」を確認したか
比を約分して最も簡単な整数比にしたか
証明では理由（対頂角、錯角など）を書いたか

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(\triangle ABC \sim \triangle DEF\) で相似比が \(3:5\)。\(BC = 12\) cm のとき、対応する辺 \(EF\) の長さを求めよ。

答えを見る

\(BC : EF = 3 : 5\) より

\(12 : EF = 3 : 5\)

\(EF = 12 \times \frac{5}{3} = \boxed{20 \text{ cm}}\)

練習2

\(\triangle ABC\) で \(AB = 8\), \(BC = 6\), \(CA = 10\)。\(\triangle DEF\) で \(DE = 12\), \(EF = 9\), \(FD = 15\)。この2つは相似か。相似ならば相似比を答えよ。

答えを見る

3辺の比を調べる：

\(AB : DE = 8 : 12 = 2 : 3\)

\(BC : EF = 6 : 9 = 2 : 3\)

\(CA : FD = 10 : 15 = 2 : 3\)

3辺の比がすべて \(2:3\) で等しいので、SSS相似より

\(\boxed{\text{相似である。相似比は } 2:3}\)

📌 まとめ：相似条件は「AA（2角）、SSS（3辺の比）、SAS（2辺の比と間の角）」の3つ。まずAAを疑え！

おまけ：歴史ストーリー「影でピラミッドを測った男」

📖 タレスとピラミッドの謎（紀元前6世紀・エジプト）

紀元前6世紀、ギリシャの哲学者タレスがエジプトを訪れた。そこで彼はギザの大ピラミッドを見上げて、こう聞かれた。

「この巨大なピラミッドの高さ、あなたには測れますか？」

ピラミッドは高さ約140メートル。直接登るのは危険だし、そもそも斜面を測っても本当の高さにはならない。どうする？

タレスは笑って言った。「影を使えばいい」

彼がやったことはシンプルだった：

  1. 自分の身長と、自分の影の長さを測る（例：身長180cm、影の長さ200cm）
  2. 同じ時刻にピラミッドの影の長さを測る（例：160m）
  3. 「身長：影の長さ」と「ピラミッドの高さ：ピラミッドの影の長さ」は同じ比！
\[ \frac{\text{身長}}{\text{自分の影}} = \frac{\text{ピラミッドの高さ}}{\text{ピラミッドの影}} \]

これはAA相似だ。太陽光線は平行だから、自分の影を作る三角形と、ピラミッドの影を作る三角形は相似になる。

\[ \frac{180}{200} = \frac{x}{160} \] \[ x = 180 \times \frac{160}{200} = 144 \text{ m} \]

タレスは定規1本とロープだけで、140メートル級のピラミッドの高さを言い当てた。

これが人類が「相似」を実用的に使った最古の記録の一つ。2600年経った今でも、測量や地図作りの基本原理は変わっていない。タレスは「最初の科学者」とも呼ばれている。

※タレスのピラミッド測量については、古代ギリシャの歴史家プルタルコスや哲学者ディオゲネス・ラエルティオスが記録を残しています。
