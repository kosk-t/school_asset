1. テーマの定義

気温が「-5℃」のとき、どれくらい寒いかイメージできる？借金が「-1000円」あるとき、あといくら返せばいい？正負の数は、「0より大きい」「0より小さい」を区別して、反対向きの量を扱うための数です。

このページでは、正負の数の計算ルールを学びます：

  • 正負の数の意味：符号・絶対値を理解する
  • 加法・減法：正負の数の足し算・引き算
  • 乗法・除法：正負の数の掛け算・割り算
  • 四則混合計算：累乗・かっこを含む計算

2. 覚えるルール

【最重要】符号の掛け算ルール：
同符号 → プラス（＋）、異符号 → マイナス（−）
符号の掛け算ルール（割り算も同じ）
📌 正の数・負の数・絶対値
  • 正の数：0より大きい数（+3, +7.5 など）。＋は省略できる
  • 負の数：0より小さい数（-2, -0.5 など）。−は省略できない
  • 絶対値：数直線で0からの距離。記号は \(|a|\)（「絶対値 a」と読む）
\[ |+5| = 5, \quad |-5| = 5, \quad |0| = 0 \]

→ だから結果は： 絶対値は常に0以上の数になる

💡 考え方を見る

絶対値は「符号を取り去った数」と考えてもOK。正の数はそのまま、負の数は符号を取るだけ。

📌 加法（足し算）のルール → 例1
パターン 計算方法 例
同符号 絶対値を足して、共通の符号をつける \((+3) + (+5) = +8\)
\((-3) + (-5) = -8\)
異符号 絶対値の差を求め、絶対値が大きい方の符号をつける \((+7) + (-3) = +4\)
\((-7) + (+3) = -4\)
📌 減法（引き算）のルール → 例3
\[ a - b = a + (-b) \]

「引く」は「反対符号を足す」に変換できる！

→ だから結果は： 引き算は全て足し算に変えて計算できる

💡 考え方を見る

「5から3を引く」は「5に-3を足す」と同じ結果になる。数直線で考えると、「右に3進む」の反対は「左に3進む」＝「-3を足す」。

📌 乗法（掛け算）のルール → 例5
  1. 符号を決める：同符号→＋、異符号→−
  2. 絶対値同士を掛ける
\[ (+3) \times (-4) = -12, \quad (-3) \times (-4) = +12 \]
📌 除法（割り算）のルール → 例7

符号のルールは乗法と同じ！

\[ (+12) \div (-4) = -3, \quad (-12) \div (-4) = +3 \]
📌 累乗のルール → 例9
  • \((-a)^2 = (-a) \times (-a) = +a^2\)（偶数乗は＋）
  • \((-a)^3 = (-a) \times (-a) \times (-a) = -a^3\)（奇数乗は−）
\[ (-2)^2 = 4, \quad (-2)^3 = -8, \quad -2^2 = -4 \]

⚠️ 注意： \((-2)^2\) と \(-2^2\) は違う！かっこの有無で意味が変わる

📌 四則混合計算の優先順位
  1. かっこの中を先に計算
  2. 累乗を計算
  3. 乗法・除法を左から順に
  4. 加法・減法を左から順に

3. 例題と解き方

符号のルールを使って、実際に計算してみましょう。

▶ 加法（足し算）

例1同符号の加法

\((-4) + (-7)\) を計算せよ。

解き方：

  1. 両方マイナス → 結果もマイナス
  2. 絶対値を足す：\(4 + 7 = 11\)
  3. 符号をつける
\[ (-4) + (-7) = -11 \]
例2異符号の加法

\((+8) + (-3)\) を計算せよ。

解き方：

  1. 符号が違う → 絶対値の差を求める
  2. 絶対値の差：\(8 - 3 = 5\)
  3. 絶対値が大きい方（+8）の符号をつける → プラス
\[ (+8) + (-3) = +5 \]

▶ 減法（引き算）

例3減法の基本

\((+5) - (+8)\) を計算せよ。

解き方：

  1. 「引く」を「反対符号を足す」に変換：\((+5) + (-8)\)
  2. 異符号の加法として計算：\(|8| - |5| = 3\)
  3. 絶対値が大きい方（-8）の符号 → マイナス
\[ (+5) - (+8) = (+5) + (-8) = -3 \]
例4負の数を引く

\((-2) - (-6)\) を計算せよ。

解き方：

  1. 「-(-6)」を「+(+6)」に変換：\((-2) + (+6)\)
  2. 異符号の加法：\(|6| - |2| = 4\)
  3. 絶対値が大きい方（+6）の符号 → プラス
\[ (-2) - (-6) = (-2) + (+6) = +4 \]

▶ 乗法（掛け算）

例5異符号の乗法

\((+6) \times (-5)\) を計算せよ。

解き方：

  1. 異符号（＋と−）→ 結果はマイナス
  2. 絶対値を掛ける：\(6 \times 5 = 30\)
\[ (+6) \times (-5) = -30 \]
例63つ以上の乗法

\((-2) \times (-3) \times (-4)\) を計算せよ。

解き方：

  1. 負の数が3つ（奇数個）→ 結果はマイナス
  2. 絶対値を掛ける：\(2 \times 3 \times 4 = 24\)
\[ (-2) \times (-3) \times (-4) = -24 \]

コツ： 負の数の個数を数える！偶数個→＋、奇数個→−

▶ 除法（割り算）

例7除法の基本

\((-18) \div (+3)\) を計算せよ。

解き方：

  1. 異符号（−と＋）→ 結果はマイナス
  2. 絶対値を割る：\(18 \div 3 = 6\)
\[ (-18) \div (+3) = -6 \]
例8分数を含む除法

\((-6) \div \left(-\frac{2}{3}\right)\) を計算せよ。

解き方：

  1. 同符号（−と−）→ 結果はプラス
  2. 逆数を掛ける：\(6 \times \frac{3}{2} = 9\)
\[ (-6) \div \left(-\frac{2}{3}\right) = (+6) \times \frac{3}{2} = +9 \]

▶ 累乗

例9累乗の計算

\((-3)^2\) と \(-3^2\) をそれぞれ計算せよ。

解き方：

  1. \((-3)^2 = (-3) \times (-3) = +9\)　←かっこ全体を2乗
  2. \(-3^2 = -(3 \times 3) = -9\)　←3だけを2乗してから−をつける
\[ (-3)^2 = 9, \quad -3^2 = -9 \]

▶ 四則混合

例10四則混合計算

\((-2)^3 + 12 \div (-4) \times 3\) を計算せよ。

解き方：

  1. 累乗を先に：\((-2)^3 = -8\)
  2. 乗除を左から：\(12 \div (-4) = -3\)、\((-3) \times 3 = -9\)
  3. 加減：\((-8) + (-9) = -17\)
\[ (-2)^3 + 12 \div (-4) \times 3 = -8 + (-9) = -17 \]
例11かっこを含む計算

\(5 - 3 \times (2 - 7)\) を計算せよ。

解き方：

  1. かっこの中を先に：\(2 - 7 = -5\)
  2. 乗法：\(3 \times (-5) = -15\)
  3. 減法：\(5 - (-15) = 5 + 15 = 20\)
\[ 5 - 3 \times (2 - 7) = 5 - 3 \times (-5) = 5 + 15 = 20 \]

4. 簡単にするコツ

✏️ コツ1：符号は先に決める
掛け算・割り算では、まず符号を決めてから絶対値を計算する。計算の途中で符号を忘れにくくなる。
\[ (-4) \times (+7) \rightarrow \text{異符号だから−} \rightarrow 4 \times 7 = 28 \rightarrow \textbf{-28} \]
✏️ コツ2：引き算は足し算に変換
引き算で混乱したら、「ひく」を「反対符号を足す」に置き換える。
\[ 3 - 8 = 3 + (-8) = -5 \]
✏️ コツ3：負の数の個数を数える
掛け算・割り算が連続するとき、負の数の個数で最終的な符号が決まる。
  • 偶数個 → プラス
  • 奇数個 → マイナス
✏️ コツ4：0を含む計算
  • 何を掛けても0：\((-5) \times 0 = 0\)
  • 0を何で割っても0：\(0 \div (-3) = 0\)
  • 0で割ることはできない（定義されない）

5. よくあるつまづき

⚠️ つまづき1：\((-a)^2\) と \(-a^2\) の混同

ダメな例： \(-3^2 = 9\) と計算してしまう

対策： かっこの有無を確認！

  • \((-3)^2 = (-3) \times (-3) = 9\)　←かっこ全体を2乗
  • \(-3^2 = -(3^2) = -9\)　←3だけを2乗
⚠️ つまづき2：引き算の符号変換ミス

ダメな例： \(5 - (-3)\) を \(5 + (-3) = 2\) と計算

対策： 「−(−)」は「＋」に変わる！

\[ 5 - (-3) = 5 + (+3) = 8 \]
⚠️ つまづき3：計算順序のミス

ダメな例： \(2 + 3 \times 4\) を \((2 + 3) \times 4 = 20\) と計算

対策： 掛け算・割り算を先に！

\[ 2 + 3 \times 4 = 2 + 12 = 14 \]
⚠️ つまづき4：符号の数え忘れ

ダメな例： \((-2) \times (-3) \times (-1)\) を \(+6\) と計算

対策： 負の数は3個（奇数）だからマイナス！

\[ (-2) \times (-3) \times (-1) = -6 \]

6. 仕上げチェックリスト

同符号の加法ができる（絶対値を足す）
異符号の加法ができる（絶対値の差を取る）
引き算を足し算に変換できる
乗法・除法の符号を正しく判定できる
\((-a)^n\) と \(-a^n\) の違いがわかる

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の計算をせよ。

(1) \((-8) + (+3)\)

(2) \((-5) - (-9)\)

(3) \((-4) \times (-6)\)

(4) \((+20) \div (-5)\)

答えを見る

(1) \(-5\)　（異符号：\(8-3=5\)、絶対値が大きい方の符号−）

(2) \(+4\)　（\(-5 + 9 = 4\)）

(3) \(+24\)　（同符号→＋）

(4) \(-4\)　（異符号→−）

練習2

次の計算をせよ。

(1) \((-2)^4\)

(2) \(-2^4\)

(3) \((-3)^2 - 4 \times (-2) + 6 \div (-3)\)

答えを見る

(1) \(+16\)　（偶数乗→プラス）

(2) \(-16\)　（2だけを4乗して−をつける）

(3) \(9 + 8 - 2 = 15\)

解説：\((-3)^2 = 9\)、\(4 \times (-2) = -8\) なので引くと+8、\(6 \div (-3) = -2\)

📌 まとめ：「同符号はプラス、異符号はマイナス」さえ覚えれば、あとは絶対値の計算だけ！

おまけ：歴史ストーリー「悪魔の数」

📖 マイナス×マイナスは悪魔の仕業？

1500年代のヨーロッパ。数学者たちは、ある計算結果に頭を抱えていた。

\[ (-3) \times (-2) = +6 \]

「マイナスとマイナスを掛けると、なぜプラスになるんだ？」

当時の学者たちは、この結果を「不自然で、直感に反する」と考えた。一部の数学者は「そんな計算は悪魔の仕業だ！」と本気で拒絶。負の数そのものを「偽物の数」「ありえない数」と呼んで、使うことを避けようとした。

イギリスの数学者フランシス・マセレスは、1759年にこう書いている：

「負の数は混乱を招く。数学の方程式から追放すべきだ」

ところが、インドや中国では古くから負の数を「借金」や「損失」として普通に使っていた。紀元前から！

ヨーロッパが負の数を完全に受け入れたのは、なんと19世紀になってから。約300年も抵抗したことになる。

結論：「0より小さい数がある」という事実を認めるのに、人類は数百年かかった
