1. テーマの定義

\((x + 3)(x + 5)\) を展開するとき、毎回分配法則で計算するのは面倒。でも「乗法公式」を使えば、一瞬で答えが出せる！式の展開を素早く正確に行うための「ショートカット」を身につけよう。

このページでは、乗法公式を学びます：

  • 和と差の積：\((a + b)(a - b) = a^2 - b^2\)
  • 和の平方：\((a + b)^2 = a^2 + 2ab + b^2\)
  • 差の平方：\((a - b)^2 = a^2 - 2ab + b^2\)
  • 一般の展開：\((x + a)(x + b) = x^2 + (a+b)x + ab\)

2. 覚えるルール

【最重要】4つの乗法公式：
\((a + b)(a - b) = a^2 - b^2\)　←和と差の積
\((a + b)^2 = a^2 + 2ab + b^2\)　←和の平方
\((a - b)^2 = a^2 - 2ab + b^2\)　←差の平方
\((x + a)(x + b) = x^2 + (a+b)x + ab\)　←一般形
和と差の積のイメージ
📌 公式1：和と差の積 → 例1
\[ (a + b)(a - b) = a^2 - b^2 \]

同じ数の「＋と−」をかけると、2乗の差になる。

✅ メリット： 真ん中の項（\(ab\)）が消える！計算がシンプル。

💡 考え方を見る

分配法則で展開すると：
\((a+b)(a-b) = a^2 - ab + ab - b^2 = a^2 - b^2\)
\(-ab\) と \(+ab\) が打ち消し合うから、\(a^2 - b^2\) だけが残る。

📌 公式2：和の平方 → 例3
\[ (a + b)^2 = a^2 + 2ab + b^2 \]

「たす」の2乗は、真ん中にプラス2abが入る。

語呂合わせ： 「エー2乗、ニーエービー、ビー2乗」

💡 考え方を見る

\((a+b)^2 = (a+b)(a+b)\)
分配法則で \(= a^2 + ab + ab + b^2 = a^2 + 2ab + b^2\)

📌 公式3：差の平方 → 例4
\[ (a - b)^2 = a^2 - 2ab + b^2 \]

「ひく」の2乗は、真ん中にマイナス2abが入る。

⚠️ 注意： 最後の項は \(+b^2\)（マイナスではない！）

📌 公式4：(x + a)(x + b) の展開 → 例5
\[ (x + a)(x + b) = x^2 + (a + b)x + ab \]
  • \(x^2\) の係数は常に1
  • \(x\) の係数は \(a + b\)（足し算）
  • 定数項は \(ab\)（掛け算）

語呂合わせ： 「x2乗、足してx、かける」

公式4の展開パターン
📌 分配法則（基本）

乗法公式が使えないときは、分配法則で展開する。

\[ (a + b)(c + d) = ac + ad + bc + bd \]

すべての項同士を掛けて足す（たすき掛け）。

3. 例題と解き方

乗法公式を使って、式を素早く展開してみましょう。

▶ 和と差の積

例1和と差の積の基本

\((x + 5)(x - 5)\) を展開せよ。

解き方：

  1. 公式 \((a + b)(a - b) = a^2 - b^2\) を使う
  2. \(a = x, b = 5\) と考える
  3. \(x^2 - 5^2 = x^2 - 25\)
\[ (x + 5)(x - 5) = x^2 - 25 \]
例2係数がある場合

\((3x + 2)(3x - 2)\) を展開せよ。

解き方：

  1. \(a = 3x, b = 2\) と考える
  2. \((3x)^2 - 2^2 = 9x^2 - 4\)
\[ (3x + 2)(3x - 2) = 9x^2 - 4 \]

▶ 和の平方・差の平方

例3和の平方

\((x + 4)^2\) を展開せよ。

解き方：

  1. 公式 \((a + b)^2 = a^2 + 2ab + b^2\) を使う
  2. \(a = x, b = 4\)
  3. \(x^2 + 2 \cdot x \cdot 4 + 4^2 = x^2 + 8x + 16\)
\[ (x + 4)^2 = x^2 + 8x + 16 \]
例4差の平方

\((x - 6)^2\) を展開せよ。

解き方：

  1. 公式 \((a - b)^2 = a^2 - 2ab + b^2\) を使う
  2. \(a = x, b = 6\)
  3. \(x^2 - 2 \cdot x \cdot 6 + 6^2 = x^2 - 12x + 36\)
\[ (x - 6)^2 = x^2 - 12x + 36 \]

注意： 最後は \(+36\) であって \(-36\) ではない！

▶ 一般の展開

例5(x + a)(x + b) の展開

\((x + 3)(x + 7)\) を展開せよ。

解き方：

  1. 公式 \((x + a)(x + b) = x^2 + (a+b)x + ab\)
  2. \(a = 3, b = 7\)
  3. \(a + b = 10\)、\(ab = 21\)
  4. \(x^2 + 10x + 21\)
\[ (x + 3)(x + 7) = x^2 + 10x + 21 \]
例6負の数を含む場合

\((x + 5)(x - 2)\) を展開せよ。

解き方：

  1. \(a = 5, b = -2\) と考える
  2. \(a + b = 5 + (-2) = 3\)
  3. \(ab = 5 \times (-2) = -10\)
  4. \(x^2 + 3x - 10\)
\[ (x + 5)(x - 2) = x^2 + 3x - 10 \]
例7両方マイナスの場合

\((x - 4)(x - 3)\) を展開せよ。

解き方：

  1. \(a = -4, b = -3\) と考える
  2. \(a + b = -4 + (-3) = -7\)
  3. \(ab = (-4) \times (-3) = +12\)
  4. \(x^2 - 7x + 12\)
\[ (x - 4)(x - 3) = x^2 - 7x + 12 \]

▶ 応用

例8数の計算への利用

\(102 \times 98\) を暗算で計算せよ。

解き方：

  1. \(102 = 100 + 2\)、\(98 = 100 - 2\)
  2. 和と差の積：\((100 + 2)(100 - 2) = 100^2 - 2^2\)
  3. \(= 10000 - 4 = 9996\)
\[ 102 \times 98 = (100 + 2)(100 - 2) = 10000 - 4 = 9996 \]
例9複雑な式の展開

\((2x + 3)^2\) を展開せよ。

解き方：

  1. \(a = 2x, b = 3\) と考える
  2. \((2x)^2 + 2 \cdot 2x \cdot 3 + 3^2\)
  3. \(= 4x^2 + 12x + 9\)
\[ (2x + 3)^2 = 4x^2 + 12x + 9 \]

4. 簡単にするコツ

✏️ コツ1：公式の形を見分ける
  • \((a + b)(a - b)\) → 和と差の積（同じ数の＋と−）
  • \((a + b)^2\) または \((a - b)^2\) → 平方公式
  • \((x + a)(x + b)\) → 一般の展開
✏️ コツ2：2abの計算を忘れない
平方公式で最もミスしやすいのが、真ん中の項「2ab」。
「ニバイ、エー、ビー」と声に出して確認しよう。
✏️ コツ3：数の計算に応用
  • \(51 \times 49 = (50+1)(50-1) = 2500 - 1 = 2499\)
  • \(99^2 = (100-1)^2 = 10000 - 200 + 1 = 9801\)
✏️ コツ4：符号を先に確認
展開後の符号は次のパターン：
式の形展開後
\((x+a)(x+b)\)（両方＋）\(x^2 + (正)x + (正)\)
\((x-a)(x-b)\)（両方−）\(x^2 - (正)x + (正)\)
\((x+a)(x-b)\)（＋と−）符号は値による

5. よくあるつまづき

⚠️ つまづき1：\((a-b)^2\) の最後を \(-b^2\) にする

ダメな例： \((x-3)^2 = x^2 - 6x - 9\)

対策： \((-3)^2 = 9\) なので、最後は必ずプラス！

\[ (x - 3)^2 = x^2 - 6x + 9 \]
⚠️ つまづき2：2abを忘れる

ダメな例： \((x+4)^2 = x^2 + 16\)

対策： 真ん中の \(2ab = 2 \times x \times 4 = 8x\) を忘れない！

\[ (x + 4)^2 = x^2 + 8x + 16 \]
⚠️ つまづき3：係数を2乗し忘れる

ダメな例： \((2x+3)^2 = 2x^2 + 12x + 9\)

対策： \((2x)^2 = 4x^2\) であって、\(2x^2\) ではない！

\[ (2x + 3)^2 = 4x^2 + 12x + 9 \]
⚠️ つまづき4：和と差の積と平方を混同

ダメな例： \((x+3)(x-3)\) を \(x^2 + 9\) や \(x^2 - 6x + 9\) にする

対策： \((+3)(-3)\) は和と差の積なので \(x^2 - 9\)！

6. 仕上げチェックリスト

\((a+b)(a-b) = a^2 - b^2\) が使える
\((a+b)^2 = a^2 + 2ab + b^2\) が使える
\((a-b)^2 = a^2 - 2ab + b^2\) が使える
\((x+a)(x+b)\) の展開ができる
係数がある式（\((2x+3)^2\) など）も展開できる

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の式を展開せよ。

(1) \((x + 7)(x - 7)\)

(2) \((x + 5)^2\)

(3) \((x - 8)^2\)

答えを見る

(1) \(x^2 - 49\)　（和と差の積）

(2) \(x^2 + 10x + 25\)　（和の平方）

(3) \(x^2 - 16x + 64\)　（差の平方、最後は+64）

練習2

次の式を展開せよ。

(1) \((x + 2)(x + 9)\)

(2) \((x - 3)(x + 7)\)

(3) \((x - 5)(x - 4)\)

答えを見る

(1) \(x^2 + 11x + 18\)　（2+9=11、2×9=18）

(2) \(x^2 + 4x - 21\)　（-3+7=4、-3×7=-21）

(3) \(x^2 - 9x + 20\)　（-5-4=-9、-5×-4=20）

練習3

次の式を展開せよ。

(1) \((3x - 4)^2\)

(2) \((5x + 2)(5x - 2)\)

答えを見る

(1) \(9x^2 - 24x + 16\)　（\((3x)^2=9x^2\)、\(2 \times 3x \times 4 = 24x\)）

(2) \(25x^2 - 4\)　（\((5x)^2 - 2^2\)）

📌 まとめ：乗法公式は「和と差→2乗の差」「平方→2abを忘れずに」「一般形→足してx、かける」！

おまけ：数学雑学「面積で公式を証明」

📐 正方形で乗法公式を「見える化」する

乗法公式は丸暗記するもの？いいえ、実は図形の面積で証明できます！

【和の平方】 \((a + b)^2 = a^2 + 2ab + b^2\)

一辺が \(a + b\) の正方形を考えてみましょう。

正方形の面積は \((a + b)^2\)。これを4つの部分に分けると...

  • 左上：\(a \times a = a^2\)（青）
  • 右上：\(a \times b = ab\)（オレンジ）
  • 左下：\(a \times b = ab\)（オレンジ）
  • 右下：\(b \times b = b^2\)（緑）

合計すると \(a^2 + ab + ab + b^2 = a^2 + 2ab + b^2\)。証明完了！

【和と差の積】 \((a + b)(a - b) = a^2 - b^2\) も図で証明できます。

長方形の面積を「追加して引く」テクニックで考えると、真ん中の \(ab\) が打ち消し合って、\(a^2 - b^2\) だけが残ることがわかります。

結論：乗法公式は「面積」で理解すれば、忘れても自分で導き出せる！
