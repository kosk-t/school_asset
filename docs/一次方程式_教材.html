1. テーマの定義

「ある数に5を足したら12になった。ある数はいくつ？」——こんな問題を解くとき、頭の中で計算するよりも、式を立てて解いた方が確実で速い。それが方程式だ。

このページでは、一次方程式の解き方を学びます：

  • 等式の性質：天秤のように左右のバランスを保つルール
  • 移項：項を反対側に移動させるテクニック
  • 一次方程式の解き方：\(x\)の値を求める手順
  • 検算：答えが正しいか確かめる方法

2. 覚えるルール

【最重要】等式の性質：両辺に同じ操作をしても等式は成り立つ
両方の皿から同じ数だけ取っても、天秤はつりあったまま
📌 等式の性質（4つのルール） → 例1
操作 式で書くと イメージ
両辺に同じ数を足す \(A = B \Rightarrow A + C = B + C\) 両方の皿に同じ重りを置く
両辺から同じ数を引く \(A = B \Rightarrow A - C = B - C\) 両方の皿から同じ重りを取る
両辺に同じ数をかける \(A = B \Rightarrow A \times C = B \times C\) 両方の皿の中身を同じ倍にする
両辺を同じ数で割る \(A = B \Rightarrow A \div C = B \div C\) 両方の皿の中身を同じ等分にする

→ だから結果は： 「\(=\)」の左右に同じ操作をすれば、いつでも等式は成り立つ。これを使って\(x\)を求めていく。

💡 考え方を見る

天秤をイメージしよう。左右の皿がつりあっているとき（\(=\)の状態）、両方の皿に同じ重りを足しても、同じ重りを取っても、つりあいは崩れない。これが等式の性質の正体だ。

📌 移項のルール → 例2

項を「\(=\)」の反対側に移動させることを移項という。

\[ x + 3 = 7 \quad \Rightarrow \quad x = 7 - 3 \]

移項すると符号が変わる！

  • \(+3\) を移項すると \(-3\) になる
  • \(-5\) を移項すると \(+5\) になる

✅ メリット： 「両辺から3を引く」と考えるより、「+3を移項して-3にする」と考える方が速い。

💡 考え方を見る

移項は「両辺から同じ数を引く」操作の省略形だ。\(x + 3 = 7\) の両辺から3を引くと \(x + 3 - 3 = 7 - 3\) → \(x = 4\)。毎回これを書くのは面倒なので、「+3を右辺に移項して-3」と一発で書く。

📌 一次方程式の解き方の基本ステップ → 例3
  1. カッコを外す（分配法則）
  2. \(x\)の項を左辺に、数の項を右辺に移項
  3. 両辺をまとめる
  4. \(x\)の係数で両辺を割る
  5. 検算する（答えを元の式に代入）

→ ゴール： 「\(x = \)（数）」の形にする！

📌 分数・小数を含む方程式 → 例5

最初に分母を払う（両辺に分母の最小公倍数をかける）

\[ \frac{x}{2} + \frac{x}{3} = 5 \quad \xrightarrow{\text{両辺に6をかける}} \quad 3x + 2x = 30 \]

小数は10倍、100倍して整数にする

\[ 0.3x + 1.2 = 2.1 \quad \xrightarrow{\text{両辺に10をかける}} \quad 3x + 12 = 21 \]

✅ メリット： 分数や小数のまま計算するより、整数にした方がミスが減る！

3. 例題と解き方

基本から順番に解いていこう。最後の検算まで忘れずに！

▶ 基本の方程式

例1等式の性質で解く

\(x + 5 = 12\) を解け。

解き方：

  1. 両辺から5を引く
  2. \(x + 5 - 5 = 12 - 5\)
  3. \(x = 7\)
\[ x + 5 = 12 \quad \Rightarrow \quad x = 12 - 5 = 7 \]

検算： \(7 + 5 = 12\) ✓

例2移項を使って解く

\(3x - 4 = 11\) を解け。

解き方：

  1. \(-4\) を右辺に移項（符号を変える）
  2. \(3x = 11 + 4 = 15\)
  3. 両辺を3で割る
  4. \(x = 5\)
\[ 3x - 4 = 11 \quad \Rightarrow \quad 3x = 15 \quad \Rightarrow \quad x = 5 \]

検算： \(3 \times 5 - 4 = 15 - 4 = 11\) ✓

▶ カッコを含む方程式

例3カッコを外して解く

\(2(x + 3) = 5x - 6\) を解け。

解き方：

  1. カッコを外す：\(2x + 6 = 5x - 6\)
  2. \(x\)の項を左辺に、数を右辺に移項
  3. \(2x - 5x = -6 - 6\)
  4. \(-3x = -12\)
  5. 両辺を\(-3\)で割る：\(x = 4\)
\[ 2(x + 3) = 5x - 6 \quad \Rightarrow \quad 2x + 6 = 5x - 6 \quad \Rightarrow \quad -3x = -12 \quad \Rightarrow \quad x = 4 \]

検算： 左辺 \(= 2(4 + 3) = 2 \times 7 = 14\)、右辺 \(= 5 \times 4 - 6 = 14\) ✓

例4両辺にxがある場合

\(5x + 2 = 3x + 10\) を解け。

解き方：

  1. 右辺の\(3x\)を左辺に移項
  2. 左辺の\(+2\)を右辺に移項
  3. \(5x - 3x = 10 - 2\)
  4. \(2x = 8\)
  5. \(x = 4\)
\[ 5x + 2 = 3x + 10 \quad \Rightarrow \quad 2x = 8 \quad \Rightarrow \quad x = 4 \]

検算： 左辺 \(= 5 \times 4 + 2 = 22\)、右辺 \(= 3 \times 4 + 10 = 22\) ✓

▶ 分数・小数を含む方程式

例5分数を含む方程式

\(\displaystyle \frac{x}{2} - \frac{x}{3} = 2\) を解け。

解き方：

  1. 分母2と3の最小公倍数は6
  2. 両辺に6をかけて分母を払う
  3. \(3x - 2x = 12\)
  4. \(x = 12\)
\[ \frac{x}{2} - \frac{x}{3} = 2 \quad \xrightarrow{\times 6} \quad 3x - 2x = 12 \quad \Rightarrow \quad x = 12 \]

検算： \(\displaystyle \frac{12}{2} - \frac{12}{3} = 6 - 4 = 2\) ✓

例6小数を含む方程式

\(0.2x + 0.5 = 1.3\) を解け。

解き方：

  1. 小数第一位まであるので、両辺に10をかける
  2. \(2x + 5 = 13\)
  3. \(2x = 8\)
  4. \(x = 4\)
\[ 0.2x + 0.5 = 1.3 \quad \xrightarrow{\times 10} \quad 2x + 5 = 13 \quad \Rightarrow \quad x = 4 \]

検算： \(0.2 \times 4 + 0.5 = 0.8 + 0.5 = 1.3\) ✓

4. 簡単にするコツ

✏️ コツ1：xは左辺、数は右辺に集める
\(x\)の項をすべて左辺に、数だけの項をすべて右辺に移項する。最終形「\(x = \)（数）」に近づく！
✏️ コツ2：先に分母を払う
分数が出てきたら、まず最初に「両辺に分母の最小公倍数をかける」。分数のまま計算するより圧倒的に楽。
✏️ コツ3：マイナスの係数は両辺に-1をかける
\(-2x = 8\) のように\(x\)の係数がマイナスになったら、両辺に\(-1\)をかけて \(2x = -8\) にすると見やすい。または最後に符号を意識して割ればOK。
✏️ コツ4：検算は必ずやる
答えを元の式に代入して、左辺と右辺が等しくなるか確認。これで計算ミスを防げる。テストでは必ず検算！

5. よくあるつまづき

⚠️ つまづき1：移項で符号を変え忘れる

ダメな例： \(x + 3 = 7\) → \(x = 7 + 3 = 10\) ✗

対策： 移項したら必ず符号を逆にする！ \(+3\) を移項したら \(-3\) になる。

\[ x + 3 = 7 \quad \Rightarrow \quad x = 7 - 3 = 4 \quad \text{○} \]
⚠️ つまづき2：カッコを外すときの分配を忘れる

ダメな例： \(2(x + 3) = 2x + 3\) ✗

対策： カッコの前の数は、中のすべての項にかける！

\[ 2(x + 3) = 2 \times x + 2 \times 3 = 2x + 6 \quad \text{○} \]
⚠️ つまづき3：マイナス×マイナスを間違える

ダメな例： \(-3(x - 2) = -3x - 6\) ✗

対策： \(-3 \times (-2) = +6\) マイナス×マイナスはプラス！

\[ -3(x - 2) = -3x + 6 \quad \text{○} \]
⚠️ つまづき4：分数の一部だけに数をかける

ダメな例： \(\displaystyle \frac{x + 2}{3} = 4\) の両辺に3をかけて \(\displaystyle \frac{x + 6}{3} = 12\) ✗

対策： 分数全体に3をかける。分子全体が3倍になって分母が消える。

\[ \frac{x + 2}{3} = 4 \quad \Rightarrow \quad x + 2 = 12 \quad \text{○} \]

6. 仕上げチェックリスト

等式の性質4つを説明できる
移項するときに符号を変えることを理解している
カッコを正しく外せる（分配法則）
分数・小数を含む方程式で、最初に分母を払える
解いた後に検算する習慣がついている

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(4x - 7 = 2x + 5\) を解け。

答えを見る

解き方：

  1. \(2x\)を左辺に、\(-7\)を右辺に移項
  2. \(4x - 2x = 5 + 7\)
  3. \(2x = 12\)
  4. \(x = 6\)

検算： 左辺 \(= 4 \times 6 - 7 = 17\)、右辺 \(= 2 \times 6 + 5 = 17\) ✓

答え：\(x = 6\)

練習2

\(\displaystyle \frac{2x - 1}{3} = \frac{x + 2}{2}\) を解け。

答えを見る

解き方：

  1. 分母3と2の最小公倍数は6
  2. 両辺に6をかける：\(2(2x - 1) = 3(x + 2)\)
  3. カッコを外す：\(4x - 2 = 3x + 6\)
  4. \(3x\)を左辺に、\(-2\)を右辺に移項
  5. \(4x - 3x = 6 + 2\)
  6. \(x = 8\)

検算： 左辺 \(= \displaystyle \frac{2 \times 8 - 1}{3} = \frac{15}{3} = 5\)、右辺 \(= \displaystyle \frac{8 + 2}{2} = \frac{10}{2} = 5\) ✓

答え：\(x = 8\)

📌 まとめ：方程式は「天秤のバランス」。両辺に同じ操作をして、\(x = \)（数）の形にすれば解ける！

おまけ：数学雑学「ディオファントスの墓」

📜 墓石に刻まれた一次方程式

古代ギリシャの数学者ディオファントス（紀元後3世紀頃）。彼の墓石には、こんな謎かけが刻まれていたという：

「旅人よ、ここにディオファントスが眠る。
彼の人生の6分の1は少年時代、
12分の1の後にあごひげが生えた。
さらに7分の1後に結婚し、
5年後に息子が生まれた。
息子は父の半分の年齢で亡くなり、
父はその4年後に息子のもとへ旅立った。
さて、彼は何歳まで生きただろうか？」

ディオファントスの生涯を\(x\)年とすると......これ、一次方程式で解ける！

\[ \frac{x}{6} + \frac{x}{12} + \frac{x}{7} + 5 + \frac{x}{2} + 4 = x \]

分母6, 12, 7, 2の最小公倍数は84。両辺に84をかけると：

\[ 14x + 7x + 12x + 420 + 42x + 336 = 84x \]
\[ 75x + 756 = 84x \quad \Rightarrow \quad 756 = 9x \quad \Rightarrow \quad x = 84 \]

答え：84歳

ちなみに息子は42歳で亡くなったことになる。古代としては長寿だ。

「代数学の父」と呼ばれるディオファントス。彼の著書『算術』は後の数学に大きな影響を与え、フェルマーの最終定理もこの本の余白に書かれたメモから始まった。墓石の謎かけは後世の創作とも言われるが、一次方程式の良問として2000年近く語り継がれている。

※ディオファントスの生没年は正確には不明。紀元後200〜290年頃と推定されています。墓碑銘は『ギリシア詞華集』（10世紀頃の編纂）に収録されています（Wikipedia: ディオファントス）。
