1. テーマの定義

ボールを投げると、どんな軌道を描くでしょうか？噴水の水は、どんな形で飛んでいくでしょうか？——答えは「放物線」です。

このページでは、二次関数のグラフと性質を学びます：

  • 二次関数とは何か：\(y = ax^2 + bx + c\) の形の関数
  • グラフの形：放物線と呼ばれるなめらかな曲線
  • 対称軸と頂点：グラフの中心線と折り返し点
  • 最大値・最小値：グラフの最も高い（または低い）点

2. 覚えるルール

二次関数とは、\(x\) の2乗を含む関数です：

\[ y = ax^2 + bx + c \quad (a \neq 0) \]

→ \(a, b, c\) は定数。\(a = 0\) だと2乗がなくなり一次関数になるので、\(a \neq 0\) が条件。

🎯 最重要：二次関数のグラフは「放物線」、対称軸は \(x = -\frac{b}{2a}\)
二次関数のグラフは放物線（対称軸で左右対称）
📌 \(a\) の符号とグラフの向き → 例1
  • \(a > 0\)（正）：下に凸（∪型）、頂点が最小値
  • \(a < 0\)（負）：上に凸（∩型）、頂点が最大値

💡 なぜそうなる？ \(y = ax^2\) で \(x^2\) は常に正（または0）。\(a > 0\) なら \(y\) も正の方向（上）に大きくなり、\(a < 0\) なら負の方向（下）に小さくなる。だから \(a\) の符号でグラフの向きが決まる。

a > 0 なら下に凸、a < 0 なら上に凸
📌 対称軸の公式 → 例2

\(y = ax^2 + bx + c\) の対称軸は：

\[ x = -\frac{b}{2a} \]

→ グラフを左右に分ける縦の直線。この直線で折り返すと左右が重なる。

💡 なぜこの公式？ 平方完成すると \(y = a(x - p)^2 + q\) の形になり、\(x = p\) が対称軸。計算すると \(p = -\frac{b}{2a}\) になる（平方完成の公式から導ける）。

📌 頂点の座標 → 例3

方法1：対称軸から求める（おすすめ）

  1. 対称軸 \(x = -\frac{b}{2a}\) を計算
  2. その \(x\) を元の式に代入して \(y\) を計算
  3. → 頂点の座標は \((x, y)\)

方法2：平方完成で求める → 例3-2

\(y = a(x - p)^2 + q\) の形に変形できれば、頂点は \((p, q)\)

💡 なぜ頂点で最大・最小？ 放物線は対称軸を中心に左右対称。頂点から離れるほど \(y\) の値が変わるので、頂点が折り返し点（最大値か最小値）になる。

📌 最大値・最小値の求め方 → 例4 → 例6（応用）
  1. 頂点の座標 \((p, q)\) を求める
  2. \(a > 0\) なら下に凸 → 頂点で最小値 \(q\)（最大値なし）
  3. \(a < 0\) なら上に凸 → 頂点で最大値 \(q\)（最小値なし）

💡 「定義域」がある場合は？ 「\(x\) の範囲が \(a \leq x \leq b\)」のように制限があるときは、頂点だけでなく端点 \(x = a, x = b\) でも値を計算し、その中で最大・最小を判定する。

3. 例題と解き方

ルールを使って実際に解いてみましょう。

▶ グラフの向きを判定する

例1グラフの向きを判定する

次の二次関数のグラフは上に凸か、下に凸か判定せよ。

（1）\(y = 2x^2 + 3x - 1\)　　（2）\(y = -x^2 + 4x + 5\)

解き方：

  1. \(x^2\) の係数 \(a\) の符号を確認する
  2. \(a > 0\) なら下に凸、\(a < 0\) なら上に凸

答え：

（1）\(a = 2 > 0\) なので下に凸

（2）\(a = -1 < 0\) なので上に凸

▶ 対称軸を求める

例2対称軸を求める

\(y = x^2 - 4x + 3\) の対称軸の方程式を求めよ。

解き方：

  1. 係数を確認する → \(a = 1, b = -4, c = 3\)
  2. 対称軸の公式 \(x = -\frac{b}{2a}\) に代入
\[ x = -\frac{-4}{2 \cdot 1} = \frac{4}{2} = 2 \]

答え：\(x = 2\)

▶ 頂点を求める

例3頂点を求める（対称軸から）

同じ関数 \(y = x^2 - 4x + 3\) の頂点の座標を求めよ。

解き方：

  1. 対称軸 \(x = 2\) を求める（例2より）
  2. この \(x\) の値を元の式に代入して \(y\) を求める
\[ y = 2^2 - 4 \cdot 2 + 3 = 4 - 8 + 3 = -1 \]

答え：頂点 \((2, -1)\)

例3-2頂点を求める（平方完成で）

同じ関数 \(y = x^2 - 4x + 3\) を平方完成して頂点を求めよ。

解き方：

  1. \(x\) の係数（-4）を半分にして2乗 → \(\left(\frac{-4}{2}\right)^2 = (-2)^2 = 4\)
  2. この「4」を足して、同じ分だけ引く（式の値を変えないため）
  3. カッコでまとめて完成
\[ y = x^2 - 4x + 3 \]

↓ \(+4\) を足して、\(-4\) を引く（プラマイゼロ）

\[ y = x^2 - 4x \textcolor{blue}{+ 4} + 3 \textcolor{blue}{- 4} \]

↓ 前半3つが \((x-2)^2\) になる

\[ y = (x - 2)^2 + 3 - 4 \] \[ y = (x - 2)^2 - 1 \]

\(y = (x - 2)^2 - 1\) より、頂点は \((2, -1)\)

💡 ポイント：「足した分だけ引く」から式の値は変わらない。これで \((x - ◯)^2\) の形が作れる！

▶ 最大値・最小値を求める

例4最大値・最小値を求める

次の二次関数の最大値または最小値を求めよ。

（1）\(y = x^2 - 4x + 3\)　　（2）\(y = -2x^2 + 8x - 5\)

解き方：

  1. 頂点の座標を求める
  2. \(a\) の符号を確認して、最大値か最小値かを判定

答え：

（1）頂点 \((2, -1)\)、\(a = 1 > 0\) より下に凸 → 最小値 -1（\(x = 2\) のとき）

（2）平方完成すると \(y = -2(x - 2)^2 + 3\) より頂点 \((2, 3)\)、\(a = -2 < 0\) より上に凸 → 最大値 3（\(x = 2\) のとき）

例5定義域がある場合の最大最小

\(y = x^2 - 4x + 3\)（\(0 \leq x \leq 3\)）の最大値と最小値を求めよ。

解き方：

  1. 頂点 \((2, -1)\) が範囲内にあるか確認 → \(0 \leq 2 \leq 3\) で範囲内
  2. 頂点と端点（\(x = 0, 3\)）での値を比較
\[ x = 0: y = 0 - 0 + 3 = 3 \] \[ x = 2: y = -1 \quad \text{（頂点）} \] \[ x = 3: y = 9 - 12 + 3 = 0 \]

答え：最小値 -1（\(x = 2\) のとき）、最大値 3（\(x = 0\) のとき）

▶ 実生活への応用（放物運動）

例6コインを投げる問題

オリバーがオフィスビルの窓からコインを水平な地面に向かって投げた。コインの軌道は次の二次関数でモデル化される：

\[ y = 8 + 3x - 0.5x^2 \]

ここで \(y\) はコインの高さ（メートル）、\(x\) は水平距離（メートル）を表す。建物の壁は \(y\) 軸に沿っている。

コインの軌道（放物線）

(a) コインを投げた高さを求めよ。

解き方：

  1. 出発点 A は壁の位置、つまり \(x = 0\) の点
  2. \(x = 0\) を式に代入する
\[ y = 8 + 3(0) - 0.5(0)^2 = 8 \]

答え：8 メートル

(b) コインが達する最高点の高さを求めよ。

解き方：

  1. 式を整理：\(y = -0.5x^2 + 3x + 8\)（\(a = -0.5, b = 3, c = 8\)）
  2. \(a = -0.5 < 0\) なので上に凸 → 頂点が最大値
  3. 対称軸 \(x = -\frac{b}{2a} = -\frac{3}{2 \times (-0.5)} = -\frac{3}{-1} = 3\)
  4. \(x = 3\) を代入して最高点の高さを求める
\[ y = -0.5(3)^2 + 3(3) + 8 = -0.5 \times 9 + 9 + 8 = -4.5 + 17 = 12.5 \]

答え：12.5 メートル

(c) コインが地面に着く地点の、壁からの水平距離を求めよ。

解き方：

  1. 地面は \(y = 0\) の位置
  2. \(y = 0\) として方程式を解く
\[ 0 = -0.5x^2 + 3x + 8 \]

両辺を \(-0.5\) で割って整理：

\[ x^2 - 6x - 16 = 0 \]

因数分解または解の公式で解く：

\[ (x - 8)(x + 2) = 0 \] \[ x = 8 \quad \text{または} \quad x = -2 \]

\(x > 0\)（水平距離は正）なので：

答え：8 メートル

4. 簡単にするコツ

✏️ コツ1：まず \(a\) の符号を確認
\(a > 0\) か \(a < 0\) かで、グラフの向きと最大最小が決まる。最初に確認する習慣を。
✏️ コツ2：対称軸は「マイナスb割る2a」と覚える
\(x = -\frac{b}{2a}\) を何度も使うので、丸暗記しておく。
✏️ コツ3：平方完成で頂点を一発で求める
\(y = a(x - p)^2 + q\) の形にすれば、頂点は \((p, q)\)。対称軸から計算するより早い。
✏️ コツ4：定義域があるときは端点も確認
範囲が制限されているときは、頂点だけでなく両端の値も計算して比較する。
✏️ コツ5：グラフを簡単に描いてイメージする
対称軸・頂点・\(y\) 切片（\(x = 0\) のとき）の3点を打てば、だいたいの形が分かる。

5. よくあるつまづき

⚠️ つまづき1：\(a\) の符号を逆にする

ダメな例： \(a = -2\) なのに下に凸と答える

対策： \(a > 0\) は下、\(a < 0\) は上。マイナスは上に凸！

⚠️ つまづき2：対称軸の公式で符号ミス

ダメな例： \(b = -4\) のとき、\(x = -\frac{-4}{2}\) を \(-2\) とする

対策： マイナス×マイナスはプラス。\(-\frac{-4}{2} = \frac{4}{2} = 2\)

⚠️ つまづき3：頂点の \(y\) 座標を求め忘れる

ダメな例： 対称軸 \(x = 2\) を求めて、頂点を \((2, 0)\) とする

対策： 必ず \(x = 2\) を元の式に代入して \(y\) を計算！

⚠️ つまづき4：平方完成で定数項の調整ミス

ダメな例： \(x^2 - 4x + 3 = (x - 2)^2\) と変形する

対策： \((x - 2)^2 = x^2 - 4x + 4\) だから、\(+3\) と合わせて \((x - 2)^2 - 1\)

⚠️ つまづき5：定義域の端点を忘れる

ダメな例： 範囲が \(0 \leq x \leq 3\) なのに、頂点だけで最大最小を判定

対策： 頂点と \(x = 0, 3\) での値を比較して、本当の最大最小を決める

6. 仕上げチェックリスト

\(a\) の符号からグラフの向きを正しく判定したか
対称軸 \(x = -\frac{b}{2a}\) の計算で符号ミスをしていないか
頂点の \(y\) 座標を求めるとき、ちゃんと代入計算したか
最大値・最小値は \(a\) の符号から正しく判定したか

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(y = -x^2 + 6x - 5\) の対称軸と頂点を求めなさい。

答えを見る

対称軸：\(x = -\frac{6}{2 \cdot (-1)} = \frac{6}{2} = 3\)

\(y\) 座標：\(y = -3^2 + 6 \cdot 3 - 5 = -9 + 18 - 5 = 4\)

答え：対称軸 \(x = 3\)、頂点 \((3, 4)\)

練習2

\(y = 2x^2 - 8x + 10\) の最小値を求めなさい。

答えを見る

平方完成：\(y = 2(x^2 - 4x) + 10 = 2(x^2 - 4x + 4 - 4) + 10 = 2(x - 2)^2 - 8 + 10 = 2(x - 2)^2 + 2\)

頂点：\((2, 2)\)

\(a = 2 > 0\) より下に凸

答え：最小値 2（\(x = 2\) のとき）

📌 まとめ：対称軸は「−b÷2a」、頂点で折り返し、a の符号で向きが決まる！

おまけ：歴史ストーリー「ガリレオの大発見と大ピンチ」

📖 ガリレオの大発見と大ピンチ（1600年代イタリア）

1600年代のイタリア。ガリレオ・ガリレイは、当時の「常識」に真っ向から挑戦していた天才科学者だった。

当時の常識：「大砲の弾は、まっすぐ飛んでから急に落ちる」
ガリレオの主張：「いや、放物線を描いて飛ぶはずだ！」

当時の人々は「弾はまっすぐ飛び続け、力が尽きたら真下に落ちる」と信じていた。でもガリレオは実験と計算から、「弾道は放物線（二次関数のグラフ）だ！」と気づいた。

【ガリレオの発見】

斜め上に投げた物体の軌道は、水平方向の動きと垂直方向の動き（重力）を組み合わせると...

\[ y = (\text{初速による上昇}) - (\text{重力による落下}) \] \[ y = ax - bx^2 \quad \text{（二次関数！）} \]

これを証明するため、ガリレオは斜面で球を転がす実験を繰り返した。でも...

カトリック教会：「は？何を言ってるんだガリレオ。そもそもお前、地動説を唱えただろ？『地球が動く』なんて聖書に反する。放物線とかいう前に、まず考えを改めろ」
ガリレオ：「でも望遠鏡で木星の衛星が回ってるの見えたし...地球も回ってると思うんですけど」
教会：「...裁判な」

ガリレオは宗教裁判にかけられ、地動説を撤回させられた。彼の発見した放物線の理論も、すぐには認められなかった。

【でも、後の世界は...】

ガリレオの死後、ニュートンが「運動の法則」を確立し、放物線の理論が正しいことが完全に証明された。今では：

  • ミサイルの弾道計算
  • 野球のボールの軌道予測
  • ロケットの打ち上げ

すべて放物線（二次関数）を使って計算されている。

ガリレオの最期の言葉（伝説）：
裁判で地動説の撤回を強制された後、つぶやいた。
「それでも地球は回っている（Eppur si muove）」

教訓：正しい数学は、たとえ権力に潰されても、いつか必ず認められる。ガリレオが命がけで証明した放物線の理論を、今、君は教科書で普通に学んでいる。数学、時代を超えて最強。

※ガリレオの弾道が放物線であるという発見は『新科学対話』（1638年）に記されています。地動説を巡る裁判については、1633年の宗教裁判で有罪となりましたが、1992年にローマ教皇ヨハネ・パウロ2世によって名誉回復されました（ガリレオ・ガリレイ）。
