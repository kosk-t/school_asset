1. テーマの定義

「80点以上で合格」「予算は1万円以下」——日常でも「以上」「以下」「未満」といった不等号をよく使います。高校では、これを数式で扱う不等式を学びます。

このページでは、一次不等式を学びます：

  • 一次不等式の解き方：不等号の向きに注意しながら解く
  • 連立不等式：複数の不等式を同時に満たす範囲を求める
  • 絶対値を含む不等式：場合分けまたは定型パターンで解く

2. 覚えるルール

🎯 最重要：負の数を掛けたり割ったりすると、不等号の向きが逆になる！
負の数で割ると不等号の向きが逆になる
📌 ルール1：不等式の性質 → 例1

\(a < b\) のとき、以下が成り立つ：

\[ a + c < b + c, \quad a - c < b - c \] \[ ac < bc, \quad \frac{a}{c} < \frac{b}{c} \quad (c > 0 \text{ のとき}) \] \[ ac > bc, \quad \frac{a}{c} > \frac{b}{c} \quad (c < 0 \text{ のとき：向き逆転！}) \]

ポイント：足し算・引き算は向き変わらない。掛け算・割り算は負の数なら逆転。

📌 ルール2：一次不等式の解き方 → 例1・2

一次方程式と同じ手順で解く。ただし負の数で割るときは向き逆転。

  1. カッコを外す・分数を払う
  2. \(x\) の項を左辺に、定数項を右辺に移項
  3. \(x\) の係数で割る（負なら不等号逆転）

→ だから結果は： \(x > a\) や \(x \leq b\) の形になる

📌 ルール3：連立不等式 → 例3・4

複数の不等式を同時に満たす \(x\) の範囲を求める。

\[ \begin{cases} x > 2 \\ x < 5 \end{cases} \Rightarrow 2 < x < 5 \]

手順：

  1. 各不等式を個別に解く
  2. 数直線に図示する
  3. 共通部分（重なっている部分）を求める

✅ メリット： 数直線を使うと共通部分が一目で分かる

📌 ルール4：絶対値を含む不等式 → 例5・6
\[ |x| < a \Leftrightarrow -a < x < a \quad (a > 0) \] \[ |x| > a \Leftrightarrow x < -a \text{ または } x > a \quad (a > 0) \]

覚え方：

  • \(|x| < a\)：原点からの距離が \(a\) 未満 → 「挟まれる」形
  • \(|x| > a\)：原点からの距離が \(a\) より大きい → 「外側」に飛び出す
📌 ルール5：不等号の記号と意味
記号 意味 端点を含む？
\(<\) より小さい（未満） 含まない（○）
\(\leq\) 以下 含む（●）
\(>\) より大きい 含まない（○）
\(\geq\) 以上 含む（●）

3. 例題と解き方

ルールを使って実際に不等式を解いてみましょう。

▶ 一次不等式の基本

例1\(3x - 5 > 7\) を解く

解き方：

  1. \(-5\) を右辺に移項
  2. 3で割る（正の数なので向き変わらず）
\[ 3x - 5 > 7 \Rightarrow 3x > 12 \Rightarrow x > 4 \]
例2\(-2x + 3 \leq 9\) を解く

解き方：

  1. 3を右辺に移項
  2. \(-2\) で割る（負の数なので向き逆転！）
\[ -2x + 3 \leq 9 \Rightarrow -2x \leq 6 \Rightarrow x \geq -3 \]

▶ 連立不等式

例3\(\begin{cases} 2x - 1 > 3 \\ x + 4 \leq 10 \end{cases}\) を解く

解き方：

  1. 1つ目：\(2x - 1 > 3 \Rightarrow 2x > 4 \Rightarrow x > 2\)
  2. 2つ目：\(x + 4 \leq 10 \Rightarrow x \leq 6\)
  3. 共通部分を求める
\[ 2 < x \leq 6 \]
例4\(\begin{cases} x > 5 \\ x < 2 \end{cases}\) を解く

解き方：

  1. \(x > 5\) と \(x < 2\) を数直線に図示
  2. 共通部分がない！
\[ \text{解なし} \]

▶ 絶対値を含む不等式

例5\(|x - 2| < 3\) を解く

解き方：

  1. \(|A| < a \Leftrightarrow -a < A < a\) を使う
  2. \(-3 < x - 2 < 3\)
  3. 各辺に2を足す
\[ -3 < x - 2 < 3 \Rightarrow -1 < x < 5 \]
例6\(|2x + 1| \geq 5\) を解く

解き方：

  1. \(|A| \geq a \Leftrightarrow A \leq -a\) または \(A \geq a\) を使う
  2. \(2x + 1 \leq -5\) または \(2x + 1 \geq 5\)
  3. 各不等式を解く
\[ 2x + 1 \leq -5 \Rightarrow x \leq -3 \] \[ 2x + 1 \geq 5 \Rightarrow x \geq 2 \] \[ \therefore x \leq -3 \text{ または } x \geq 2 \]

▶ 分数・小数を含む不等式

例7\(\dfrac{x-1}{3} > \dfrac{2x+1}{4}\) を解く

解き方：

  1. 両辺に12（分母の最小公倍数）を掛ける
  2. 整理して解く
\[ 4(x-1) > 3(2x+1) \] \[ 4x - 4 > 6x + 3 \] \[ -2x > 7 \] \[ x < -\frac{7}{2} \]

4. 簡単にするコツ

✏️ コツ1：xの係数を正にしてから割る
\(-3x > 6\) を解くとき、「\(-3\) で割って向き逆転」と考えるより、先に両辺に \(-1\) を掛けて \(3x < -6\) にする方が間違えにくい。
✏️ コツ2：連立不等式は数直線を描く
2つ以上の不等式があるときは、必ず数直線を描いて共通部分を確認する。頭の中だけで考えると間違えにくい。
✏️ コツ3：絶対値は「挟む or 外側」で覚える
\(|x| < a\) は「挟まれる」（\(-a\) と \(a\) の間）。\(|x| > a\) は「外側に飛び出す」（\(-a\) より左、または \(a\) より右）。
✏️ コツ4：端点の確認を忘れない
\(<\) と \(\leq\) の違いは重要。「等号を含むか」を常に確認し、数直線では ○（含まない）と ●（含む）を使い分ける。

5. よくあるつまづき

⚠️ つまづき1：負の数で割って向き逆転を忘れる

ダメな例： \(-2x > 4\) を \(x > -2\) と解く

対策： 負の数で割ったら必ず不等号をひっくり返す。正解は \(x < -2\)。

⚠️ つまづき2：連立不等式で「または」と「かつ」を混同

ダメな例： \(\begin{cases} x > 2 \\ x < 5 \end{cases}\) を「\(x > 2\) または \(x < 5\)」と解釈

対策： 連立不等式は「かつ」（同時に満たす）。答えは \(2 < x < 5\)。

⚠️ つまづき3：絶対値不等式の向きを逆に覚える

ダメな例： \(|x| > 3\) を \(-3 < x < 3\) と解く

対策： \(|x| > a\) は「外側」。\(x < -3\) または \(x > 3\) が正解。

6. 仕上げチェックリスト

負の数で割ると不等号が逆転することを覚えている
連立不等式を数直線で解ける
\(|x| < a\) と \(|x| > a\) の違いを説明できる
\(<\) と \(\leq\) の違い（端点を含むか）を意識している
分数を含む不等式を分母を払って解ける

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(-4x + 5 > 1\) を解け。

答えを見る

\(-4x + 5 > 1 \Rightarrow -4x > -4 \Rightarrow x < 1\)

（負の数で割ったので向き逆転）

練習2

\(\begin{cases} 3x - 2 \geq 4 \\ 2x + 1 < 9 \end{cases}\) を解け。

答えを見る

1つ目：\(3x - 2 \geq 4 \Rightarrow 3x \geq 6 \Rightarrow x \geq 2\)

2つ目：\(2x + 1 < 9 \Rightarrow 2x < 8 \Rightarrow x < 4\)

共通部分：\(2 \leq x < 4\)

練習3

\(|x + 3| > 2\) を解け。

答えを見る

\(|A| > a \Leftrightarrow A < -a\) または \(A > a\)

\(x + 3 < -2\) または \(x + 3 > 2\)

\(x < -5\) または \(x > -1\)

📌 まとめ：負の数で割ったら不等号逆転！絶対値は「挟む or 外側」で覚える！

おまけ：数学と社会「不等式で最適解を見つける」

📖 第二次世界大戦と線形計画法

1940年代、第二次世界大戦中のアメリカ軍は深刻な問題を抱えていた。

「限られた資源で、どうやって効率よく物資を運ぶか？」

船の数、燃料、人員、時間——すべてに制限（不等式）がある。その制限の中で「最も効率的な方法」を見つけなければならない。

この問題を数学的に解決したのが、数学者ジョージ・ダンツィグ。彼が開発した「線形計画法」は、複数の不等式の条件を満たしながら、目的（コスト最小化や効率最大化）を達成する方法だった。

\[ \text{制約条件：} \begin{cases} x + y \leq 100 \\ 2x + y \leq 150 \\ x, y \geq 0 \end{cases} \] \[ \text{この範囲で「}3x + 2y\text{」を最大にする}(x, y)\text{を見つける} \]

戦後、この技術は企業の経営にも応用された。

  • 航空会社：パイロットのシフトを最適化
  • 物流会社：配送ルートの最短化
  • 工場：生産計画の効率化

今でもAmazonの倉庫、Googleの広告配信、Uberの配車システムなど、私たちの生活を支える裏側で不等式の数学が働いている。

高校の不等式は、実は世界を動かす数学の入り口だった。
