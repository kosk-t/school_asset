1. テーマの定義

円は古代から最も美しい図形とされ、建築・工学・天文学など様々な分野で使われています。座標平面で円を方程式として扱えるようになると、直線との交点や接線が計算だけで求められるようになります。

このページでは、円の方程式の扱い方を学びます：

  • 円の方程式（標準形）：中心と半径から円の式を作る
  • 円の方程式（一般形）：展開した形から中心と半径を読み取る
  • 円と直線の位置関係：交わる・接する・離れるを判定
  • 円の接線：円上の点・円外の点から接線を求める
  • 2円の位置関係：外接・内接・交わる・離れるを判定

2. 覚えるルール

【最重要】円の方程式（標準形）：中心 \((a, b)\)、半径 \(r\) の円は \[ (x - a)^2 + (y - b)^2 = r^2 \]
円の標準形：中心(a, b)から半径rの点の集まり
考え方を見る

円とは「中心からの距離が一定（= 半径）の点の集まり」です。中心 \((a, b)\) から点 \((x, y)\) までの距離が \(r\) ということは、三平方の定理（距離公式）から：

\[ \sqrt{(x - a)^2 + (y - b)^2} = r \]

両辺を2乗すると標準形になります。

円の方程式（一般形） → 例2
\[ x^2 + y^2 + lx + my + n = 0 \]

標準形を展開した形。平方完成で標準形に戻すと中心と半径がわかる。

→ 中心と半径： \(\left(-\frac{l}{2}, -\frac{m}{2}\right)\)、\(r = \sqrt{\frac{l^2 + m^2}{4} - n}\)

注意： \(\frac{l^2 + m^2}{4} - n > 0\) でないと円にならない。

円と直線の位置関係 → 例3

中心 \((a, b)\) から直線 \(px + qy + r = 0\) への距離 \(d\) を求め、半径 \(R\) と比較：

\[ d = \frac{|pa + qb + r|}{\sqrt{p^2 + q^2}} \]
条件位置関係交点の個数
\(d < R\)2点で交わる2個
\(d = R\)接する1個
\(d > R\)離れている0個
円の接線（円上の点から） → 例4

円 \(x^2 + y^2 = r^2\) 上の点 \((x_1, y_1)\) における接線：

\[ x_1 x + y_1 y = r^2 \]

覚え方： 円の式の \(x^2\) を \(x_1 x\)、\(y^2\) を \(y_1 y\) に置き換える。

円の接線（円外の点から） → 例5

円外の点 \((x_0, y_0)\) から引いた接線は2本ある。

解法1： 接線 \(y - y_0 = m(x - x_0)\) とおき、中心との距離 = 半径で \(m\) を求める。

解法2： 接点を \((x_1, y_1)\) とおき、接線の式と「接点は円上」の条件を連立。

2円の位置関係 → 例6

半径 \(r_1, r_2\)（\(r_1 \geq r_2\) とする）の2円の中心間距離を \(d\) とすると：

条件位置関係
\(d > r_1 + r_2\)離れている（外部）
\(d = r_1 + r_2\)外接
\(r_1 - r_2 < d < r_1 + r_2\)2点で交わる
\(d = r_1 - r_2\)内接
\(d < r_1 - r_2\)一方が他方の内部
2円の位置関係（中心間距離と半径の和・差で決まる）

3. 例題と解き方

円の方程式と接線・位置関係の問題を実際に解いてみましょう。

▶ 円の方程式を求める

例1中心と半径から円の方程式（標準形）

中心 \((3, -2)\)、半径 \(5\) の円の方程式を求めよ。

解き方：

  1. 標準形 \((x - a)^2 + (y - b)^2 = r^2\) に代入
  2. \(a = 3\)、\(b = -2\)、\(r = 5\) を代入
\[ (x - 3)^2 + (y + 2)^2 = 25 \]

→ 答え： \((x - 3)^2 + (y + 2)^2 = 25\)

例2一般形から中心と半径を求める

円 \(x^2 + y^2 - 6x + 4y - 3 = 0\) の中心と半径を求めよ。

解き方：

  1. \(x\) と \(y\) の項をそれぞれ平方完成
  2. \(x^2 - 6x = (x - 3)^2 - 9\)
  3. \(y^2 + 4y = (y + 2)^2 - 4\)
  4. 元の式に代入して整理
\begin{align} (x - 3)^2 - 9 + (y + 2)^2 - 4 - 3 &= 0 \\ (x - 3)^2 + (y + 2)^2 &= 16 \end{align}

→ 答え： 中心 \((3, -2)\)、半径 \(4\)

▶ 円と直線の位置関係

例3円と直線の位置関係を判定

円 \(x^2 + y^2 = 25\) と直線 \(3x + 4y - 15 = 0\) の位置関係を調べよ。

解き方：

  1. 円の中心 \((0, 0)\)、半径 \(R = 5\) を確認
  2. 中心から直線への距離 \(d\) を求める（点と直線の距離公式）
  3. \(d\) と \(R\) を比較
\[ d = \frac{|3 \cdot 0 + 4 \cdot 0 - 15|}{\sqrt{3^2 + 4^2}} = \frac{15}{5} = 3 \]

\(d = 3 < R = 5\) なので、2点で交わる

▶ 接線を求める

例4円上の点における接線

円 \(x^2 + y^2 = 25\) 上の点 \((3, 4)\) における接線の方程式を求めよ。

解き方：

  1. 接線公式 \(x_1 x + y_1 y = r^2\) を使う
  2. \(x_1 = 3\)、\(y_1 = 4\)、\(r^2 = 25\) を代入
\[ 3x + 4y = 25 \]

→ 答え： \(3x + 4y = 25\)

なぜこの公式が使えるのか

接線は中心と接点を結ぶ線分（半径）に垂直です。中心 \((0, 0)\) から \((3, 4)\) へのベクトルは \((3, 4)\)。これに垂直な直線は \(3(x - 3) + 4(y - 4) = 0\) つまり \(3x + 4y = 25\) となります。

例5円外の点からの接線

円 \(x^2 + y^2 = 4\) に点 \((4, 0)\) から引いた接線の方程式を求めよ。

解き方：

  1. 接点を \((x_1, y_1)\) とおく（円上なので \(x_1^2 + y_1^2 = 4\)）
  2. 接線の式 \(x_1 x + y_1 y = 4\) が点 \((4, 0)\) を通る → \(4x_1 = 4\) → \(x_1 = 1\)
  3. \(x_1 = 1\) を円の式に代入 → \(1 + y_1^2 = 4\) → \(y_1 = \pm\sqrt{3}\)
  4. 接点 \((1, \sqrt{3})\) と \((1, -\sqrt{3})\) から接線を求める
\[ x + \sqrt{3}y = 4 \quad \text{または} \quad x - \sqrt{3}y = 4 \]

▶ 2円の位置関係

例62円の位置関係を判定

円 \(C_1: x^2 + y^2 = 9\) と円 \(C_2: (x - 4)^2 + y^2 = 1\) の位置関係を調べよ。

解き方：

  1. \(C_1\)：中心 \((0, 0)\)、半径 \(r_1 = 3\)
  2. \(C_2\)：中心 \((4, 0)\)、半径 \(r_2 = 1\)
  3. 中心間距離 \(d = \sqrt{(4-0)^2 + (0-0)^2} = 4\)
  4. \(r_1 + r_2 = 4\)、\(r_1 - r_2 = 2\) と比較

\(d = 4 = r_1 + r_2\) なので、外接する

例73点を通る円の方程式

3点 \((0, 0)\)、\((4, 0)\)、\((0, 3)\) を通る円の方程式を求めよ。

解き方：

  1. 一般形 \(x^2 + y^2 + lx + my + n = 0\) に3点を代入
  2. \((0, 0)\) → \(n = 0\)
  3. \((4, 0)\) → \(16 + 4l = 0\) → \(l = -4\)
  4. \((0, 3)\) → \(9 + 3m = 0\) → \(m = -3\)
\[ x^2 + y^2 - 4x - 3y = 0 \]

標準形に直すと：\((x - 2)^2 + (y - \frac{3}{2})^2 = \frac{25}{4}\)

→ 答え： 中心 \((2, \frac{3}{2})\)、半径 \(\frac{5}{2}\)

4. 簡単にするコツ

コツ1：原点中心の円は計算がシンプル
\(x^2 + y^2 = r^2\) の形なら、接線は \(x_1 x + y_1 y = r^2\) とすぐ書ける。中心が原点でない場合は、座標を平行移動して計算すると楽なことがある。
コツ2：位置関係は「距離と半径」で考える
円と直線、2円の位置関係は、中心からの距離を計算して半径と比較するだけ。連立方程式を解くより速い。
コツ3：平方完成は係数の半分の2乗を足し引き
\(x^2 + 6x\) → 6の半分は3、\((x + 3)^2 - 9\)。これを使えば一般形から標準形への変換は一瞬。
コツ4：接線公式は「\(x^2\) を \(x_1 x\) に置き換え」
円 \(x^2 + y^2 = r^2\) → 接線 \(x_1 x + y_1 y = r^2\)
円 \((x-a)^2 + (y-b)^2 = r^2\) → 接線 \((x_1-a)(x-a) + (y_1-b)(y-b) = r^2\)

5. よくあるつまづき

つまづき1：半径と半径の2乗を混同

ダメな例： 半径3の円を \((x-1)^2 + (y-2)^2 = 3\) と書く

対策： 右辺は \(r^2\)！半径3なら右辺は \(9\)

つまづき2：一般形から中心を読み間違える

ダメな例： \(x^2 + y^2 - 6x + 4y = 0\) の中心を \((6, 4)\) とする

対策： 中心は \(\left(-\frac{l}{2}, -\frac{m}{2}\right)\)。\(l = -6, m = 4\) なので中心は \((3, -2)\)

つまづき3：接線が円の外を通るか確認しない

ダメな例： 円内の点から接線を引こうとする

対策： まず点が円の外にあるか確認。\(x_0^2 + y_0^2 > r^2\) なら円外で接線は2本引ける

つまづき4：2円の位置関係で \(r_1 - r_2\) を忘れる

ダメな例： \(d < r_1 + r_2\) だけで「交わる」と判断

対策： \(r_1 - r_2 < d < r_1 + r_2\) のとき交わる。\(d < r_1 - r_2\) だと一方が他方の内部

6. 仕上げチェックリスト

中心と半径から標準形 \((x-a)^2 + (y-b)^2 = r^2\) を書ける
一般形を平方完成して中心と半径を求められる
中心から直線への距離と半径の比較で位置関係を判定できる
円上の点における接線の公式 \(x_1 x + y_1 y = r^2\) を使える

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

円 \(x^2 + y^2 + 2x - 8y + 8 = 0\) の中心と半径を求めよ。

答えを見る

平方完成：\((x+1)^2 - 1 + (y-4)^2 - 16 + 8 = 0\)

\((x+1)^2 + (y-4)^2 = 9\)

中心 \((-1, 4)\)、半径 \(3\)

練習2

円 \(x^2 + y^2 = 10\) 上の点 \((1, 3)\) における接線の方程式を求めよ。

答えを見る

接線公式 \(x_1 x + y_1 y = r^2\) を使う

\(1 \cdot x + 3 \cdot y = 10\)

\(x + 3y = 10\)

練習3

円 \(x^2 + y^2 = 16\) と円 \((x-5)^2 + y^2 = 4\) の位置関係を調べよ。

答えを見る

円1：中心 \((0, 0)\)、半径 \(r_1 = 4\)

円2：中心 \((5, 0)\)、半径 \(r_2 = 2\)

中心間距離 \(d = 5\)

\(r_1 + r_2 = 6\)、\(r_1 - r_2 = 2\)

\(2 < 5 < 6\) なので、2点で交わる

円の方程式は「中心からの距離 = 半径」を式で表したもの。標準形で中心と半径を読み取り、距離公式で位置関係を判定しよう！

おまけ：歴史ストーリー「アポロニウスの円」

アポロニウスの円

紀元前3世紀、古代ギリシャの数学者アポロニウス（Apollonius of Perga）は「円錐曲線論」という大著を書きました。楕円・放物線・双曲線という名前を付けたのも彼です。

その中で、彼はこんな問題を考えました：

「2点 A, B からの距離の比が一定（m : n）となる点の軌跡は？」

たとえば「A からの距離が B からの距離の2倍」という条件を満たす点はどんな軌跡を描くでしょうか？

直感的には「A に近づくほど条件を満たしにくそう」と思うかもしれませんが、実際に計算すると...

\[ \text{点 } P \text{ が満たす条件: } \frac{PA}{PB} = m : n \]

なんと、この軌跡は円になるのです！これを「アポロニウスの円」といいます。

現代の座標幾何で確認してみましょう。A = (0, 0)、B = (3, 0)として、PA : PB = 1 : 2 の場合：

\begin{align} PA &= 2 \cdot PB \\ \sqrt{x^2 + y^2} &= 2\sqrt{(x-3)^2 + y^2} \end{align}

両辺を2乗して整理すると：

\[ (x + 1)^2 + y^2 = 4 \]

確かに、中心 (-1, 0)、半径 2 の円になります！

アポロニウスは座標もない時代に、純粋な幾何学だけでこれを証明しました。2300年後の私たちは方程式で数行の計算。便利な時代になったものです。

でも、座標を使わずに証明できたアポロニウスの方がすごいのかも？
