1. テーマの定義

テストの点数、身長、気温の変動...自然界や社会の多くのデータは、ある特徴的な「釣り鐘型」の分布に従います。この分布を理解すれば、「上位何%に入るか」「何点以上取る確率は」といった問いに答えられるようになります。

このページでは、統計学の最重要概念である正規分布を学びます：

  • 正規分布とは：平均を中心に左右対称な釣り鐘型の分布
  • 標準正規分布：平均0、標準偏差1の基準となる正規分布
  • 標準化（z値）：どんな正規分布も標準正規分布に変換する方法
  • 正規分布表の使い方：確率を求めるための表の読み方

2. 覚えるルール

【最重要】標準化の公式：\( Z = \dfrac{X - \mu}{\sigma} \)

データ \(X\) を、平均 \(\mu\)（ミュー）と標準偏差 \(\sigma\)（シグマ）を使って、標準正規分布に従う値 \(Z\) に変換する

正規分布の曲線と確率の範囲（68-95-99.7ルール）
📌 正規分布 \(N(\mu, \sigma^2)\) → 例1

正規分布は2つのパラメータで決まる：

  • \(\mu\)（ミュー）：平均（分布の中心位置）
  • \(\sigma^2\)（シグマの2乗）：分散（\(\sigma\) は標準偏差）

「確率変数 \(X\) が正規分布に従う」ことを \(X \sim N(\mu, \sigma^2)\) と書く。

→ だから結果は： 平均 \(\mu\) を中心に、データの約68%が \(\mu \pm \sigma\) の範囲に、約95%が \(\mu \pm 2\sigma\) の範囲に収まる。

📌 標準正規分布 \(N(0, 1)\) → 例2

平均 \(\mu = 0\)、標準偏差 \(\sigma = 1\) の正規分布を標準正規分布という。

\[ Z \sim N(0, 1) \]

✅ メリット： 正規分布表はすべて標準正規分布用なので、どんな正規分布も標準化すれば表が使える！

📌 標準化の公式 → 例3
\[ Z = \frac{X - \mu}{\sigma} \]

\(X \sim N(\mu, \sigma^2)\) のとき、\(Z = \dfrac{X - \mu}{\sigma}\) は \(N(0, 1)\) に従う。

→ だから結果は： 「\(X\) が \(a\) 以上」の確率は「\(Z\) が \(\dfrac{a - \mu}{\sigma}\) 以上」の確率と同じ。

💡 考え方を見る

標準化は「平均からどれだけ離れているかを、標準偏差何個分で表す」操作です。

たとえば、平均50点・標準偏差10点のテストで70点を取った場合：

\( Z = \dfrac{70 - 50}{10} = 2 \)

これは「平均より標準偏差2個分だけ上」という意味です。

📌 正規分布表の見方 → 例4

正規分布表には \(P(0 \leq Z \leq z)\) の値が載っている（0からzまでの確率）。

正規分布表が示す確率（緑色の部分）

→ よく使う値：

  • \(P(0 \leq Z \leq 1) = 0.3413\)（約34%）
  • \(P(0 \leq Z \leq 1.96) = 0.4750\)（約47.5%）
  • \(P(0 \leq Z \leq 2) = 0.4772\)（約48%）
📌 確率の計算パターン → 例5

正規分布は左右対称なので、以下の関係を使う：

求めたい確率計算方法
\(P(Z \leq z)\)\(0.5 + P(0 \leq Z \leq z)\)
\(P(Z \geq z)\)\(0.5 - P(0 \leq Z \leq z)\)
\(P(a \leq Z \leq b)\)\(P(0 \leq Z \leq b) - P(0 \leq Z \leq a)\)（\(0 < a < b\) のとき）
\(P(Z \leq -z)\)\(P(Z \geq z) = 0.5 - P(0 \leq Z \leq z)\)

3. 例題と解き方

標準化と正規分布表を使って、確率を求める手順を見ていきましょう。

▶ 基本：標準正規分布の確率

例1正規分布のパラメータを読み取る

ある高校の数学テストの点数は、平均60点、標準偏差12点の正規分布に従う。この正規分布を記号で表せ。

解き方：

  1. 平均 \(\mu = 60\) を確認する
  2. 標準偏差 \(\sigma = 12\) なので、分散 \(\sigma^2 = 144\)
  3. 正規分布の記号 \(N(\mu, \sigma^2)\) に代入する
\[ X \sim N(60, 12^2) = N(60, 144) \]

注意： \(N(\mu, \sigma^2)\) の2つ目は分散（標準偏差の2乗）です。

例2標準正規分布表を使う（基本）

\(Z \sim N(0, 1)\) のとき、\(P(Z \leq 1.5)\) を求めよ。

解き方：

  1. \(P(Z \leq 1.5) = P(Z \leq 0) + P(0 \leq Z \leq 1.5)\) と分ける
  2. \(P(Z \leq 0) = 0.5\)（左右対称なので半分）
  3. 正規分布表で \(z = 1.5\) の値を読む → \(P(0 \leq Z \leq 1.5) = 0.4332\)
  4. 足し算する
\[ P(Z \leq 1.5) = 0.5 + 0.4332 = 0.9332 \]

▶ 標準化して確率を求める

例3標準化の計算

例1のテスト（平均60点、標準偏差12点）で、84点以上取る確率を求めよ。

解き方：

  1. 標準化する：\(Z = \dfrac{X - \mu}{\sigma} = \dfrac{84 - 60}{12} = \dfrac{24}{12} = 2\)
  2. 「\(X \geq 84\)」は「\(Z \geq 2\)」と同じ
  3. \(P(Z \geq 2) = 0.5 - P(0 \leq Z \leq 2)\)
  4. 正規分布表で \(P(0 \leq Z \leq 2) = 0.4772\)
  5. 計算する
\[ P(X \geq 84) = P(Z \geq 2) = 0.5 - 0.4772 = 0.0228 \]

答え： 約2.3%（上位約2%に相当）

例4正規分布表の詳しい読み方

正規分布表で \(P(0 \leq Z \leq 1.64)\) を求めよ。

解き方：

  1. \(z = 1.64\) を整数部分と小数部分に分ける
  2. 表の「行」で \(1.6\) を見つける
  3. 表の「列」で \(0.04\) を見つける
  4. 交点の値を読む

▶ 応用：範囲の確率

例5範囲の確率を求める

身長が平均170cm、標準偏差6cmの正規分布に従うとき、身長が164cm以上182cm以下である確率を求めよ。

解き方：

  1. 両端を標準化する
    • \(Z_1 = \dfrac{164 - 170}{6} = -1\)
    • \(Z_2 = \dfrac{182 - 170}{6} = 2\)
  2. \(P(164 \leq X \leq 182) = P(-1 \leq Z \leq 2)\)
  3. 分けて計算する：\(P(-1 \leq Z \leq 2) = P(-1 \leq Z \leq 0) + P(0 \leq Z \leq 2)\)
  4. 左右対称性より \(P(-1 \leq Z \leq 0) = P(0 \leq Z \leq 1) = 0.3413\)
  5. 表より \(P(0 \leq Z \leq 2) = 0.4772\)
\[ P(-1 \leq Z \leq 2) = 0.3413 + 0.4772 = 0.8185 \]

答え： 約81.9%

例6上位何%かを求める

あるテストの得点が平均55点、標準偏差8点の正規分布に従う。75点は上位何%に入るか。

解き方：

  1. 標準化する：\(Z = \dfrac{75 - 55}{8} = \dfrac{20}{8} = 2.5\)
  2. 「75点以上」の確率を求める：\(P(Z \geq 2.5)\)
  3. 表より \(P(0 \leq Z \leq 2.5) = 0.4938\)
  4. \(P(Z \geq 2.5) = 0.5 - 0.4938 = 0.0062\)
\[ P(X \geq 75) = P(Z \geq 2.5) = 0.0062 \approx 0.6\% \]

答え： 上位約0.6%

4. 簡単にするコツ

✏️ コツ1：68-95-99.7ルールで概算する
正規分布表を見なくても、以下の値は覚えておくと便利：
  • \(\mu \pm \sigma\) の範囲に約68%
  • \(\mu \pm 2\sigma\) の範囲に約95%
  • \(\mu \pm 3\sigma\) の範囲に約99.7%
例：平均60、標準偏差10なら、40点〜80点の範囲に約95%が入る。
✏️ コツ2：図を描いて求める範囲を確認
正規分布曲線を描き、求めたい範囲を塗りつぶす。
  • 0より右側だけ → 表の値をそのまま使う
  • 0をまたぐ → 左右に分けて足す
  • 「以上」→ 0.5から引く
✏️ コツ3：標準化の計算は分数のままでOK
\(Z = \dfrac{X - \mu}{\sigma}\) の計算では、無理に小数にしなくてよい。
例：\(Z = \dfrac{76 - 60}{8} = \dfrac{16}{8} = 2\) のように、約分して整数や小数第1位で済むことが多い。

5. よくあるつまづき

⚠️ つまづき1：分散と標準偏差を混同する

ダメな例： 標準偏差10のとき、\(N(\mu, 10)\) と書く

対策： \(N(\mu, \sigma^2)\) の2つ目は分散！標準偏差10なら \(N(\mu, 100)\) と書く。

⚠️ つまづき2：正規分布表の読み方を間違える

ダメな例： \(P(Z \leq 1.5)\) を求めるとき、表の値 0.4332 をそのまま答える

対策： 表は \(P(0 \leq Z \leq z)\) を示す。\(P(Z \leq z) = 0.5 + P(0 \leq Z \leq z)\) に注意！

⚠️ つまづき3：負のz値で混乱する

ダメな例： \(P(Z \leq -1)\) を求めるとき、表に \(-1\) がないので困る

対策： 左右対称性を使う！\(P(Z \leq -1) = P(Z \geq 1) = 0.5 - P(0 \leq Z \leq 1)\)

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

あるメーカーのボルトの長さは、平均50mm、標準偏差2mmの正規分布に従う。長さが54mm以上のボルトは全体の何%か。正規分布表より \(P(0 \leq Z \leq 2) = 0.4772\) を使え。

答えを見る

解答：

  1. 標準化：\(Z = \dfrac{54 - 50}{2} = 2\)
  2. \(P(X \geq 54) = P(Z \geq 2) = 0.5 - P(0 \leq Z \leq 2)\)
  3. \(= 0.5 - 0.4772 = 0.0228\)

答え：約2.28%

練習2

ある試験の得点は平均65点、標準偏差10点の正規分布に従う。得点が55点以上75点以下の受験生は全体の何%か。正規分布表より \(P(0 \leq Z \leq 1) = 0.3413\) を使え。

答えを見る

解答：

  1. 両端を標準化：
    • \(Z_1 = \dfrac{55 - 65}{10} = -1\)
    • \(Z_2 = \dfrac{75 - 65}{10} = 1\)
  2. \(P(55 \leq X \leq 75) = P(-1 \leq Z \leq 1)\)
  3. \(= P(-1 \leq Z \leq 0) + P(0 \leq Z \leq 1)\)
  4. 左右対称より \(P(-1 \leq Z \leq 0) = P(0 \leq Z \leq 1) = 0.3413\)
  5. \(= 0.3413 + 0.3413 = 0.6826\)

答え：約68.26%（これは68-95-99.7ルールの68%に相当！）

おまけ：歴史ストーリー「天文学者が見つけた誤差の法則」

📖 星を観測していたら、数学の大発見

18世紀、天文学者たちはある問題に悩んでいました。星の位置を何度測っても、毎回少しずつ違う値が出てしまうのです。「真の位置」はどれなのか？

1809年、ドイツの数学者カール・フリードリヒ・ガウス（1777-1855）は、惑星ケレスの軌道を計算する中で、この「測定誤差」に法則があることを発見しました。

彼が見つけた法則とは：

\[ f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \]

これが正規分布（別名：ガウス分布）の確率密度関数です。

ガウスは、誤差には以下の特徴があることを見抜きました：

  • 小さい誤差は大きい誤差より起こりやすい
  • プラスの誤差とマイナスの誤差は同じくらい起こる
  • 複数の測定の平均をとると、真の値に近づく

面白いことに、この分布は星の観測だけでなく、身長、テストの点数、工場の製品の寸法...あらゆる自然現象に現れることがわかりました。

なぜこれほど普遍的なのでしょうか？その答えは「中心極限定理」にあります。多くの小さな要因が積み重なった結果は、正規分布に近づくのです。

ちなみに、ガウスはあまりに多くの業績を残したため、「数学の王子」と呼ばれています。正規分布の発見は、その業績のほんの一部にすぎません。

次に正規分布を使うときは、200年以上前の天文学者たちが星を見上げて悩んでいた姿を思い浮かべてみてください。誤差の中に法則を見出すのは、科学の醍醐味かもしれませんね。
