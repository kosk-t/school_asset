1. テーマの定義

テレビの視聴率調査では、日本全国の約4000万世帯すべてを調べることはできません。そこで、一部の世帯（標本）を調べて、全体（母集団）の傾向を「推測」します。これが統計的推測の考え方です。

このページでは、以下の内容を学びます：

  • 母集団と標本：調査対象全体と、そこから選んだ一部の関係
  • 母平均の推定：標本平均から母平均を「点」で推定する方法
  • 信頼区間：母平均が「この範囲にある」と95%の確率で言える区間
  • 仮説検定：「この仮説は正しいか？」を統計的に判断する方法

2. 覚えるルール

【最重要】標本平均 \(\bar{X}\) は正規分布 \(N\left(m, \dfrac{\sigma^2}{n}\right)\) に従う（中心極限定理）
母集団から無作為に抽出した標本で、母集団の特性を推測する
📌 母集団と標本の用語
用語意味記号
母集団調査対象の全体大きさ \(N\)
標本母集団から抽出した一部大きさ \(n\)
母平均母集団全体の平均\(m\)（または \(\mu\)）
母分散母集団全体の分散\(\sigma^2\)
母標準偏差母集団全体の標準偏差\(\sigma\)
標本平均標本の平均値\(\bar{X}\)

無作為抽出とは、母集団のどの要素も同じ確率で選ばれるように抽出すること。偏りのない推測のために必須！

📌 標本平均の分布（中心極限定理） → 例1

母平均 \(m\)、母標準偏差 \(\sigma\) の母集団から、大きさ \(n\) の標本を無作為抽出するとき：

\[ \bar{X} \sim N\left(m, \frac{\sigma^2}{n}\right) \]

つまり、標本平均 \(\bar{X}\) の：

  • 期待値（平均）= \(m\)（母平均と同じ）
  • 分散 = \(\dfrac{\sigma^2}{n}\)（母分散を \(n\) で割る）
  • 標準偏差 = \(\dfrac{\sigma}{\sqrt{n}}\)（これを標準誤差という）

→ だから結果は： 標本サイズ \(n\) が大きいほど、標本平均のばらつきは小さくなる！

💡 考え方を見る

たくさんのデータを集めれば、平均値は安定する。100人調べた平均より、10000人調べた平均の方が信頼できる、というイメージ。

📌 標準化と正規分布表 → 例2

標本平均 \(\bar{X}\) を標準化すると、標準正規分布 \(N(0, 1)\) に従う：

\[ Z = \frac{\bar{X} - m}{\dfrac{\sigma}{\sqrt{n}}} \sim N(0, 1) \]

正規分布表で重要な値：

  • \(P(|Z| \leq 1.96) = 0.95\)（95%が \(-1.96 \leq Z \leq 1.96\) に入る）
  • \(P(|Z| \leq 2.58) = 0.99\)（99%が \(-2.58 \leq Z \leq 2.58\) に入る）
正規分布の中央95%の範囲が信頼区間の基礎
📌 95%信頼区間の公式 → 例3

母平均 \(m\) の95%信頼区間は：

\[ \bar{X} - 1.96 \cdot \frac{\sigma}{\sqrt{n}} \leq m \leq \bar{X} + 1.96 \cdot \frac{\sigma}{\sqrt{n}} \]

または、まとめて書くと：

\[ \bar{X} \pm 1.96 \cdot \frac{\sigma}{\sqrt{n}} \]

→ だから結果は： 「この区間に母平均が含まれる」と95%の信頼度で言える！

✅ メリット： 「母平均は〇〇です」（点推定）より、「〇〇から△△の間です」（区間推定）の方が信頼性を伝えやすい。

💡 考え方を見る

「95%信頼区間」とは、「同じ調査を100回やれば、そのうち95回はこの区間に母平均が入る」という意味。1回の調査で「絶対に入る」わけではないことに注意！

📌 仮説検定の手順 → 例4

「母平均は \(m_0\) である」という仮説を検定する手順：

  1. 帰無仮説 \(H_0\) を立てる：「母平均 \(m = m_0\)」（否定したい仮説）
  2. 対立仮説 \(H_1\) を立てる：「母平均 \(m \neq m_0\)」（主張したい仮説）
  3. 有意水準 \(\alpha\) を決める（通常は5%）
  4. 検定統計量 \(Z = \dfrac{\bar{X} - m_0}{\sigma / \sqrt{n}}\) を計算
  5. 棄却域と比較して判定：
    • \(|Z| > 1.96\) なら \(H_0\) を棄却（有意差あり）
    • \(|Z| \leq 1.96\) なら \(H_0\) を棄却しない

→ だから結果は： 仮説が正しいとしたら起こりにくいこと（5%未満）が起きたら、仮説を疑う！

有意水準5%の両側検定：\(|Z| > 1.96\) なら帰無仮説を棄却

3. 例題と解き方

標本から母集団の特性を推測する具体的な方法を見ていきましょう。

▶ 標本平均の分布

例1標本平均の確率を求める

ある工場で生産される製品の重さは、母平均 50g、母標準偏差 4g の正規分布に従う。この製品から16個を無作為抽出したとき、標本平均が 49g 以上 51g 以下になる確率を求めよ。

解き方：

  1. 標本平均 \(\bar{X}\) の分布を確認
    \[ \bar{X} \sim N\left(50, \frac{4^2}{16}\right) = N(50, 1) \]
    標準偏差（標準誤差）は \(\dfrac{4}{\sqrt{16}} = 1\)
  2. 標準化する
    \[ Z = \frac{\bar{X} - 50}{1} = \bar{X} - 50 \]
  3. \(\bar{X} = 49\) のとき \(Z = -1\)、\(\bar{X} = 51\) のとき \(Z = 1\)
  4. 正規分布表から確率を求める
    \[ P(-1 \leq Z \leq 1) = 2 \times 0.3413 = 0.6826 \]

答え：約 0.683（約68.3%）

▶ 標準化

例2標本サイズによる違い

母平均 100、母標準偏差 20 の母集団から標本を抽出する。標本平均が 96 以上 104 以下になる確率が 95% 以上になるには、標本サイズ \(n\) はいくつ以上必要か。

解き方：

  1. 条件を式で表す。\(P(96 \leq \bar{X} \leq 104) \geq 0.95\) より
    \[ P\left(\frac{96 - 100}{\sigma/\sqrt{n}} \leq Z \leq \frac{104 - 100}{\sigma/\sqrt{n}}\right) \geq 0.95 \]
  2. \(\sigma = 20\) を代入して整理
    \[ P\left(-\frac{4\sqrt{n}}{20} \leq Z \leq \frac{4\sqrt{n}}{20}\right) \geq 0.95 \]
    \[ P\left(-\frac{\sqrt{n}}{5} \leq Z \leq \frac{\sqrt{n}}{5}\right) \geq 0.95 \]
  3. 95%の範囲は \(|Z| \leq 1.96\) なので
    \[ \frac{\sqrt{n}}{5} \geq 1.96 \]
  4. \(\sqrt{n} \geq 9.8\) より \(n \geq 96.04\)

答え：\(n \geq 97\)（97個以上）

▶ 信頼区間

例3母平均の95%信頼区間を求める

ある缶ジュースの内容量は母標準偏差 5mL の正規分布に従う。100本を無作為抽出したところ、標本平均は 248mL だった。母平均の95%信頼区間を求めよ。

解き方：

  1. 標準誤差を計算する
    \[ \frac{\sigma}{\sqrt{n}} = \frac{5}{\sqrt{100}} = \frac{5}{10} = 0.5 \]
  2. 信頼区間の幅を計算する
    \[ 1.96 \times 0.5 = 0.98 \]
  3. 信頼区間を求める
    \[ 248 - 0.98 \leq m \leq 248 + 0.98 \]
    \[ 247.02 \leq m \leq 248.98 \]

答え：247.02mL 以上 248.98mL 以下

（または「\(248 \pm 0.98\) mL」「約 247mL から 249mL」と書いてもよい）

▶ 仮説検定

例4仮説検定（両側検定）

ある工場では、製品の重さの母平均は 500g とされている。品質管理のため 64 個を無作為抽出したところ、標本平均は 503g だった。母標準偏差は 16g である。母平均が 500g と異なるといえるか。有意水準 5% で検定せよ。

解き方：

  1. 仮説を立てる
    • 帰無仮説 \(H_0\)：\(m = 500\)（母平均は 500g）
    • 対立仮説 \(H_1\)：\(m \neq 500\)（母平均は 500g ではない）
  2. 検定統計量を計算する
    \[ Z = \frac{\bar{X} - m_0}{\sigma / \sqrt{n}} = \frac{503 - 500}{16 / \sqrt{64}} = \frac{3}{16/8} = \frac{3}{2} = 1.5 \]
  3. 棄却域と比較する

    有意水準 5%（両側検定）の棄却域は \(|Z| > 1.96\)

    \(|1.5| = 1.5 < 1.96\) なので、棄却域に入らない

  4. 結論を述べる

答え：帰無仮説を棄却しない。母平均が 500g と異なるとはいえない。

（注意：「500g と等しい」とは言っていない。「異なるとはいえない」という消極的な結論）

例5仮説検定（棄却される場合）

例4と同じ条件で、標本平均が 508g だった場合はどうなるか。

解き方：

  1. 検定統計量を計算する
    \[ Z = \frac{508 - 500}{16 / \sqrt{64}} = \frac{8}{2} = 4 \]
  2. 棄却域と比較する

    \(|4| = 4 > 1.96\) なので、棄却域に入る

答え：帰無仮説を棄却する。有意水準5%で、母平均は 500g と異なるといえる。

4. 簡単にするコツ

✏️ コツ1：1.96 と 2.58 を覚える
95%信頼区間 → 1.96（ほぼ2）
99%信頼区間 → 2.58（ほぼ2.6）
概算なら「95%は約2」と覚えておくと便利！
✏️ コツ2：標準誤差の計算は先にやる
\(\dfrac{\sigma}{\sqrt{n}}\) を先に計算しておくと、信頼区間も検定統計量もスムーズに出せる。
特に \(\sqrt{n}\) が整数になる \(n = 4, 9, 16, 25, 36, 49, 64, 81, 100\) は計算しやすい！
✏️ コツ3：仮説検定は「背理法」と同じ発想
「帰無仮説が正しいと仮定 → ありえないことが起きた → 仮説は間違い」
数学の背理法と同じ論理構造！「正しいと仮定して矛盾を導く」イメージで理解しよう。

5. よくあるつまづき

⚠️ つまづき1：分散と標準偏差を混同

ダメな例： 「母分散が 16 なので、標準誤差は \(\dfrac{16}{\sqrt{n}}\)」

対策： 母分散 \(\sigma^2 = 16\) なら、母標準偏差 \(\sigma = 4\) を使う。標準誤差は \(\dfrac{4}{\sqrt{n}}\)。公式には「\(\sigma\)（標準偏差）」が入ることを確認！

⚠️ つまづき2：信頼区間の解釈ミス

ダメな例： 「母平均が信頼区間に入る確率が95%」

対策： 母平均は固定値（動かない）。「同じ調査を100回やれば、95回の区間が母平均を含む」が正しい解釈。区間が動くのであって、母平均が動くのではない！

⚠️ つまづき3：仮説検定の結論の書き方

ダメな例： 「帰無仮説を棄却しないので、母平均は500gである」

対策： 「棄却しない」は「正しいと証明した」わけではない。「異なるとはいえない」「500gでないとはいえない」のように消極的に書く。

6. 仕上げチェックリスト

標本平均の分布 \(N\left(m, \dfrac{\sigma^2}{n}\right)\) を書ける
標準誤差 \(\dfrac{\sigma}{\sqrt{n}}\) を正しく計算できる
95%信頼区間の公式 \(\bar{X} \pm 1.96 \cdot \dfrac{\sigma}{\sqrt{n}}\) を使える
仮説検定で帰無仮説・対立仮説を正しく立てられる
検定統計量 \(Z\) を計算し、棄却域と比較できる

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

ある商品の重さは母平均 \(m\)、母標準偏差 10g の正規分布に従う。25個を無作為抽出したところ、標本平均は 202g だった。母平均 \(m\) の95%信頼区間を求めよ。

答えを見る

標準誤差：\(\dfrac{10}{\sqrt{25}} = \dfrac{10}{5} = 2\)

信頼区間の幅：\(1.96 \times 2 = 3.92\)

\(202 - 3.92 \leq m \leq 202 + 3.92\)

答え：198.08g 以上 205.92g 以下

練習2

ある電球の寿命は母平均 1000 時間、母標準偏差 100 時間の正規分布に従うとされている。新しい製法で作った電球 36 個を調べたところ、標本平均は 1040 時間だった。新しい製法で寿命が変わったといえるか。有意水準 5% で検定せよ。

答えを見る

帰無仮説 \(H_0\)：\(m = 1000\)、対立仮説 \(H_1\)：\(m \neq 1000\)

検定統計量：\(Z = \dfrac{1040 - 1000}{100 / \sqrt{36}} = \dfrac{40}{100/6} = \dfrac{40 \times 6}{100} = 2.4\)

\(|2.4| > 1.96\) なので棄却域に入る

答え：帰無仮説を棄却する。有意水準5%で、新しい製法で寿命が変わったといえる。

📌 まとめ：標本平均は \(N\left(m, \dfrac{\sigma^2}{n}\right)\) に従い、95%信頼区間は \(\bar{X} \pm 1.96 \cdot \dfrac{\sigma}{\sqrt{n}}\) で求める。仮説検定では \(|Z| > 1.96\) なら棄却！

おまけ：統計パラドックス「世論調査のウソ？」

📖 1936年のアメリカ大統領選挙

1936年、アメリカでは大統領選挙が行われ、現職のフランクリン・ルーズベルトと共和党のアルフレッド・ランドンが対決しました。

当時、「リテラリー・ダイジェスト」という有名な雑誌が、約240万人という史上最大規模の世論調査を実施。その結果、「ランドンが57%で圧勝する」と予測しました。

一方、新興の調査会社ギャラップは、たった約5万人の調査で「ルーズベルトが56%で勝つ」と予測。

結果は？

ルーズベルトが62%を獲得して圧勝！

なぜ240万人の調査が大外れし、5万人の調査が的中したのでしょうか？

原因は「無作為抽出」ではなかったこと。

リテラリー・ダイジェストは、電話帳や自動車登録名簿から回答者を選びました。しかし、1936年当時、電話や自動車を持っているのは比較的裕福な人々だけ。つまり、富裕層に偏った標本だったのです。不況下でルーズベルトを支持した低所得層の声は、ほとんど含まれていませんでした。

一方、ギャラップは所得・地域・年齢のバランスを考慮した「割り当て法」で調査対象を選定。人数は少なくても、母集団を代表する標本を作ることに成功したのです。

\[ \text{大量データ} + \text{偏った抽出} < \text{少量データ} + \text{無作為抽出} \]

教訓：サンプルサイズが大きくても、無作為抽出でなければ意味がない。統計的推測で最も大事なのは「どう選んだか」なのです！

ちなみに、この大失敗でリテラリー・ダイジェストは信用を失い、まもなく廃刊。ギャラップは一躍有名になり、今も「ギャラップ調査」として世界的に知られています。
