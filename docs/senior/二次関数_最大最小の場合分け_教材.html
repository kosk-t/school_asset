1. テーマの定義

二次関数の最大最小問題で一番の悩みは「場合分けがいるの？」「何で分けるの？」という判断。この教材で迷わず判断できるようになろう。

このページでは、場合分けの判断力を学びます：

  • 場合分けが必要か：問題文に文字があるか？どこに文字があるか？
  • 何で場合分けするか：軸の位置？定義域の端点？
  • 場合分けの境界：等号はどこにつける？

2. 判断フロー

🎯 最重要：「文字がどこにあるか」で場合分けの必要性が決まる！
まず「文字がどこにあるか」を確認！
📌 パターン1：文字なし → 場合分け不要 → 例1

問題文に文字（パラメータ）がなければ、場合分けは不要。

例：\(y = x^2 - 4x + 3\)（\(0 \leq x \leq 5\)）の最大最小

→ やること：

  1. 平方完成して軸と頂点を出す
  2. 軸が定義域の中か外かを確認
  3. 最大最小を求める
📌 パターン2：軸に文字 → 軸の位置で場合分け → 例2

軸にパラメータがある場合、軸が定義域のどこにあるかで場合分け。

例：\(y = x^2 - 2ax + 1\)（\(0 \leq x \leq 2\)）の最小値

→ 場合分けの境界：

  • 軸 = 定義域の左端（この例では \(a = 0\)）
  • 軸 = 定義域の右端（この例では \(a = 2\)）
\[ \text{場合分け：} \begin{cases} a < 0 & \text{（軸が左端の左）} \\ 0 \leq a \leq 2 & \text{（軸が定義域内）} \\ a > 2 & \text{（軸が右端の右）} \end{cases} \]
📌 パターン3：定義域に文字 → 端点の位置で場合分け → 例3

定義域の端点にパラメータがある場合、端点が軸のどこにあるかで場合分け。

例：\(y = x^2 - 4x + 3\)（\(0 \leq x \leq a\)）の最小値

→ 場合分けの境界：

  • 右端 = 軸（この例では \(a = 2\)）
\[ \text{場合分け：} \begin{cases} 0 < a < 2 & \text{（軸が定義域の右）} \\ a \geq 2 & \text{（軸が定義域内）} \end{cases} \]
📌 パターン4：軸と定義域の両方に文字 → 複合的に場合分け → 例4

軸と定義域の両方にパラメータがある場合、両方の関係で場合分け。

例：\(y = x^2 - 2ax\)（\(0 \leq x \leq 1\)）の最大値を \(a\) で表す

→ やること：

  1. 軸の位置を確認（この例では \(x = a\)）
  2. 軸と定義域の位置関係で場合分け
  3. 各場合で端点の値を比較
境界値は「軸が定義域の端に来るとき」
📌 境界値の決め方まとめ
文字の場所 境界の条件 例（定義域 [0, 2]）
軸に文字 \(x = a\) 軸 = 左端、軸 = 右端 \(a = 0, a = 2\)
右端に文字（〜a） 右端 = 軸 \(a = \text{軸の値}\)
左端に文字（a〜） 左端 = 軸 \(a = \text{軸の値}\)

3. 例題と解き方

パターンごとに例題を解いてみましょう。

▶ パターン1：文字なし

例1\(y = x^2 - 6x + 5\)（\(1 \leq x \leq 4\)）の最大最小

判断： 文字なし → 場合分け不要

解き方：

  1. 平方完成：\(y = (x - 3)^2 - 4\)、軸 \(x = 3\)
  2. 軸 \(x = 3\) は \([1, 4]\) の中にある
  3. 最小値：頂点で \(-4\)（\(x = 3\)）
  4. 最大値：\(f(1) = 0\)、\(f(4) = -3\) → 大きい方は \(0\)（\(x = 1\)）
\[ \text{最小値}：-4 \;(x=3), \quad \text{最大値}：0 \;(x=1) \]

▶ パターン2：軸に文字

例2\(y = x^2 - 2ax + 3\)（\(0 \leq x \leq 2\)）の最小値を \(a\) で表す

判断： 軸 \(x = a\) に文字あり → 軸の位置で場合分け

境界値： \(a = 0\)（軸 = 左端）、\(a = 2\)（軸 = 右端）

解き方：

  1. 平方完成：\(y = (x - a)^2 - a^2 + 3\)、軸 \(x = a\)、頂点の \(y\) 座標は \(-a^2 + 3\)
  2. 3つに場合分け

(i) \(a < 0\) のとき：軸が左端の左

　→ 定義域内で増加、最小は左端 \(f(0) = 3\)

(ii) \(0 \leq a \leq 2\) のとき：軸が定義域内

　→ 最小は頂点で \(-a^2 + 3\)

(iii) \(a > 2\) のとき：軸が右端の右

　→ 定義域内で減少、最小は右端 \(f(2) = 7 - 4a\)

\[ \text{最小値} = \begin{cases} 3 & (a < 0) \\ -a^2 + 3 & (0 \leq a \leq 2) \\ 7 - 4a & (a > 2) \end{cases} \]

▶ パターン3：定義域に文字

例3\(y = x^2 - 4x + 5\)（\(0 \leq x \leq a\)、\(a > 0\)）の最小値を \(a\) で表す

判断： 定義域の右端に文字 \(a\) あり → 右端の位置で場合分け

解き方：

  1. 平方完成：\(y = (x - 2)^2 + 1\)、軸 \(x = 2\)、頂点 \((2, 1)\)
  2. 境界値は「右端 = 軸」つまり \(a = 2\)

(i) \(0 < a < 2\) のとき：軸が定義域の右側

　→ 定義域内で減少、最小は右端 \(f(a) = a^2 - 4a + 5\)

(ii) \(a \geq 2\) のとき：軸が定義域内

　→ 最小は頂点で \(1\)

\[ \text{最小値} = \begin{cases} a^2 - 4a + 5 & (0 < a < 2) \\ 1 & (a \geq 2) \end{cases} \]

▶ パターン4：両方に文字

例4\(y = x^2 - 2ax\)（\(0 \leq x \leq 1\)）の最大値を \(a\) で表す

判断： 軸に文字 \(a\)、定義域は固定だが最大値なので端点比較が必要 → 場合分け

解き方：

  1. 平方完成：\(y = (x - a)^2 - a^2\)、軸 \(x = a\)
  2. 下に凸なので、最大値は端点 \(f(0)\) と \(f(1)\) の比較
  3. \(f(0) = 0\)、\(f(1) = 1 - 2a\)
  4. \(f(0) = f(1)\) となるのは \(0 = 1 - 2a\)、つまり \(a = \dfrac{1}{2}\)

(i) \(a < \dfrac{1}{2}\) のとき：\(f(1) = 1 - 2a > 0\)

　→ 最大値は \(f(1) = 1 - 2a\)

(ii) \(a \geq \dfrac{1}{2}\) のとき：\(f(1) = 1 - 2a \leq 0\)

　→ 最大値は \(f(0) = 0\)

\[ \text{最大値} = \begin{cases} 1 - 2a & \left(a < \dfrac{1}{2}\right) \\ 0 & \left(a \geq \dfrac{1}{2}\right) \end{cases} \]

4. 簡単にするコツ

✏️ コツ1：「文字がどこにあるか」を最初にチェック
問題を読んだら、すぐに「軸に文字？」「定義域に文字？」を確認。これで場合分けの必要性がわかる。
✏️ コツ2：境界値を先に計算
「軸 = 左端」「軸 = 右端」を解いて境界値を出しておく。場合分けの式を書くのが楽になる。
✏️ コツ3：グラフを3つ描く
場合分けが3つなら、3つのグラフを並べて描く。視覚的に確認すると間違いが減る。
✏️ コツ4：等号はどちらでもOK
境界で \(a < 2\) と \(a \leq 2\) のどちらにするか迷うなら、実際に代入してどちらのケースでも答えが一致することを確認。等号は片方に入れればOK。
✏️ コツ5：最大値は「端点比較」を忘れずに
下に凸の最大値（上に凸の最小値）は、軸の位置に関係なく端点の比較。\(f(\text{左端})\) と \(f(\text{右端})\) が等しくなる条件で場合分け。

5. よくあるつまづき

⚠️ つまづき1：場合分けの必要性を見落とす

ダメな例： \(y = x^2 - 2ax + 1\) で「軸は \(x = a\) だから頂点で最小」と即答

対策： 軸にパラメータがあったら場合分けが必要！定義域との位置関係を確認。

⚠️ つまづき2：境界値を間違える

ダメな例： 定義域 \([0, 3]\) で「\(a < 0, 0 < a < 3, a > 3\)」と場合分け（\(a = 0, 3\) を忘れている）

対策： 境界は「軸 = 端点」。その値を含めて「\(a \leq 0\)」「\(0 < a < 3\)」「\(a \geq 3\)」のように書く。

⚠️ つまづき3：最大値なのに「頂点で最大」と答える

ダメな例： 下に凸の関数で「軸が定義域内だから最大値は頂点」

対策： 下に凸 → 頂点は最小。最大値は端点のどちらか。

⚠️ つまづき4：場合分けの結果を統合し忘れる

ダメな例： 3つの場合について別々に答えを書くが、最終的な答えとしてまとめない

対策： 最後に「よって最小値は〜」と場合分けの結果を統合して書く。

6. 仕上げチェックリスト

問題文の「どこに文字があるか」を確認したか
場合分けの境界値（軸 = 端点）を正しく計算したか
すべてのケースについて答えを出したか
最大値と最小値を混同していないか
場合分けの結果を最終的にまとめたか

7. 練習問題

まず「場合分けが必要か」「何で分けるか」を判断してから解こう。

練習1

\(y = x^2 + 2x - 3\)（\(-2 \leq x \leq 1\)）の最大値と最小値を求めよ。

答えを見る

判断：文字なし → 場合分け不要

\(y = (x + 1)^2 - 4\)、軸 \(x = -1\) は \([-2, 1]\) の中

最小値：\(-4\)（\(x = -1\)）

\(f(-2) = -3\)、\(f(1) = 0\) → 最大値：\(0\)（\(x = 1\)）

練習2

\(y = x^2 - 4ax + 2\)（\(0 \leq x \leq 1\)）の最小値を \(a\) で表せ。

答えを見る

判断：軸 \(x = 2a\) に文字 → 軸の位置で場合分け

境界：\(2a = 0\) → \(a = 0\)、\(2a = 1\) → \(a = \dfrac{1}{2}\)

(i) \(a \leq 0\)：最小 = \(f(0) = 2\)

(ii) \(0 < a < \dfrac{1}{2}\)：最小 = 頂点 \(= -4a^2 + 2\)

(iii) \(a \geq \dfrac{1}{2}\)：最小 = \(f(1) = 3 - 4a\)

📌 まとめ：文字が軸にあれば「軸の位置」、定義域にあれば「端点の位置」で場合分け！

おまけ：数学雑学「ロケット軌道と場合分け」

🚀 アポロ計画と二次関数

1969年、NASAは人類初の月面着陸を目指していた。ロケットの軌道計算は、まさに二次関数の最大最小問題だった。

ロケットは打ち上げ後、放物線を描いて飛んでいく。燃料を使い切るタイミング、地球の重力から逃れる速度、月に到達するための軌道修正...すべてが計算だ。

特に重要なのは「場合分け」の発想。

  • 打ち上げ角度が浅すぎる → 大気圏を抜けられない
  • 打ち上げ角度が深すぎる → 燃料が足りなくなる
  • 打ち上げ角度がちょうどいい → 月に到達できる

この「ちょうどいい範囲」を見つけるのが、まさに定義域と軸の位置関係の問題だった。

アポロ11号の成功は、こうした計算の積み重ねの結果。アームストロングが月面に降り立つ前に、何百人もの数学者が二次関数と場合分けと格闘していたのだ。

...ちなみにアポロ計画当時のコンピュータは、今のスマホよりはるかに性能が低かった。それでも月に行けたのは、正確な数学があったからこそ。
