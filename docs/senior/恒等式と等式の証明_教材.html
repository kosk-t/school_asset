1. テーマの定義

数学では「いつでも成り立つ等式」と「証明すべき等式・不等式」を区別します。恒等式の判定や、有名な不等式を使いこなせると、大学入試で強力な武器になります。

このページでは、以下の内容を学びます：

  • 恒等式の判定：係数比較法と数値代入法で未知の係数を決定
  • 等式の証明：条件付き等式を証明するテクニック
  • 相加相乗平均の不等式：2数の平均から最小値を求める
  • コーシー・シュワルツの不等式：内積と絶対値の関係を使う強力な不等式

2. 覚えるルール

【最重要】恒等式は「すべての \(x\) で成り立つ」→ 係数を比較できる！
方程式と恒等式の違い
📌 ルール1：係数比較法 → 例1

恒等式 \(ax^2 + bx + c = a'x^2 + b'x + c'\) がすべての \(x\) で成り立つ

\[ \Leftrightarrow a = a', \quad b = b', \quad c = c' \]

→ だから： 両辺の同じ次数の係数を比較して連立方程式を作る

✅ メリット： どんな恒等式でも確実に解ける

📌 ルール2：数値代入法 → 例2

恒等式は「すべての \(x\)」で成立 → 都合の良い値を代入してOK

\[ x = 0, 1, -1, 2, \ldots \text{を代入して係数を決定} \]

→ だから： 計算が簡単になる値を選んで代入

✅ メリット： 係数比較より計算が楽なことが多い

💡 考え方を見る

分数式の部分分数分解では、分母が0になる値を代入すると式が大幅に簡略化されます。

📌 ルール3：相加相乗平均の不等式（AM-GM不等式） → 例3

\(a > 0, b > 0\) のとき

\[ \frac{a + b}{2} \geq \sqrt{ab} \]

（左辺：相加平均、右辺：相乗平均）

→ だから： 「和が一定なら積は最大」「積が一定なら和は最小」

等号成立条件： \(a = b\) のとき

💡 考え方を見る

\((\sqrt{a} - \sqrt{b})^2 \geq 0\) を展開すると \(a - 2\sqrt{ab} + b \geq 0\)、整理すると \(\frac{a+b}{2} \geq \sqrt{ab}\)

📌 ルール4：コーシー・シュワルツの不等式 → 例4

実数 \(a, b, c, d\) に対して

\[ (a^2 + b^2)(c^2 + d^2) \geq (ac + bd)^2 \]

→ だから： 「2乗の和」×「2乗の和」≧「内積」²

等号成立条件： \(\frac{a}{c} = \frac{b}{d}\)（比例関係）のとき

💡 考え方を見る

展開すると \(a^2c^2 + a^2d^2 + b^2c^2 + b^2d^2 \geq a^2c^2 + 2abcd + b^2d^2\)、整理すると \((ad - bc)^2 \geq 0\)

📌 ルール5：等式の証明の基本方針 → 例5

「A = B」を証明する3つの方法：

  1. A を変形して B を導く（最も基本的）
  2. A - B = 0 を示す（差をとって0になることを示す）
  3. A = C かつ B = C を示す（共通の式Cに変形）

3. 例題と解き方

恒等式の係数決定と不等式の証明を、実際の例題で確認しましょう。

▶ 恒等式の係数決定

例1係数比較法

\(x^2 + 5x + 6 = a(x+1)^2 + b(x+1) + c\) がすべての \(x\) で成り立つとき、定数 \(a, b, c\) を求めよ。

解き方：

  1. 右辺を展開して整理する
  2. 両辺の係数を比較する
  3. 連立方程式を解く

右辺を展開：

\[ a(x^2 + 2x + 1) + bx + b + c = ax^2 + (2a+b)x + (a+b+c) \]

係数比較：

\[ x^2: a = 1, \quad x: 2a + b = 5, \quad \text{定数}: a + b + c = 6 \]

\(a = 1\) を代入：\(b = 3\)、\(c = 2\)

答え：\(a = 1, b = 3, c = 2\)

例2数値代入法（部分分数分解）

\(\displaystyle\frac{1}{x(x+1)} = \frac{a}{x} + \frac{b}{x+1}\) が \(x \neq 0, -1\) で成り立つとき、定数 \(a, b\) を求めよ。

解き方：

  1. 両辺に \(x(x+1)\) をかけて分母を払う
  2. 都合の良い値（分母が0になる値に近い値）を代入
  3. \(a, b\) を決定する

分母を払う：

\[ 1 = a(x+1) + bx \]

\(x = 0\) を代入：\(1 = a \cdot 1 + b \cdot 0\) より \(a = 1\)

\(x = -1\) を代入：\(1 = a \cdot 0 + b \cdot (-1)\) より \(b = -1\)

答え：\(a = 1, b = -1\)

※ 実際に \(\displaystyle\frac{1}{x} - \frac{1}{x+1} = \frac{1}{x(x+1)}\) を確認できます

▶ 相加相乗平均の不等式

例3最小値を求める

\(x > 0\) のとき、\(x + \displaystyle\frac{4}{x}\) の最小値を求めよ。

解き方：

  1. 相加相乗平均の不等式を適用する形を確認
  2. \(a = x, b = \frac{4}{x}\) として不等式を適用
  3. 等号成立条件を確認

相加相乗平均より：

\[ x + \frac{4}{x} \geq 2\sqrt{x \cdot \frac{4}{x}} = 2\sqrt{4} = 4 \]

等号成立：\(x = \frac{4}{x}\) つまり \(x^2 = 4\)、\(x > 0\) より \(x = 2\)

答え：最小値は \(4\)（\(x = 2\) のとき）

例3'条件付き最大値

\(x > 0, y > 0\) かつ \(x + y = 10\) のとき、\(xy\) の最大値を求めよ。

解き方：

  1. 「和が一定なら積は最大」を使う（相加相乗の逆利用）
  2. 相加相乗平均を変形
  3. 等号成立条件を確認

相加相乗平均より \(\displaystyle\frac{x + y}{2} \geq \sqrt{xy}\)

\(x + y = 10\) を代入：

\[ 5 \geq \sqrt{xy} \quad \Rightarrow \quad xy \leq 25 \]

等号成立：\(x = y = 5\) のとき

答え：最大値は \(25\)（\(x = y = 5\) のとき）

▶ コーシー・シュワルツの不等式

例4コーシー・シュワルツの適用

\(a^2 + b^2 = 1\) のとき、\(3a + 4b\) の最大値を求めよ。

解き方：

  1. コーシー・シュワルツを適用する形を作る
  2. \((a^2 + b^2)(3^2 + 4^2) \geq (3a + 4b)^2\) を利用
  3. 等号成立条件を確認

コーシー・シュワルツより：

\[ (a^2 + b^2)(3^2 + 4^2) \geq (3a + 4b)^2 \]

\(a^2 + b^2 = 1\) を代入：

\[ 1 \times 25 \geq (3a + 4b)^2 \quad \Rightarrow \quad |3a + 4b| \leq 5 \]

等号成立：\(\displaystyle\frac{a}{3} = \frac{b}{4}\) かつ \(a^2 + b^2 = 1\)

つまり \(a = \frac{3}{5}, b = \frac{4}{5}\) のとき

答え：最大値は \(5\)

▶ 等式の証明

例5条件付き等式の証明

\(a + b + c = 0\) のとき、\(a^3 + b^3 + c^3 = 3abc\) を証明せよ。

解き方：

  1. 左辺 - 右辺 = 0 を示す方針
  2. 因数分解の公式を利用
  3. 条件 \(a + b + c = 0\) を代入

因数分解の公式：

\[ a^3 + b^3 + c^3 - 3abc = (a + b + c)(a^2 + b^2 + c^2 - ab - bc - ca) \]

\(a + b + c = 0\) のとき、右辺 = \(0 \times (\cdots) = 0\)

よって \(a^3 + b^3 + c^3 - 3abc = 0\)

したがって \(a^3 + b^3 + c^3 = 3abc\)（証明完了）

4. 簡単にするコツ

✏️ コツ1：係数比較と数値代入の使い分け
  • 多項式の恒等式 → 係数比較法が確実
  • 分数式の部分分数分解 → 数値代入法が楽
部分分数分解では「分母が0になる値」を代入すると、項が消えて計算が楽になります。
✏️ コツ2：相加相乗平均は「積が定数」を作る
\(x + \frac{k}{x}\) の形なら、積は \(x \cdot \frac{k}{x} = k\) で定数。
この形を見つけたら、すぐに相加相乗平均を適用！
最小値 = \(2\sqrt{k}\)、等号成立は \(x = \sqrt{k}\)
✏️ コツ3：コーシー・シュワルツは「2乗の制約」があるとき
条件に \(a^2 + b^2 = 1\) のような「2乗の和」が出たら、コーシー・シュワルツを疑う！
求めたい式が \(pa + qb\) の形なら、\((a^2 + b^2)(p^2 + q^2) \geq (pa + qb)^2\) を使う。
✏️ コツ4：等号成立条件を忘れずに
不等式で最大値・最小値を求めるときは、必ず等号成立条件を確認！
条件を満たす \(x\) の値が存在するかどうかで、その最大値・最小値が実際に達成されるかが決まります。

5. よくあるつまづき

⚠️ つまづき1：恒等式と方程式の混同

ダメな例： 「\(x^2 + 2x + 1 = (x+1)^2\) を解け」と言われて \(x\) を求めようとする

対策： これは恒等式（すべての \(x\) で成立）なので、「解」は存在しない。恒等式は「成り立つことを確認する」または「係数を求める」問題。

⚠️ つまづき2：相加相乗平均を負の数に適用

ダメな例： \(x < 0\) のとき \(x + \frac{1}{x} \geq 2\sqrt{x \cdot \frac{1}{x}} = 2\)

対策： 相加相乗平均は \(a > 0, b > 0\) のときのみ成立！
\(x < 0\) なら \(-x > 0, -\frac{1}{x} > 0\) として適用するか、別の方法を考える。

⚠️ つまづき3：等号成立条件の確認忘れ

ダメな例： 「最小値は2」と答えるが、等号が成立する条件を書かない

対策： 不等式の最大・最小を求めたら、必ず「等号成立は○○のとき」を明記。
等号が成立しない場合は「最小値」ではなく「下限」と言う。

⚠️ つまづき4：部分分数分解で係数の数が合わない

ダメな例： \(\displaystyle\frac{1}{x^2(x+1)}\) を \(\frac{a}{x} + \frac{b}{x+1}\) と置く

対策： 分母の重複度に応じて項を増やす。\(x^2\) があれば \(\frac{a}{x} + \frac{b}{x^2}\) の両方が必要！
正しくは \(\displaystyle\frac{a}{x} + \frac{b}{x^2} + \frac{c}{x+1}\)

6. 仕上げチェックリスト

恒等式の係数比較法で、同じ次数の係数を正しく比較できる
数値代入法で、計算が楽になる値を選べる
相加相乗平均を \(a > 0, b > 0\) を確認してから適用できる
コーシー・シュワルツの不等式を「2乗の制約」で使える
不等式の最大・最小を求めたとき、等号成立条件を書ける

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(\displaystyle\frac{2x+1}{(x-1)(x+2)} = \frac{a}{x-1} + \frac{b}{x+2}\) が \(x \neq 1, -2\) で成り立つとき、定数 \(a, b\) を求めよ。

答えを見る

両辺に \((x-1)(x+2)\) をかけて：

\(2x + 1 = a(x+2) + b(x-1)\)

\(x = 1\) を代入：\(3 = 3a\) より \(a = 1\)

\(x = -2\) を代入：\(-3 = -3b\) より \(b = 1\)

答え：\(a = 1, b = 1\)

練習2

\(x > 0\) のとき、\(2x + \displaystyle\frac{8}{x}\) の最小値とそのときの \(x\) の値を求めよ。

答えを見る

相加相乗平均より：

\(2x + \frac{8}{x} \geq 2\sqrt{2x \cdot \frac{8}{x}} = 2\sqrt{16} = 8\)

等号成立：\(2x = \frac{8}{x}\) より \(x^2 = 4\)、\(x > 0\) より \(x = 2\)

答え：最小値は \(8\)（\(x = 2\) のとき）

練習3

\(x^2 + y^2 = 2\) のとき、\(x + y\) の最大値を求めよ。

答えを見る

コーシー・シュワルツより：

\((x^2 + y^2)(1^2 + 1^2) \geq (x \cdot 1 + y \cdot 1)^2\)

\(2 \times 2 \geq (x + y)^2\) より \((x + y)^2 \leq 4\)

よって \(|x + y| \leq 2\)、つまり \(x + y \leq 2\)

等号成立：\(\frac{x}{1} = \frac{y}{1}\) より \(x = y\)、\(x^2 + y^2 = 2\) より \(x = y = 1\)

答え：最大値は \(2\)（\(x = y = 1\) のとき）

📌 まとめ：恒等式は「係数比較」、不等式は「相加相乗とコーシー・シュワルツ」で攻略！

おまけ：数学雑学「相加相乗と古代バビロニア」

📖 4000年前の粘土板に眠る不等式

相加相乗平均の不等式 \(\frac{a+b}{2} \geq \sqrt{ab}\) は、高校数学で初めて習う人が多いですが、実はこの不等式の「精神」は古代バビロニア（紀元前2000年頃）まで遡ります。

バビロニアの数学者たちは、「長方形の面積が一定なら、正方形に近いほど周の長さが短い」ことを知っていました。これは相加相乗平均の幾何学的な表現です。

\[ \text{面積} = ab \text{ が一定} \Rightarrow \text{周の長さ} = 2(a+b) \text{ は } a = b \text{ で最小} \]

面白いのは、彼らがこれを「実用的な理由」で発見したこと。古代の土地測量では、同じ面積の土地を囲むなら、正方形が最も柵の材料が少なくて済むことが重要でした。数学の定理が「節約」から生まれたのです。

ちなみに、この不等式の現代的な証明は、フランスの数学者コーシー（1789-1857）によって一般化されました。コーシー・シュワルツの不等式も、実は相加相乗平均の「親戚」なのです。

結論：数学は「ケチな人」のために発展した。（材料を節約したい古代人が、最適化問題を解いていたのですから！）
