1. テーマの定義

「雨が降ったとき、傘を忘れる確率は？」「コインを5回投げて3回表が出る確率は？」こうした「条件がついた確率」や「繰り返しの試行」を扱うのがこのテーマです。

このページでは、条件付き確率と独立試行を学びます：

  • 条件付き確率：「Bが起きたとき」Aが起きる確率
  • 乗法定理：「AかつB」の確率の求め方
  • 独立試行：結果が他に影響しない試行
  • 反復試行の確率：n回中r回成功する確率
  • 期待値：「平均的にどうなるか」を数値化

2. 覚えるルール

🔗 最重要：条件付き確率 \(P(A|B) = \frac{P(A \cap B)}{P(B)}\)（Bが起きたとき、Aが起きる確率）
条件付き確率は「Bを全体として見たときのA∩Bの割合」
📌 ルール1：条件付き確率 → 例1
\[ P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{n(A \cap B)}{n(B)} \]

\(P(A|B)\)：「Bが起きたという条件のもとでAが起きる確率」

読み方：「BのもとでのAの確率」「Bが与えられたときのAの確率」

→ だから結果は：Bが起きた場合に限定して、その中でAも起きる割合を計算

💡 考え方を見る

条件付き確率は「視点の変更」。全事象Uではなく、Bを「新しい全体」として考える。

Bの中でA∩Bが占める割合が、条件付き確率になる。

📌 ルール2：乗法定理 → 例2
\[ P(A \cap B) = P(B) \cdot P(A|B) = P(A) \cdot P(B|A) \]

意味：「AかつB」の確率 = Bの確率 × BのもとでAが起きる確率

✅ メリット：条件付き確率がわかっているとき、同時確率を計算できる

📌 ルール3：独立な事象 → 例3
\[ \text{AとBが独立} \Leftrightarrow P(A \cap B) = P(A) \cdot P(B) \]

独立：一方の結果が他方に影響しない

独立のとき：\(P(A|B) = P(A)\)、\(P(B|A) = P(B)\)

例：サイコロ2回の結果は独立。くじを戻さずに引くのは独立でない。

📌 ルール4：反復試行の確率 → 例4
\[ P(\text{n回中r回成功}) = {}_nC_r \cdot p^r \cdot (1-p)^{n-r} \]

反復試行：同じ試行を独立に繰り返す

\(p\)：1回の試行で成功する確率

\(_nC_r\)：n回中どのr回で成功するかの組合せ

例：コインを5回投げて3回表が出る確率

💡 考え方を見る

「◯◯◯××」のような特定のパターンの確率は \(p^3 \times (1-p)^2\)。

でも「3回成功」は「◯◯◯××」「◯◯×◯×」など複数パターンがある。

その数が \(_5C_3 = 10\) 通りなので、掛け合わせる。

📌 ルール5：期待値 → 例5
\[ E(X) = x_1 p_1 + x_2 p_2 + \cdots + x_n p_n = \sum_{i=1}^{n} x_i p_i \]

期待値：確率変数の「平均的な値」

\(x_i\)：起こりうる値、\(p_i\)：その確率

例：サイコロの期待値 = \(1 \times \frac{1}{6} + 2 \times \frac{1}{6} + \cdots + 6 \times \frac{1}{6} = 3.5\)

📌 ルール6：ベイズの定理（発展）
\[ P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)} \]

意味：結果Bから原因Aを推測する「逆確率」

使いどころ：「検査が陽性だったとき、本当に病気である確率」など

3. 例題と解き方

条件付き確率と独立試行の計算を実際にやってみましょう。

▶ 条件付き確率

例1表で見る条件付き確率

40人のクラスで、数学が好きな生徒は24人、理科が好きな生徒は20人、両方好きな生徒は15人いる。理科が好きな生徒の中で、数学も好きな生徒の割合（確率）を求めよ。

解き方：

  1. A：数学が好き、B：理科が好き とする
  2. 求めるのは \(P(A|B)\)（Bのもとでのaの確率）
  3. \(P(A \cap B) = \frac{15}{40}\)、\(P(B) = \frac{20}{40}\)
\[ P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{15/40}{20/40} = \frac{15}{20} = \frac{3}{4} \]

または、場合の数で直接計算：\(P(A|B) = \frac{n(A \cap B)}{n(B)} = \frac{15}{20} = \frac{3}{4}\)

▶ 乗法定理

例2くじを戻さずに引く

当たりが3本、はずれが7本の計10本のくじがある。くじを1本引き、戻さずにもう1本引くとき、2本とも当たりである確率を求めよ。

解き方：

  1. A：1本目が当たり、B：2本目が当たり とする
  2. 1本目が当たりの確率：\(P(A) = \frac{3}{10}\)
  3. 1本目が当たりだったとき、2本目が当たりの確率：\(P(B|A) = \frac{2}{9}\)（残り9本中2本が当たり）
  4. 乗法定理を適用
\[ P(A \cap B) = P(A) \cdot P(B|A) = \frac{3}{10} \times \frac{2}{9} = \frac{6}{90} = \frac{1}{15} \]

▶ 独立な事象

例3サイコロ2回の独立試行

サイコロを2回投げるとき、1回目が偶数、2回目が3以上である確率を求めよ。

解き方：

  1. 2回の結果は独立（1回目の結果が2回目に影響しない）
  2. A：1回目が偶数 → \(P(A) = \frac{3}{6} = \frac{1}{2}\)
  3. B：2回目が3以上 → \(P(B) = \frac{4}{6} = \frac{2}{3}\)
  4. 独立なので、単純に掛ける
\[ P(A \cap B) = P(A) \times P(B) = \frac{1}{2} \times \frac{2}{3} = \frac{1}{3} \]

▶ 反復試行

例4コインを5回投げて3回表

コインを5回投げるとき、ちょうど3回表が出る確率を求めよ。

解き方：

  1. n = 5回、r = 3回成功、p = \(\frac{1}{2}\)
  2. 反復試行の公式を適用
\[ P = {}_5C_3 \times \left(\frac{1}{2}\right)^3 \times \left(\frac{1}{2}\right)^2 = 10 \times \frac{1}{8} \times \frac{1}{4} = \frac{10}{32} = \frac{5}{16} \]
例4bサイコロで6が2回出る

サイコロを4回投げるとき、6の目がちょうど2回出る確率を求めよ。

解き方：

  1. n = 4回、r = 2回成功、p = \(\frac{1}{6}\)（6が出る確率）
  2. 1 - p = \(\frac{5}{6}\)（6以外が出る確率）
\[ P = {}_4C_2 \times \left(\frac{1}{6}\right)^2 \times \left(\frac{5}{6}\right)^2 = 6 \times \frac{1}{36} \times \frac{25}{36} = \frac{150}{1296} = \frac{25}{216} \]

▶ 期待値

例5くじの期待値

1等1000円が1本、2等200円が3本、はずれ（0円）が6本のくじがある。1回くじを引いたときの賞金の期待値を求めよ。

解き方：

  1. 確率変数X（賞金）の取りうる値：1000円、200円、0円
  2. それぞれの確率：\(\frac{1}{10}\)、\(\frac{3}{10}\)、\(\frac{6}{10}\)
  3. 期待値を計算
\[ E(X) = 1000 \times \frac{1}{10} + 200 \times \frac{3}{10} + 0 \times \frac{6}{10} = 100 + 60 + 0 = 160 \text{円} \]

解釈：このくじを何度も引くと、平均して1回あたり160円もらえる

▶ ベイズの定理

例6製造ラインの不良品

工場のライン1は全体の60%、ライン2は40%を製造する。ライン1の不良率は2%、ライン2の不良率は5%である。製品が不良品だったとき、それがライン1で作られた確率を求めよ。

解き方：

  1. A：ライン1で製造、B：不良品 とする
  2. \(P(A) = 0.6\)、\(P(B|A) = 0.02\)
  3. \(P(\overline{A}) = 0.4\)、\(P(B|\overline{A}) = 0.05\)
  4. まず \(P(B)\) を計算（全確率の公式）
\[ P(B) = P(A) \cdot P(B|A) + P(\overline{A}) \cdot P(B|\overline{A}) = 0.6 \times 0.02 + 0.4 \times 0.05 = 0.012 + 0.02 = 0.032 \]

ベイズの定理を適用：

\[ P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)} = \frac{0.02 \times 0.6}{0.032} = \frac{0.012}{0.032} = \frac{3}{8} = 0.375 \]

答え：37.5%（不良品がライン1製である確率）

4. 簡単にするコツ

✏️ コツ1：「独立」か「従属」かを最初に判断
独立（前の結果が後に影響しない）→ 単純に掛け算 P(A)×P(B)
従属（影響する）→ 乗法定理 P(A)×P(B|A)
✏️ コツ2：条件付き確率は「分母を変える」
\(P(A|B)\) は「Bの中でのA∩Bの割合」。
全事象ではなく、Bを新しい「全体」として計算する。
✏️ コツ3：反復試行は「組合せ × 確率のべき乗」
n回中r回成功 = \(_nC_r\)（どのr回か）× \(p^r\)（成功r回）× \((1-p)^{n-r}\)（失敗n-r回）
✏️ コツ4：期待値は「値×確率の和」
各結果の「値」と「確率」を掛けて全部足す。
確率の合計が1になっているか確認すると計算ミスを防げる。

5. よくあるつまづき

⚠️ つまづき1：独立と従属を間違える

ダメな例：くじを戻さずに引くとき、P(2本とも当たり) = \(\frac{3}{10} \times \frac{3}{10}\)

対策：戻さない → 従属。2本目は \(\frac{2}{9}\) になる

⚠️ つまづき2：\(P(A|B)\) と \(P(B|A)\) を混同

ダメな例：「病気のとき陽性の確率」と「陽性のとき病気の確率」を同じと思う

対策：条件（分母）が違えば、確率は違う。「〜のとき」が何かを確認

⚠️ つまづき3：反復試行で \(_nC_r\) を忘れる

ダメな例：5回中3回表の確率を \((\frac{1}{2})^3 \times (\frac{1}{2})^2 = \frac{1}{32}\) とする

対策：「◯◯◯××」以外にも「◯◯×◯×」などがある。\(_5C_3 = 10\) 通りを掛ける

⚠️ つまづき4：期待値の計算で確率を確認しない

ダメな例：確率の合計が1にならないのに気づかず計算を進める

対策：確率の和が1か確認。1でなければどこか間違っている

6. 仕上げチェックリスト

条件付き確率 \(P(A|B) = \frac{P(A \cap B)}{P(B)}\) が使える
乗法定理 \(P(A \cap B) = P(A) \cdot P(B|A)\) が使える
独立と従属を正しく判断できる
反復試行の公式 \(_nC_r \cdot p^r \cdot (1-p)^{n-r}\) が使える
期待値 \(E(X) = \sum x_i p_i\) が計算できる

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

赤玉4個、白玉6個が入った袋から、1個取り出し、戻さずにもう1個取り出す。2個とも赤玉である確率を求めよ。

答えを見る

戻さないので従属。乗法定理を使う。

\(P = \frac{4}{10} \times \frac{3}{9} = \frac{12}{90} = \frac{2}{15}\)

練習2

コインを6回投げるとき、ちょうど4回表が出る確率を求めよ。

答えを見る

反復試行の公式：n=6, r=4, p=1/2

\(P = {}_6C_4 \times \left(\frac{1}{2}\right)^4 \times \left(\frac{1}{2}\right)^2 = 15 \times \frac{1}{16} \times \frac{1}{4} = \frac{15}{64}\)

練習3

1等500円が2本、2等100円が5本、はずれが3本のくじがある。1回引いたときの賞金の期待値を求めよ。

答えを見る

\(E(X) = 500 \times \frac{2}{10} + 100 \times \frac{5}{10} + 0 \times \frac{3}{10}\)

\(= 100 + 50 + 0 = 150\) 円

📌 まとめ：条件付き確率は「視点の変更」、独立なら掛けるだけ、反復試行は組合せ×べき乗！

おまけ：確率パラドックス「誕生日の奇跡」

📖 23人で誕生日が一致する確率は？

ある日のクラスで、先生が質問した。「このクラス40人の中に、同じ誕生日の人がいると思う人、手を挙げて！」

生徒たちは首をかしげた。「365日もあるのに、40人で一致するわけないでしょ」

先生はニヤリと笑って言った。「じゃあ実際に調べてみよう」

結果、なんと3組も同じ誕生日のペアがいた！

実は、確率を計算すると...

23人いれば、同じ誕生日の人がいる確率は50%を超える。40人なら約89%、50人では97%にもなる！

なぜこんな「意外な」結果になるのか？

ポイントは「ペアの数」だ。23人いると、誕生日を比べるペアは \({_{23}}C_2 = 253\) 組もある。「誰かと誰かが一致」のチャンスがこれだけあるから、50%を超えるのだ。

余事象を使って計算しよう

「全員バラバラ」の確率を求めて、1から引く方が簡単：

\[ P(\text{全員バラバラ}) = \frac{365}{365} \times \frac{364}{365} \times \frac{363}{365} \times \cdots \times \frac{343}{365} \]

これを計算すると約0.493。つまり「誰かが一致」する確率は 1 - 0.493 = 約50.7%！

サッカーW杯での検証

2014年ブラジルW杯の32チーム中、なんと16チームで同じ誕生日の選手がいた。各チーム23人なので、理論値50%とほぼ一致！

教訓

「365日もあるから大丈夫」という直感は、「比べるペアの数」を見落としている。人間の直感と確率計算の結果は、しばしば大きく食い違う。だから確率は面白いのだ。

...ちなみに、「自分と同じ誕生日の人がいる」確率は、23人でもたった約6%。「誰かと誰か」と「自分と誰か」では全然違うのだ！
