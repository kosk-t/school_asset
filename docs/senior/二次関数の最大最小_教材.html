1. テーマの定義

「ボールを投げたとき、最も高くなるのは何秒後？」「利益を最大にする価格設定は？」——こうした問題は二次関数の最大・最小で解けます。

このページでは、二次関数の最大最小を学びます：

  • 定義域がない場合：頂点が最大または最小になる
  • 定義域がある場合：軸と定義域の位置関係で場合分け
  • 軸が動く問題：パラメータを含む最大最小

2. 覚えるルール

🎯 最重要：最大・最小は「軸が定義域の中にあるかどうか」で決まる！
軸と定義域の位置関係で最小値の場所が変わる
📌 ルール1：定義域がない場合 → 例1

\(y = a(x - p)^2 + q\) で定義域が全実数のとき：

  • \(a > 0\)（下に凸）：最小値 \(q\)（\(x = p\) で）、最大値なし
  • \(a < 0\)（上に凸）：最大値 \(q\)（\(x = p\) で）、最小値なし

→ だから結果は： 頂点の \(y\) 座標が最大または最小になる

📌 ルール2：定義域がある場合（下に凸） → 例2・3

定義域 \([a, b]\) で \(y = f(x)\)（\(a > 0\)、下に凸）の最大・最小：

【最小値】軸が定義域の中にあるかで判定

  • 軸 \(x = p\) が \([a, b]\) 内 → 最小値は頂点
  • 軸が \(a\) より左 → 最小値は \(f(a)\)
  • 軸が \(b\) より右 → 最小値は \(f(b)\)

【最大値】端点で比較

  • 最大値は \(f(a)\) と \(f(b)\) の大きい方
📌 ルール3：定義域がある場合（上に凸）

上に凸（\(a < 0\)）の場合は、最大と最小が逆になる：

  • 軸が定義域内 → 最大値が頂点
  • 最小値は端点 \(f(a)\) と \(f(b)\) の小さい方
📌 ルール4：軸が動く問題 → 例4

軸にパラメータ（文字）が含まれる場合、軸の位置で場合分け：

  1. 軸が定義域の左端より左のとき
  2. 軸が定義域の中にあるとき
  3. 軸が定義域の右端より右のとき

✅ メリット： どんな問題でも、場合分けで確実に解ける

3. 例題と解き方

ルールを使って実際に問題を解いてみましょう。

▶ 定義域がない場合

例1\(y = 2x^2 - 4x + 5\) の最大値・最小値を求める

解き方：

  1. 平方完成：\(y = 2(x^2 - 2x) + 5 = 2(x - 1)^2 - 2 + 5 = 2(x - 1)^2 + 3\)
  2. 頂点 \((1, 3)\)、\(a = 2 > 0\) で下に凸
  3. 定義域がないので、頂点で最小
\[ \text{最小値}：3 \quad (x = 1 \text{ のとき}), \quad \text{最大値}：\text{なし} \]

▶ 定義域がある場合

例2\(y = x^2 - 4x + 3\)（\(0 \leq x \leq 3\)）の最大値・最小値を求める

解き方：

  1. 平方完成：\(y = (x - 2)^2 - 1\)、頂点 \((2, -1)\)
  2. 軸 \(x = 2\) は定義域 \([0, 3]\) の中にある
  3. 最小値は頂点で \(-1\)
  4. 最大値は端点で比較：\(f(0) = 3\)、\(f(3) = 0\) → 大きい方は \(3\)
\[ \text{最小値}：-1 \quad (x = 2), \quad \text{最大値}：3 \quad (x = 0) \]
例3\(y = x^2 - 4x + 3\)（\(3 \leq x \leq 5\)）の最大値・最小値を求める

解き方：

  1. 平方完成：\(y = (x - 2)^2 - 1\)、軸 \(x = 2\)
  2. 軸 \(x = 2\) は定義域 \([3, 5]\) の左にある
  3. 下に凸なので、定義域内では右に行くほど増加
  4. 最小値は左端 \(f(3) = 0\)、最大値は右端 \(f(5) = 8\)
\[ \text{最小値}：0 \quad (x = 3), \quad \text{最大値}：8 \quad (x = 5) \]

▶ 軸が動く問題

例4\(y = x^2 - 2ax + 1\)（\(0 \leq x \leq 2\)）の最小値を \(a\) で表す

解き方：

  1. 平方完成：\(y = (x - a)^2 - a^2 + 1\)、軸 \(x = a\)
  2. 軸 \(x = a\) と定義域 \([0, 2]\) の関係で場合分け

場合分け：

(i) \(a < 0\) のとき：軸が定義域の左 → 最小値は \(f(0) = 1\)

(ii) \(0 \leq a \leq 2\) のとき：軸が定義域の中 → 最小値は頂点で \(-a^2 + 1\)

(iii) \(a > 2\) のとき：軸が定義域の右 → 最小値は \(f(2) = 5 - 4a\)

\[ \text{最小値} = \begin{cases} 1 & (a < 0) \\ -a^2 + 1 & (0 \leq a \leq 2) \\ 5 - 4a & (a > 2) \end{cases} \]

▶ 上に凸の場合

例5\(y = -x^2 + 4x + 1\)（\(0 \leq x \leq 3\)）の最大値・最小値を求める

解き方：

  1. 平方完成：\(y = -(x^2 - 4x) + 1 = -(x - 2)^2 + 5\)
  2. \(a = -1 < 0\) で上に凸、軸 \(x = 2\) は \([0, 3]\) 内
  3. 最大値は頂点で \(5\)
  4. 最小値は端点比較：\(f(0) = 1\)、\(f(3) = 4\) → 小さい方は \(1\)
\[ \text{最大値}：5 \quad (x = 2), \quad \text{最小値}：1 \quad (x = 0) \]

4. 簡単にするコツ

✏️ コツ1：まず平方完成で軸を出す
どんな問題でも、まず \(y = a(x - p)^2 + q\) の形にして軸 \(x = p\) を求める。軸が分かれば場合分けができる。
✏️ コツ2：グラフの概形を描く
放物線の概形と定義域を図示すると、どこで最大・最小になるか一目で分かる。
✏️ コツ3：「中か外か」で判定
軸が定義域の「中」→ 頂点で最小（下に凸）または最大（上に凸）。軸が「外」→ 端点で最小・最大。
✏️ コツ4：場合分けは「境界値」を確認
軸 \(x = a\) が定義域 \([0, 2]\) を通過する境界は \(a = 0\) と \(a = 2\)。これを基準に場合分け。

5. よくあるつまづき

⚠️ つまづき1：軸が定義域外なのに頂点を答える

ダメな例： \(y = (x-5)^2 + 1\)（\(0 \leq x \leq 3\)）の最小値を \(1\) と答える

対策： 軸 \(x = 5\) は \([0, 3]\) の外。最小値は \(f(3) = 5\) が正解。

⚠️ つまづき2：最大値と最小値を逆にする

ダメな例： 上に凸なのに「頂点で最小値」と答える

対策： 上に凸→頂点で最大。下に凸→頂点で最小。グラフを描いて確認！

⚠️ つまづき3：場合分けの境界を間違える

ダメな例： 定義域 \([0, 3]\) で軸 \(x = a\) の場合分けを「\(a < 0, 0 < a < 3, a > 3\)」とする

対策： \(a = 0\) や \(a = 3\) を含める！「\(a \leq 0\)」「\(0 \leq a \leq 3\)」「\(a \geq 3\)」のように、等号の位置に注意。

6. 仕上げチェックリスト

定義域がない場合、頂点で最大または最小になることを理解している
軸と定義域の位置関係で場合分けができる
下に凸・上に凸で最大最小が逆になることを覚えている
端点の値を正確に計算できる
パラメータを含む問題で正しく場合分けできる

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(y = x^2 - 6x + 5\)（\(1 \leq x \leq 5\)）の最大値と最小値を求めよ。

答えを見る

\(y = (x - 3)^2 - 4\)、軸 \(x = 3\) は \([1, 5]\) の中

最小値：\(-4\)（\(x = 3\)）

\(f(1) = 0\)、\(f(5) = 0\) より最大値：\(0\)（\(x = 1, 5\)）

練習2

\(y = -x^2 + 2x + 3\)（\(-1 \leq x \leq 2\)）の最大値と最小値を求めよ。

答えを見る

\(y = -(x - 1)^2 + 4\)、上に凸、軸 \(x = 1\) は \([-1, 2]\) の中

最大値：\(4\)（\(x = 1\)）

\(f(-1) = 0\)、\(f(2) = 3\) より最小値：\(0\)（\(x = -1\)）

練習3

\(y = x^2 - 4x + 5\)（\(0 \leq x \leq a\)）の最小値が \(1\) となる \(a\) の範囲を求めよ。

答えを見る

\(y = (x - 2)^2 + 1\)、軸 \(x = 2\)、頂点の \(y\) 座標は \(1\)

最小値 \(1\) になるのは、軸 \(x = 2\) が定義域に含まれるとき

よって \(a \geq 2\)

📌 まとめ：軸が定義域の中なら頂点で最大or最小！外なら端点で比較！

おまけ：数学と社会「利益を最大化する数学」

📖 価格設定と二次関数

ある会社が新商品を売り出すことになった。価格を決めなければならない。

高く売れば1個あたりの利益は増えるが、買う人は減る。安く売れば多く売れるが、1個あたりの利益は減る。

「いくらで売れば、利益が最大になるか？」

マーケティング調査の結果、こんな関係が分かった：

  • 価格を \(x\) 円上げると、販売数が \(100 - 2x\) 個になる
  • 1個あたりの利益は \(x\) 円増える

すると、総利益 \(y\) は：

\[ y = x(100 - 2x) = -2x^2 + 100x \]

これは上に凸の二次関数！頂点で最大値を取る。

\[ y = -2(x^2 - 50x) = -2(x - 25)^2 + 1250 \]

軸は \(x = 25\)。つまり25円値上げしたとき、利益が最大になる。

このような考え方は「最適化問題」と呼ばれ、企業の経営戦略から資源配分まで、あらゆる場面で使われている。

二次関数の最大最小は、ビジネスを動かす数学だった。
