1. テーマの定義

円は古代から研究されてきた美しい図形。円周角・接線・方べきの定理を使うと、複雑な図形問題もスマートに解けます。

このページでは、円の性質（高校）を学びます：

  • 円周角の定理（復習）：同じ弧に対する円周角は等しい
  • 円に内接する四角形：対角の和が180°
  • 接弦定理：接線と弦のなす角
  • 方べきの定理：円と直線の交点に関する定理
  • 2円の位置関係：共通接線と交点

2. 覚えるルール

⭕ 最重要：方べきの定理 PA · PB = PC · PD（点Pから円への2本の直線）
📌 ルール1：円周角の定理（復習）
\[ \text{円周角} = \frac{1}{2} \times \text{中心角} \]

性質：同じ弧に対する円周角は等しい

半円の弧に対する円周角 = 90°（直径の両端を見る角）

同じ弧ABに対する円周角は等しい
📌 ルール2：円に内接する四角形 → 例1
\[ \text{対角の和} = 180° \quad (\angle A + \angle C = \angle B + \angle D = 180°) \]

逆も成立：四角形の対角の和が180°なら、円に内接する

外角 = 内対角：1つの外角は、その対角に等しい

内接四角形の対角の和は180°
📌 ルール3：接弦定理 → 例2
\[ \text{接線と弦のなす角} = \text{その弦に対する円周角} \]

接弦定理：接点を通る弦と接線がなす角は、その弦に対する円周角に等しい

接弦定理：接線と弦のなす角 = 円周角
📌 ルール4：方べきの定理 → 例3

パターン1：2本の割線（点Pが円の外部）

\[ PA \cdot PB = PC \cdot PD \]

パターン2：2本の弦（点Pが円の内部）

\[ PA \cdot PB = PC \cdot PD \]

パターン3：割線と接線（点Pが円の外部）

\[ PA \cdot PB = PT^2 \quad (\text{Tは接点}) \]
📌 ルール5：2円の位置関係

半径 r, R（r < R）、中心間距離 d のとき：

位置関係 条件 共通接線
離れている d > R + r 4本
外接 d = R + r 3本
2点で交わる R - r < d < R + r 2本
内接 d = R - r 1本
一方が内部 d < R - r 0本

3. 例題と解き方

円の性質を使って問題を解いてみましょう。

▶ 内接四角形

例1内接四角形の角度

円に内接する四角形ABCDにおいて、∠A = 70°, ∠B = 110° のとき、∠C, ∠D を求めよ。

解き方：

  1. 対角の和 = 180° を使う
  2. ∠A + ∠C = 180° → ∠C = 180° - 70° = 110°
  3. ∠B + ∠D = 180° → ∠D = 180° - 110° = 70°
\[ \angle C = 110°, \quad \angle D = 70° \]

▶ 接弦定理

例2接弦定理の利用

円Oの接線ATと弦ABがあり、接線と弦のなす角が35°である。弦ABに対する円周角∠ACBを求めよ（CはAB上にない円周上の点）。

解き方：

  1. 接弦定理より、接線と弦のなす角 = 円周角
\[ \angle ACB = 35° \]

▶ 方べきの定理

例3方べきの定理（外部の点）

円の外部の点Pから2本の割線を引き、一方は円とA, Bで交わり、他方はC, Dで交わる。PA = 3, AB = 5, PC = 2 のとき、CD を求めよ。

解き方：

  1. 方べきの定理：PA · PB = PC · PD
  2. PB = PA + AB = 3 + 5 = 8
  3. PD = PC + CD = 2 + CD
  4. 3 × 8 = 2 × (2 + CD)
\[ 24 = 2(2 + CD) \Rightarrow 2 + CD = 12 \Rightarrow CD = 10 \]
例4方べきの定理（接線）

円の外部の点Pから接線PTと割線PABを引く。PA = 4, AB = 5 のとき、接線の長さPTを求めよ。

解き方：

  1. 方べきの定理：PA · PB = PT²
  2. PB = PA + AB = 4 + 5 = 9
\[ PT^2 = 4 \times 9 = 36 \Rightarrow PT = 6 \]
例5方べきの定理（内部の点）

円の内部の点Pを通る2本の弦AB, CDがある。AP = 3, PB = 8, CP = 4 のとき、PD を求めよ。

解き方：

  1. 方べきの定理：AP · PB = CP · PD
\[ 3 \times 8 = 4 \times PD \Rightarrow PD = 6 \]

▶ 2円の位置関係

例62円の共通接線

半径3と半径5の2円があり、中心間距離が10のとき、共通接線は何本あるか。

解き方：

  1. r = 3, R = 5, d = 10
  2. R + r = 8, R - r = 2
  3. d = 10 > R + r = 8 なので、2円は離れている
\[ \text{共通接線は } 4 \text{ 本} \]

4. 簡単にするコツ

✏️ コツ1：円周角は「同じ弧」を探す
等しい円周角を見つけるには、同じ弧を見ている角を探す。
半円の弧なら90°になる。
✏️ コツ2：内接四角形は「対角」に注目
対角の和が180°、または外角 = 内対角を使う。
四角形が内接することの証明にも使える。
✏️ コツ3：方べきは「点Pからの長さの積」
点Pから円への2本の線について、交点までの長さの積が等しい。
接線のときは PT² になる（同じ点を2回通る）。
✏️ コツ4：接弦定理は「接点を探す」
接線があったら、接点を通る弦に注目。
接線と弦のなす角 = その弦に対する円周角。

5. よくあるつまづき

⚠️ つまづき1：方べきの「長さ」を間違える

ダメな例：PA · AB = PC · CD と計算（ABはABの長さではない）

対策：Pからの「距離」を使う。PA · PB = PC · PD

⚠️ つまづき2：内接四角形の条件を逆にする

ダメな例：隣り合う角の和が180°と勘違い

対策：「対角」の和が180°。向かい合う角を選ぶ

⚠️ つまづき3：円周角と中心角を混同

ダメな例：円周角を2倍して別の円周角と比較

対策：円周角 = 中心角の半分。同じ弧の円周角同士は等しい

⚠️ つまづき4：2円の位置関係の境界を間違える

ダメな例：d = R + r で「2点で交わる」と判断

対策：d = R + r は外接（1点で接する）

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

円に内接する四角形ABCDにおいて、∠A = 85° のとき、∠C を求めよ。

答えを見る

∠A + ∠C = 180°

∠C = 180° - 85° = 95°

練習2

円の外部の点Pから接線PTを引く。また、Pを通る直線が円とA, Bで交わり、PA = 2, AB = 6 である。PTの長さを求めよ。

答えを見る

PB = PA + AB = 2 + 6 = 8

PT² = PA · PB = 2 × 8 = 16

PT = 4

練習3

半径2と半径4の2円があり、中心間距離が6のとき、2円の位置関係と共通接線の本数を答えよ。

答えを見る

r = 2, R = 4, d = 6

R + r = 6 = d なので外接

共通接線は3本

📌 まとめ：方べきは「Pからの長さの積」、内接四角形は「対角の和180°」！

おまけ：数学雑学「円周率πの歴史」

🥧 πを追い求めた人類の歴史

円周率π ≈ 3.14159...は、円の直径と周の長さの比。古代から人類を魅了してきた数だ。

古代の近似値：

  • 古代バビロニア（紀元前2000年頃）：π ≈ 3.125
  • 古代エジプト（紀元前1650年頃）：π ≈ 3.16
  • アルキメデス（紀元前250年頃）：\(\frac{223}{71} < \pi < \frac{22}{7}\) → 約3.14

アルキメデスは、円に内接・外接する正多角形を使って、πの範囲を絞り込んだ。正96角形まで計算したという。

その後の発展：

  • 祖沖之（中国、5世紀）：π ≈ 3.1415926〜3.1415927（小数点以下7桁）
  • ライプニッツの公式（1674年）：\(\frac{\pi}{4} = 1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots\)
  • コンピュータ時代：2024年時点で100兆桁以上が計算されている

πの性質：

  • 無理数：分数では表せない（1761年にランベルトが証明）
  • 超越数：整数係数の代数方程式の解にならない（1882年にリンデマンが証明）
  • これにより「円の正方形化」（コンパスと定規だけで、円と同じ面積の正方形を作る）が不可能であることが証明された

円という単純な図形から、これほど奥深い数が生まれる。数学の神秘だ。
