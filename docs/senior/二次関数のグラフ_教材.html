1. テーマの定義

中学では \(y = ax^2\) という原点を頂点とする放物線を学びました。高校では頂点がずれた二次関数を扱い、どんな二次関数でもグラフを描けるようになります。

このページでは、二次関数のグラフを学びます：

  • 標準形：\(y = a(x - p)^2 + q\) から頂点と軸を読み取る
  • 平方完成：一般形 \(y = ax^2 + bx + c\) を標準形に変形する
  • グラフの移動：平行移動の仕組みを理解する

2. 覚えるルール

🎯 最重要：\(y = a(x - p)^2 + q\) の頂点は \((p, q)\)、軸は \(x = p\)
二次関数の標準形：頂点と軸が一目で分かる
📌 ルール1：標準形と頂点・軸 → 例1
\[ y = a(x - p)^2 + q \]
  • 頂点：\((p, q)\)
  • 軸（対称軸）：\(x = p\)
  • \(a > 0\)：下に凸（∪型）
  • \(a < 0\)：上に凸（∩型）

→ だから結果は： 標準形を見れば、グラフの形が分かる

💡 符号の覚え方

\(y = a(x - p)^2 + q\) の \(p\) の前はマイナスであることに注意！

\(y = (x - 3)^2\) の頂点は \((3, 0)\)（マイナスの後ろの数がそのまま \(x\) 座標）

\(y = (x + 2)^2 = (x - (-2))^2\) の頂点は \((-2, 0)\)

📌 ルール2：平方完成のやり方 → 例2・3

\(y = ax^2 + bx + c\) を標準形に変形する手順：

  1. \(x^2\) と \(x\) の項を \(a\) でくくる
  2. カッコ内で「\((x + \text{〇})^2\)」の形を作る
  3. 調整した分を外で引く
\[ y = ax^2 + bx + c = a\left(x + \frac{b}{2a}\right)^2 - \frac{b^2 - 4ac}{4a} \]

頂点の公式（覚えてもOK）：

\[ \text{頂点} = \left(-\frac{b}{2a}, -\frac{b^2 - 4ac}{4a}\right) \]

✅ メリット： どんな二次関数でも頂点と軸が求まる

📌 ルール3：グラフの平行移動 → 例4

\(y = f(x)\) のグラフを平行移動すると：

\[ \text{x軸方向に } p \text{ 移動} \Rightarrow y = f(x - p) \] \[ \text{y軸方向に } q \text{ 移動} \Rightarrow y = f(x) + q \]

覚え方：「xの方向は符号が逆」「yの方向は符号そのまま」

例：\(y = x^2\) を x軸方向に3、y軸方向に2移動 → \(y = (x-3)^2 + 2\)

📌 ルール4：aの値とグラフの形
\(a\) の値 グラフの形 開き具合
\(a > 0\) 下に凸（∪型） \(|a|\) が大きいほど細い
\(a < 0\) 上に凸（∩型）

\(|a| = 1\) が基準。\(|a| > 1\) で細く、\(|a| < 1\) で広くなる。

3. 例題と解き方

ルールを使って実際に問題を解いてみましょう。

▶ 標準形から頂点・軸を読む

例1\(y = 2(x - 3)^2 + 5\) の頂点と軸を求める

解き方：

  1. 標準形 \(y = a(x - p)^2 + q\) と比較
  2. \(p = 3\)、\(q = 5\)
\[ \text{頂点}：(3, 5), \quad \text{軸}：x = 3 \]

※ \(a = 2 > 0\) なので下に凸（∪型）

▶ 平方完成

例2\(y = x^2 - 6x + 5\) を標準形に変形し、頂点と軸を求める

解き方：

  1. \(x^2 - 6x\) の部分で平方完成
  2. \(x^2 - 6x = (x - 3)^2 - 9\)
  3. 元の式に代入して整理
\[ y = x^2 - 6x + 5 = (x - 3)^2 - 9 + 5 = (x - 3)^2 - 4 \] \[ \text{頂点}：(3, -4), \quad \text{軸}：x = 3 \]
例3\(y = 2x^2 + 8x + 3\) を標準形に変形する

解き方：

  1. \(x^2\) の係数2でくくる：\(y = 2(x^2 + 4x) + 3\)
  2. カッコ内で平方完成：\(x^2 + 4x = (x + 2)^2 - 4\)
  3. 展開して整理
\[ y = 2(x^2 + 4x) + 3 = 2\{(x + 2)^2 - 4\} + 3 \] \[ = 2(x + 2)^2 - 8 + 3 = 2(x + 2)^2 - 5 \] \[ \text{頂点}：(-2, -5), \quad \text{軸}：x = -2 \]

▶ グラフの平行移動

例4\(y = x^2\) を x軸方向に \(-2\)、y軸方向に \(3\) 平行移動した式を求める

解き方：

  1. x軸方向に \(-2\) → \(x\) を \(x - (-2) = x + 2\) に置き換え
  2. y軸方向に \(3\) → 全体に \(+3\)
\[ y = (x + 2)^2 + 3 \]

▶ グラフを描く

例5\(y = -x^2 + 4x - 3\) のグラフの概形を描く

解き方：

  1. 平方完成：\(y = -(x^2 - 4x) - 3 = -\{(x-2)^2 - 4\} - 3 = -(x-2)^2 + 1\)
  2. 頂点 \((2, 1)\)、軸 \(x = 2\)
  3. \(a = -1 < 0\) なので上に凸（∩型）
  4. y切片：\(x = 0\) を代入 → \(y = -3\)
\[ \text{頂点}：(2, 1), \quad \text{軸}：x = 2, \quad \text{上に凸} \]

4. 簡単にするコツ

✏️ コツ1：平方完成は「半分の2乗」
\(x^2 + bx\) を平方完成するとき、\(b\) を半分にして2乗した \(\left(\dfrac{b}{2}\right)^2\) を足して引く。例：\(x^2 + 6x = (x + 3)^2 - 9\)
✏️ コツ2：係数をくくるのを忘れない
\(2x^2 + 8x\) のような場合、まず係数2をくくってから平方完成。\(2(x^2 + 4x) = 2\{(x+2)^2 - 4\} = 2(x+2)^2 - 8\)
✏️ コツ3：頂点の符号に注意
\(y = (x - 3)^2\) の頂点は \((3, 0)\)、\(y = (x + 2)^2\) の頂点は \((-2, 0)\)。「マイナスの後ろの数」がx座標になる。
✏️ コツ4：グラフは3点で描く
頂点 + 軸に対称な2点を取れば、放物線の形が決まる。y切片も追加すると確実。

5. よくあるつまづき

⚠️ つまづき1：頂点の符号を逆にする

ダメな例： \(y = (x - 3)^2 + 2\) の頂点を \((-3, 2)\) と答える

対策： 標準形は \(y = a(x - p)^2 + q\) で \(p\) の前はマイナス。頂点は \((3, 2)\) が正解。

⚠️ つまづき2：係数をくくり忘れる

ダメな例： \(y = 2x^2 + 4x + 1\) を \(y = 2(x + 1)^2 + 1\) と変形

対策： くくった後の調整を忘れない。\(2(x+1)^2 = 2x^2 + 4x + 2\) なので、\(-2 + 1 = -1\)。正解は \(y = 2(x+1)^2 - 1\)

⚠️ つまづき3：平行移動の符号ミス

ダメな例： x軸方向に3移動を \(y = (x + 3)^2\) と書く

対策： x軸方向は「符号逆」。右に3移動なら \(y = (x - 3)^2\)。「xの後ろにマイナス3」=「右に3移動」と覚える。

6. 仕上げチェックリスト

標準形から頂点と軸をすぐに読み取れる
一般形から平方完成で標準形に変形できる
係数をくくる場合の平方完成ができる
平行移動の式を正しく書ける（x方向は符号逆）
aの符号からグラフの向き（凸の向き）を判断できる

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(y = 3(x + 1)^2 - 4\) の頂点と軸を求めよ。

答えを見る

\(y = 3(x - (-1))^2 + (-4)\) と考えて

頂点：\((-1, -4)\)、軸：\(x = -1\)

練習2

\(y = x^2 + 4x + 1\) を標準形に変形せよ。

答えを見る

\(y = x^2 + 4x + 1 = (x + 2)^2 - 4 + 1 = (x + 2)^2 - 3\)

頂点：\((-2, -3)\)、軸：\(x = -2\)

練習3

\(y = -2x^2 + 8x - 5\) の頂点を求めよ。

答えを見る

\(y = -2(x^2 - 4x) - 5 = -2\{(x-2)^2 - 4\} - 5\)

\(= -2(x-2)^2 + 8 - 5 = -2(x-2)^2 + 3\)

頂点：\((2, 3)\)

📌 まとめ：\(y = a(x-p)^2 + q\) の頂点は \((p, q)\)！平方完成で「半分の2乗」を使う！

おまけ：歴史ストーリー「ガリレオと放物線」

📖 大砲の弾はどう飛ぶ？（17世紀イタリア）

17世紀初頭、イタリアの科学者ガリレオ・ガリレイは、ある問題に取り組んでいた。

「大砲の弾は、どんな軌道を描いて飛ぶのか？」

当時の常識では、「弾は斜めに上がり、力が尽きたら真下に落ちる」と考えられていた。しかしガリレオは実験と数学で、全く違う結論に達した。

「弾道は放物線を描く」

横方向の運動（等速）と縦方向の運動（重力で加速）を分けて考えると、組み合わせた軌道は二次関数のグラフになる。

\[ y = -\frac{g}{2v_0^2 \cos^2\theta}x^2 + (\tan\theta)x \]

これは \(y = ax^2 + bx\) の形——まさに放物線！

この発見は軍事的にも重要だった。「どの角度で撃てば最も遠くに飛ぶか」が計算できるようになったのだ。答えは45度（空気抵抗がない場合）。

ガリレオは後に地動説を支持したことで教会と対立し、軟禁生活を送ることになる。しかし彼が発見した放物線の法則は、今でもスポーツ（ボールの軌道）から宇宙開発（ロケットの打ち上げ）まで使われている。

二次関数のグラフは、世界を動かす曲線だった。
