1. テーマの定義

\((a+b)^{10}\) を展開したら、項がいくつ出てきて、各係数はいくつ？ こんな計算を一瞬で解決するのが二項定理です。また、多項式を別の多項式で割る整式の割り算は、因数定理や高次方程式を解く上で欠かせない技術です。

このページでは、式と証明の前半を学びます：

  • 二項定理：\((a+b)^n\) の展開公式
  • 二項係数：\(_nC_r\) と展開係数の関係
  • パスカルの三角形：二項係数の規則性
  • 整式の割り算：商と余りの求め方
  • 組立除法：\((x-a)\) で割るときの簡便法

2. 覚えるルール

📐 最重要：二項定理 \((a+b)^n = \sum_{r=0}^{n} {}_nC_r \, a^{n-r}b^r\)
二項定理の係数はパスカルの三角形に現れる
📌 ルール1：二項定理 → 例1
\[ (a+b)^n = \sum_{r=0}^{n} {}_nC_r \, a^{n-r}b^r = {}_nC_0 a^n + {}_nC_1 a^{n-1}b + {}_nC_2 a^{n-2}b^2 + \cdots + {}_nC_n b^n \]

読み方：「シグマ記号 \(\sum\)」は「r=0からnまで足し合わせる」という意味

→ だから結果は： 各項の係数は \({}_nC_r\)、aの指数は \(n-r\)、bの指数は \(r\)

✅ メリット： \((a+b)^{10}\) のような高い累乗も、特定の項の係数を直接計算できる

💡 考え方を見る

\((a+b)^n = (a+b)(a+b)\cdots(a+b)\)（n個の積）を展開すると、各カッコからaかbを1つずつ選ぶ。

\(a^{n-r}b^r\) の項は「n個のカッコからr個のbを選ぶ」場合の数 \({}_nC_r\) 通りだけ出てくる。

📌 ルール2：一般項（第r+1項） → 例2
\[ (a+b)^n \text{の第} (r+1) \text{項} = {}_nC_r \, a^{n-r}b^r \]

注意：「第r+1項」なのでr=0が第1項。「第5項」を求めるならr=4を代入

✅ メリット： 展開を全部しなくても、特定の項だけピンポイントで計算できる

📌 ルール3：パスカルの三角形の性質
\[ {}_nC_r = {}_{n-1}C_{r-1} + {}_{n-1}C_r \quad \text{（加法公式）} \]

意味：上の2つの数を足すと、その下の数になる

その他の性質：

  • \({}_nC_0 = {}_nC_n = 1\)（両端は常に1）
  • \({}_nC_r = {}_nC_{n-r}\)（左右対称）
  • \({}_nC_0 + {}_nC_1 + \cdots + {}_nC_n = 2^n\)（各行の和）
📌 ルール4：整式の割り算 → 例3
\[ A = BQ + R \quad \text{（被除式 = 除式 × 商 + 余り）} \]

Aを整式B（\(B \neq 0\)）で割ると、商Qと余りRが一意に定まる

条件：余りRの次数 \(<\) 除式Bの次数（または R=0）

計算方法：筆算で最高次の項から順に消していく

📌 ルール5：組立除法（\(x-a\) で割るとき） → 例4
\[ P(x) \div (x-a) \text{ の商と余りを簡単に求める方法} \]

手順：

  1. 被除式の係数を横に並べる（欠けている次数は0を入れる）
  2. 除数 \(x-a\) の \(a\) を左端に書く
  3. 最高次の係数をそのまま下ろす
  4. 下ろした数 × a を次の係数の下に書き、足す
  5. 以下繰り返し。最後の数が余り

✅ メリット： 筆算より圧倒的に速い

商は \(2x^2 + 7x + 9\)、余りは \(22\)

3. 例題と解き方

二項定理と整式の割り算を使って実際に問題を解いてみましょう。

▶ 二項定理による展開

例1\((x+2)^5\) を展開する

\((x+2)^5\) を展開せよ。

解き方：

  1. 二項定理を適用：\(a=x, b=2, n=5\)
  2. 各項の係数 \({}_5C_r\) を計算：1, 5, 10, 10, 5, 1
  3. \(2^r\) をかけて各項を求める
\[ (x+2)^5 = {}_5C_0 x^5 + {}_5C_1 x^4 \cdot 2 + {}_5C_2 x^3 \cdot 2^2 + {}_5C_3 x^2 \cdot 2^3 + {}_5C_4 x \cdot 2^4 + {}_5C_5 \cdot 2^5 \] \[ = x^5 + 10x^4 + 40x^3 + 80x^2 + 80x + 32 \]
例2特定の項の係数を求める

\((2x-3)^6\) の展開式における \(x^4\) の係数を求めよ。

解き方：

  1. \(a=2x, b=-3, n=6\) として一般項を書く
  2. \(x^4\) になる条件を見つける → \((2x)^4(-3)^2\) なので \(r=2\)
  3. 係数を計算
\[ \text{一般項} = {}_6C_r (2x)^{6-r}(-3)^r \]

\(x^4\) の項は \(6-r=4\)、つまり \(r=2\) のとき：

\[ {}_6C_2 (2x)^4(-3)^2 = 15 \times 16x^4 \times 9 = 2160x^4 \]

答え：\(x^4\) の係数は \(\boldsymbol{2160}\)

例3差の累乗

\((a-b)^4\) を展開せよ。

解き方：

  1. \((a-b)^4 = (a+(-b))^4\) と考える
  2. \((-b)^r\) の符号に注意して展開
\[ (a-b)^4 = a^4 - 4a^3b + 6a^2b^2 - 4ab^3 + b^4 \]

ポイント：係数の符号は + - + - + と交互になる

▶ 二項係数の性質

例4二項係数の和

\({}_8C_0 + {}_8C_1 + {}_8C_2 + \cdots + {}_8C_8\) の値を求めよ。

解き方：

  1. 二項定理で \(a=1, b=1, n=8\) とおく
  2. \((1+1)^8 = 2^8\) を利用
\[ (1+1)^8 = {}_8C_0 + {}_8C_1 + {}_8C_2 + \cdots + {}_8C_8 = 2^8 = 256 \]

▶ 整式の割り算

例5筆算による整式の割り算

\((x^3 + 2x^2 - 5x + 3) \div (x^2 - x + 1)\) の商と余りを求めよ。

解き方：

  1. 最高次の項 \(x^3 \div x^2 = x\) が商の最初
  2. \(x \times (x^2 - x + 1) = x^3 - x^2 + x\) を引く
  3. 次は \(3x^2 \div x^2 = 3\) が商の次
  4. 残った式が余り
\[ x^3 + 2x^2 - 5x + 3 = (x^2 - x + 1)(x + 3) + (-3x) \]

答え：商 \(x+3\)、余り \(-3x\)

例6組立除法

\((x^3 - 4x^2 + 5x - 2) \div (x - 1)\) の商と余りを組立除法で求めよ。

解き方：

  1. \(x - 1 = 0\) より \(a = 1\)
  2. 係数 1, -4, 5, -2 を並べて組立除法を実行
1 1 -4 5 -2
1 -3 2
1 -3 2 0

答え：商 \(x^2 - 3x + 2\)、余り \(0\)

（余りが0なので、\(x-1\) は因数）

▶ 余りの応用

例7二次式で割った余り

\(P(x) = x^4 - 2x^3 + x - 5\) を \((x-1)(x+2)\) で割った余りを求めよ。

解き方：

  1. 二次式で割るので余りは一次式以下 → \(ax+b\) とおく
  2. \(P(x) = (x-1)(x+2) \cdot Q(x) + ax + b\)
  3. \(x=1\) を代入：\(P(1) = a + b\)
  4. \(x=-2\) を代入：\(P(-2) = -2a + b\)
  5. 連立方程式を解く
\[ P(1) = 1 - 2 + 1 - 5 = -5 \quad \Rightarrow \quad a + b = -5 \] \[ P(-2) = 16 + 16 - 2 - 5 = 25 \quad \Rightarrow \quad -2a + b = 25 \]

連立方程式を解くと \(a = -10, b = 5\)

答え：余りは \(-10x + 5\)

4. 簡単にするコツ

✏️ コツ1：パスカルの三角形を暗記
n=1〜5くらいまでは暗記しておくと、展開が一瞬で終わる。
1行目: 1
2行目: 1 1
3行目: 1 2 1
4行目: 1 3 3 1
5行目: 1 4 6 4 1
6行目: 1 5 10 10 5 1
✏️ コツ2：特定の項だけ求めるなら一般項
「\(x^3\) の係数は？」と聞かれたら、全部展開せずに一般項 \({}_nC_r a^{n-r}b^r\) から \(x^3\) になる \(r\) を逆算。
✏️ コツ3：\((a-b)^n\) は符号が交互
\((a+(-b))^n\) と考えると、\((-b)^r\) の符号で項の符号が + - + - ... と交互に変わる。
✏️ コツ4：組立除法は \(x-a\) 専用
除式が \(x-a\) の形なら組立除法が圧倒的に速い。\(x+3\) で割るなら \(a=-3\) を使う。
✏️ コツ5：余りの次数は除式より小さい
二次式で割れば余りは一次以下、三次式で割れば二次以下。これを使って文字を置く。

5. よくあるつまづき

⚠️ つまづき1：一般項の「第r+1項」を間違える

ダメな例：「第5項」を求めるのに r=5 を代入

対策：一般項は r=0 が第1項。「第5項」なら r=4 を代入する

⚠️ つまづき2：\((a-b)^n\) で符号を間違える

ダメな例：\((x-2)^3\) を展開して \(x^3 + 6x^2 + 12x + 8\) とする

対策：\((-2)^r\) の符号に注意。奇数乗はマイナス、偶数乗はプラス

⚠️ つまづき3：組立除法で \(x+a\) の符号を間違える

ダメな例：\(x+3\) で割るのに \(a=3\) として計算

対策：\(x+3 = x-(-3)\) なので \(a=-3\) を使う

⚠️ つまづき4：欠けている次数を飛ばす

ダメな例：\(x^3 + 5\) を割るとき、係数を 1, 5 と並べる

対策：欠けている次数（\(x^2, x\)）には 0 を入れて 1, 0, 0, 5 と並べる

学んだことを使って、以下の問題を解いてみよう。

練習1

\((x+3)^4\) を展開せよ。

答えを見る

\((x+3)^4 = x^4 + 4x^3 \cdot 3 + 6x^2 \cdot 9 + 4x \cdot 27 + 81\)

\(= x^4 + 12x^3 + 54x^2 + 108x + 81\)

練習2

\((2x-1)^5\) の展開式における \(x^3\) の係数を求めよ。

答えを見る

一般項：\({}_5C_r (2x)^{5-r}(-1)^r\)

\(x^3\) になるのは \(5-r=3\)、つまり \(r=2\)

\({}_5C_2 (2x)^3(-1)^2 = 10 \times 8x^3 \times 1 = 80x^3\)

答え：80

おまけ：歴史ストーリー「パスカルの三角形は誰のもの？」

📖 パスカルより500年も前に...

「パスカルの三角形」と聞くと、フランスの数学者ブレーズ・パスカル（1623-1662）が発見したと思うだろう。確かにパスカルは1653年に『算術三角形論』を書いた。

しかし、実はこの三角形はパスカルより500年も前に発見されていた。

中国では、11世紀（1050年頃）に賈憲（カケン）がすでにこの三角形を使っていた。さらに1303年、朱世傑が『四元玉鑑』という本でこの三角形を詳しく解説している。中国では「賈憲三角」や「楊輝三角」と呼ばれる。

インドでは、紀元前2世紀にピンガラがこの配列を知っていたという説もある。

ペルシャでは、12世紀にオマル・ハイヤームが二項展開の係数として使っていた。

つまり、パスカルは「発見者」ではなく「西洋で体系化した人」なのだ。

地域 時代 数学者
インド 紀元前2世紀? ピンガラ
中国 11世紀 賈憲、楊輝、朱世傑
ペルシャ 12世紀 オマル・ハイヤーム
フランス 17世紀 パスカル

数学の発見は、世界中で独立に起こることが多い。でも名前は「最初に体系化した人」ではなく「一番有名な人」がつくことが多い。

教訓：数学に国境はない。そして、歴史の「発見者」は必ずしも最初の発見者ではない。
