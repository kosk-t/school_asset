1. テーマの定義

音や光の波、観覧車の動き、心電図のリズム...。世の中の「繰り返す現象」は三角関数で表されます。高校数学の三角比から一歩進んで、あらゆる角度に対応できる道具を手に入れましょう。

このページでは、三角関数の基本を学びます：

  • 弧度法（ラジアン）：度数法より便利な角度の表し方
  • 三角関数のグラフ：sin・cos・tanの波形と特徴
  • 周期：関数が繰り返す間隔の求め方
  • 加法定理：2つの角の和・差の三角関数を計算する公式

2. 覚えるルール

【最重要】弧度法（ラジアン）では、180° = π（パイ）。これが角度変換の基本！
弧度法の定義：半径と弧の長さの比で角度を表す
📌 弧度法と度数法の変換 → 例1
\[ \pi \text{（ラジアン）} = 180° \]
度数法弧度法
30°\(\frac{\pi}{6}\)
45°\(\frac{\pi}{4}\)
60°\(\frac{\pi}{3}\)
90°\(\frac{\pi}{2}\)
180°\(\pi\)
360°\(2\pi\)

→ だから結果は： 度→ラジアンは \(\times \frac{\pi}{180}\)、ラジアン→度は \(\times \frac{180}{\pi}\)

✅ メリット： 微分・積分や物理で「角度」が自然な計算になる

📌 三角関数のグラフと周期 → 例2

\(y = \sin x\)、\(y = \cos x\) の周期は \(2\pi\)、\(y = \tan x\) の周期は \(\pi\)

\[ y = A\sin(Bx + C) \text{ の周期} = \frac{2\pi}{|B|} \]

→ だから結果は： \(B\) が大きいほど周期は短く（振動が速く）なる

sin・cos（周期2π）と tan（周期π）のグラフ
📌 加法定理（超重要！） → 例3
\[ \sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta \] \[ \sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta \] \[ \cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta \] \[ \cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta \]

→ だから結果は： 75°や15°など「特別な角の和・差」の三角関数が計算できる

💡 覚え方のコツ

sin の加法定理：「sin cos cos sin」（サイン・コサイン・コサイン・サイン）

cos の加法定理：「cos cos sin sin」（コサイン・コサイン・サイン・サイン）で符号が逆（+のとき−、−のとき+）

📌 tan の加法定理 → 例4
\[ \tan(\alpha + \beta) = \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta} \] \[ \tan(\alpha - \beta) = \frac{\tan\alpha - \tan\beta}{1 + \tan\alpha\tan\beta} \]

→ だから結果は： tan だけで計算できるので、傾きの問題に便利

📌 三角関数の相互関係（復習）
\[ \sin^2\theta + \cos^2\theta = 1, \quad \tan\theta = \frac{\sin\theta}{\cos\theta} \]

→ だから結果は： 1つがわかれば残りも求められる

3. 例題と解き方

公式を使って実際に計算してみましょう。

▶ 弧度法と度数法の変換

例1弧度法・度数法の変換

（1）120° をラジアンで表せ。　（2）\(\frac{5\pi}{4}\) を度数法で表せ。

解き方：

  1. 度→ラジアン：\(\times \frac{\pi}{180}\) をかける
  2. ラジアン→度：\(\times \frac{180}{\pi}\) をかける

（1）

\[ 120° = 120 \times \frac{\pi}{180} = \frac{120\pi}{180} = \frac{2\pi}{3} \]

（2）

\[ \frac{5\pi}{4} = \frac{5\pi}{4} \times \frac{180}{\pi} = \frac{5 \times 180}{4} = 225° \]

▶ 三角関数の値と周期

例2三角関数の値と周期

（1）\(\sin\frac{5\pi}{6}\) の値を求めよ。　（2）\(y = \sin 3x\) の周期を求めよ。

解き方：

  1. ラジアンを度数法に直して、基本角に帰着させる
  2. 周期は \(\frac{2\pi}{|B|}\) で計算

（1）

\[ \frac{5\pi}{6} = 150° = 180° - 30° \]

\(\sin 150° = \sin(180° - 30°) = \sin 30° = \frac{1}{2}\)

（2） \(y = \sin 3x\) で \(B = 3\) なので：

\[ \text{周期} = \frac{2\pi}{|3|} = \frac{2\pi}{3} \]

▶ 加法定理

例3sin, cos の加法定理

\(\sin 75°\) の値を求めよ。

解き方：

  1. 75° = 45° + 30° と分解する
  2. 加法定理 \(\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta\) を使う
  3. 特別な角の値を代入する
\[ \sin 75° = \sin(45° + 30°) \] \[ = \sin 45°\cos 30° + \cos 45°\sin 30° \] \[ = \frac{\sqrt{2}}{2} \cdot \frac{\sqrt{3}}{2} + \frac{\sqrt{2}}{2} \cdot \frac{1}{2} \] \[ = \frac{\sqrt{6}}{4} + \frac{\sqrt{2}}{4} = \frac{\sqrt{6} + \sqrt{2}}{4} \]
例4tan の加法定理

\(\tan 15°\) の値を求めよ。

解き方：

  1. 15° = 45° - 30° と分解する
  2. tan の加法定理を使う
  3. \(\tan 45° = 1\)、\(\tan 30° = \frac{1}{\sqrt{3}}\) を代入
\[ \tan 15° = \tan(45° - 30°) = \frac{\tan 45° - \tan 30°}{1 + \tan 45° \tan 30°} \] \[ = \frac{1 - \frac{1}{\sqrt{3}}}{1 + 1 \cdot \frac{1}{\sqrt{3}}} = \frac{\frac{\sqrt{3} - 1}{\sqrt{3}}}{\frac{\sqrt{3} + 1}{\sqrt{3}}} = \frac{\sqrt{3} - 1}{\sqrt{3} + 1} \]

有理化すると：

\[ = \frac{(\sqrt{3} - 1)^2}{(\sqrt{3} + 1)(\sqrt{3} - 1)} = \frac{3 - 2\sqrt{3} + 1}{3 - 1} = \frac{4 - 2\sqrt{3}}{2} = 2 - \sqrt{3} \]
例5加法定理の応用（値の計算）

\(\sin\alpha = \frac{4}{5}\)（\(\alpha\) は第1象限）、\(\cos\beta = \frac{12}{13}\)（\(\beta\) は第1象限）のとき、\(\sin(\alpha + \beta)\) を求めよ。

解き方：

  1. \(\sin^2\theta + \cos^2\theta = 1\) から足りない値を求める
  2. 加法定理に代入

\(\cos\alpha = \sqrt{1 - \sin^2\alpha} = \sqrt{1 - \frac{16}{25}} = \sqrt{\frac{9}{25}} = \frac{3}{5}\)（第1象限なので正）

\(\sin\beta = \sqrt{1 - \cos^2\beta} = \sqrt{1 - \frac{144}{169}} = \sqrt{\frac{25}{169}} = \frac{5}{13}\)（第1象限なので正）

\[ \sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta \] \[ = \frac{4}{5} \cdot \frac{12}{13} + \frac{3}{5} \cdot \frac{5}{13} = \frac{48}{65} + \frac{15}{65} = \frac{63}{65} \]

4. 簡単にするコツ

✏️ コツ1：よく使うラジアン値は暗記する
30° = π/6、45° = π/4、60° = π/3、90° = π/2、180° = π
これを覚えておけば、変換計算なしで使える！
✏️ コツ2：加法定理は「角度を分解」から始める
75° = 45° + 30°、15° = 45° - 30° または 60° - 45°
まず「特別な角の和・差」に分解できないか考える。
✏️ コツ3：cos の加法定理の符号は「逆」
\(\cos(\alpha \textcolor{red}{+} \beta)\) のときは \(\cos\alpha\cos\beta \textcolor{red}{-} \sin\alpha\sin\beta\)（符号が逆）
\(\cos(\alpha \textcolor{blue}{-} \beta)\) のときは \(\cos\alpha\cos\beta \textcolor{blue}{+} \sin\alpha\sin\beta\)（符号が逆）
sinは「そのまま」、cosは「逆」と覚える。
✏️ コツ4：周期を求めるときは係数だけ見る
\(y = \sin(3x + 2)\) の周期は？ → \(B = 3\) だから周期 = \(\frac{2\pi}{3}\)
\(+2\) の部分（位相）は周期に影響しない。

5. よくあるつまづき

⚠️ つまづき1：cos の加法定理で符号を間違える

ダメな例： \(\cos(A + B) = \cos A \cos B + \sin A \sin B\)（×）

対策： cosの加法定理は符号が「逆」！
\(\cos(A + B) = \cos A \cos B \textcolor{red}{-} \sin A \sin B\)（○）

⚠️ つまづき2：ラジアン変換で分母を忘れる

ダメな例： 60° = \(\frac{\pi}{60}\)（×）

対策： \(60° = 60 \times \frac{\pi}{180} = \frac{\pi}{3}\)（○）
「180° = π」を基準に考える。60°は180°の1/3だから、π/3。

⚠️ つまづき3：tan の周期を 2π と思い込む

ダメな例： \(\tan x\) の周期は \(2\pi\)（×）

対策： \(\tan x\) の周期は π（sin, cos の半分）
グラフを見れば、\(\pi\) ごとに同じパターンが繰り返されている。

6. 仕上げチェックリスト

180° = π（ラジアン）を使って、度とラジアンを変換できる
sin・cos のグラフの周期は 2π、tan の周期は π と言える
\(y = \sin(Bx)\) の周期が \(\frac{2\pi}{|B|}\) で計算できる
sin・cos の加法定理を暗唱でき、75°や15°の三角関数が計算できる
tan の加法定理を使える（分数の形で覚えている）

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

（1）135° をラジアンで表せ。　（2）\(\frac{7\pi}{6}\) を度数法で表せ。

答えを見る

（1）\(135° = 135 \times \frac{\pi}{180} = \frac{3\pi}{4}\)

（2）\(\frac{7\pi}{6} = \frac{7\pi}{6} \times \frac{180}{\pi} = \frac{7 \times 180}{6} = 210°\)

練習2

\(\cos 15°\) の値を求めよ。

答えを見る

15° = 45° - 30° と分解して、cos の加法定理を使う。

\(\cos 15° = \cos(45° - 30°)\)

\(= \cos 45° \cos 30° + \sin 45° \sin 30°\)（符号が逆に注意！）

\(= \frac{\sqrt{2}}{2} \cdot \frac{\sqrt{3}}{2} + \frac{\sqrt{2}}{2} \cdot \frac{1}{2}\)

\(= \frac{\sqrt{6}}{4} + \frac{\sqrt{2}}{4} = \frac{\sqrt{6} + \sqrt{2}}{4}\)

練習3

\(y = \cos 4x\) の周期を求めよ。

答えを見る

\(B = 4\) なので、周期 \(= \frac{2\pi}{|4|} = \frac{2\pi}{4} = \frac{\pi}{2}\)

📌 まとめ：弧度法では「180° = π」、加法定理で複雑な角度も計算できる！sinは符号そのまま、cosは符号が逆。

おまけ：歴史ストーリー「三角関数表を作った天文学者たち」

📖 電卓がなかった時代の「超人的な計算力」

今では電卓で \(\sin 37°\) と打てば一瞬で答えが出ますよね。でも電卓が発明されたのは20世紀。それ以前、人々はどうやって三角関数の値を求めていたのでしょう？

答えは「三角関数表」。分厚い本に、0°から90°まで1°刻み（時には1分=1/60°刻み！）で sin, cos, tan の値がびっしり書かれていました。

この表を最初に本格的に作ったのは、2世紀の天文学者プトレマイオス。彼は天体の運動を予測するために、なんと1万5千行以上の三角関数表を手計算で作成しました。

プトレマイオスが使ったのは、加法定理と半角の公式です：

\[ \sin 75° = \sin(45° + 30°) = \sin 45° \cos 30° + \cos 45° \sin 30° \]

こうして「特別な角」から出発し、足したり引いたり半分にしたりして、あらゆる角度の値を求めていったのです。

16世紀のドイツでは、数学者レティクスが生涯をかけて三角関数表の精度を上げ続けました。彼の弟子が完成させた表は、小数点以下15桁の精度。計算に使った紙は山のようになったそうです。

現代のスマートフォンが1秒でできる計算に、天才たちが何十年もの人生を費やした—。「加法定理って覚えるの面倒だな」と思ったら、プトレマイオスの苦労を思い出してみてください。きっと公式がちょっとだけ愛おしくなるはず...？
