1. テーマの定義

「曲線で囲まれた図形の面積を求めたい」——この自然な疑問に答えるのが積分による面積計算です。長方形や三角形の面積は小学校で習いましたが、曲線で囲まれた図形はどうでしょう？定積分を使えば、どんな曲線でも面積が計算できます。

このページでは、積分による面積計算を学びます：

  • 曲線とx軸で囲まれた面積：定積分の絶対値に注意
  • 2曲線間の面積：「上から下を引く」の原則
  • 面積公式（1/6公式）：放物線と直線で囲まれた面積の高速計算
  • 面積公式（1/12公式）：2つの放物線で囲まれた面積

2. 覚えるルール

【最重要】面積は「上の関数 − 下の関数」を積分：\(S = \int_a^b \{f(x) - g(x)\} dx\)（\(f(x) \geq g(x)\) のとき）
面積は「上の関数から下の関数を引いて積分」
📌 曲線とx軸で囲まれた面積（基本） → 例1

\(a \leq x \leq b\) で \(f(x) \geq 0\) のとき：

\[ S = \int_a^b f(x)\, dx \]

\(a \leq x \leq b\) で \(f(x) \leq 0\) のとき（x軸より下）：

\[ S = -\int_a^b f(x)\, dx = \int_a^b |f(x)|\, dx \]

→ だから結果は： x軸より下の部分は絶対値を取る（マイナスをつける）

💡 なぜマイナスをつけるのか

定積分は「符号付き面積」を計算します。x軸より下では \(f(x) < 0\) なので、積分値も負になります。面積（正の値）を求めるには絶対値を取る必要があります。

📌 2曲線間の面積 → 例2

\(a \leq x \leq b\) で \(f(x) \geq g(x)\) のとき（f が上、g が下）：

\[ S = \int_a^b \{f(x) - g(x)\}\, dx \]

→ だから結果は： 「上から下を引く」と覚える。上下が入れ替わる場合は分割して計算。

✅ メリット： 「どちらがx軸の上か下か」を気にせず、単に「上−下」で計算できる

📌 面積公式① 1/6公式 → 例3

放物線 \(y = a(x - \alpha)(x - \beta)\)（\(a \neq 0\)、\(\alpha < \beta\)）と x軸で囲まれた面積：

\[ S = \frac{|a|}{6}(\beta - \alpha)^3 \]

→ だから結果は： 2次関数がx軸と交わる2点 \(\alpha, \beta\) がわかれば、即座に面積が求まる

💡 公式の導出

\(a > 0\) のとき、\(\alpha < x < \beta\) で \(y < 0\) なので：

\(S = -\int_\alpha^\beta a(x-\alpha)(x-\beta)\,dx = -a \cdot \left(-\frac{1}{6}\right)(\beta-\alpha)^3 = \frac{a}{6}(\beta-\alpha)^3\)

積分の計算に \(\int_\alpha^\beta (x-\alpha)(x-\beta)\,dx = -\frac{(\beta-\alpha)^3}{6}\) という公式を使っています。

1/6公式：放物線とx軸、放物線と直線で囲まれた面積
📌 面積公式② 放物線と直線の1/6公式 → 例4

放物線 \(y = ax^2 + bx + c\) と直線が \(x = \alpha, \beta\) で交わるとき：

\[ S = \frac{|a|}{6}(\beta - \alpha)^3 \]

→ だから結果は： 「放物線と直線の差」は2次関数。その2次の係数と交点の差だけで面積がわかる

📌 面積公式③ 1/12公式 → 例5

2つの放物線が \(x = \alpha, \beta\) で交わり、\(x^2\) の係数の差が \(a\) のとき：

\[ S = \frac{|a|}{12}(\beta - \alpha)^3 \]

→ だから結果は： 2つの放物線の差は2次関数。その最高次係数と交点で面積が決まる

💡 1/6 と 1/12 の違い

1/6公式：放物線と「直線」（または x軸）→ 差が2次関数で、最高次係数がそのまま \(a\)

1/12公式：2つの放物線 → 差をとると2次関数になるが、放物線同士の対称性から係数が半分の効果を持ち、1/12になる

📌 便利な積分公式（導出に使う）
積分結果用途
\(\int_\alpha^\beta (x-\alpha)(x-\beta)\,dx\)\(-\frac{(\beta-\alpha)^3}{6}\)1/6公式の導出
\(\int_\alpha^\beta (x-\alpha)^2(x-\beta)\,dx\)\(-\frac{(\beta-\alpha)^4}{12}\)1/12公式の導出
\(\int_\alpha^\beta (x-\alpha)(x-\beta)^2\,dx\)\(\frac{(\beta-\alpha)^4}{12}\)1/12公式の導出

3. 例題と解き方

定積分を使って面積を求めてみましょう。

▶ 曲線とx軸で囲まれた面積

例1放物線とx軸の面積（基本）

放物線 \(y = x^2 - 4\) と x軸で囲まれた面積を求めよ。

解き方：

  1. x軸との交点を求める（\(y = 0\) を解く）
  2. グラフの上下を確認する
  3. 面積を積分で計算する（x軸より下なら絶対値）

Step 1：交点を求める

\[ x^2 - 4 = 0 \quad \Rightarrow \quad x = \pm 2 \]

Step 2：上下を確認

\(-2 < x < 2\) で \(y = x^2 - 4 < 0\)（x軸より下）

Step 3：面積を計算

\[ S = -\int_{-2}^{2} (x^2 - 4)\, dx = -\left[\frac{x^3}{3} - 4x\right]_{-2}^{2} \] \[ = -\left\{\left(\frac{8}{3} - 8\right) - \left(-\frac{8}{3} + 8\right)\right\} \] \[ = -\left\{-\frac{16}{3} - \frac{16}{3}\right\} = -\left(-\frac{32}{3}\right) = \frac{32}{3} \]

（確認：1/6公式なら \(S = \frac{1}{6} \cdot |1| \cdot (2-(-2))^3 = \frac{1}{6} \cdot 64 = \frac{32}{3}\) で一致！）

▶ 2曲線間の面積

例2放物線と直線の面積（基本）

放物線 \(y = x^2\) と直線 \(y = x + 2\) で囲まれた面積を求めよ。

解き方：

  1. 2曲線の交点を求める
  2. どちらが上かを確認する
  3. 「上 − 下」を積分する

Step 1：交点を求める

\[ x^2 = x + 2 \quad \Rightarrow \quad x^2 - x - 2 = 0 \] \[ (x-2)(x+1) = 0 \quad \Rightarrow \quad x = -1, 2 \]

Step 2：上下を確認

\(-1 < x < 2\) で \(x + 2 > x^2\)（直線が上、放物線が下）

Step 3：面積を計算

\[ S = \int_{-1}^{2} \{(x + 2) - x^2\}\, dx = \int_{-1}^{2} (-x^2 + x + 2)\, dx \] \[ = \left[-\frac{x^3}{3} + \frac{x^2}{2} + 2x\right]_{-1}^{2} \] \[ = \left(-\frac{8}{3} + 2 + 4\right) - \left(\frac{1}{3} + \frac{1}{2} - 2\right) \] \[ = \left(-\frac{8}{3} + 6\right) - \left(\frac{1}{3} + \frac{1}{2} - 2\right) \] \[ = \frac{10}{3} - \left(-\frac{7}{6}\right) = \frac{10}{3} + \frac{7}{6} = \frac{20}{6} + \frac{7}{6} = \frac{27}{6} = \frac{9}{2} \]

▶ 1/6公式の活用

例31/6公式で高速計算

放物線 \(y = x^2 - 4\) と x軸で囲まれた面積を1/6公式で求めよ。

解き方：

  1. \(y = a(x - \alpha)(x - \beta)\) の形に因数分解
  2. 公式 \(S = \frac{|a|}{6}(\beta - \alpha)^3\) に代入

Step 1：因数分解

\[ y = x^2 - 4 = (x + 2)(x - 2) \]

よって \(a = 1\)、\(\alpha = -2\)、\(\beta = 2\)

Step 2：公式に代入

\[ S = \frac{|1|}{6}(2 - (-2))^3 = \frac{1}{6} \cdot 4^3 = \frac{64}{6} = \frac{32}{3} \]

積分計算なしで一発！

例4放物線と直線に1/6公式

例2の面積を1/6公式で求めよ。

解き方：

  1. 「上 − 下」の式を作る
  2. 2次の係数 \(a\) と交点 \(\alpha, \beta\) を読み取る
  3. 公式に代入

Step 1：上 − 下の式

\[ (x + 2) - x^2 = -x^2 + x + 2 = -(x^2 - x - 2) = -(x - 2)(x + 1) \]

よって \(a = -1\)、\(\alpha = -1\)、\(\beta = 2\)

Step 2：公式に代入

\[ S = \frac{|-1|}{6}(2 - (-1))^3 = \frac{1}{6} \cdot 3^3 = \frac{27}{6} = \frac{9}{2} \]

例2と同じ答えが出た！

▶ 1/12公式の活用

例52つの放物線で囲まれた面積

放物線 \(y = x^2\) と \(y = -x^2 + 4x\) で囲まれた面積を求めよ。

解き方：

  1. 交点を求める
  2. \(x^2\) の係数の差を求める
  3. 1/12公式に代入

Step 1：交点を求める

\[ x^2 = -x^2 + 4x \quad \Rightarrow \quad 2x^2 - 4x = 0 \] \[ 2x(x - 2) = 0 \quad \Rightarrow \quad x = 0, 2 \]

Step 2：係数の差

上の放物線：\(-x^2 + 4x\)（\(x^2\) の係数は \(-1\)）

下の放物線：\(x^2\)（\(x^2\) の係数は \(1\)）

差：\(-1 - 1 = -2\)、よって \(|a| = 2\)

Step 3：1/12公式に代入

\[ S = \frac{|{-2}|}{12}(2 - 0)^3 = \frac{2}{12} \cdot 8 = \frac{16}{12} = \frac{4}{3} \]

（確認：普通に積分しても \(\int_0^2 \{(-x^2+4x) - x^2\}\,dx = \int_0^2 (-2x^2+4x)\,dx = \frac{4}{3}\) で一致）

▶ 上下が入れ替わる場合

例6上下が入れ替わる場合

\(y = x^3 - x\) と x軸で囲まれた面積を求めよ。

解き方：

  1. x軸との交点を求める
  2. 各区間で上下を確認し、分割して積分

Step 1：交点を求める

\[ x^3 - x = x(x^2 - 1) = x(x+1)(x-1) = 0 \] \[ x = -1, 0, 1 \]

Step 2：各区間の上下

  • \(-1 < x < 0\)：\(y > 0\)（x軸より上）
  • \(0 < x < 1\)：\(y < 0\)（x軸より下）

Step 3：分割して積分

\[ S = \int_{-1}^{0} (x^3 - x)\, dx + \left(-\int_{0}^{1} (x^3 - x)\, dx\right) \]

対称性より、2つの積分の絶対値は等しい：

\[ \int_{-1}^{0} (x^3 - x)\, dx = \left[\frac{x^4}{4} - \frac{x^2}{2}\right]_{-1}^{0} = 0 - \left(\frac{1}{4} - \frac{1}{2}\right) = \frac{1}{4} \]
\[ S = \frac{1}{4} + \frac{1}{4} = \frac{1}{2} \]

4. 簡単にするコツ

✏️ コツ1：1/6公式が使えるか最初にチェック
「放物線とx軸」「放物線と直線」→ 1/6公式が使える！
「2つの放物線」→ 1/12公式が使える！
公式が使えれば積分計算が不要で、計算ミスも激減。
✏️ コツ2：グラフの概形を描いてから計算
どちらが上でどちらが下か、必ず確認する。
「上 − 下」を間違えると、符号が逆になり答えが変わってしまう。
✏️ コツ3：「上 − 下」の式は因数分解してから
\(f(x) - g(x)\) を計算したら、まず因数分解を試みる。
因数分解できれば、1/6公式や1/12公式が使えることが多い。
✏️ コツ4：交点の計算は丁寧に
面積計算の多くは「交点を求める」→「積分区間が決まる」という流れ。
交点を間違えると、すべてが狂う。必ず検算しよう。

5. よくあるつまづき

⚠️ つまづき1：x軸より下の部分の符号を忘れる

ダメな例： \(y = x^2 - 4\) と x軸の面積を \(\int_{-2}^{2} (x^2 - 4)\, dx = -\frac{32}{3}\) として終わり

対策： 面積は必ず正！x軸より下ならマイナスをつける。
\(S = -\int_{-2}^{2} (x^2 - 4)\, dx = \frac{32}{3}\)（○）

⚠️ つまづき2：「上から下」を逆にする

ダメな例： 下の関数から上の関数を引いてしまう

対策： 必ずグラフを描いて確認。「上 − 下」で正の値になる。
迷ったら、区間内の適当な点で値を代入して確認。

⚠️ つまづき3：1/6公式の係数を間違える

ダメな例： \(y = 2x^2 - 8\) で \(a = 1\) として計算

対策： 元の式の \(x^2\) の係数が \(a\)。
\(y = 2x^2 - 8 = 2(x^2 - 4) = 2(x+2)(x-2)\) なので \(a = 2\)。

⚠️ つまづき4：上下が入れ替わる場合に分割を忘れる

ダメな例： \(y = x^3 - x\) と x軸の面積を一括で積分

対策： 交点が3つ以上あれば上下が入れ替わる可能性あり。
各区間で上下を確認し、分割して計算する。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

放物線 \(y = -x^2 + 6x - 5\) と x軸で囲まれた面積を求めよ。

答えを見る

交点：\(-x^2 + 6x - 5 = 0\) → \(x^2 - 6x + 5 = 0\) → \((x-1)(x-5) = 0\) → \(x = 1, 5\)

\(y = -(x-1)(x-5)\) なので \(a = -1\)、\(\alpha = 1\)、\(\beta = 5\)

1/6公式：\(S = \frac{|-1|}{6}(5-1)^3 = \frac{1}{6} \cdot 64 = \frac{32}{3}\)

練習2

放物線 \(y = x^2 - 2x\) と直線 \(y = x\) で囲まれた面積を求めよ。

答えを見る

交点：\(x^2 - 2x = x\) → \(x^2 - 3x = 0\) → \(x(x-3) = 0\) → \(x = 0, 3\)

上 − 下：\(x - (x^2 - 2x) = -x^2 + 3x = -x(x-3)\)

\(a = -1\)、\(\alpha = 0\)、\(\beta = 3\)

1/6公式：\(S = \frac{|-1|}{6}(3-0)^3 = \frac{1}{6} \cdot 27 = \frac{9}{2}\)

練習3

2つの放物線 \(y = x^2 - 4x + 3\) と \(y = -x^2 + 2x + 3\) で囲まれた面積を求めよ。

答えを見る

交点：\(x^2 - 4x + 3 = -x^2 + 2x + 3\) → \(2x^2 - 6x = 0\) → \(2x(x-3) = 0\) → \(x = 0, 3\)

上 − 下：\((-x^2 + 2x + 3) - (x^2 - 4x + 3) = -2x^2 + 6x = -2x(x-3)\)

\(x^2\) の係数の差は \(-2\)、よって \(|a| = 2\)

1/12公式：\(S = \frac{2}{12}(3-0)^3 = \frac{1}{6} \cdot 27 = \frac{9}{2}\)

📌 まとめ：面積は「上 − 下」を積分。1/6公式・1/12公式で高速計算！

おまけ：歴史ストーリー「ケプラーのワイン樽問題」

📖 ケプラーのワイン樽問題（17世紀・オーストリア）

1612年、天文学者ヨハネス・ケプラーは結婚式のためにワインを大量に購入しました。ところが、ワイン商人が樽の容量を測る方法を見て、彼は首をかしげました。

商人は樽に斜めに棒を差し込み、その長さだけで容量を判断していたのです。

「いやいや、それじゃ正確じゃないだろ」——天文学者の血が騒ぎました。

ケプラーは「曲線で囲まれた立体の体積をどう計算するか」という問題に取り組み、樽を無数の薄い円盤の積み重ねと考えました。これは現代の積分の考え方そのものです。

\[ V = \int_a^b \pi r(x)^2 \, dx \]

彼の研究は『ワイン樽の新立体幾何学』（1615年）として出版され、微分積分学の先駆けとなりました。ニュートンやライプニッツが微積分を完成させる50年以上も前のことです。

惑星の運動法則で有名なケプラーですが、実は「ワインの値段が気になって」積分学の種を蒔いていたとは...。科学の発展は意外なところから始まるものですね。

ちなみに、結婚式は大成功だったそうです。数学的に正確な量のワインのおかげかもしれません。
