1. テーマの定義

地震のマグニチュードや音の大きさ（デシベル）、星の明るさ（等級）には対数が使われています。桁違いの大きさを扱いやすくするのが対数の力です。

このページでは、対数関数を学びます：

  • 対数の定義：「\(a\)を何乗すると\(b\)になるか」を\(\log_a b\)と表す
  • 対数の性質：積・商・累乗を簡単にする3つの法則
  • 対数関数のグラフ：底が1より大きいか小さいかで形が変わる
  • 対数方程式・不等式：真数条件に注意して解く
  • 常用対数：桁数計算や科学的応用に活用

2. 覚えるルール

【最重要】対数の定義：\(a^p = M\) ⇔ \(p = \log_a M\)（「\(a\)を何乗すると\(M\)になるか」が対数）
指数形式と対数形式の相互変換
📌 対数の基本値 → 例1
\[ \log_a 1 = 0, \quad \log_a a = 1 \]

\(a^0 = 1\)より\(\log_a 1 = 0\)、\(a^1 = a\)より\(\log_a a = 1\)となる。

→ だから結果は： 真数が1なら対数は0、真数が底と同じなら対数は1。

💡 考え方を見る

対数の定義に当てはめると、\(a^? = 1\)を満たす?は0、\(a^? = a\)を満たす?は1です。

📌 対数の3つの法則 → 例2
\[ \log_a MN = \log_a M + \log_a N \quad \text{（積 → 和）} \]
\[ \log_a \frac{M}{N} = \log_a M - \log_a N \quad \text{（商 → 差）} \]
\[ \log_a M^r = r \log_a M \quad \text{（累乗 → 係数）} \]

→ だから結果は： かけ算を足し算に、割り算を引き算に、累乗を係数に変換できる！

✅ メリット： 大きな数の計算が劇的に簡単になる。

💡 考え方を見る

\(a^p = M\)、\(a^q = N\)とすると、\(MN = a^p \cdot a^q = a^{p+q}\)より、\(\log_a MN = p + q = \log_a M + \log_a N\)。

📌 底の変換公式 → 例3
\[ \log_a b = \frac{\log_c b}{\log_c a} \]

異なる底の対数を、共通の底（よく使うのは10やe）に揃えられる。

→ だから結果は： 電卓やプログラムで常用対数（底10）や自然対数（底e）を使って計算できる。

📌 対数関数のグラフ → 例4

底 \(a > 1\) のとき： 右上がりの曲線、xが大きくなるとyも増加（単調増加）

底 \(0 < a < 1\) のとき： 右下がりの曲線、xが大きくなるとyは減少（単調減少）

共通点： 点(1, 0)を通り、y軸が漸近線。定義域は\(x > 0\)、値域は実数全体。

対数関数のグラフ：底が1より大きいか小さいかで増減が逆転
📌 対数方程式・不等式の解き方 → 例5, 例6

真数条件を必ず確認： 真数 > 0 を満たす範囲で解く

方程式： 両辺の対数を取るか、指数形式に変換

不等式： 底が1より大きいか小さいかで不等号の向きに注意

\[ a > 1 \text{ のとき } \log_a M < \log_a N \Leftrightarrow M < N \]
\[ 0 < a < 1 \text{ のとき } \log_a M < \log_a N \Leftrightarrow M > N \text{（不等号が逆転！）} \]
📌 常用対数と桁数 → 例7
\[ \text{正の整数 } N \text{ の桁数} = \lfloor \log_{10} N \rfloor + 1 \]

\(\log_{10} 2 \approx 0.3010\)、\(\log_{10} 3 \approx 0.4771\)を使って計算。

→ だから結果は： \(2^{100}\)のような巨大な数の桁数も計算できる！

3. 例題と解き方

対数の定義と法則を使って、さまざまな問題を解いていきましょう。

▶ 対数の計算

例1対数の値を求める

\(\log_2 8\)、\(\log_3 \frac{1}{9}\)、\(\log_{10} 0.01\)の値を求めよ。

解き方：

  1. 対数の定義「\(a\)を何乗すると真数になるか」を考える
  2. \(\log_2 8\)：\(2^3 = 8\)より \(\log_2 8 = 3\)
  3. \(\log_3 \frac{1}{9}\)：\(3^{-2} = \frac{1}{9}\)より \(\log_3 \frac{1}{9} = -2\)
  4. \(\log_{10} 0.01\)：\(10^{-2} = 0.01\)より \(\log_{10} 0.01 = -2\)
\[ \log_2 8 = 3, \quad \log_3 \frac{1}{9} = -2, \quad \log_{10} 0.01 = -2 \]
例2対数法則を使った計算

\(\log_2 6 + \log_2 \frac{8}{3}\)を簡単にせよ。

解き方：

  1. 和の法則で1つにまとめる：\(\log_a M + \log_a N = \log_a MN\)
  2. \(\log_2 6 + \log_2 \frac{8}{3} = \log_2 \left(6 \times \frac{8}{3}\right)\)
  3. \(= \log_2 16 = \log_2 2^4 = 4\)
\[ \log_2 6 + \log_2 \frac{8}{3} = \log_2 16 = 4 \]
例3底の変換公式

\(\log_4 8\)を求めよ。

解き方：

  1. 底を2に揃える：\(\log_4 8 = \frac{\log_2 8}{\log_2 4}\)
  2. それぞれ計算：\(\log_2 8 = 3\)、\(\log_2 4 = 2\)
  3. \(\log_4 8 = \frac{3}{2}\)
\[ \log_4 8 = \frac{\log_2 8}{\log_2 4} = \frac{3}{2} \]

▶ 対数関数のグラフ

例4グラフの読み取り

\(y = \log_3 x\)のグラフについて、次の問いに答えよ。

(1) \(x = 9\)のときの\(y\)の値

(2) \(y = -1\)のときの\(x\)の値

(3) グラフが通る点で座標がともに整数であるものを2つ

解き方：

  1. (1) \(y = \log_3 9 = \log_3 3^2 = 2\)
  2. (2) \(-1 = \log_3 x\)より\(x = 3^{-1} = \frac{1}{3}\)
  3. (3) \(\log_3 1 = 0\)より(1, 0)、\(\log_3 3 = 1\)より(3, 1)、\(\log_3 9 = 2\)より(9, 2)など
\[ (1)\ y = 2, \quad (2)\ x = \frac{1}{3}, \quad (3)\ (1, 0), (3, 1), (9, 2) \]

▶ 対数方程式・不等式

例5対数方程式

\(\log_2 (x-1) + \log_2 (x+2) = 2\)を解け。

解き方：

  1. 真数条件を確認：\(x - 1 > 0\)かつ\(x + 2 > 0\) → \(x > 1\)
  2. 対数法則で1つにまとめる：\(\log_2 (x-1)(x+2) = 2\)
  3. 指数形式に変換：\((x-1)(x+2) = 2^2 = 4\)
  4. 展開して整理：\(x^2 + x - 2 = 4\) → \(x^2 + x - 6 = 0\)
  5. 因数分解：\((x+3)(x-2) = 0\) → \(x = -3, 2\)
  6. 真数条件を確認：\(x > 1\)を満たすのは\(x = 2\)のみ
\[ x = 2 \]
例6対数不等式

\(\log_{\frac{1}{2}} (x-3) > -2\)を解け。

解き方：

  1. 真数条件：\(x - 3 > 0\) → \(x > 3\)
  2. 右辺を対数で表す：\(-2 = \log_{\frac{1}{2}} \left(\frac{1}{2}\right)^{-2} = \log_{\frac{1}{2}} 4\)
  3. 不等式は\(\log_{\frac{1}{2}} (x-3) > \log_{\frac{1}{2}} 4\)
  4. 底が\(0 < \frac{1}{2} < 1\)なので不等号が逆転：\(x - 3 < 4\)
  5. \(x < 7\)
  6. 真数条件と合わせて：\(3 < x < 7\)
\[ 3 < x < 7 \]

▶ 常用対数の応用

例7桁数の計算

\(2^{50}\)は何桁の整数か。ただし\(\log_{10} 2 = 0.3010\)とする。

解き方：

  1. 常用対数を取る：\(\log_{10} 2^{50} = 50 \log_{10} 2 = 50 \times 0.3010 = 15.05\)
  2. これは\(10^{15.05}\)を意味する → \(10^{15} < 2^{50} < 10^{16}\)
  3. \(10^{15}\)は16桁、\(10^{16}\)は17桁だから、\(2^{50}\)は16桁
\[ \log_{10} 2^{50} = 15.05 \quad \therefore \text{16桁} \]
例8最高位の数字

\(2^{10}\)の最高位の数字を求めよ。ただし\(\log_{10} 2 = 0.3010\)とする。

解き方：

  1. \(\log_{10} 2^{10} = 10 \times 0.3010 = 3.010\)
  2. \(2^{10} = 10^{3.010} = 10^{0.010} \times 10^3\)
  3. \(10^{0.010} \approx 1.023\)（\(\log_{10} 1.023 \approx 0.010\)より）
  4. よって\(2^{10} \approx 1.023 \times 10^3 = 1023\)
  5. 最高位の数字は1（実際、\(2^{10} = 1024\)）
\[ 2^{10} = 1024 \quad \therefore \text{最高位は 1} \]

4. 簡単にするコツ

✏️ コツ1：指数と対数は「翻訳」の関係
\(a^p = M\)と\(p = \log_a M\)は同じことを違う書き方で表している。「\(a\)を何乗すると\(M\)?」と考えれば、指数形式と対数形式を自由に行き来できる。
✏️ コツ2：対数の法則は「指数法則の翻訳版」
積→和、商→差、累乗→係数。これらは指数法則\(a^m \cdot a^n = a^{m+n}\)などを対数で表現したもの。指数と対数はセットで覚えよう。
✏️ コツ3：底を揃えてから計算
\(\log_2 \)と\(\log_4 \)のように底が違う対数が混ざったら、底の変換公式で揃える。特に\(\log_4 = \frac{\log_2}{\log_2 4} = \frac{\log_2}{2}\)のように2のべき乗の底は2に揃えると計算しやすい。
✏️ コツ4：不等式は底で場合分け
対数不等式で最も多いミスは底による不等号の向きの変化。底 > 1 なら不等号そのまま、0 < 底 < 1 なら不等号逆転。これを忘れると答えが真逆になる！
✏️ コツ5：常用対数の近似値は丸暗記
\(\log_{10} 2 \approx 0.3010\)、\(\log_{10} 3 \approx 0.4771\)を覚えておくと、桁数計算がスムーズ。\(\log_{10} 5 = \log_{10} \frac{10}{2} = 1 - 0.3010 = 0.6990\)と計算することもできる。

5. よくあるつまづき

⚠️ つまづき1：真数条件を忘れる

ダメな例： \(\log_2 (x-1) = 3\)を解いて\(x = 9\)としただけ。

対策： 対数方程式・不等式では必ず最初に真数 > 0の条件を書き出す。解が出たら必ず真数条件を満たすか確認！

⚠️ つまづき2：対数法則の誤用

ダメな例： \(\log_a (M + N) = \log_a M + \log_a N\)と計算してしまう。

対策： 対数法則が使えるのは積・商・累乗のみ！和や差には使えない。\(\log_a (M + N)\)はこれ以上分解できない。

⚠️ つまづき3：底による不等号の向きを間違える

ダメな例： \(\log_{\frac{1}{2}} x < 2\)を解いて\(x < \frac{1}{4}\)としてしまう。

対策： 底が1より小さいとき、対数関数は減少関数。\(\log_a M < \log_a N\)のとき\(M > N\)（不等号逆転）。正解は\(x > \frac{1}{4}\)（真数条件\(x > 0\)と合わせて）。

⚠️ つまづき4：\(\log_a 1 = 0\)と\(\log_a a = 1\)を混同

ダメな例： \(\log_3 1 = 1\)としてしまう。

対策： 「\(3\)を何乗すると\(1\)になる?」→ \(3^0 = 1\)だから答えは\(0\)。真数が\(1\)なら答えは常に\(0\)、真数が底と同じなら答えは常に\(1\)。

⚠️ つまづき5：桁数計算で1ずれる

ダメな例： \(\log_{10} N = 5.3\)だから5桁としてしまう。

対策： \(10^5 < N < 10^6\)のとき、\(N\)は\(10^5 = 100000\)（6桁）以上\(10^6\)（7桁）未満。つまり6桁。桁数 = 整数部分 + 1 と覚えよう。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の値を求めよ。

(1) \(\log_5 125\)

(2) \(\log_4 2 + \log_4 8\)

(3) \(\log_9 27\)

答えを見る

(1) \(5^3 = 125\)より \(\log_5 125 = 3\)

(2) \(\log_4 2 + \log_4 8 = \log_4 16 = \log_4 4^2 = 2\)

(3) 底を3に変換：\(\log_9 27 = \frac{\log_3 27}{\log_3 9} = \frac{3}{2}\)

練習2

方程式 \(\log_3 (2x+1) = 2\) を解け。

答えを見る

真数条件：\(2x + 1 > 0\) → \(x > -\frac{1}{2}\)

指数形式に変換：\(2x + 1 = 3^2 = 9\)

\(2x = 8\) → \(x = 4\)

真数条件を満たす。よって \(x = 4\)

練習3

不等式 \(\log_2 (x-2) \leq 3\) を解け。

答えを見る

真数条件：\(x - 2 > 0\) → \(x > 2\)

底が\(2 > 1\)なので不等号の向きはそのまま

\(x - 2 \leq 2^3 = 8\) → \(x \leq 10\)

真数条件と合わせて：\(2 < x \leq 10\)

練習4

\(3^{20}\)は何桁の整数か。ただし\(\log_{10} 3 = 0.4771\)とする。

答えを見る

\(\log_{10} 3^{20} = 20 \times 0.4771 = 9.542\)

\(10^9 < 3^{20} < 10^{10}\)

\(10^9\)は10桁、よって\(3^{20}\)は10桁

📌 まとめ：対数は「何乗?」を求める逆演算。積を和に変える法則と真数条件を押さえれば、桁違いの計算も自由自在！

おまけ：歴史ストーリー「対数を発明した男」

📖 20年かけて計算を革命した数学者

1614年、スコットランドの数学者ジョン・ネイピア（John Napier）は「対数」を発表しました。当時、天文学者たちは膨大な計算に悩まされていました。惑星の軌道計算には何桁もの数のかけ算・割り算が必要で、1つの計算に何時間もかかることも珍しくありませんでした。

ネイピアは「かけ算を足し算に変えられないか」と考えました。

\[ \log(A \times B) = \log A + \log B \]

この発想のために、彼は20年以上をかけて対数表を作成しました。手計算だけで膨大な数値を計算し、表にまとめたのです。

この対数表が発表されると、天文学者たちは熱狂しました。有名な天文学者ケプラーは「ネイピアの発明によって、私たちの寿命が2倍になった」と言ったとか。計算時間が劇的に短縮され、その時間を新しい発見に使えるようになったからです。

さらに、対数の発明は計算尺の誕生にもつながりました。計算尺は20世紀後半まで科学者やエンジニアの必須道具で、アポロ計画で月に行った宇宙飛行士たちも持っていました。

現代では電卓やコンピュータがありますが、対数は今でも地震のマグニチュード（\(M = \log E\)の形）、音の大きさ（デシベル）、pHの計算、情報理論など、あらゆる分野で活躍しています。

ネイピアが20年かけた対数表、今なら数行のプログラムで作れますが、彼の「かけ算を足し算に」という発想がなければ、現代科学の発展はもっと遅れていたかもしれませんね。
