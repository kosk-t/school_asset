1. テーマの定義

中学で学んだ二次方程式を発展させ、「解が何個あるか」を判定したり、「不等式」を解いたりする方法を学びます。

このページでは、二次方程式と二次不等式を学びます：

  • 判別式：解の個数（2個・1個・0個）を判定する
  • 解と係数の関係：解を求めずに和・積を計算する
  • 二次不等式：グラフを使って不等式を解く

2. 覚えるルール

🎯 最重要：判別式 \(D = b^2 - 4ac\) の符号で解の個数が決まる！
判別式 D の符号と放物線の x 軸との交点の関係
📌 ルール1：判別式 → 例1

二次方程式 \(ax^2 + bx + c = 0\) の判別式：

\[ D = b^2 - 4ac \]
  • \(D > 0\)：異なる2つの実数解
  • \(D = 0\)：重解（1つの実数解）
  • \(D < 0\)：実数解なし（虚数解のみ）

→ だから結果は： 解を求めなくても、解の個数が分かる

💡 考え方を見る

解の公式 \(x = \dfrac{-b \pm \sqrt{b^2-4ac}}{2a}\) の \(\sqrt{\ }\) の中身が \(D\)。

\(D > 0\) なら \(\pm\) で2つの解、\(D = 0\) なら \(\pm 0\) で1つ、\(D < 0\) なら \(\sqrt{\text{負}}\) で実数解なし。

📌 ルール2：解と係数の関係 → 例2

\(ax^2 + bx + c = 0\) の2つの解を \(\alpha, \beta\) とすると：

\[ \alpha + \beta = -\frac{b}{a}, \quad \alpha\beta = \frac{c}{a} \]

✅ メリット： 解を求めなくても、解の和・積が分かる

💡 覚え方を見る

\((x - \alpha)(x - \beta) = x^2 - (\alpha + \beta)x + \alpha\beta\) と展開。

\(ax^2 + bx + c = a(x^2 + \frac{b}{a}x + \frac{c}{a})\) と比較すれば導ける。

📌 ルール3：二次不等式の解き方 → 例3・4

\(ax^2 + bx + c > 0\)（または \(< 0\)）の解き方：

  1. \(y = ax^2 + bx + c\) のグラフを考える
  2. \(y = 0\) となる点（x軸との交点）を求める
  3. グラフが x軸より上（\(> 0\)）か下（\(< 0\)）かで判断
📌 ルール4：二次不等式の解のパターン（\(a > 0\) の場合）

\(ax^2 + bx + c = 0\) の解を \(\alpha, \beta\)（\(\alpha < \beta\)）とすると：

不等式 \(D > 0\)（2解） \(D = 0\)（重解） \(D < 0\)（解なし）
\(ax^2 + bx + c > 0\) \(x < \alpha, \beta < x\) \(x \neq \alpha\) の全実数 すべての実数
\(ax^2 + bx + c < 0\) \(\alpha < x < \beta\) 解なし 解なし

※ \(a < 0\) の場合は不等号の向きが逆になる

📌 ルール5：常に正・常に負の条件 → 例5

\(ax^2 + bx + c\) がすべての実数 \(x\) で正になる条件：

\[ a > 0 \text{ かつ } D < 0 \]

\(ax^2 + bx + c\) がすべての実数 \(x\) で負になる条件：

\[ a < 0 \text{ かつ } D < 0 \]

3. 例題と解き方

ルールを使って実際に問題を解いてみましょう。

▶ 判別式

例1\(x^2 - 4x + k = 0\) が異なる2つの実数解を持つ \(k\) の範囲

解き方：

  1. 判別式 \(D = b^2 - 4ac = (-4)^2 - 4 \cdot 1 \cdot k = 16 - 4k\)
  2. 異なる2つの実数解 → \(D > 0\)
  3. \(16 - 4k > 0\) を解く
\[ 16 - 4k > 0 \Rightarrow k < 4 \]

▶ 解と係数の関係

例2\(x^2 - 5x + 3 = 0\) の2解を \(\alpha, \beta\) とするとき、\(\alpha^2 + \beta^2\) の値

解き方：

  1. 解と係数の関係：\(\alpha + \beta = 5\)、\(\alpha\beta = 3\)
  2. \(\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta\) に代入
\[ \alpha^2 + \beta^2 = 5^2 - 2 \cdot 3 = 25 - 6 = 19 \]

▶ 二次不等式

例3\(x^2 - 5x + 6 > 0\) を解く

解き方：

  1. \(x^2 - 5x + 6 = 0\) を解く → \((x-2)(x-3) = 0\) → \(x = 2, 3\)
  2. \(a = 1 > 0\) なので下に凸の放物線
  3. 「> 0」は x軸より上の部分 → 2つの解の外側
\[ x < 2 \text{ または } x > 3 \]
例4\(x^2 - 5x + 6 < 0\) を解く

解き方：

  1. 例3と同じく、\(x = 2, 3\) で x軸と交わる
  2. 「< 0」は x軸より下の部分 → 2つの解の内側
\[ 2 < x < 3 \]
例5\(x^2 + 2x + k > 0\) がすべての実数 \(x\) で成り立つ \(k\) の範囲

解き方：

  1. \(a = 1 > 0\) なので「常に正」の条件は \(D < 0\)
  2. \(D = 4 - 4k < 0\)
  3. \(4 - 4k < 0\) を解く
\[ k > 1 \]

▶ 解がない場合

例6\(x^2 + 2x + 5 < 0\) を解く

解き方：

  1. 判別式：\(D = 4 - 20 = -16 < 0\) → 実数解なし
  2. \(a = 1 > 0\) で下に凸、かつ x軸と交わらない → 常に正
  3. 「< 0」を満たす \(x\) は存在しない
\[ \text{解なし} \]

4. 簡単にするコツ

✏️ コツ1：グラフを描く
二次不等式は必ずグラフを描く。下に凸なら「∪」、上に凸なら「∩」。x軸との交点を求めれば、どこが正・負か一目瞭然。
✏️ コツ2：「外側・内側」で覚える
下に凸（\(a > 0\)）の場合：「> 0」は外側、「< 0」は内側。上に凸は逆。
✏️ コツ3：解と係数の関係で式変形
\(\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta\) など、和・積だけで表せる形に変形する。
✏️ コツ4：「常に正」は D < 0 と a > 0
x軸と交わらず（\(D < 0\)）、下に凸（\(a > 0\)）なら常に正。両方の条件を確認！

5. よくあるつまづき

⚠️ つまづき1：不等式の向きとグラフの対応ミス

ダメな例： \(x^2 - 4 > 0\) の解を「\(-2 < x < 2\)」と答える

対策： 「> 0」は x軸より上。下に凸なら外側。正解は「\(x < -2, x > 2\)」。

⚠️ つまづき2：判別式の計算ミス

ダメな例： \(D = b^2 - 4ac\) で \(b\) を2倍してしまう（\(2bx\) の場合）

対策： \(ax^2 + bx + c\) の形にそろえてから \(D\) を計算。\(x^2 - 4x + 3\) なら \(b = -4\)。

⚠️ つまづき3：「常に正」で a の符号を忘れる

ダメな例： \(D < 0\) だけで「常に正」と判断

対策： \(D < 0\) かつ \(a > 0\) が必要。\(a < 0\) なら常に負になってしまう。

6. 仕上げチェックリスト

判別式 \(D = b^2 - 4ac\) を正確に計算できる
\(D\) の符号と解の個数の関係を覚えている
解と係数の関係（和と積）を使える

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(x^2 + 6x + k = 0\) が重解を持つとき、\(k\) の値を求めよ。

答えを見る

重解 → \(D = 0\)

\(D = 36 - 4k = 0\)

\(k = 9\)

練習2

\(x^2 - 3x - 10 < 0\) を解け。

答えを見る

\(x^2 - 3x - 10 = 0\) → \((x-5)(x+2) = 0\) → \(x = -2, 5\)

下に凸で「< 0」なので内側

\(-2 < x < 5\)

練習3

\(2x^2 - 3x + 1 = 0\) の2解を \(\alpha, \beta\) とするとき、\(\dfrac{1}{\alpha} + \dfrac{1}{\beta}\) の値を求めよ。

答えを見る

解と係数の関係：\(\alpha + \beta = \dfrac{3}{2}\)、\(\alpha\beta = \dfrac{1}{2}\)

\(\dfrac{1}{\alpha} + \dfrac{1}{\beta} = \dfrac{\alpha + \beta}{\alpha\beta} = \dfrac{3/2}{1/2} = 3\)

📌 まとめ：判別式 D で解の個数を判定！二次不等式はグラフで「外側・内側」を見る！

おまけ：数学雑学「判別式の正体」

🔍 なぜ \(b^2 - 4ac\) なのか？

判別式 \(D = b^2 - 4ac\)。この式、どこから来たのか？

実は、解の公式のルートの中身がそのまま判別式だ：

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

\(\pm\sqrt{D}\) の部分を見れば、すべてが分かる：

  • \(D > 0\) → \(\sqrt{D}\) は正の数 → \(\pm\) で2つの解
  • \(D = 0\) → \(\sqrt{0} = 0\) → \(\pm 0\) は同じ → 重解
  • \(D < 0\) → \(\sqrt{\text{負}}\) は実数にならない → 実数解なし

つまり判別式とは、「解の公式が実数を出力できるかどうか」を判定しているのだ。

ちなみに、\(D < 0\) のときに出てくる「虚数解」は、高校数学IIで学ぶ。\(\sqrt{-1} = i\)（虚数単位）を使えば、\(D < 0\) でも解が求まる。

判別式は、解の公式の「門番」だった。
