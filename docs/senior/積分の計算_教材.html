1. テーマの定義

微分は「変化率を求める」操作でした。では、変化率から元の関数を復元するには？それが積分です。曲線で囲まれた面積を求めるなど、様々な場面で威力を発揮します。

このページでは、積分の基本を学びます：

  • 不定積分：微分の逆操作で、原始関数を求める
  • 積分公式：べき乗・定数倍・和差の積分
  • 定積分：区間を指定して数値を計算する

2. 覚えるルール

【最重要】積分は微分の逆操作！\(F'(x) = f(x)\) のとき \(\displaystyle \int f(x)\,dx = F(x) + C\)
微分と積分は逆の関係
📌 べき乗の積分公式 → 例1
\[ \int x^n\,dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1) \]

指数に1を足して、その数で割る。微分の逆操作そのもの！

→ だから結果は： \(x^3\)を積分すると\(\dfrac{x^4}{4}+C\)、\(x^2\)を積分すると\(\dfrac{x^3}{3}+C\)

💡 考え方を見る

微分すると指数が1減って前に降りてくるから、積分は逆に指数を1増やして、増えた数で割れば元に戻る。

例：\(\left(\dfrac{x^4}{4}\right)' = 4 \cdot \dfrac{x^3}{4} = x^3\) だから \(\int x^3\,dx = \dfrac{x^4}{4} + C\)

📌 定数・定数倍・和差の積分 → 例2
\[ \int k\,dx = kx + C, \quad \int kf(x)\,dx = k\int f(x)\,dx \]
\[ \int \{f(x) \pm g(x)\}\,dx = \int f(x)\,dx \pm \int g(x)\,dx \]

→ だから結果は： 各項を別々に積分して足し引きすればOK！

✅ メリット： 複雑な式も項ごとに分解して計算できる

📌 定積分の計算 → 例3
\[ \int_a^b f(x)\,dx = \Big[F(x)\Big]_a^b = F(b) - F(a) \]

\(F(x)\)は\(f(x)\)の原始関数（+Cは不要）。上端の値から下端の値を引く！

→ だから結果は： 定積分は具体的な数値になる（積分定数Cは消える）

定積分は曲線とx軸で囲まれた面積を表す
📌 定積分の性質 → 例4
\[ \int_a^a f(x)\,dx = 0, \quad \int_a^b f(x)\,dx = -\int_b^a f(x)\,dx \]
\[ \int_a^b f(x)\,dx + \int_b^c f(x)\,dx = \int_a^c f(x)\,dx \]

→ だから結果は： 区間の分割や向きの反転が自在にできる

📌 よく使う積分公式まとめ
関数 \(f(x)\)不定積分 \(\int f(x)\,dx\)
\(x^n\) \((n \neq -1)\)\(\dfrac{x^{n+1}}{n+1} + C\)
\(1\)（定数）\(x + C\)
\(k\)（定数）\(kx + C\)

3. 例題と解き方

積分公式を使って実際に計算してみましょう。

▶ 不定積分の計算

例1べき乗の積分

\(\displaystyle \int x^4\,dx\) を計算せよ。

解き方：

  1. 指数4に1を足して5
  2. 5で割る
  3. 積分定数+Cを忘れずに
\[ \int x^4\,dx = \frac{x^5}{5} + C \]

検算： \(\left(\dfrac{x^5}{5}\right)' = \dfrac{5x^4}{5} = x^4\) ✓

例2多項式の積分

\(\displaystyle \int (3x^2 - 4x + 5)\,dx\) を計算せよ。

解き方：

  1. 各項を別々に積分する
  2. \(3x^2 \to 3 \cdot \dfrac{x^3}{3} = x^3\)
  3. \(-4x \to -4 \cdot \dfrac{x^2}{2} = -2x^2\)
  4. \(5 \to 5x\)
  5. 最後に+Cをつける
\[ \int (3x^2 - 4x + 5)\,dx = x^3 - 2x^2 + 5x + C \]

▶ 定積分の計算

例3基本の定積分

\(\displaystyle \int_1^3 x^2\,dx\) を計算せよ。

解き方：

  1. まず不定積分を求める：\(x^2 \to \dfrac{x^3}{3}\)
  2. 上端x=3を代入：\(\dfrac{3^3}{3} = \dfrac{27}{3} = 9\)
  3. 下端x=1を代入：\(\dfrac{1^3}{3} = \dfrac{1}{3}\)
  4. 上端 - 下端を計算
\[ \int_1^3 x^2\,dx = \left[\frac{x^3}{3}\right]_1^3 = \frac{27}{3} - \frac{1}{3} = \frac{26}{3} \]
例4定積分の性質

\(\displaystyle \int_0^2 (x^2 + 1)\,dx + \int_2^3 (x^2 + 1)\,dx\) を計算せよ。

解き方：

  1. 区間の性質より、2つをつなげる
  2. \(\displaystyle \int_0^3 (x^2 + 1)\,dx\) として計算
\[ \int_0^3 (x^2 + 1)\,dx = \left[\frac{x^3}{3} + x\right]_0^3 = \left(9 + 3\right) - 0 = 12 \]
例5分数・負の指数の積分

\(\displaystyle \int \frac{1}{x^2}\,dx\) を計算せよ。

解き方：

  1. \(\dfrac{1}{x^2} = x^{-2}\) と変形
  2. 指数-2に1を足して-1
  3. -1で割る
\[ \int x^{-2}\,dx = \frac{x^{-1}}{-1} + C = -\frac{1}{x} + C \]
例6展開してから積分

\(\displaystyle \int (x+1)(x-2)\,dx\) を計算せよ。

解き方：

  1. まず展開：\((x+1)(x-2) = x^2 - x - 2\)
  2. 各項を積分
\[ \int (x^2 - x - 2)\,dx = \frac{x^3}{3} - \frac{x^2}{2} - 2x + C \]

4. 簡単にするコツ

✏️ コツ1：微分で検算する
積分の答えが不安なときは、微分して元に戻るか確認！

例：\(\int x^3\,dx = \dfrac{x^4}{4} + C\) → 微分すると \(\dfrac{4x^3}{4} = x^3\) ✓

✏️ コツ2：分数は指数に直す
\(\dfrac{1}{x^n} = x^{-n}\)、\(\sqrt{x} = x^{1/2}\) と変形してから公式を使う！

例：\(\int \sqrt{x}\,dx = \int x^{1/2}\,dx = \dfrac{x^{3/2}}{3/2} + C = \dfrac{2}{3}x^{3/2} + C\)

✏️ コツ3：定積分は途中式を丁寧に
\(\Big[F(x)\Big]_a^b = F(b) - F(a)\) の形を書いてから代入！符号ミスを防げる。
✏️ コツ4：積分前に因数分解・展開
\((x+1)^2\) はそのまま積分できないので \(x^2+2x+1\) に展開してから！

5. よくあるつまづき

⚠️ つまづき1：積分定数Cを忘れる

ダメな例： \(\int x^2\,dx = \dfrac{x^3}{3}\)

対策： 不定積分では必ず+Cをつける！定積分では不要。

⚠️ つまづき2：定積分の上下を間違える

ダメな例： \(\Big[F(x)\Big]_a^b = F(a) - F(b)\)

対策： 「上端 - 下端」と覚える。上が先！

⚠️ つまづき3：指数計算のミス

ダメな例： \(\int x^3\,dx = \dfrac{x^3}{3}\) (指数が変わっていない！)

対策： 「指数に1を足す」を忘れない。\(3+1=4\)で\(\dfrac{x^4}{4}\)

⚠️ つまづき4：積の積分で公式を間違える

ダメな例： \(\int x \cdot x^2\,dx = \int x\,dx \cdot \int x^2\,dx\) (これは間違い！)

対策： 積は展開してから！\(\int x \cdot x^2\,dx = \int x^3\,dx = \dfrac{x^4}{4} + C\)

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の不定積分を計算せよ。

(1) \(\displaystyle \int (2x^3 - 3x + 1)\,dx\)

(2) \(\displaystyle \int \frac{2}{x^3}\,dx\)

答えを見る

(1) \(\displaystyle \int (2x^3 - 3x + 1)\,dx = \frac{x^4}{2} - \frac{3x^2}{2} + x + C\)

(2) \(\dfrac{2}{x^3} = 2x^{-3}\) より、\(\displaystyle \int 2x^{-3}\,dx = 2 \cdot \frac{x^{-2}}{-2} + C = -\frac{1}{x^2} + C\)

練習2

次の定積分を計算せよ。

(1) \(\displaystyle \int_0^2 (x^2 - 2x)\,dx\)

(2) \(\displaystyle \int_1^4 (3x - 1)\,dx\)

答えを見る

(1) \(\displaystyle \int_0^2 (x^2 - 2x)\,dx = \left[\frac{x^3}{3} - x^2\right]_0^2 = \left(\frac{8}{3} - 4\right) - 0 = -\frac{4}{3}\)

(2) \(\displaystyle \int_1^4 (3x - 1)\,dx = \left[\frac{3x^2}{2} - x\right]_1^4 = \left(24 - 4\right) - \left(\frac{3}{2} - 1\right) = 20 - \frac{1}{2} = \frac{39}{2}\)

📌 まとめ：積分は微分の逆！べき乗は「指数+1してその数で割る」、不定積分には+C、定積分は「上端-下端」

おまけ：歴史ストーリー「ニュートンとライプニッツの優先権争い」

📖 数学史上最大のスキャンダル

17世紀後半、微分積分学は同時期に2人の天才によって発明されました。イギリスのアイザック・ニュートンと、ドイツのゴットフリート・ライプニッツです。

ニュートンは1665年頃に「流率法」として微積分を発見しましたが、すぐには発表しませんでした。一方、ライプニッツは1684年に独自に微積分を発見し、論文を発表。

\[ \int, \frac{d}{dx} \leftarrow \text{これらの記号はライプニッツが考案！} \]

やがて「どちらが先に発明したか」という優先権争いが勃発。イギリスとドイツの国家的な対立にまで発展し、両国の数学者が敵対することに...

結論から言うと、2人は独立に発見したというのが現代の見解。ただし、ライプニッツの記号（\(\int\)や\(\dfrac{dy}{dx}\)）の方が使いやすかったため、現代ではこちらが標準になっています。

💡 豆知識：\(\int\)という記号は、「Sum（和）」のSを縦に伸ばしたもの。微小な面積を足し合わせる（summation）という意味が込められています。ライプニッツのネーミングセンスに感謝！
