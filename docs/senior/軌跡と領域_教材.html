1. テーマの定義

「2点からの距離が等しい点を全部集めるとどんな図形になる？」「時速60km以下で進める範囲は？」こんな問いに答えるのが軌跡と領域です。条件を方程式や不等式で表し、図形として可視化する技術は、最適化問題など実社会でも活躍します。

このページでは、以下の内容を学びます：

  • 軌跡の求め方：「条件を満たす点の集合」を方程式で表す
  • 不等式と領域：不等式が表す座標平面上の範囲を図示する
  • 線形計画法：領域内で一次式の最大・最小を求める

2. 覚えるルール

【最重要】軌跡を求める3ステップ：
(1) 求める点を\((x, y)\)とおく
(2) 条件を\(x, y\)の式で表す
(3) 式を整理して図形の方程式を得る
軌跡の例：2点から等距離にある点の集合
📌 不等式と領域 → 例4

直線 \(ax + by + c = 0\) について：

  • \(ax + by + c > 0\) → 直線の片側の領域
  • \(ax + by + c < 0\) → 直線の反対側の領域

→ 判定法：原点\((0, 0)\)を代入して、不等式が成り立つ側が領域

✅ メリット：グラフを描かなくても領域がわかる

📌 境界線の扱い
  • \(\geq\) または \(\leq\) → 境界線を含む（実線）
  • \(>\) または \(<\) → 境界線を含まない（破線）
📌 線形計画法 → 例6
\[ k = ax + by \text{ の最大・最小} \Rightarrow \text{領域の頂点で調べる} \]

直線 \(y = -\frac{a}{b}x + \frac{k}{b}\) を平行移動させて、領域と共有点を持つ範囲で\(k\)の値域を求める。

→ だから結果は：最大・最小は必ず頂点で達成される

💡 考え方を見る

直線\(ax + by = k\)を\(k\)を変化させながら平行移動すると、領域との共有点がある範囲で\(k\)が動きます。凸多角形領域なら、最も端に接触するのは頂点です。

📌 円の内部・外部 → 例5

円 \((x-a)^2 + (y-b)^2 = r^2\) について：

  • \((x-a)^2 + (y-b)^2 < r^2\) → 円の内部
  • \((x-a)^2 + (y-b)^2 > r^2\) → 円の外部

→ 覚え方：「距離 < 半径」なら内部、「距離 > 半径」なら外部

📌 代表的な軌跡パターン
条件軌跡
定点Aからの距離が一定（\(= r\)）Aを中心とする半径\(r\)の円
2点A, Bから等距離線分ABの垂直二等分線
2点A, Bからの距離の和が一定楕円（\(|PA| + |PB| = 2a\)）
2点A, Bからの距離の比が一定（\(\neq 1\)）アポロニウスの円

3. 例題と解き方

軌跡と領域の問題を、パターン別に解いていきましょう。

▶ 軌跡の求め方

例12点から等距離にある点の軌跡

2点 \(A(1, 2)\), \(B(5, 4)\) から等距離にある点\(P\)の軌跡を求めよ。

解き方：

  1. 求める点を\(P(x, y)\)とおく
  2. 条件「\(PA = PB\)」を式で表す：\(\sqrt{(x-1)^2+(y-2)^2} = \sqrt{(x-5)^2+(y-4)^2}\)
  3. 両辺を2乗して展開：\((x-1)^2+(y-2)^2 = (x-5)^2+(y-4)^2\)
  4. 整理：\(x^2 - 2x + 1 + y^2 - 4y + 4 = x^2 - 10x + 25 + y^2 - 8y + 16\)
  5. \(8x + 4y - 36 = 0\) → \(2x + y - 9 = 0\)
\[ \text{答：直線 } 2x + y - 9 = 0 \text{（線分ABの垂直二等分線）} \]
例2定点と定直線から等距離にある点の軌跡

点 \(F(0, 1)\) と直線 \(y = -1\) から等距離にある点\(P\)の軌跡を求めよ。

解き方：

  1. 点\(P(x, y)\)から点\(F\)までの距離：\(\sqrt{x^2 + (y-1)^2}\)
  2. 点\(P(x, y)\)から直線\(y = -1\)までの距離：\(|y + 1|\)
  3. 条件「等距離」：\(\sqrt{x^2 + (y-1)^2} = |y + 1|\)
  4. 両辺を2乗（\(y \geq -1\)のとき）：\(x^2 + (y-1)^2 = (y+1)^2\)
  5. \(x^2 + y^2 - 2y + 1 = y^2 + 2y + 1\) → \(x^2 = 4y\)
\[ \text{答：放物線 } x^2 = 4y \]

※これは焦点\(F(0, 1)\)、準線\(y = -1\)の放物線の定義そのものです。

例3アポロニウスの円

2点 \(A(0, 0)\), \(B(6, 0)\) からの距離の比が \(1:2\) である点\(P\)の軌跡を求めよ。

解き方：

  1. 条件：\(PA : PB = 1 : 2\) → \(2 \cdot PA = PB\)
  2. \(2\sqrt{x^2 + y^2} = \sqrt{(x-6)^2 + y^2}\)
  3. 両辺を2乗：\(4(x^2 + y^2) = (x-6)^2 + y^2\)
  4. 展開：\(4x^2 + 4y^2 = x^2 - 12x + 36 + y^2\)
  5. \(3x^2 + 3y^2 + 12x - 36 = 0\) → \(x^2 + y^2 + 4x - 12 = 0\)
  6. 平方完成：\((x+2)^2 + y^2 = 16\)
\[ \text{答：中心 } (-2, 0)\text{、半径 } 4 \text{ の円} \]

▶ 不等式と領域

例4直線で分けられる領域

不等式 \(2x - y + 3 > 0\) の表す領域を図示せよ。

解き方：

  1. 境界線 \(2x - y + 3 = 0\) を描く → \(y = 2x + 3\)
  2. 原点\((0, 0)\)を代入：\(2 \cdot 0 - 0 + 3 = 3 > 0\) ✓
  3. 原点を含む側が領域（境界線は含まない → 破線）
2x - y + 3 > 0 の領域（斜線部分、境界を含まない）
例5円の内部

不等式 \(x^2 + y^2 \leq 4\) の表す領域を図示せよ。

解き方：

  1. 境界線：\(x^2 + y^2 = 4\) → 原点中心、半径2の円
  2. \(x^2 + y^2 \leq 4\) → 「中心からの距離が2以下」→ 円の内部
  3. \(\leq\) なので境界を含む（実線）
\[ \text{答：原点中心、半径2の円の内部（周を含む）} \]

▶ 線形計画法

例6線形計画法の基本

条件 \(x \geq 0\), \(y \geq 0\), \(x + y \leq 4\), \(x + 2y \leq 6\) のもとで、\(k = x + 3y\) の最大値を求めよ。

解き方：

  1. 4つの不等式で囲まれる領域を図示する
  2. 領域の頂点を求める：\((0, 0)\), \((4, 0)\), \((2, 2)\), \((0, 3)\)
  3. 各頂点で\(k = x + 3y\)の値を計算：
    • \((0, 0)\)：\(k = 0\)
    • \((4, 0)\)：\(k = 4\)
    • \((2, 2)\)：\(k = 2 + 6 = 8\)
    • \((0, 3)\)：\(k = 0 + 9 = 9\)
  4. 最大値は\((0, 3)\)で達成
\[ \text{答：最大値 } 9 \text{（点 }(0, 3)\text{ で達成）} \]
線形計画法：頂点で最大・最小を調べる
例7連立不等式の領域

連立不等式 \(\begin{cases} x^2 + y^2 \leq 9 \\ y \geq x \end{cases}\) の表す領域を図示せよ。

解き方：

  1. \(x^2 + y^2 \leq 9\) → 原点中心、半径3の円の内部（周を含む）
  2. \(y \geq x\) → 直線\(y = x\)の上側（直線を含む）
  3. 2つの領域の共通部分が答え
\[ \text{答：円の内部のうち、直線 } y = x \text{ より上側の部分} \]

4. 簡単にするコツ

✏️ コツ1：軌跡は「2乗して整理」が基本
距離の条件が出てきたら、ルートを外すために両辺を2乗。その後、丁寧に展開・整理すれば図形の方程式が現れます。
✏️ コツ2：領域の判定は「原点代入」が最速
直線 \(ax + by + c = 0\) で領域を判定するとき、原点\((0, 0)\)が境界上になければ原点を代入。符号で領域がわかります。原点が境界上なら\((1, 0)\)など別の点で判定。
✏️ コツ3：線形計画法は「頂点だけチェック」
領域が凸多角形なら、\(k = ax + by\)の最大・最小は必ず頂点で達成されます。頂点の座標を全て求めて代入比較するだけでOK。
✏️ コツ4：連立不等式は「1つずつ図示して重ねる」
複数の不等式がある場合、それぞれの領域を順番に図示して、共通部分を見つけます。色分けやハッチングを使い分けると見やすくなります。

5. よくあるつまづき

⚠️ つまづき1：境界線の実線・破線を間違える

ダメな例：\(x + y > 3\)の境界を実線で描いてしまう

対策：\(>\)や\(<\)は「含まない」→ 破線、\(\geq\)や\(\leq\)は「含む」→ 実線

⚠️ つまづき2：領域の向きを逆にする

ダメな例：\(y > 2x + 1\)で直線の下側を塗る

対策：必ず具体的な点（原点など）を代入して確認。\(0 > 2 \cdot 0 + 1 = 1\)は偽なので、原点は含まない → 原点と反対側が領域

⚠️ つまづき3：軌跡で条件の確認を忘れる

ダメな例：計算で得た方程式をそのまま答えにする

対策：軌跡の問題では「条件を満たす点が本当にその図形上にあるか」「その図形上の点が条件を満たすか」を最後に確認。特に除外点（分母=0など）に注意。

⚠️ つまづき4：線形計画法で頂点を見落とす

ダメな例：連立方程式を解くとき、交点の計算ミスで頂点を間違える

対策：領域の図を正確に描き、境界線同士の交点を丁寧に計算。見つけた頂点の個数が領域の形と一致するか確認。

6. 仕上げチェックリスト

軌跡の3ステップ（点を置く → 条件を式に → 整理）を使いこなせる
不等式の領域を「原点代入」で素早く判定できる
境界線の実線・破線を正しく使い分けられる
連立不等式の共通部分を正確に図示できる
線形計画法で頂点を列挙して最大・最小を求められる

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

点 \(A(2, 0)\) からの距離と、\(y\)軸からの距離が等しい点\(P\)の軌跡を求めよ。

答えを見る

解答：

点\(P(x, y)\)とする。

  • Aからの距離：\(\sqrt{(x-2)^2 + y^2}\)
  • \(y\)軸からの距離：\(|x|\)

条件より：\(\sqrt{(x-2)^2 + y^2} = |x|\)

両辺を2乗：\((x-2)^2 + y^2 = x^2\)

\(x^2 - 4x + 4 + y^2 = x^2\)

\(y^2 = 4x - 4 = 4(x-1)\)

答：放物線 \(y^2 = 4(x-1)\)（頂点\((1, 0)\)）

練習2

条件 \(x \geq 0\), \(y \geq 0\), \(2x + y \leq 8\), \(x + 2y \leq 10\) のもとで、\(k = 3x + 2y\) の最大値と、そのときの\((x, y)\)を求めよ。

答えを見る

解答：

領域の頂点を求める：

  • \((0, 0)\)：原点
  • \((4, 0)\)：\(2x + y = 8\)と\(y = 0\)の交点
  • \((2, 4)\)：\(2x + y = 8\)と\(x + 2y = 10\)の交点（連立して\(x = 2, y = 4\)）
  • \((0, 5)\)：\(x + 2y = 10\)と\(x = 0\)の交点

各頂点で\(k = 3x + 2y\)を計算：

  • \((0, 0)\)：\(k = 0\)
  • \((4, 0)\)：\(k = 12\)
  • \((2, 4)\)：\(k = 6 + 8 = 14\)
  • \((0, 5)\)：\(k = 10\)

答：最大値 \(14\)（点\((2, 4)\)で達成）

📌 まとめ：軌跡は「点を置いて条件を式に」、領域は「境界線を引いて点で判定」、線形計画法は「頂点で勝負」！

おまけ：歴史ストーリー「デカルトと座標幾何学」

📖 天井のハエが数学を変えた

1637年、フランスの哲学者ルネ・デカルトは病床に伏せていました。天井を眺めていると、1匹のハエが飛んでいるのが目に入ります。

「あのハエの位置を、壁の角からの距離で表せないだろうか...」

この何気ない発想が、解析幾何学（座標幾何学）の誕生につながりました。デカルトは「図形」と「数式」を結びつける革命的なアイデアを生み出したのです。

それまで、図形の問題と方程式の問題は別々に扱われていました。円は「中心から等距離にある点の集合」として幾何学的に理解され、方程式は「未知数を求める計算」でした。

デカルトの座標系を使えば：

\[ \text{「中心から等距離」} \Leftrightarrow x^2 + y^2 = r^2 \]

このように、幾何学の問題が代数の計算で解けるようになったのです。

デカルトが著した『方法序説』の付録『幾何学』には、「我思う、ゆえに我あり」の哲学だけでなく、この座標幾何学のアイデアが記されています。

今日学んだ「軌跡と領域」は、まさにデカルトの遺産。「条件を満たす点の集合」という幾何学的な考えを、方程式や不等式で表現する技術です。

ちなみに、同時代の数学者フェルマーも独立に同じアイデアに到達していましたが、論文を発表しなかったため、座標系は「デカルト座標」と呼ばれることになりました。発表って大事ですね...
