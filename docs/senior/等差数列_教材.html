1. テーマの定義

階段を上るときの段数、カレンダーの日付、給料の昇給額。身の回りには「同じ量ずつ増えていく」数の並びがあふれています。この規則を数学的に扱うのが等差数列です。

このページでは、等差数列を学びます：

  • 等差数列とは何か：「隣同士の差が一定」という特徴を理解する
  • 一般項の公式：第n項を直接求める方法
  • 和の公式：初項から第n項までの合計を素早く計算する方法

2. 覚えるルール

【最重要】等差数列の一般項：\(a_n = a + (n-1)d\)
等差数列は公差dずつ増える数の並び
📌 等差数列の定義

等差数列とは、隣り合う項の差が一定の数列のこと。

たとえば、\(2, 5, 8, 11, 14, \ldots\) は差が常に3なので等差数列です。

→ ポイント： この一定の差を公差（common difference）といい、\(d\) で表します。

📌 一般項の公式 → 例1
\[ a_n = a + (n-1)d \]

ここで、

  • \(a_n\)（エーエヌ）：第n項の値
  • \(a\)：初項（最初の項、\(a_1\) とも書く）
  • \(d\)：公差（隣り合う項の差）
  • \(n\)：項番号（何番目か）

→ だから結果は： 第n項は「初項から公差を\((n-1)\)回足したもの」

💡 考え方を見る

第1項：\(a\)（公差を0回足す）

第2項：\(a + d\)（公差を1回足す）

第3項：\(a + 2d\)（公差を2回足す）

：

第n項：\(a + (n-1)d\)（公差を(n-1)回足す）

📌 等差数列の和の公式 → 例3
\[ S_n = \frac{n(a + l)}{2} = \frac{n\{2a + (n-1)d\}}{2} \]

ここで、

  • \(S_n\)（エスエヌ）：初項から第n項までの和
  • \(a\)：初項
  • \(l\)（エル）：末項（最後の項、\(a_n\) のこと）
  • \(n\)：項数

✅ 使い分け：

  • 末項がわかっているとき → \(\displaystyle S_n = \frac{n(a + l)}{2}\)
  • 公差がわかっているとき → \(\displaystyle S_n = \frac{n\{2a + (n-1)d\}}{2}\)
💡 考え方を見る

和を2回書いて逆順に足すと、各項のペアがすべて \(a + l\) になります。

\(S_n = a + (a+d) + (a+2d) + \cdots + l\)

\(S_n = l + (l-d) + (l-2d) + \cdots + a\)

足すと、\(2S_n = n(a + l)\)

よって、\(\displaystyle S_n = \frac{n(a + l)}{2}\)

正順と逆順を足すと各項のペアが (a+l) になる
📌 等差中項 → 例5
\[ b = \frac{a + c}{2} \]

\(a, b, c\) がこの順で等差数列のとき、\(b\) を \(a\) と \(c\) の等差中項（相加平均）といいます。

→ ポイント： 等差数列の真ん中の項は「両端の平均」

3. 例題と解き方

等差数列の問題は「一般項を求める」→「和を求める」の順で解きます。

▶ 一般項を求める

例1一般項を求める（基本）

初項3、公差4の等差数列の一般項を求めよ。

解き方：

  1. \(a = 3\)、\(d = 4\) を公式に代入
  2. 一般項 \(a_n = a + (n-1)d\) を計算
\[ a_n = 3 + (n-1) \cdot 4 = 3 + 4n - 4 = 4n - 1 \]

答え：\(a_n = 4n - 1\)

例22つの条件から一般項を求める

等差数列で \(a_3 = 7\)、\(a_7 = 19\) のとき、一般項を求めよ。

解き方：

  1. 一般項 \(a_n = a + (n-1)d\) に条件を代入
    • \(a_3 = a + 2d = 7\) ... ①
    • \(a_7 = a + 6d = 19\) ... ②
  2. ② - ① で公差を求める：\(4d = 12\) より \(d = 3\)
  3. ① に代入して初項を求める：\(a + 6 = 7\) より \(a = 1\)
  4. 一般項を書く
\[ a_n = 1 + (n-1) \cdot 3 = 3n - 2 \]

答え：\(a_n = 3n - 2\)

▶ 和を求める

例3初項から第n項までの和（末項既知）

初項5、末項95、項数10の等差数列の和を求めよ。

解き方：

  1. 末項がわかっているので \(\displaystyle S_n = \frac{n(a + l)}{2}\) を使う
  2. \(a = 5\)、\(l = 95\)、\(n = 10\) を代入
\[ S_{10} = \frac{10(5 + 95)}{2} = \frac{10 \times 100}{2} = 500 \]

答え：500

例4初項から第n項までの和（公差既知）

初項2、公差3の等差数列の初項から第20項までの和を求めよ。

解き方：

  1. 公差がわかっているので \(\displaystyle S_n = \frac{n\{2a + (n-1)d\}}{2}\) を使う
  2. \(a = 2\)、\(d = 3\)、\(n = 20\) を代入
\[ S_{20} = \frac{20\{2 \cdot 2 + (20-1) \cdot 3\}}{2} = \frac{20(4 + 57)}{2} = \frac{20 \times 61}{2} = 610 \]

答え：610

▶ 等差中項

例5等差中項

\(x, 12, 3x-4\) がこの順で等差数列をなすとき、\(x\) の値を求めよ。

解き方：

  1. 真ん中の12が等差中項 → \(12 = \displaystyle\frac{x + (3x-4)}{2}\)
  2. 方程式を解く
\[ 12 = \frac{x + 3x - 4}{2} = \frac{4x - 4}{2} = 2x - 2 \]
\[ 2x = 14 \quad \Rightarrow \quad x = 7 \]

答え：\(x = 7\)

（検算：\(7, 12, 17\) で公差5の等差数列 ✓）

▶ 応用問題

例61から100までの和

1から100までの自然数の和を求めよ。

解き方：

  1. \(1, 2, 3, \ldots, 100\) は初項1、公差1、末項100、項数100の等差数列
  2. 末項がわかっているので \(\displaystyle S_n = \frac{n(a + l)}{2}\)
\[ S_{100} = \frac{100(1 + 100)}{2} = \frac{100 \times 101}{2} = 5050 \]

答え：5050

例7和が最大になる項数

初項50、公差-3の等差数列の初項から第n項までの和が最大になるnを求めよ。

解き方：

  1. 公差が負 → 数列は減少していく
  2. 正の項を足している間は和が増え、負の項を足し始めると和は減る
  3. \(a_n \geq 0\) となる最大のnを求める
    \[ a_n = 50 + (n-1)(-3) = 53 - 3n \geq 0 \]
    \[ 53 \geq 3n \quad \Rightarrow \quad n \leq \frac{53}{3} = 17.67... \]
  4. よって \(n = 17\) まで正（\(a_{17} = 53 - 51 = 2\)）

答え：n = 17 のとき和が最大

4. 簡単にするコツ

✏️ コツ1：公式は「形」で覚える
一般項 \(a_n = a + (n-1)d\) は「初項 + (項数 - 1) × 公差」。
「第n項は初項からスタートして公差を(n-1)回足す」と覚えましょう。
✏️ コツ2：和の公式は「項数×平均」
\(\displaystyle S_n = \frac{n(a + l)}{2}\) は「項数 × (初項と末項の平均)」。
等差数列では平均値が真ん中の値になるので、これは「項数 × 平均」という意味です。
✏️ コツ3：2条件があれば連立方程式
「\(a_3 = 7\)、\(a_7 = 19\)」のように2つの条件があれば、\(a\) と \(d\) の連立方程式を作って解く。
項番号の差を使うと公差が直接求まります。
✏️ コツ4：和の最大は「正の項まで」
公差が負の等差数列で和が最大になるのは、項が負になる直前まで足したとき。
\(a_n \geq 0\) を解いて、正の項の最後を見つけましょう。

5. よくあるつまづき

⚠️ つまづき1：(n-1) を n と間違える

ダメな例： 一般項を \(a_n = a + nd\) と書く

対策： 第1項で確認！ \(a_1 = a + (1-1)d = a\) なら正しい。\(a + d\) になったら間違い。

⚠️ つまづき2：項数の数え間違い

ダメな例： 「3から15までの奇数」の項数を \(15 - 3 = 12\) と計算

対策： 一般項を \(a_n = 3 + (n-1) \cdot 2 = 2n + 1\) として、\(2n + 1 = 15\) を解くと \(n = 7\)。項数は7個。

⚠️ つまづき3：和の公式を間違える

ダメな例： \(\displaystyle S_n = \frac{n(a + l)}{2}\) で \(l\) に公差を入れる

対策： \(l\) は末項の値（最後の項）。公差ではない！公差がわかっているなら \(\displaystyle S_n = \frac{n\{2a + (n-1)d\}}{2}\) を使う。

⚠️ つまづき4：公差の符号を逆にする

ダメな例： \(10, 7, 4, 1, \ldots\) で公差を \(d = 3\) とする

対策： 公差 = (後の項) - (前の項) = \(7 - 10 = -3\)。減っていれば公差は負！

6. 仕上げチェックリスト

等差数列とは「隣り合う項の差が一定」と言える
一般項 \(a_n = a + (n-1)d\) を正しく使える
和の公式を末項既知と公差既知で使い分けられる
等差中項は「両端の平均」と理解している
和が最大になる条件を求められる

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

初項-2、公差5の等差数列について、

(1) 一般項を求めよ。

(2) 初項から第15項までの和を求めよ。

答えを見る

(1) \(a_n = -2 + (n-1) \cdot 5 = 5n - 7\)

(2) \(\displaystyle S_{15} = \frac{15\{2 \cdot (-2) + 14 \cdot 5\}}{2} = \frac{15(-4 + 70)}{2} = \frac{15 \times 66}{2} = 495\)

練習2

等差数列で \(a_5 = 13\)、\(a_{10} = 28\) のとき、

(1) 初項と公差を求めよ。

(2) 初項から第n項までの和が初めて200を超えるのは第何項までか。

答えを見る

(1) \(a_5 = a + 4d = 13\) ... ①、\(a_{10} = a + 9d = 28\) ... ②

② - ① より \(5d = 15\)、よって \(d = 3\)

① に代入：\(a + 12 = 13\)、よって \(a = 1\)

答え：初項1、公差3

(2) \(\displaystyle S_n = \frac{n(2 \cdot 1 + (n-1) \cdot 3)}{2} = \frac{n(3n - 1)}{2} > 200\)

\(3n^2 - n > 400\) より \(3n^2 - n - 400 > 0\)

\(n = 12\) のとき \(S_{12} = \frac{12 \times 35}{2} = 210 > 200\) ✓

\(n = 11\) のとき \(S_{11} = \frac{11 \times 32}{2} = 176 < 200\) ✗

答え：第12項まで

📌 まとめ：等差数列は「公差」がキーワード。一般項は \(a + (n-1)d\)、和は「項数×平均」で一発計算！

おまけ：歴史ストーリー「9歳の天才ガウス」

📖 1から100までを5秒で足した少年

1786年、ドイツ。小学校の算数の授業で、先生は生徒たちを静かにさせようと、こんな課題を出しました。

「1から100まで全部足しなさい。終わった人は手を挙げて。」

先生は「これで30分は静かにしていてくれるだろう」と思っていました。

ところが...

5秒後、9歳のカール・フリードリヒ・ガウス少年が手を挙げました。

「先生、答えは5050です。」

信じられない先生が「どうやって計算したんだ？」と聞くと、ガウスはこう説明しました。

\[ 1 + 100 = 101 \]
\[ 2 + 99 = 101 \]
\[ 3 + 98 = 101 \]

「こうやってペアを作ると、全部で50組できて、どのペアも101になります。だから答えは \(101 \times 50 = 5050\) です。」

これこそ、私たちが学んだ等差数列の和の公式そのものです！

\[ S_n = \frac{n(a + l)}{2} = \frac{100 \times (1 + 100)}{2} = 5050 \]

ガウスは後に「数学の王」と呼ばれ、数論、統計学、天文学など多くの分野で革命的な発見をしました。ガウス分布（正規分布）も彼の名前から来ています。

ちなみに、ガウスの天才ぶりを示す逸話はまだあります。3歳のとき、父親が給料計算を間違えたのを指摘したとか。算数を習う前に足し算ができていたらしいです。
