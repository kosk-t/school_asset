1. テーマの定義

「\(x^2 = -1\) を満たす数は存在しない」と中学では習いました。しかし、\(i^2 = -1\) となる「虚数単位 \(i\)」を導入すると、すべての方程式が解けるようになります。

このページでは、複素数と高次方程式について学びます：

  • 複素数の計算：虚数単位 \(i\) を使った四則演算
  • 解と係数の関係：方程式の解と係数の間に成り立つ公式
  • 因数定理・剰余の定理：多項式を因数分解するための強力な定理
  • 高次方程式：3次・4次方程式を解くテクニック

2. 覚えるルール

【最重要】虚数単位 \(i\) の定義：\(i^2 = -1\)（つまり \(i = \sqrt{-1}\)）
複素数 3+2i を複素数平面上に表示
📌 複素数の定義 → 例1

\(a + bi\)（\(a, b\) は実数）の形で表される数を複素数という。

  • \(a\)：実部（Real part）
  • \(b\)：虚部（Imaginary part）
  • \(b \neq 0\) のとき虚数、\(a = 0, b \neq 0\) のとき純虚数

→ だから結果は： すべての実数は \(b = 0\) の複素数と見なせる

📌 複素数の相等
\[ a + bi = c + di \Leftrightarrow a = c \text{ かつ } b = d \]

実部どうし、虚部どうしが等しいときに限り、2つの複素数は等しい。

📌 共役複素数 → 例2
\[ \overline{a + bi} = a - bi \]

\(z = a + bi\) に対し、\(\bar{z} = a - bi\) を \(z\) の共役複素数という。

重要な性質：

  • \(z \cdot \bar{z} = a^2 + b^2\)（常に実数で非負）
  • \(z + \bar{z} = 2a\)（実部の2倍）
  • \(\overline{z_1 \cdot z_2} = \bar{z_1} \cdot \bar{z_2}\)
📌 \(i\) の累乗
\[ i^1 = i, \quad i^2 = -1, \quad i^3 = -i, \quad i^4 = 1, \quad i^5 = i, \ldots \]

\(i\) の累乗は4周期で繰り返す。\(i^n\) を求めるには \(n\) を4で割った余りを見る。

📌 負の数の平方根 → 例3
\[ \sqrt{-a} = \sqrt{a} \cdot i \quad (a > 0) \]

⚠️ 注意： \(\sqrt{-a} \cdot \sqrt{-b} \neq \sqrt{ab}\)（負の数では成り立たない）

→ 正しい計算法： 先に \(i\) を出してから計算する

📌 解と係数の関係（2次方程式） → 例4

\(ax^2 + bx + c = 0\) の2つの解を \(\alpha, \beta\) とすると：

\[ \alpha + \beta = -\frac{b}{a}, \quad \alpha\beta = \frac{c}{a} \]

✅ メリット： 解を求めなくても、解の和・積がわかる

💡 考え方を見る

\(ax^2 + bx + c = a(x - \alpha)(x - \beta)\) と因数分解できる。
展開すると \(a(x^2 - (\alpha + \beta)x + \alpha\beta)\) となるので、係数を比較すると公式が得られる。

📌 解と係数の関係（3次方程式） → 例5

\(ax^3 + bx^2 + cx + d = 0\) の3つの解を \(\alpha, \beta, \gamma\) とすると：

\[ \alpha + \beta + \gamma = -\frac{b}{a}, \quad \alpha\beta + \beta\gamma + \gamma\alpha = \frac{c}{a}, \quad \alpha\beta\gamma = -\frac{d}{a} \]
📌 剰余の定理 → 例6

多項式 \(P(x)\) を \((x - a)\) で割った余りは \(P(a)\) に等しい。

\[ P(x) = (x - a)Q(x) + P(a) \]

✅ メリット： 割り算をせずに余りがわかる

📌 因数定理 → 例7
\[ P(a) = 0 \Leftrightarrow P(x) \text{ は } (x - a) \text{ で割り切れる} \]

剰余の定理の特別な場合。余りが0なら、\((x - a)\) は \(P(x)\) の因数。

→ 使い方： 高次方程式で \(P(a) = 0\) となる \(a\) を見つけ、\((x - a)\) でくくる

📌 2次方程式の解と共役

実数係数の2次方程式が虚数解 \(\alpha\) を持つとき、その共役 \(\bar{\alpha}\) も解となる。

→ 理由： 解の公式で \(b^2 - 4ac < 0\) のとき、\(\pm\sqrt{b^2-4ac}\) の部分が共役の関係になる

3. 例題と解き方

ルールを使って実際に解いてみましょう。

▶ 複素数の計算

例1複素数の四則演算

\((3 + 2i) + (1 - 4i)\)、\((3 + 2i)(1 - 4i)\)、\(\dfrac{3 + 2i}{1 - 4i}\) を計算せよ。

解き方：

  1. 加法：実部どうし、虚部どうしを足す
    \[ (3 + 2i) + (1 - 4i) = (3+1) + (2-4)i = \mathbf{4 - 2i} \]
  2. 乗法：展開して \(i^2 = -1\) を使う
    \[ (3 + 2i)(1 - 4i) = 3 - 12i + 2i - 8i^2 = 3 - 10i - 8(-1) = \mathbf{11 - 10i} \]
  3. 除法：分母の共役を分母・分子にかけて、分母を実数化
    \[ \frac{3 + 2i}{1 - 4i} = \frac{(3 + 2i)(1 + 4i)}{(1 - 4i)(1 + 4i)} = \frac{3 + 12i + 2i + 8i^2}{1 + 16} = \frac{-5 + 14i}{17} = \mathbf{-\frac{5}{17} + \frac{14}{17}i} \]
例2共役複素数の活用

\(z = 2 + 3i\) のとき、\(z\bar{z}\) と \(z + \bar{z}\) を求めよ。

解き方：

  1. \(\bar{z} = 2 - 3i\) を求める
  2. \(z\bar{z} = (2 + 3i)(2 - 3i) = 4 - 9i^2 = 4 + 9 = \mathbf{13}\)
  3. \(z + \bar{z} = (2 + 3i) + (2 - 3i) = \mathbf{4}\)

※ \(z\bar{z} = |z|^2 = a^2 + b^2\) は複素数の絶対値の2乗になる

例3負の数の平方根の計算

\(\sqrt{-3} \cdot \sqrt{-12}\) を計算せよ。

解き方：

  1. 先に \(i\) を出す：\(\sqrt{-3} = \sqrt{3}i\)、\(\sqrt{-12} = 2\sqrt{3}i\)
  2. 計算：\(\sqrt{3}i \cdot 2\sqrt{3}i = 2 \cdot 3 \cdot i^2 = 6 \cdot (-1) = \mathbf{-6}\)

⚠️ 誤り： \(\sqrt{-3} \cdot \sqrt{-12} = \sqrt{36} = 6\) としてはいけない！

▶ 解と係数の関係

例4解と係数の関係（2次）

2次方程式 \(x^2 - 5x + 3 = 0\) の2つの解を \(\alpha, \beta\) とするとき、\(\alpha^2 + \beta^2\) の値を求めよ。

解き方：

  1. 解と係数の関係より：\(\alpha + \beta = 5\)、\(\alpha\beta = 3\)
  2. \(\alpha^2 + \beta^2\) を \(\alpha + \beta\) と \(\alpha\beta\) で表す：
    \[ \alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta \]
  3. 代入：\(\alpha^2 + \beta^2 = 5^2 - 2 \cdot 3 = 25 - 6 = \mathbf{19}\)
例52数を解とする方程式

2数 \(3 + 2i\)、\(3 - 2i\) を解とする \(x\) の2次方程式を求めよ。

解き方：

  1. 解の和：\((3 + 2i) + (3 - 2i) = 6\)
  2. 解の積：\((3 + 2i)(3 - 2i) = 9 + 4 = 13\)
  3. 解と係数の関係より：\(\mathbf{x^2 - 6x + 13 = 0}\)

※ 共役な複素数を解に持つ2次方程式は、係数がすべて実数になる

▶ 因数定理・剰余の定理

例6剰余の定理

\(P(x) = x^3 - 2x^2 + 3x - 4\) を \(x - 2\) で割った余りを求めよ。

解き方：

  1. 剰余の定理より、余り = \(P(2)\)
  2. \(P(2) = 2^3 - 2 \cdot 2^2 + 3 \cdot 2 - 4 = 8 - 8 + 6 - 4 = \mathbf{2}\)
例7因数定理で因数分解

\(P(x) = x^3 - 6x^2 + 11x - 6\) を因数分解せよ。

解き方：

  1. \(P(a) = 0\) となる整数 \(a\) を探す（定数項 \(-6\) の約数 \(\pm 1, \pm 2, \pm 3, \pm 6\) を試す）
  2. \(P(1) = 1 - 6 + 11 - 6 = 0\) → \((x - 1)\) は因数
  3. 組立除法（または筆算）で割る：
    1-611-6
    11-56
    1-560
  4. \(P(x) = (x - 1)(x^2 - 5x + 6) = (x - 1)(x - 2)(x - 3)\)
\[ \mathbf{P(x) = (x - 1)(x - 2)(x - 3)} \]

▶ 高次方程式

例83次方程式

\(x^3 + 2x^2 - 5x - 6 = 0\) を解け。

解き方：

  1. \(P(x) = x^3 + 2x^2 - 5x - 6\) として、\(P(a) = 0\) となる \(a\) を探す
  2. \(P(2) = 8 + 8 - 10 - 6 = 0\) → \((x - 2)\) は因数
  3. 組立除法：\(P(x) = (x - 2)(x^2 + 4x + 3)\)
  4. \(x^2 + 4x + 3 = (x + 1)(x + 3)\)
  5. \(P(x) = (x - 2)(x + 1)(x + 3) = 0\)
\[ \mathbf{x = 2, -1, -3} \]
例9虚数解を持つ3次方程式

\(x^3 - 3x^2 + 4x - 12 = 0\) を解け。

解き方：

  1. \(P(3) = 27 - 27 + 12 - 12 = 0\) → \((x - 3)\) は因数
  2. 組立除法：\(P(x) = (x - 3)(x^2 + 4)\)
  3. \(x^2 + 4 = 0\) より \(x^2 = -4\)、\(x = \pm 2i\)
\[ \mathbf{x = 3, 2i, -2i} \]
例104次方程式（複二次式）

\(x^4 - 5x^2 + 4 = 0\) を解け。

解き方：

  1. \(x^2 = t\) とおく（\(t \geq 0\) の条件に注意）
  2. \(t^2 - 5t + 4 = 0\) を解く：\((t - 1)(t - 4) = 0\)、\(t = 1, 4\)
  3. \(x^2 = 1\) より \(x = \pm 1\)、\(x^2 = 4\) より \(x = \pm 2\)
\[ \mathbf{x = \pm 1, \pm 2} \]

4. 簡単にするコツ

✏️ コツ1：除法は「共役を掛ける」で実数化
\(\dfrac{a + bi}{c + di}\) を計算するとき、分母分子に \(c - di\) を掛ける。分母が \(c^2 + d^2\)（実数）になる。
✏️ コツ2：因数定理では「定数項の約数」を試す
\(P(x) = x^n + \cdots + d\) の整数解は、定数項 \(d\) の約数に限られる。まず \(\pm 1, \pm 2, \ldots\) を代入して \(P(a) = 0\) となる \(a\) を探す。
✏️ コツ3：対称式は基本対称式で表す
\(\alpha^2 + \beta^2\)、\(\alpha^3 + \beta^3\) などは、\(\alpha + \beta\) と \(\alpha\beta\) だけで表せる。
  • \(\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta\)
  • \(\alpha^3 + \beta^3 = (\alpha + \beta)^3 - 3\alpha\beta(\alpha + \beta)\)
  • \(\dfrac{1}{\alpha} + \dfrac{1}{\beta} = \dfrac{\alpha + \beta}{\alpha\beta}\)
✏️ コツ4：複二次式は \(t = x^2\) で置換
\(ax^4 + bx^2 + c = 0\) の形なら、\(t = x^2\) と置いて2次方程式に帰着。ただし \(t \geq 0\) の解のみ有効。
✏️ コツ5：\(i\) の累乗は4で割った余りで判定
\(i^{100}\) なら、\(100 \div 4 = 25\) 余り \(0\) → \(i^0 = 1\)

5. よくあるつまづき

⚠️ つまづき1：\(\sqrt{-a} \cdot \sqrt{-b} = \sqrt{ab}\) と計算

ダメな例： \(\sqrt{-2} \cdot \sqrt{-8} = \sqrt{16} = 4\)

対策： 先に \(i\) を出す。\(\sqrt{-2} \cdot \sqrt{-8} = \sqrt{2}i \cdot 2\sqrt{2}i = 4i^2 = -4\)

⚠️ つまづき2：解と係数の関係の符号ミス

ダメな例： \(x^2 + 5x + 6 = 0\) で \(\alpha + \beta = 5\)

対策： \(\alpha + \beta = -\dfrac{b}{a}\) なので、\(\alpha + \beta = -5\)。符号を逆にするのを忘れない。

⚠️ つまづき3：因数定理で約数以外を試す

ダメな例： \(x^3 + 2x - 3 = 0\) で \(x = 2\) を最初に試す

対策： 定数項 \(-3\) の約数は \(\pm 1, \pm 3\) のみ。効率的に \(1, -1, 3, -3\) の順で試す。

⚠️ つまづき4：複二次式で \(t < 0\) の解を使う

ダメな例： \(x^4 + 5x^2 + 4 = 0\) で \(t = -1, -4\) から \(x = \pm i, \pm 2i\)

対策： 実数解を求める場合は \(t \geq 0\) の解のみ。この場合は「実数解なし」。複素数解を求める場合は \(t < 0\) も使える。

⚠️ つまづき5：3次方程式の解と係数の関係で積の符号

ダメな例： \(x^3 - 2x^2 + 3x - 4 = 0\) で \(\alpha\beta\gamma = -4\)

対策： \(\alpha\beta\gamma = -\dfrac{d}{a}\) なので、\(\alpha\beta\gamma = -\dfrac{-4}{1} = 4\)。符号が2回変わることに注意。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(\dfrac{1 + 2i}{3 - i}\) を \(a + bi\) の形で表せ。

答えを見る

分母の共役 \(3 + i\) を掛ける：

\(\dfrac{(1 + 2i)(3 + i)}{(3 - i)(3 + i)} = \dfrac{3 + i + 6i + 2i^2}{9 + 1} = \dfrac{1 + 7i}{10} = \mathbf{\dfrac{1}{10} + \dfrac{7}{10}i}\)

練習2

2次方程式 \(x^2 - 3x + 5 = 0\) の2つの解を \(\alpha, \beta\) とするとき、\(\alpha^2 + \beta^2\) と \(\dfrac{1}{\alpha} + \dfrac{1}{\beta}\) を求めよ。

答えを見る

解と係数の関係より：\(\alpha + \beta = 3\)、\(\alpha\beta = 5\)

\(\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta = 9 - 10 = \mathbf{-1}\)

\(\dfrac{1}{\alpha} + \dfrac{1}{\beta} = \dfrac{\alpha + \beta}{\alpha\beta} = \mathbf{\dfrac{3}{5}}\)

練習3

\(x^3 - 7x + 6 = 0\) を解け。

答えを見る

\(P(1) = 1 - 7 + 6 = 0\) より \((x - 1)\) は因数

組立除法で割ると：\(P(x) = (x - 1)(x^2 + x - 6) = (x - 1)(x + 3)(x - 2)\)

よって \(\mathbf{x = 1, -3, 2}\)

📌 まとめ：\(i^2 = -1\) で負の数の平方根が扱え、因数定理で高次方程式が解ける

おまけ：歴史ストーリー「虚数を発見した男たち」

📖 「ありえない数」の誕生

16世紀のイタリア。数学者たちは3次方程式の解法を競い合っていました。

カルダノ（Gerolamo Cardano, 1501-1576）は、3次方程式の一般解法を発表しましたが、ある問題に直面しました。

\[ x^3 = 15x + 4 \]

この方程式の解は明らかに \(x = 4\) です（代入すれば確かめられます）。しかし、カルダノの公式を使うと、計算の途中で負の数の平方根が現れてしまいます。

「負の数の平方根なんて存在しない！」と当時の数学者は考えました。しかし、最終的な答えは正しい実数になる...。

カルダノの弟子ボンベリ（Rafael Bombelli, 1526-1572）は、大胆なアイデアを思いつきました。

「存在しない数でも、計算を続ければいい」

彼は \(\sqrt{-1}\) を「あたかも存在するかのように」扱い、計算を進めました。すると、途中に現れた「ありえない数」が最後には打ち消し合い、正しい答え \(x = 4\) が得られたのです！

この「ありえない数」は長い間「虚数（imaginary number）」と呼ばれ、実在しない数として扱われました。しかし18世紀、オイラーが \(i = \sqrt{-1}\) という記号を導入し、ガウスが複素数を平面上の点として可視化したことで、虚数は立派な「市民権」を得ました。

教訓： 今では、電気工学、量子力学、信号処理など、虚数なしでは成り立たない分野がたくさんあります。「ありえない」と思われた数が、現代科学を支えているのです！
