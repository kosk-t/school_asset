1. テーマの定義

人口増加、細菌の増殖、複利計算、放射性物質の減衰...これらはすべて「倍々ゲーム」で増えたり減ったりします。この「倍々」の仕組みを数学的に扱うのが指数関数です。

このページでは、以下の内容を学びます：

  • 指数法則の拡張：整数指数から有理数指数へ
  • 指数関数のグラフ：\(y = a^x\) の形とその特徴
  • 指数方程式：底を揃えて指数を比較する解法
  • 指数不等式：底が1より大きいか小さいかで向きが変わる

2. 覚えるルール

【最重要】指数法則は「累乗のかけ算・割り算・累乗」の3パターン！
指数法則の3つの基本パターン
📌 指数法則（基本） → 例1
\[ a^m \times a^n = a^{m+n}, \quad \frac{a^m}{a^n} = a^{m-n}, \quad (a^m)^n = a^{mn} \]

\(a > 0, a \neq 1\) で、\(m, n\) は任意の実数で成り立ちます。

→ だから結果は： 指数のかけ算は指数を足す、割り算は引く、累乗の累乗はかける

📌 追加公式 → 例2
\[ (ab)^n = a^n b^n, \quad \left(\frac{a}{b}\right)^n = \frac{a^n}{b^n} \]

積や商の累乗は、それぞれに分配できます。

📌 負の指数と0乗 → 例3
\[ a^0 = 1, \quad a^{-n} = \frac{1}{a^n} \]

「0乗は1」「負の指数は逆数」と覚えましょう。

💡 考え方を見る

\(a^m \div a^m = a^{m-m} = a^0\)。また \(a^m \div a^m = 1\) なので \(a^0 = 1\)。

\(a^{-n} = a^{0-n} = a^0 \div a^n = 1 \div a^n = \frac{1}{a^n}\) となります。

📌 分数指数（累乗根） → 例4
\[ a^{\frac{1}{n}} = \sqrt[n]{a}, \quad a^{\frac{m}{n}} = \sqrt[n]{a^m} = (\sqrt[n]{a})^m \]

\(\sqrt[n]{a}\)（「n乗根」と読む）は「n乗すると \(a\) になる正の数」。

→ だから結果は： 分母はルートの次数、分子は累乗の指数

【最重要】指数関数 \(y = a^x\) のグラフは、\(a > 1\) なら右上がり、\(0 < a < 1\) なら右下がり！
指数関数のグラフ：底aの大きさで増減が決まる
📌 指数関数のグラフの特徴 → 例5
特徴 \(a > 1\) のとき \(0 < a < 1\) のとき
グラフの形 右上がり（単調増加） 右下がり（単調減少）
必ず通る点 \((0, 1)\)
漸近線 \(x\) 軸（\(y = 0\)）
\(y\) の値の範囲 \(y > 0\)（常に正）

✅ メリット： グラフの増減を見れば、指数の大小関係がわかる

📌 指数方程式の解法 → 例6
\[ a^{f(x)} = a^{g(x)} \Rightarrow f(x) = g(x) \]

底を揃えて指数を比較するのが基本戦略です。

→ だから結果は： \(2^x = 8\) なら \(2^x = 2^3\) なので \(x = 3\)

📌 指数不等式の解法 → 例7
\[ a > 1 のとき：a^{f(x)} > a^{g(x)} \Leftrightarrow f(x) > g(x) \] \[ 0 < a < 1 のとき：a^{f(x)} > a^{g(x)} \Leftrightarrow f(x) < g(x) \]

底が1より小さいと不等号が逆転することに注意！

💡 考え方を見る

\(a > 1\) のときグラフは右上がりなので、\(x\) が大きいほど \(y\) も大きい。

\(0 < a < 1\) のときグラフは右下がりなので、\(x\) が大きいほど \(y\) は小さくなる。

3. 例題と解き方

指数法則から方程式・不等式まで、順番に見ていきましょう。

▶ 指数法則の計算

例1指数法則（基本）

\(2^3 \times 2^5\) と \(\dfrac{3^7}{3^4}\) と \((5^2)^3\) を計算せよ。

解き方：

  1. 積 → 指数を足す：\(2^3 \times 2^5 = 2^{3+5} = 2^8 = 256\)
  2. 商 → 指数を引く：\(\dfrac{3^7}{3^4} = 3^{7-4} = 3^3 = 27\)
  3. 累乗の累乗 → 指数をかける：\((5^2)^3 = 5^{2 \times 3} = 5^6 = 15625\)
例2積・商の累乗

\((2 \times 3)^4\) と \(\left(\dfrac{4}{5}\right)^3\) を計算せよ。

解き方：

  1. 積の累乗：\((2 \times 3)^4 = 2^4 \times 3^4 = 16 \times 81 = 1296\)
  2. 商の累乗：\(\left(\dfrac{4}{5}\right)^3 = \dfrac{4^3}{5^3} = \dfrac{64}{125}\)
例30乗と負の指数

\(7^0\) と \(2^{-3}\) と \(\left(\dfrac{1}{3}\right)^{-2}\) を計算せよ。

解き方：

  1. 0乗は常に1：\(7^0 = 1\)
  2. 負の指数は逆数：\(2^{-3} = \dfrac{1}{2^3} = \dfrac{1}{8}\)
  3. 分数の負の指数：\(\left(\dfrac{1}{3}\right)^{-2} = \dfrac{1}{\left(\frac{1}{3}\right)^2} = \dfrac{1}{\frac{1}{9}} = 9\)

ポイント： \(\left(\dfrac{a}{b}\right)^{-n} = \left(\dfrac{b}{a}\right)^n\) で分子分母を入れ替えてもOK！

例4分数指数

\(8^{\frac{1}{3}}\) と \(27^{\frac{2}{3}}\) と \(16^{-\frac{3}{4}}\) を計算せよ。

解き方：

  1. \(8^{\frac{1}{3}} = \sqrt[3]{8} = 2\)（\(2^3 = 8\) だから）
  2. \(27^{\frac{2}{3}} = (\sqrt[3]{27})^2 = 3^2 = 9\)（\(3^3 = 27\) だから）
  3. \(16^{-\frac{3}{4}} = \dfrac{1}{16^{\frac{3}{4}}} = \dfrac{1}{(\sqrt[4]{16})^3} = \dfrac{1}{2^3} = \dfrac{1}{8}\)

ポイント： 分数指数は「分母がルートの次数、分子が累乗の指数」と覚える

▶ 指数関数のグラフ

例5グラフの特徴

\(y = 3^x\) と \(y = \left(\dfrac{1}{3}\right)^x\) のグラフについて、共通点と相違点を述べよ。

解き方：

  1. 共通点：
    • 必ず点 \((0, 1)\) を通る
    • \(x\) 軸が漸近線（\(y = 0\) には達しない）
    • \(y > 0\)（常に正の値をとる）
  2. 相違点：
    • \(y = 3^x\)：\(3 > 1\) なので右上がり（\(x\) が増えると \(y\) も増加）
    • \(y = \left(\frac{1}{3}\right)^x\)：\(0 < \frac{1}{3} < 1\) なので右下がり（\(x\) が増えると \(y\) は減少）

ポイント： \(\left(\dfrac{1}{3}\right)^x = 3^{-x}\) なので、2つのグラフは \(y\) 軸に関して対称！

▶ 指数方程式

例6指数方程式

次の方程式を解け。

(1) \(4^x = 32\)　　(2) \(9^x = 27\)　　(3) \(2^{2x+1} = 8^{x-1}\)

解き方：

  1. (1) 底を2に統一：\(4^x = (2^2)^x = 2^{2x}\)、\(32 = 2^5\)
    \[ 2^{2x} = 2^5 \Rightarrow 2x = 5 \Rightarrow x = \frac{5}{2} \]
  2. (2) 底を3に統一：\(9^x = (3^2)^x = 3^{2x}\)、\(27 = 3^3\)
    \[ 3^{2x} = 3^3 \Rightarrow 2x = 3 \Rightarrow x = \frac{3}{2} \]
  3. (3) 両辺を2のべき乗に：左辺 \(2^{2x+1}\)、右辺 \(8^{x-1} = (2^3)^{x-1} = 2^{3x-3}\)
    \[ 2^{2x+1} = 2^{3x-3} \Rightarrow 2x+1 = 3x-3 \Rightarrow x = 4 \]

▶ 指数不等式

例7指数不等式

次の不等式を解け。

(1) \(2^{x+1} > 8\)　　(2) \(\left(\dfrac{1}{3}\right)^x \geq 9\)

解き方：

  1. (1) 底が2（> 1）なので、不等号の向きはそのまま：
    \[ 2^{x+1} > 2^3 \Rightarrow x+1 > 3 \Rightarrow x > 2 \]
  2. (2) 底が \(\frac{1}{3}\)（< 1）なので、不等号の向きが逆転：
    まず \(9 = 3^2 = \left(\frac{1}{3}\right)^{-2}\) と変形
    \[ \left(\frac{1}{3}\right)^x \geq \left(\frac{1}{3}\right)^{-2} \]
    底が1より小さいので：
    \[ x \leq -2 \]

ポイント： 底が1より小さいときは「不等号が逆転」を忘れずに！

例8置き換えによる方程式

\(4^x - 3 \cdot 2^x - 4 = 0\) を解け。

解き方：

  1. \(4^x = (2^2)^x = (2^x)^2\) なので、\(t = 2^x\) とおく（\(t > 0\)）
  2. 方程式は \(t^2 - 3t - 4 = 0\) になる
  3. 因数分解：\((t-4)(t+1) = 0\) より \(t = 4\) または \(t = -1\)
  4. \(t > 0\) なので \(t = -1\) は不適。\(t = 4\) より \(2^x = 4 = 2^2\)
  5. よって \(x = 2\)
\[ 答え：x = 2 \]

ポイント： \(a^{2x}\) と \(a^x\) が混在 → \(t = a^x\) とおいて二次方程式に！

4. 簡単にするコツ

✏️ コツ1：よく使う累乗を覚えておく
2の累乗 \(2^1=2\)\(2^2=4\)\(2^3=8\)\(2^4=16\)\(2^5=32\)\(2^6=64\)\(2^{10}=1024\)
3の累乗 \(3^1=3\)\(3^2=9\)\(3^3=27\)\(3^4=81\)
5の累乗 \(5^1=5\)\(5^2=25\)\(5^3=125\)
✏️ コツ2：底を統一するときは素因数分解
\(4 = 2^2\)、\(8 = 2^3\)、\(16 = 2^4\)、\(32 = 2^5\) のように素因数分解すれば、底を揃えやすくなります。
例：\(4^x = 32\) → \((2^2)^x = 2^5\) → \(2^{2x} = 2^5\)
✏️ コツ3：負の指数は「ひっくり返す」
\(a^{-n} = \dfrac{1}{a^n}\) を使えば、負の指数を正の指数に変換できます。
また \(\left(\dfrac{a}{b}\right)^{-n} = \left(\dfrac{b}{a}\right)^n\) で分子分母を入れ替えられます。
✏️ コツ4：グラフの増減は底で決まる
不等式を解くとき「底が1より大きい？小さい？」をまず確認。
  • \(a > 1\)：不等号そのまま（右上がりだから）
  • \(0 < a < 1\)：不等号逆転（右下がりだから）

5. よくあるつまづき

⚠️ つまづき1：指数の足し算と底の足し算を混同

ダメな例： \(2^3 + 2^4 = 2^7\) としてしまう

対策： 指数法則は「かけ算・割り算」のときだけ使える！足し算のときは使えません。

正しくは：\(2^3 + 2^4 = 8 + 16 = 24\)

⚠️ つまづき2：底が違うのに指数法則を使う

ダメな例： \(2^3 \times 3^2 = 6^5\) としてしまう

対策： 指数法則は「底が同じ」ときだけ使える！底が違うときはそのまま計算。

正しくは：\(2^3 \times 3^2 = 8 \times 9 = 72\)

⚠️ つまづき3：底が1より小さいときの不等号

ダメな例： \(\left(\dfrac{1}{2}\right)^x > \left(\dfrac{1}{2}\right)^3\) のとき \(x > 3\) とする

対策： 底が1より小さいときは、グラフが右下がりなので不等号が逆転！

正しくは：\(x < 3\)

⚠️ つまづき4：置き換えで範囲を忘れる

ダメな例： \(t = 2^x\) とおいて \(t = -1\) も答えに含めてしまう

対策： \(a^x > 0\) は常に正なので、置き換えた変数の範囲 \(t > 0\) を忘れずに！

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の計算をせよ。

(1) \(3^2 \times 3^{-5}\)　　(2) \(16^{\frac{3}{4}}\)　　(3) \(\left(\dfrac{2}{3}\right)^{-2}\)

答えを見る

(1) \(3^2 \times 3^{-5} = 3^{2-5} = 3^{-3} = \dfrac{1}{27}\)

(2) \(16^{\frac{3}{4}} = (\sqrt[4]{16})^3 = 2^3 = 8\)

(3) \(\left(\dfrac{2}{3}\right)^{-2} = \left(\dfrac{3}{2}\right)^2 = \dfrac{9}{4}\)

練習2

次の方程式・不等式を解け。

(1) \(8^x = 4^{x+1}\)　　(2) \(\left(\dfrac{1}{2}\right)^{2x-1} \leq 4\)

答えを見る

(1) 底を2に統一：\(8^x = (2^3)^x = 2^{3x}\)、\(4^{x+1} = (2^2)^{x+1} = 2^{2x+2}\)

\(2^{3x} = 2^{2x+2}\) より \(3x = 2x + 2\)、よって \(x = 2\)

(2) \(4 = 2^2 = \left(\frac{1}{2}\right)^{-2}\) なので

\(\left(\frac{1}{2}\right)^{2x-1} \leq \left(\frac{1}{2}\right)^{-2}\)

底が \(\frac{1}{2} < 1\) なので不等号逆転：\(2x - 1 \geq -2\)

よって \(x \geq -\dfrac{1}{2}\)

📌 まとめ：指数法則で「かけ算は足す、割り算は引く、累乗はかける」を使い、指数関数では「底 > 1 なら右上がり、0 < 底 < 1 なら右下がり」を覚えれば方程式・不等式も攻略できる！

おまけ：数学雑学「細菌の爆発的増殖」

📖 1匹の細菌が地球を覆うまで

ある種の細菌は、理想的な環境では20分ごとに2倍に分裂します。1匹の細菌からスタートしたら、24時間後には何匹になるでしょう？

20分で2倍ということは、1時間で3回分裂。24時間では...

\[ 24 \times 3 = 72 \text{ 回の分裂} \]

つまり細菌の数は：

\[ 2^{72} \approx 4.7 \times 10^{21} \text{ 匹！} \]

これは4700兆兆匹です。地球上の砂粒の数（約10^18個）よりも多い！

もし1匹の細菌の体積が 1 立方マイクロメートルだとすると、\(2^{72}\) 匹の総体積は...

\[ 4.7 \times 10^{21} \times 10^{-18} \text{ m}^3 = 4700 \text{ m}^3 \]

これはプール（25m×10m×2m = 500 m³）約10杯分。たった24時間でこの増殖量！

実際には栄養や空間の制約があるので無限には増えませんが、これが「指数関数的増加」の恐ろしさ（と面白さ）です。感染症が急激に広がるのも、まさにこの指数関数の性質なのです。

📝 ちなみに、同じ原理で複利計算も「お金が勝手に増える」仕組み。アインシュタインは複利を「人類最大の発明」と呼んだとか...真偽は定かではありませんが、指数関数のパワーを表す良いエピソードですね。
