1. テーマの定義

「正弦定理と余弦定理、どっちを使えばいい？」——三角形の問題で一番の悩みはこれ。この教材で迷わず判断できるようになろう。

このページでは、定理の選び方を学びます：

  • 与えられた情報から判断：辺と角のペア？3辺？2辺と間の角？
  • 求めるものから判断：外接円の半径？残りの辺？角度？
  • パターン別の解法：どの情報でどの定理を使うか

2. 判断フロー

🎯 最重要：「辺と対角のペア」があれば正弦定理、なければ余弦定理！
外接円R → 正弦定理、ペアあり → 正弦定理、それ以外 → 余弦定理
📌 条件別・使う定理まとめ
与えられた条件 使う定理 理由
1辺と2角 正弦定理 辺と対角のペアができる
2辺と対角（2辺の向かいの角） 正弦定理 辺と対角のペアがある
外接円の半径Rが絡む 正弦定理 a/sin A = 2R を使う
2辺とその間の角 余弦定理 残りの1辺を直接求められる
3辺 余弦定理 cos を求めて角度を出す
📌 「辺と対角のペア」とは？

三角形ABCで、各辺と向かい合う角の組み合わせ：

  • 辺 a（BCの長さ）と角 A（頂点Aの角）
  • 辺 b（CAの長さ）と角 B（頂点Bの角）
  • 辺 c（ABの長さ）と角 C（頂点Cの角）

ペアがある例： a = 5, A = 30° がわかっている → ペア(a, A)あり → 正弦定理

ペアがない例： a = 5, b = 7, C = 60° → aとAのペアなし、bとBのペアなし → 余弦定理

問題を見たらまず図を描いて、何がわかっているか確認！
📌 複合パターン：両方使うこともある！

複雑な問題では、正弦定理と余弦定理を両方使うことがある。

例： 2辺と1角（間の角ではない）が与えられた場合

  1. まず余弦定理で残りの1辺を求める
  2. 次に正弦定理で残りの角を求める

または逆に、正弦定理で角を求めてから余弦定理で辺を求めることも。

3. 例題と解き方

パターンごとに「どちらを使うか」を判断しながら解いてみましょう。

▶ パターン：1辺と2角 → 正弦定理

例1△ABCで A = 45°, B = 60°, c = 4 のとき、辺 a を求める

判断： cとCのペア（C = 180° − 45° − 60° = 75°）がある → 正弦定理

解き方：

  1. C = 75° を求める
  2. 正弦定理 \(\dfrac{a}{\sin A} = \dfrac{c}{\sin C}\) を使う
  3. \(\dfrac{a}{\sin 45°} = \dfrac{4}{\sin 75°}\)
\[ a = \frac{4 \sin 45°}{\sin 75°} = \frac{4 \cdot \frac{\sqrt{2}}{2}}{\frac{\sqrt{6}+\sqrt{2}}{4}} = \frac{2\sqrt{2} \cdot 4}{\sqrt{6}+\sqrt{2}} \]

有理化して \(a = \sqrt{6} - \sqrt{2}\) または約2.05

▶ パターン：2辺とその間の角 → 余弦定理

例2△ABCで b = 5, c = 8, A = 60° のとき、辺 a を求める

判断： bとc、その間の角A → 辺と対角のペアなし → 余弦定理

解き方：

  1. 余弦定理 \(a^2 = b^2 + c^2 - 2bc\cos A\) を使う
  2. \(a^2 = 25 + 64 - 2 \cdot 5 \cdot 8 \cdot \cos 60°\)
  3. \(a^2 = 89 - 80 \cdot \dfrac{1}{2} = 89 - 40 = 49\)
\[ a = 7 \]

▶ パターン：3辺 → 余弦定理

例3△ABCで a = 7, b = 5, c = 8 のとき、角 B を求める

判断： 3辺のみ、角の情報なし → 余弦定理

解き方：

  1. 余弦定理を変形：\(\cos B = \dfrac{c^2 + a^2 - b^2}{2ca}\)
  2. \(\cos B = \dfrac{64 + 49 - 25}{2 \cdot 8 \cdot 7} = \dfrac{88}{112} = \dfrac{11}{14}\)
\[ \cos B = \frac{11}{14} \approx 0.786 \quad \therefore B \approx 38.2° \]

▶ パターン：外接円の半径 → 正弦定理

例4△ABCで a = 6, A = 30° のとき、外接円の半径 R を求める

判断： 外接円の半径R を求める → 正弦定理

解き方：

  1. \(\dfrac{a}{\sin A} = 2R\) を使う
  2. \(\dfrac{6}{\sin 30°} = 2R\)
  3. \(\dfrac{6}{1/2} = 2R\) → \(12 = 2R\)
\[ R = 6 \]

▶ 複合パターン：両方使う

例5△ABCで a = 7, b = 5, C = 60° のとき、角 A を求める

判断：

  • 辺aと角Aのペア？ → Aはまだ不明なのでペアなし
  • 2辺(a, b)とその間の角(C)がある → まず余弦定理で辺cを求める
  • その後、正弦定理で角Aを求める

解き方：

  1. Step 1（余弦定理）：辺 c を求める
    \[ c^2 = a^2 + b^2 - 2ab\cos C = 49 + 25 - 70 \cdot \frac{1}{2} = 39 \] \[ c = \sqrt{39} \]
  2. Step 2（正弦定理）：角 A を求める
    \[ \frac{a}{\sin A} = \frac{c}{\sin C} \] \[ \sin A = \frac{a \sin C}{c} = \frac{7 \cdot \frac{\sqrt{3}}{2}}{\sqrt{39}} = \frac{7\sqrt{3}}{2\sqrt{39}} \]

計算すると \(\sin A \approx 0.972\)、よって \(A \approx 76.3°\)

4. 簡単にするコツ

✏️ コツ1：まず「ペア」を探す
問題を読んだら、「辺aと角A」「辺bと角B」のようなペアがあるか確認。ペアがあれば正弦定理が使える可能性大。
✏️ コツ2：図を描いて情報を整理
三角形を描いて、わかっている辺と角を書き込む。視覚的に「何がわかっていて何を求めるか」を確認。
✏️ コツ3：外接円が出たら正弦定理一択
「外接円の半径」「外接円」というキーワードが出たら、迷わず正弦定理。余弦定理では外接円は扱えない。
✏️ コツ4：「〜を求めよ」に注目
  • 「辺を求めよ」で2辺と間の角がある → 余弦定理
  • 「角を求めよ」で3辺がある → 余弦定理
  • 「外接円の半径を求めよ」 → 正弦定理
✏️ コツ5：迷ったら両方試す
どちらを使うか迷う問題は、実際に式を立ててみる。立式できなければ別の定理を試す。

5. よくあるつまづき

⚠️ つまづき1：2辺と1角で正弦定理を使う（間の角でないとき）

ダメな例： a = 5, b = 7, A = 30° で「ペア(a, A)があるから正弦定理」

問題点： 正弦定理で b/sin B = a/sin A から sin B を求めると、B が鋭角か鈍角か2通りある

対策： 「2辺と対角でない角」のときは、余弦定理で先に辺を求める方が安全

⚠️ つまづき2：3辺で正弦定理を使おうとする

ダメな例： a = 5, b = 7, c = 8 で正弦定理を使おうとする

問題点： 角が1つもわからないので、a/sin A = b/sin B が立てられない

対策： 3辺のみなら余弦定理一択。cos を求めて角度を出す。

⚠️ つまづき3：sinから角度を求めるとき、2通りを忘れる

ダメな例： sin A = 1/2 から「A = 30°」とだけ答える

問題点： sin 150° = 1/2 もあるので、A = 30° または 150° の可能性

対策： 三角形の角の和が180°になるか、他の条件から絞り込む

⚠️ つまづき4：余弦定理の公式を間違える

ダメな例： \(a^2 = b^2 + c^2 - 2bc\cos \textcolor{red}{B}\) とする（Aでなく）

対策： 「求める辺の対角」のcosを使う。a²なら cos A、c²なら cos C。

7. 練習問題

まず「どちらの定理を使うか」を判断してから解こう。

練習1

△ABCで a = 8, B = 45°, C = 75° のとき、辺 b を求めよ。

答えを見る

判断：1辺と2角 → A = 180° − 45° − 75° = 60°、ペア(a, A)あり → 正弦定理

\(\dfrac{a}{\sin A} = \dfrac{b}{\sin B}\)

\(\dfrac{8}{\sin 60°} = \dfrac{b}{\sin 45°}\)

\(b = \dfrac{8 \sin 45°}{\sin 60°} = \dfrac{8 \cdot \frac{\sqrt{2}}{2}}{\frac{\sqrt{3}}{2}} = \dfrac{4\sqrt{2} \cdot 2}{\sqrt{3}} = \dfrac{8\sqrt{2}}{\sqrt{3}} = \boxed{\dfrac{8\sqrt{6}}{3}}\)

練習2

△ABCで a = 5, b = 6, c = 7 のとき、最大の角を求めよ。

答えを見る

判断：3辺のみ → 余弦定理

最大の辺 c = 7 の対角 C が最大角

\(\cos C = \dfrac{a^2 + b^2 - c^2}{2ab} = \dfrac{25 + 36 - 49}{60} = \dfrac{12}{60} = \dfrac{1}{5}\)

\(C = \arccos\left(\dfrac{1}{5}\right) \approx \boxed{78.5°}\)

📌 まとめ：外接円R → 正弦定理、辺と対角のペアあり → 正弦定理、それ以外 → 余弦定理！

おまけ：数学雑学「測量士の二刀流」

📐 エベレストの高さはどうやって測った？

世界最高峰エベレスト（8,848m）の高さは、誰も山頂に登らずに初めて測定された。

1856年、イギリスの測量士たちは、インドの平地から約200km離れた場所でエベレストを見上げた。

「あの山の高さを測りたい」

しかし登るのは不可能。彼らが使ったのは三角測量——まさに正弦定理と余弦定理だった。

Step 1：まず地上で2地点を選び、その間の距離を正確に測る（これが「基線」）

Step 2：2地点それぞれから山頂を見上げる角度を測定

Step 3：三角形ができる！基線の長さと2つの角がわかれば、正弦定理で山頂までの距離がわかる

Step 4：さらに仰角（見上げる角度）を使って、三角比で高さを計算

\[ \text{高さ} = \text{距離} \times \sin(\text{仰角}) \]

200km以上離れた場所から測定して、誤差はわずか数メートル。現代のGPS測定とほぼ一致している。

...ちなみに、このときエベレストは「Peak XV（15号峰）」と呼ばれていた。測量責任者ジョージ・エベレストの名前がつけられたのは、彼の引退後のこと。本人は「現地の名前を使うべき」と反対していたという。
