1. テーマの定義

「直角三角形でなくても辺の長さを求めたい」「外接円の半径は？」——三角比を一般の三角形に拡張するのが正弦定理・余弦定理です。

このページでは、三角形の辺と角の関係を学びます：

  • 正弦定理：辺と対角のsinの比は外接円の直径に等しい
  • 余弦定理：三平方の定理を一般の三角形に拡張した公式
  • 三角形の面積公式：2辺とその間の角から面積を求める

2. 覚えるルール

🎯 最重要：正弦定理は「辺÷sin = 直径」、余弦定理は「三平方＋cosの補正」！
正弦定理：辺と対角のsinの比はすべて等しく、外接円の直径2Rに等しい
📌 ルール1：正弦定理 → 例1・2
\[ \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R \]

三角形ABCで、a, b, c は各頂点の対辺の長さ、Rは外接円の半径。

→ だから結果は：

  • 辺を求める：\(a = 2R \sin A\)
  • 角を求める：\(\sin A = \dfrac{a}{2R}\)
  • 外接円の半径を求める：\(R = \dfrac{a}{2\sin A}\)

✅ メリット： 1つの辺と対角がわかれば、他の辺や角も芋づる式に求まる

💡 どんなときに使う？

正弦定理が有効な場面：

  • 「角度とその対辺」のペアが1組わかっている
  • 外接円の半径Rを求めたい
  • 辺の長さから角度（のsin）を求めたい
余弦定理：2辺とその間の角から、残りの辺の長さを求められる
📌 ルール2：余弦定理 → 例3・4
\[ c^2 = a^2 + b^2 - 2ab\cos C \]

他の辺についても同様に：

\[ a^2 = b^2 + c^2 - 2bc\cos A, \quad b^2 = c^2 + a^2 - 2ca\cos B \]

→ だから結果は：

  • 辺を求める：2辺とその間の角がわかれば、残りの1辺が求まる
  • 角を求める：3辺がわかれば、\(\cos C = \dfrac{a^2 + b^2 - c^2}{2ab}\) で角が求まる

✅ メリット： 三平方の定理の拡張版！直角三角形でなくても使える

💡 考え方を見る

三平方の定理との関係：

C = 90° のとき cos C = 0 なので、余弦定理は c² = a² + b² となり、三平方の定理と一致する。

C < 90°（鋭角）のとき cos C > 0 なので、c² < a² + b²（引き算される）

C > 90°（鈍角）のとき cos C < 0 なので、c² > a² + b²（足し算される）

📌 ルール3：三角形の面積公式 → 例5
\[ S = \frac{1}{2}ab\sin C = \frac{1}{2}bc\sin A = \frac{1}{2}ca\sin B \]

S は三角形ABCの面積。

→ だから結果は： 2辺とその間の角がわかれば面積が求まる

✅ メリット： 高さを求めなくても面積が計算できる！

💡 考え方を見る

「底辺×高さ÷2」の公式で、高さ = b × sin C として代入したもの。

底辺を a、高さを h とすると、h = b sin C なので S = (1/2) × a × b sin C

📌 ルール4：正弦定理と余弦定理の使い分け
与えられた条件 使う定理 求められるもの
1辺と2角 正弦定理 他の辺
2辺と対角 正弦定理 他の角
外接円の半径が絡む 正弦定理 R、または辺・角
2辺とその間の角 余弦定理 残りの1辺
3辺 余弦定理 角（のcos）

3. 例題と解き方

正弦定理・余弦定理を使って、三角形の辺や角を求めてみましょう。

▶ 正弦定理を使う

例1辺の長さを求める

△ABCで、A = 45°、B = 60°、a = 4 のとき、辺 b の長さを求めよ。

解き方：

  1. 正弦定理 \(\dfrac{a}{\sin A} = \dfrac{b}{\sin B}\) を使う
  2. 値を代入：\(\dfrac{4}{\sin 45°} = \dfrac{b}{\sin 60°}\)
  3. sin 45° = √2/2、sin 60° = √3/2 を代入
  4. b について解く
\[ \frac{4}{\frac{\sqrt{2}}{2}} = \frac{b}{\frac{\sqrt{3}}{2}} \] \[ \frac{4 \times 2}{\sqrt{2}} = \frac{b \times 2}{\sqrt{3}} \] \[ b = \frac{4\sqrt{3}}{\sqrt{2}} = \frac{4\sqrt{3} \times \sqrt{2}}{2} = 2\sqrt{6} \]

答え： \(b = 2\sqrt{6}\)

例2外接円の半径を求める

△ABCで、a = 6、A = 30° のとき、外接円の半径 R を求めよ。

解き方：

  1. 正弦定理 \(\dfrac{a}{\sin A} = 2R\) を使う
  2. 値を代入：\(\dfrac{6}{\sin 30°} = 2R\)
  3. sin 30° = 1/2 を代入
\[ \frac{6}{\frac{1}{2}} = 2R \] \[ 12 = 2R \] \[ R = 6 \]

答え： R = 6

▶ 余弦定理を使う

例3辺の長さを求める（2辺とその間の角）

△ABCで、b = 5、c = 7、A = 60° のとき、辺 a の長さを求めよ。

解き方：

  1. 求めたい辺 a の対角が A なので、余弦定理 \(a^2 = b^2 + c^2 - 2bc\cos A\) を使う
  2. 値を代入：\(a^2 = 5^2 + 7^2 - 2 \cdot 5 \cdot 7 \cdot \cos 60°\)
  3. cos 60° = 1/2 を代入して計算
\[ a^2 = 25 + 49 - 2 \cdot 5 \cdot 7 \cdot \frac{1}{2} \] \[ a^2 = 74 - 35 = 39 \] \[ a = \sqrt{39} \]

答え： \(a = \sqrt{39}\)

例4角度を求める（3辺がわかっている）

△ABCで、a = 7、b = 5、c = 3 のとき、角 A を求めよ。

解き方：

  1. 余弦定理を変形：\(\cos A = \dfrac{b^2 + c^2 - a^2}{2bc}\)
  2. 値を代入して cos A を計算
  3. cos A の値から角 A を特定
\[ \cos A = \frac{5^2 + 3^2 - 7^2}{2 \cdot 5 \cdot 3} = \frac{25 + 9 - 49}{30} = \frac{-15}{30} = -\frac{1}{2} \]

cos A = −1/2 より、A = 120°

答え： A = 120°

▶ 面積公式を使う

例5三角形の面積を求める

△ABCで、b = 6、c = 8、A = 30° のとき、面積 S を求めよ。

解き方：

  1. 面積公式 \(S = \dfrac{1}{2}bc\sin A\) を使う
  2. 値を代入：\(S = \dfrac{1}{2} \cdot 6 \cdot 8 \cdot \sin 30°\)
  3. sin 30° = 1/2 を代入
\[ S = \frac{1}{2} \cdot 6 \cdot 8 \cdot \frac{1}{2} = \frac{48}{4} = 12 \]

答え： S = 12

▶ 複合問題

例6辺と面積を両方求める

△ABCで、a = 8、B = 60°、C = 45° のとき、辺 b と面積 S を求めよ。

解き方：

  1. A = 180° − 60° − 45° = 75° を求める
  2. 正弦定理で b を求める：\(\dfrac{a}{\sin A} = \dfrac{b}{\sin B}\)
  3. 面積公式で S を求める
\[ \frac{8}{\sin 75°} = \frac{b}{\sin 60°} \]

※ sin 75° = sin(45° + 30°) = (√6 + √2)/4（加法定理）を使うか、電卓で約0.966

\[ b = \frac{8 \sin 60°}{\sin 75°} = \frac{8 \cdot \frac{\sqrt{3}}{2}}{\frac{\sqrt{6}+\sqrt{2}}{4}} = \frac{4\sqrt{3} \cdot 4}{\sqrt{6}+\sqrt{2}} = \frac{16\sqrt{3}}{\sqrt{6}+\sqrt{2}} \]

有理化すると \(b = \frac{16\sqrt{3}(\sqrt{6}-\sqrt{2})}{4} = 4\sqrt{3}(\sqrt{6}-\sqrt{2}) = 4\sqrt{18} - 4\sqrt{6} = 12\sqrt{2} - 4\sqrt{6}\)

面積は正弦定理の結果を使わず、\(S = \dfrac{1}{2}ab\sin C\) で求める（bの式を代入して計算）

4. 簡単にするコツ

✏️ コツ1：「どの定理を使うか」を最初に決める
  • 辺と対角のペアがある → 正弦定理
  • 外接円が出てくる → 正弦定理
  • 2辺とその間の角または3辺がわかっている → 余弦定理
✏️ コツ2：余弦定理の公式は「左辺の対角」で覚える
\(c^2 = a^2 + b^2 - 2ab\cos C\) で、左辺の c の対角が C。
「求めたい辺を左辺に、その対角のcosを右辺に」と覚えれば迷わない。
✏️ コツ3：cosが負なら鈍角
余弦定理で cos A を計算したとき：
  • cos A > 0 → A は鋭角（0° < A < 90°）
  • cos A = 0 → A = 90°
  • cos A < 0 → A は鈍角（90° < A < 180°）
✏️ コツ4：面積公式は「挟まれた角」
\(S = \dfrac{1}{2}bc\sin A\) で使う角 A は、辺 b と辺 c に挟まれた角。
「2辺をかけて、間の角のsinをかけて、2で割る」と覚える。

5. よくあるつまづき

⚠️ つまづき1：正弦定理と余弦定理を混同する

ダメな例： 3辺がわかっているのに正弦定理を使おうとする

対策： 「辺と対角のペア」があるかどうかで判断。ペアがあれば正弦定理、なければ余弦定理。

⚠️ つまづき2：余弦定理の符号を間違える

ダメな例： \(c^2 = a^2 + b^2 \textcolor{red}{+} 2ab\cos C\) と書いてしまう

対策： 必ずマイナス！「三平方から引き算する」と覚える。C = 90°で確認すると、cos 90° = 0 で三平方と一致する。

⚠️ つまづき3：面積公式で「対角」を使ってしまう

ダメな例： a = 5, b = 6, C = 60° で S = (1/2) × 5 × 6 × sin A としてしまう

対策： 面積公式は「2辺に挟まれた角」のsinを使う。上の例では \(S = \dfrac{1}{2} \cdot 5 \cdot 6 \cdot \sin 60°\) ではなく、辺aと辺bが角Cに挟まれていないので注意。a = BC, b = CA なら、A が挟まれた角。

⚠️ つまづき4：sinの値から角度を求めるとき、鋭角・鈍角を間違える

ダメな例： sin A = 1/2 から「A = 30°」とだけ答える

対策： sin A = 1/2 を満たす角は 30° と 150° の2つ。三角形の他の角との関係から、鋭角か鈍角かを判断する必要がある。

6. 仕上げチェックリスト

正弦定理と余弦定理のどちらを使うか正しく選んだか
辺と角の対応関係（aとA、bとB、cとC）を正しく把握したか
余弦定理の符号（マイナス）を間違えていないか
面積公式で「挟まれた角」のsinを使ったか
答えが三角形として成り立つか（辺の長さが正、角の和が180°）

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

△ABCで、a = 10、A = 30°、B = 45° のとき、辺 b と外接円の半径 R を求めよ。

答えを見る

正弦定理より \(\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = 2R\)

\(\dfrac{10}{\sin 30°} = \dfrac{10}{1/2} = 20 = 2R\) より R = 10

\(\dfrac{b}{\sin 45°} = 20\) より \(b = 20 \cdot \dfrac{\sqrt{2}}{2} = \boxed{10\sqrt{2}}\)

練習2

△ABCで、a = 5、b = 7、c = 8 のとき、角 C と面積 S を求めよ。

答えを見る

余弦定理より \(\cos C = \dfrac{a^2 + b^2 - c^2}{2ab} = \dfrac{25 + 49 - 64}{70} = \dfrac{10}{70} = \dfrac{1}{7}\)

\(\sin^2 C = 1 - \cos^2 C = 1 - \dfrac{1}{49} = \dfrac{48}{49}\)

\(\sin C = \dfrac{4\sqrt{3}}{7}\)（Cは鋭角なので正）

\(S = \dfrac{1}{2}ab\sin C = \dfrac{1}{2} \cdot 5 \cdot 7 \cdot \dfrac{4\sqrt{3}}{7} = \boxed{10\sqrt{3}}\)

📌 まとめ：正弦定理は「辺/sin = 2R」、余弦定理は「c² = a² + b² − 2ab cos C」。ペアがあれば正弦、なければ余弦！

おまけ：歴史ストーリー「星の距離を測った男」

📖 ヒッパルコスと月への距離（紀元前150年頃・ギリシャ）

古代ギリシャの天文学者ヒッパルコスは、夜空を見上げてこう考えた。

「あの月は一体どれくらい遠いのだろう？」

当時、宇宙に出かける方法など存在しない。しかしヒッパルコスには三角比があった。

彼は日食を利用した。ある日食が、ある場所では「皆既日食」（太陽が完全に隠れる）だったが、別の場所では「部分日食」だった。この見え方の違いを使えば、月までの距離が計算できると気づいたのだ。

2地点から月を見たときの角度の差（視差）を測定し、三角形を作る。地球上の2点間の距離はわかっている。あとは正弦定理を応用すれば...

\[ \text{月への距離} = \frac{\text{2地点間の距離}}{\sin(\text{視差})} \times \sin(\text{別の角}) \]

ヒッパルコスの計算結果は「月は地球の約30倍の距離にある」。現代の測定では約30.2倍なので、2000年以上前にほぼ正解を出していたのだ！

彼はさらに、星の位置を記録した「星表」を作成。これは近代天文学の基礎となった。

...三角比だけで月までの距離を当てるとは、古代ギリシャ人、恐るべし。ちなみにヒッパルコスは「三角法の父」とも呼ばれている。正弦定理・余弦定理の原型を作った人物だ。

※ヒッパルコスの業績については、プトレマイオスの『アルマゲスト』に記録が残されています（ヒッパルコス - Wikipedia）。
