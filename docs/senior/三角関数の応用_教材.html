1. テーマの定義

「\(a\sin\theta + b\cos\theta\) を1つの三角関数にまとめたい」「\(\sin x = \dfrac{1}{2}\) を満たす \(x\) は？」——三角関数を自在に操るには、合成と方程式・不等式の解法が欠かせません。

このページでは、三角関数の応用を学びます：

  • 三角関数の合成：\(a\sin\theta + b\cos\theta\) を \(r\sin(\theta + \alpha)\) の形に変換
  • 三角方程式：\(\sin x = k\) や \(\cos x = k\) を満たす角を求める
  • 三角不等式：\(\sin x > k\) や \(\cos x \leq k\) を満たす範囲を求める

2. 覚えるルール

🎯 最重要：三角関数の合成は \(a\sin\theta + b\cos\theta = \sqrt{a^2+b^2}\sin(\theta + \alpha)\) の形にまとめる！
三角関数の合成：点P(a, b)へのベクトルの長さrと角度αを使う
📌 ルール1：三角関数の合成 → 例1
\[ a\sin\theta + b\cos\theta = \sqrt{a^2+b^2}\sin(\theta + \alpha) \]

ただし、\(\alpha\) は \(\cos\alpha = \dfrac{a}{\sqrt{a^2+b^2}}\)、\(\sin\alpha = \dfrac{b}{\sqrt{a^2+b^2}}\) を満たす角。

→ だから結果は： 2つの三角関数の和を1つにまとめて、最大値・最小値や方程式が解きやすくなる。

💡 考え方を見る

加法定理 \(\sin(\theta + \alpha) = \sin\theta\cos\alpha + \cos\theta\sin\alpha\) を使うと：

\(r\sin(\theta + \alpha) = r\cos\alpha \cdot \sin\theta + r\sin\alpha \cdot \cos\theta\)

\(r\cos\alpha = a\)、\(r\sin\alpha = b\) となるように \(r\) と \(\alpha\) を決めればよい。

\(r^2 = (r\cos\alpha)^2 + (r\sin\alpha)^2 = a^2 + b^2\) より \(r = \sqrt{a^2 + b^2}\)

📌 ルール1'：cos型の合成
\[ a\sin\theta + b\cos\theta = \sqrt{a^2+b^2}\cos(\theta - \beta) \]

ただし、\(\cos\beta = \dfrac{b}{\sqrt{a^2+b^2}}\)、\(\sin\beta = \dfrac{a}{\sqrt{a^2+b^2}}\)。

✅ 使い分け： sin型かcos型かは問題に合わせて選ぶ。どちらでも結果は同じ。

📌 ルール2：三角方程式の解き方 → 例2・3

\(\sin x = k\) の解法（\(-1 \leq k \leq 1\) のとき）：

  1. 単位円で \(y = k\) となる点を探す
  2. その点に対応する角を求める（基本角 \(\alpha\) を使う）
  3. 周期 \(2\pi\) を考慮して一般解を書く：\(x = \alpha + 2n\pi\) または \(x = (\pi - \alpha) + 2n\pi\)

\(\cos x = k\) の解法：

  1. 単位円で \(x = k\) となる点を探す
  2. 基本角 \(\alpha\)（\(0 \leq \alpha \leq \pi\) で \(\cos\alpha = k\)）を求める
  3. 一般解：\(x = \pm\alpha + 2n\pi\)

→ だから結果は： 単位円を描けば、解がどこにあるか視覚的にわかる。

単位円を使った三角方程式の解法：直線との交点が解
📌 ルール3：三角不等式の解き方 → 例4・5

\(\sin x > k\) の解法：

  1. 単位円で \(\sin x = k\) の解（境界）を求める
  2. \(y > k\) となる弧を特定（上半分）
  3. その弧に対応する角の範囲を答える

\(\cos x \leq k\) の解法：

  1. 単位円で \(\cos x = k\) の解（境界）を求める
  2. \(x \leq k\) となる弧を特定（左半分）
  3. その弧に対応する角の範囲を答える

→ だから結果は： 方程式を解いて境界を出し、「上か下か」「右か左か」で範囲を決める。

三角不等式：境界を求めて、条件を満たす弧を特定
📌 ルール4：tan の方程式・不等式 → 例6

\(\tan x = k\) の解法：

\[ \tan x = k \quad \Rightarrow \quad x = \alpha + n\pi \quad (\text{n は整数}) \]

tan の周期は \(\pi\) なので、1つ解を見つければ \(\pi\) ごとに繰り返す。

→ だから結果は： sin, cos より周期が短い分、解の個数が多くなることに注意。

3. 例題と解き方

三角関数の合成、方程式、不等式を順番にマスターしましょう。

▶ 三角関数の合成

例1三角関数の合成

\(\sin\theta + \sqrt{3}\cos\theta\) を \(r\sin(\theta + \alpha)\) の形に変形せよ。

解き方：

  1. \(a = 1\)、\(b = \sqrt{3}\) として \(r = \sqrt{a^2 + b^2} = \sqrt{1 + 3} = 2\)
  2. \(\cos\alpha = \dfrac{a}{r} = \dfrac{1}{2}\)、\(\sin\alpha = \dfrac{b}{r} = \dfrac{\sqrt{3}}{2}\)
  3. これを満たす \(\alpha\) は \(\alpha = \dfrac{\pi}{3}\)（60°）
\[ \sin\theta + \sqrt{3}\cos\theta = \boxed{2\sin\left(\theta + \frac{\pi}{3}\right)} \]

✅ 検算： \(\theta = 0\) を代入。左辺 = \(0 + \sqrt{3} = \sqrt{3}\)、右辺 = \(2\sin\dfrac{\pi}{3} = 2 \cdot \dfrac{\sqrt{3}}{2} = \sqrt{3}\)。一致！

▶ 三角方程式

例2sin の方程式

\(0 \leq x < 2\pi\) のとき、\(\sin x = \dfrac{1}{2}\) を解け。

解き方：

  1. 単位円で \(y = \dfrac{1}{2}\) となる点を探す
  2. 基本角：\(\sin\dfrac{\pi}{6} = \dfrac{1}{2}\) なので \(x = \dfrac{\pi}{6}\)
  3. もう1つは対称性から：\(x = \pi - \dfrac{\pi}{6} = \dfrac{5\pi}{6}\)
\[ x = \boxed{\frac{\pi}{6}, \frac{5\pi}{6}} \]
例3cos の方程式

\(0 \leq x < 2\pi\) のとき、\(\cos x = -\dfrac{\sqrt{3}}{2}\) を解け。

解き方：

  1. \(\cos\dfrac{\pi}{6} = \dfrac{\sqrt{3}}{2}\) なので、\(\cos x = -\dfrac{\sqrt{3}}{2}\) は第2象限と第3象限
  2. 第2象限：\(x = \pi - \dfrac{\pi}{6} = \dfrac{5\pi}{6}\)
  3. 第3象限：\(x = \pi + \dfrac{\pi}{6} = \dfrac{7\pi}{6}\)
\[ x = \boxed{\frac{5\pi}{6}, \frac{7\pi}{6}} \]

▶ 三角不等式

例4sin の不等式

\(0 \leq x < 2\pi\) のとき、\(\sin x > \dfrac{1}{2}\) を解け。

解き方：

  1. 境界：\(\sin x = \dfrac{1}{2}\) の解は \(x = \dfrac{\pi}{6}, \dfrac{5\pi}{6}\)
  2. 単位円で \(y > \dfrac{1}{2}\) となる範囲は、上側の弧
  3. \(\dfrac{\pi}{6}\) から \(\dfrac{5\pi}{6}\) の間
\[ \boxed{\frac{\pi}{6} < x < \frac{5\pi}{6}} \]
例5cos の不等式

\(0 \leq x < 2\pi\) のとき、\(\cos x \leq \dfrac{1}{2}\) を解け。

解き方：

  1. 境界：\(\cos x = \dfrac{1}{2}\) の解は \(x = \dfrac{\pi}{3}, \dfrac{5\pi}{3}\)
  2. 単位円で \(x \leq \dfrac{1}{2}\) となる範囲は、左側の弧
  3. \(\dfrac{\pi}{3}\) から \(\dfrac{5\pi}{3}\) の間（境界含む）
\[ \boxed{\frac{\pi}{3} \leq x \leq \frac{5\pi}{3}} \]
例6tan の方程式

\(0 \leq x < 2\pi\) のとき、\(\tan x = 1\) を解け。

解き方：

  1. \(\tan\dfrac{\pi}{4} = 1\) なので \(x = \dfrac{\pi}{4}\) が1つの解
  2. tan の周期は \(\pi\) なので、\(x = \dfrac{\pi}{4} + \pi = \dfrac{5\pi}{4}\) も解
\[ x = \boxed{\frac{\pi}{4}, \frac{5\pi}{4}} \]

▶ 合成を使った応用

例7合成を使った方程式

\(0 \leq x < 2\pi\) のとき、\(\sin x + \cos x = 1\) を解け。

解き方：

  1. まず合成：\(\sin x + \cos x = \sqrt{2}\sin\left(x + \dfrac{\pi}{4}\right)\)
  2. \(\sqrt{2}\sin\left(x + \dfrac{\pi}{4}\right) = 1\) より \(\sin\left(x + \dfrac{\pi}{4}\right) = \dfrac{1}{\sqrt{2}} = \dfrac{\sqrt{2}}{2}\)
  3. \(x + \dfrac{\pi}{4} = \dfrac{\pi}{4}\) または \(x + \dfrac{\pi}{4} = \dfrac{3\pi}{4}\)（\(0 \leq x < 2\pi\) より \(\dfrac{\pi}{4} \leq x + \dfrac{\pi}{4} < \dfrac{9\pi}{4}\)）
  4. \(x = 0\) または \(x = \dfrac{\pi}{2}\)
\[ x = \boxed{0, \frac{\pi}{2}} \]
例8合成を使った最大最小

\(y = 3\sin\theta - 4\cos\theta\) の最大値と最小値を求めよ。

解き方：

  1. 合成：\(r = \sqrt{3^2 + (-4)^2} = \sqrt{9 + 16} = 5\)
  2. \(y = 5\sin(\theta + \alpha)\) の形になる（\(\alpha\) は \(\cos\alpha = \dfrac{3}{5}\)、\(\sin\alpha = \dfrac{-4}{5}\) を満たす角）
  3. \(-1 \leq \sin(\theta + \alpha) \leq 1\) より \(-5 \leq y \leq 5\)
\[ \text{最大値} = \boxed{5}, \quad \text{最小値} = \boxed{-5} \]

4. 簡単にするコツ

✏️ コツ1：単位円を必ず描く
方程式・不等式を解くとき、単位円を描けば解が視覚的にわかる。「上か下か」「右か左か」で範囲を判断。
✏️ コツ2：合成の符号に注意
\(a\sin\theta - b\cos\theta\) のようにマイナスがある場合、\(b\) を負として計算するか、cos型で合成するとミスが減る。
✏️ コツ3：範囲の確認を忘れずに
合成後 \(\sin(\theta + \alpha)\) を解くとき、\(\theta + \alpha\) の範囲を先に求めておく。範囲外の解を出さないように注意。
✏️ コツ4：特定の値で検算
合成した結果は、\(\theta = 0\) や \(\theta = \dfrac{\pi}{2}\) などを代入して元の式と一致するか確認。
✏️ コツ5：対称性を活用
\(\sin x = k\) の2つの解は、\(x = \alpha\) と \(x = \pi - \alpha\) の関係。\(\cos x = k\) は \(x = \pm\alpha\) の関係。

5. よくあるつまづき

⚠️ つまづき1：合成の \(\alpha\) を間違える

ダメな例： \(\sin\theta + \sqrt{3}\cos\theta\) で \(\alpha = \dfrac{\pi}{6}\) としてしまう

対策： \(\cos\alpha = \dfrac{a}{r}\)、\(\sin\alpha = \dfrac{b}{r}\) を正確に計算。\(a = 1, b = \sqrt{3}, r = 2\) なら \(\cos\alpha = \dfrac{1}{2}, \sin\alpha = \dfrac{\sqrt{3}}{2}\) だから \(\alpha = \dfrac{\pi}{3}\)。

⚠️ つまづき2：解の個数を間違える

ダメな例： \(\sin x = \dfrac{1}{2}\) で \(x = \dfrac{\pi}{6}\) だけを答える

対策： 単位円を描いて交点の数を確認。\(\sin x = k\) は通常2つ、\(\cos x = k\) も通常2つ（\(k = \pm 1\) のときは1つ）。

⚠️ つまづき3：不等式の境界を含むかどうか

ダメな例： \(\sin x > \dfrac{1}{2}\) で境界を含めてしまう

対策： 不等号に「=」があるかどうかで、\(<\) か \(\leq\) かを使い分ける。等号があれば境界を含む。

⚠️ つまづき4：tan の周期を間違える

ダメな例： \(\tan x = 1\) で周期 \(2\pi\) として \(x = \dfrac{\pi}{4}\) だけを答える

対策： tan の周期は \(\pi\)。\(0 \leq x < 2\pi\) なら \(\dfrac{\pi}{4}\) と \(\dfrac{5\pi}{4}\) の2つ。

⚠️ つまづき5：象限を間違える

ダメな例： \(\cos x = -\dfrac{1}{2}\) で第1象限の解を書いてしまう

対策： cos が負 → 第2象限か第3象限。sin が負 → 第3象限か第4象限。符号から象限を特定。

6. 仕上げチェックリスト

合成の \(r = \sqrt{a^2 + b^2}\) を正確に計算したか
合成の \(\alpha\) の象限を確認したか
方程式で解の個数を確認したか（単位円を描いたか）
不等式で境界を含むか確認したか
指定された範囲内の解だけを答えたか

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(\sqrt{3}\sin\theta + \cos\theta\) を \(r\sin(\theta + \alpha)\) の形に変形せよ。

答えを見る

\(a = \sqrt{3}\)、\(b = 1\) より \(r = \sqrt{3 + 1} = 2\)

\(\cos\alpha = \dfrac{\sqrt{3}}{2}\)、\(\sin\alpha = \dfrac{1}{2}\) より \(\alpha = \dfrac{\pi}{6}\)

\(\sqrt{3}\sin\theta + \cos\theta = \boxed{2\sin\left(\theta + \dfrac{\pi}{6}\right)}\)

練習2

\(0 \leq x < 2\pi\) のとき、\(2\sin x - 1 \geq 0\) を解け。

答えを見る

\(\sin x \geq \dfrac{1}{2}\) となる

境界：\(\sin x = \dfrac{1}{2}\) より \(x = \dfrac{\pi}{6}, \dfrac{5\pi}{6}\)

等号を含むので、境界も含めて

\(\boxed{\dfrac{\pi}{6} \leq x \leq \dfrac{5\pi}{6}}\)

練習3

\(0 \leq x < 2\pi\) のとき、\(\sin x - \cos x = 1\) を解け。

答えを見る

合成：\(\sin x - \cos x = \sqrt{2}\sin\left(x - \dfrac{\pi}{4}\right)\)

\(\sqrt{2}\sin\left(x - \dfrac{\pi}{4}\right) = 1\) より \(\sin\left(x - \dfrac{\pi}{4}\right) = \dfrac{1}{\sqrt{2}}\)

\(x - \dfrac{\pi}{4} = \dfrac{\pi}{4}\) または \(x - \dfrac{\pi}{4} = \dfrac{3\pi}{4}\)

\(x = \dfrac{\pi}{2}\) または \(x = \pi\)

\(\boxed{x = \dfrac{\pi}{2}, \pi}\)

📌 まとめ：合成で2つを1つにまとめ、単位円で方程式・不等式を解く。範囲と解の個数に注意！

おまけ：物理と数学「振動の合成と共鳴」

📖 2つの波が出会うとき

音楽の世界で「うなり」という現象がある。2つの音叉を同時に鳴らすと、音が大きくなったり小さくなったりを繰り返すのだ。

これを数学で表すと、2つの振動の合成になる。例えば：

\[ y_1 = \sin(100t), \quad y_2 = \sin(102t) \]

これらを足し合わせると：

\[ y_1 + y_2 = 2\cos(t)\sin(101t) \]

——なんと、ゆっくり変化する \(\cos(t)\) と速く振動する \(\sin(101t)\) の積になった！

\(\cos(t)\) の部分が「音の大きさ」の変化を表し、これが1秒に1回のペースで変化する。だから音が「ウワンウワン」と聞こえるのだ。

さらに面白いのは「共鳴」。

ブランコを押すとき、ブランコの揺れに合わせて押すと、どんどん大きく揺れる。これは「外力の振動数 = 固有振動数」のとき起こる。

1940年、アメリカのタコマ・ナローズ橋は風との共鳴で崩壊した。風の振動数と橋の固有振動数が一致し、橋が激しく揺れ始めたのだ。

三角関数の合成は、単なる数学の技巧ではない。波動、音響、電気回路、建築——あらゆる「振動」を理解するための基本言語なのだ。

...ちなみに、タコマ・ナローズ橋の崩壊映像は今でもYouTubeで見られる。物理学を学ぶ者なら一度は見ておくべき、三角関数が現実を動かす瞬間だ。
