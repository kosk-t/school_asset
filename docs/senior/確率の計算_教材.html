1. テーマの定義

サイコロを振って6が出る確率は？ くじを引いて当たる確率は？ 雨が降る確率は？ 私たちは日常的に「確率」を使っている。

このページでは、確率の計算方法を学びます：

  • 確率の基本公式：P(A) = （Aが起こる場合の数）÷（全体の場合の数）
  • 余事象：「〜でない確率」= 1 −「〜の確率」
  • 加法定理：「AまたはB」の確率の求め方
  • 排反事象：同時に起こらない事象の扱い方

2. 覚えるルール

🎲 最重要：確率 \(P(A) = \frac{\text{事象Aの場合の数}}{\text{全事象の場合の数}} = \frac{n(A)}{n(U)}\)
確率は「全体の中でAが占める割合」
📌 ルール1：確率の基本公式 → 例1
\[ P(A) = \frac{n(A)}{n(U)} = \frac{\text{事象Aが起こる場合の数}}{\text{起こりうるすべての場合の数}} \]

前提：すべての結果が同様に確からしいとき（どの結果も同じ確率で起こる）

確率の性質：

  • \(0 \leq P(A) \leq 1\)（確率は0以上1以下）
  • \(P(U) = 1\)（何かは必ず起こる）
  • \(P(\emptyset) = 0\)（起こりえないことの確率は0）
📌 ルール2：余事象の確率 → 例2
\[ P(\overline{A}) = 1 - P(A) \]

余事象 \(\overline{A}\)：事象Aが起こらないという事象

→ だから結果は：「Aでない」確率 = 1 −「Aの確率」

✅ メリット：「少なくとも1つ〜」「〜でない」は余事象が楽！

💡 考え方を見る

全体（100%）からAが起こる確率を引けば、Aが起こらない確率が残る。

「少なくとも1つ当たる」の余事象は「1つも当たらない」で計算が単純。

📌 ルール3：加法定理（和事象の確率） → 例3
\[ P(A \cup B) = P(A) + P(B) - P(A \cap B) \]

和事象 \(A \cup B\)：AまたはBが起こる事象

積事象 \(A \cap B\)：AかつBが起こる事象

→ だから結果は：単純に足すと重複する部分を2回数えるので、引く

📌 ルール4：排反事象の加法定理 → 例4
\[ \text{AとBが排反のとき：} P(A \cup B) = P(A) + P(B) \]

排反事象：AとBが同時に起こらない（\(A \cap B = \emptyset\)）

例：サイコロで「1が出る」と「6が出る」は排反（同時に起こらない）

→ だから結果は：重複がないので、単純に足せる

📌 ルール5：「少なくとも〜」は余事象 → 例5
\[ P(\text{少なくとも1つ〜}) = 1 - P(\text{1つも〜ない}) \]

理由：「少なくとも1つ」を直接数えると大変。「1つもない」の方が簡単。

例：「少なくとも1回当たる」= 1 −「1回も当たらない」

3. 例題と解き方

確率の計算を実際にやってみましょう。

▶ 基本の確率計算

例1サイコロの確率

1個のサイコロを投げるとき、3以下の目が出る確率を求めよ。

解き方：

  1. 全事象を確認：目の出方は 1, 2, 3, 4, 5, 6 の6通り
  2. 事象Aを確認：3以下の目は 1, 2, 3 の3通り
  3. 公式に代入
\[ P(A) = \frac{n(A)}{n(U)} = \frac{3}{6} = \frac{1}{2} \]
例1bトランプの確率

52枚のトランプから1枚引くとき、ハートが出る確率を求めよ。

解き方：

  1. 全事象：52枚（どのカードも同様に確からしい）
  2. ハートのカード：13枚
\[ P(\text{ハート}) = \frac{13}{52} = \frac{1}{4} \]

▶ 余事象を使う

例2余事象の基本

1個のサイコロを投げるとき、1以外の目が出る確率を求めよ。

解き方：

  1. 直接計算：2, 3, 4, 5, 6 の5通りで \(\frac{5}{6}\)
  2. または余事象を使う：\(P(\text{1以外}) = 1 - P(\text{1}) = 1 - \frac{1}{6} = \frac{5}{6}\)
\[ P(\overline{A}) = 1 - P(A) = 1 - \frac{1}{6} = \frac{5}{6} \]
例5少なくとも1回当たる

当たりが2本、はずれが3本のくじがある。くじを1本ずつ2回引く（引いたくじは戻す）とき、少なくとも1回当たる確率を求めよ。

解き方：

  1. 「少なくとも1回当たる」の余事象は「2回ともはずれ」
  2. 1回引いてはずれる確率：\(\frac{3}{5}\)
  3. 2回ともはずれる確率：\(\frac{3}{5} \times \frac{3}{5} = \frac{9}{25}\)
  4. 余事象で求める
\[ P(\text{少なくとも1回当たる}) = 1 - P(\text{2回ともはずれ}) = 1 - \frac{9}{25} = \frac{16}{25} \]

▶ 加法定理を使う

例3「または」の確率

52枚のトランプから1枚引くとき、ハートまたは絵札が出る確率を求めよ。

解き方：

  1. ハートの枚数：13枚
  2. 絵札の枚数：12枚（J, Q, K × 4マーク）
  3. ハートかつ絵札：3枚（ハートのJ, Q, K）
  4. 加法定理を適用
\begin{align} P(\text{ハート} \cup \text{絵札}) &= P(\text{ハート}) + P(\text{絵札}) - P(\text{ハート} \cap \text{絵札}) \\ &= \frac{13}{52} + \frac{12}{52} - \frac{3}{52} = \frac{22}{52} = \frac{11}{26} \end{align}
例4排反事象

1個のサイコロを投げるとき、1または6が出る確率を求めよ。

解き方：

  1. 「1が出る」と「6が出る」は排反（同時に起こらない）
  2. 排反のとき、単純に足せる
\[ P(1 \cup 6) = P(1) + P(6) = \frac{1}{6} + \frac{1}{6} = \frac{2}{6} = \frac{1}{3} \]

ポイント：「1かつ6」は起こりえない（\(P(1 \cap 6) = 0\)）ので引く必要なし

▶ 組合せを使った確率

例6くじ引きの確率

10本のくじの中に当たりが3本ある。3本同時に引くとき、2本が当たりである確率を求めよ。

解き方：

  1. 全事象：10本から3本を選ぶ = \(_{10}C_3\) 通り
  2. 事象A：当たり3本から2本、はずれ7本から1本を選ぶ = \(_3C_2 \times _7C_1\) 通り
\[ P(\text{当たり2本}) = \frac{_3C_2 \times _7C_1}{_{10}C_3} = \frac{3 \times 7}{120} = \frac{21}{120} = \frac{7}{40} \]
例7グループ分けの確率

男子4人、女子3人の計7人から3人を選ぶとき、男子2人、女子1人が選ばれる確率を求めよ。

解き方：

  1. 全事象：7人から3人を選ぶ = \(_7C_3 = 35\) 通り
  2. 事象A：男子4人から2人、女子3人から1人 = \(_4C_2 \times _3C_1 = 6 \times 3 = 18\) 通り
\[ P = \frac{_4C_2 \times _3C_1}{_7C_3} = \frac{6 \times 3}{35} = \frac{18}{35} \]

4. 簡単にするコツ

✏️ コツ1：「少なくとも〜」は余事象
「少なくとも1回当たる」を直接数えると、1回当たる、2回当たる...と面倒。
→ 1 −「1回も当たらない」の方が圧倒的に楽！
✏️ コツ2：「または」は加法定理
「AまたはB」= P(A) + P(B) − P(A∩B)
ただしAとBが排反なら、P(A) + P(B) でOK（引く必要なし）
✏️ コツ3：分母と分子を分けて考える
確率 = （条件を満たす場合の数）÷（全体の場合の数）
それぞれ独立に計算してから割り算する
✏️ コツ4：「同時に」と「順番に」で考え方を変える
「3枚同時に引く」→ 組合せ \(_nC_r\)
「1枚ずつ順に引く」→ 順列 or 積の法則

5. よくあるつまづき

⚠️ つまづき1：加法定理で重複を引き忘れる

ダメな例：「ハートまたは絵札」で \(\frac{13}{52} + \frac{12}{52} = \frac{25}{52}\) と計算

対策：「ハートかつ絵札」（3枚）を引く。答えは \(\frac{22}{52}\)

⚠️ つまづき2：排反でないのに単純に足す

ダメな例：「偶数または4以下」でP(偶数) + P(4以下) と計算

対策：2と4は両方に含まれる → 排反ではない → 重複を引く必要あり

⚠️ つまづき3：余事象を使わず直接計算して混乱

ダメな例：「少なくとも1回当たる」で「1回だけ当たる」「2回当たる」...を全部足す

対策：1 −「1回も当たらない」で一発計算

⚠️ つまづき4：「同様に確からしい」を確認しない

ダメな例：歪んだサイコロで \(P(6) = \frac{1}{6}\) と計算

対策：「同様に確からしい」が前提。問題文で確認する

6. 仕上げチェックリスト

確率の基本公式 \(P(A) = \frac{n(A)}{n(U)}\) が使える
余事象の公式 \(P(\overline{A}) = 1 - P(A)\) が使える
加法定理 \(P(A \cup B) = P(A) + P(B) - P(A \cap B)\) が使える
排反事象のとき、単純に足せることを知っている
「少なくとも〜」は余事象で解く習慣がある
組合せを使った確率計算ができる

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

2個のサイコロを投げるとき、目の和が7になる確率を求めよ。

答えを見る

全事象：6 × 6 = 36通り

和が7：(1,6), (2,5), (3,4), (4,3), (5,2), (6,1) の6通り

\(P = \frac{6}{36} = \frac{1}{6}\)

練習2

52枚のトランプから1枚引くとき、スペードまたはエースが出る確率を求めよ。

答えを見る

スペード：13枚、エース：4枚、スペードのエース：1枚

\(P = \frac{13}{52} + \frac{4}{52} - \frac{1}{52} = \frac{16}{52} = \frac{4}{13}\)

練習3

コインを3回投げるとき、少なくとも1回表が出る確率を求めよ。

答えを見る

余事象「3回とも裏」を使う

3回とも裏：\(\left(\frac{1}{2}\right)^3 = \frac{1}{8}\)

\(P(\text{少なくとも1回表}) = 1 - \frac{1}{8} = \frac{7}{8}\)

📌 まとめ：確率は「場合の数の割合」。余事象・加法定理を使いこなせ！

おまけ：実践ストーリー「モンティ・ホール問題」

🚪 ドアを変えるべきか？

有名なテレビ番組「Let's Make a Deal」から生まれた、モンティ・ホール問題。確率の直感がいかに当てにならないかを示す、数学史上最も議論を呼んだ問題のひとつだ。

ルール：

  • 3つのドアがあり、1つの後ろに車、残り2つの後ろにヤギがいる
  • あなたは1つのドアを選ぶ
  • 司会者（モンティ）は、あなたが選ばなかったドアのうち、ヤギがいるドアを1つ開ける
  • 「選択を変えますか？」と聞かれる

問題：ドアを変えるべきか？ それとも最初の選択を維持すべきか？

直感的な答え：「2つのドアが残っているから、確率は \(\frac{1}{2}\) ずつ。変えても変えなくても同じ」

しかし、これは間違い！

\[ P(\text{変えて当たる}) = \frac{2}{3}, \quad P(\text{変えずに当たる}) = \frac{1}{3} \]

なぜ？

  • 最初に車を選ぶ確率：\(\frac{1}{3}\)（この場合、変えると負け）
  • 最初にヤギを選ぶ確率：\(\frac{2}{3}\)（この場合、変えると必ず勝ち）

司会者は必ずヤギのドアを開けるというルールがポイント。これにより、最初にヤギを選んでいた場合、変えれば100%車に当たる。

教訓：確率は直感に反することがある。計算して確かめよう。

この問題は1990年にマリリン・ボス・サバントがパレード誌のコラムで「変えた方がいい」と答え、 数学者を含む約1万人から「間違いだ」という抗議の手紙が届いた。 しかしシミュレーションで確認すると...彼女が正しかった。
