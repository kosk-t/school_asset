1. テーマの定義

「あの木の高さは何メートル？」「この坂道の傾斜は何度？」——直接測れないものでも、角度と辺の関係を使えば計算で求められます。

このページでは、三角比を学びます：

  • 三角比の定義：sin（サイン）、cos（コサイン）、tan（タンジェント）の3つ
  • 特別な角の値：30°、45°、60°は暗記必須
  • 相互関係：sin、cos、tanの間に成り立つ便利な公式

2. 覚えるルール

🎯 最重要：SOH-CAH-TOA（ソー・カー・トア）で覚える！ sin = 対/斜、cos = 隣/斜、tan = 対/隣
三角比の定義：角θから見て「対辺」「隣辺」「斜辺」を使い分ける
📌 ルール1：三角比の定義（SOH-CAH-TOA） → 例1
\[ \sin\theta = \frac{\text{対辺}}{\text{斜辺}}, \quad \cos\theta = \frac{\text{隣辺}}{\text{斜辺}}, \quad \tan\theta = \frac{\text{対辺}}{\text{隣辺}} \]

直角三角形で、注目する角θから見て：

  • 対辺（opposite）：θの向かい側にある辺
  • 隣辺（adjacent）：θのとなりにある辺（斜辺ではない方）
  • 斜辺（hypotenuse）：直角の向かい側、一番長い辺

→ だから結果は： 辺の比を計算すれば、角度の情報が数値で得られる

💡 覚え方を見る

SOH-CAH-TOA（ソー・カー・トア）で覚える！英語の頭文字を取った語呂合わせ：

  • SOH：Sin = Opposite / Hypotenuse（対辺÷斜辺）
  • CAH：Cos = Adjacent / Hypotenuse（隣辺÷斜辺）
  • TOA：Tan = Opposite / Adjacent（対辺÷隣辺）

「ソー・カー・トア」と3回唱えれば覚えられる！

📌 ルール2：特別な角の三角比（暗記必須！） → 例2・3

30°、45°、60°の三角比は頻出なので必ず暗記：

θ sin θ cos θ tan θ
30° \(\dfrac{1}{2}\) \(\dfrac{\sqrt{3}}{2}\) \(\dfrac{1}{\sqrt{3}} = \dfrac{\sqrt{3}}{3}\)
45° \(\dfrac{1}{\sqrt{2}} = \dfrac{\sqrt{2}}{2}\) \(\dfrac{1}{\sqrt{2}} = \dfrac{\sqrt{2}}{2}\) \(1\)
60° \(\dfrac{\sqrt{3}}{2}\) \(\dfrac{1}{2}\) \(\sqrt{3}\)
特別な直角三角形の辺の比：これを使えば三角比の値が出る
💡 覚え方を見る

「1, 2, 3, √」の法則で覚える：

  • sin：分子は 30° → 1、45° → √2、60° → √3（1, √2, √3 と増える）
  • cos：分子は 30° → √3、45° → √2、60° → 1（√3, √2, 1 と減る）
  • 分母は sin, cos ともに 2（45°のときは √2 だが、有理化すると分母2）

または図で覚える：45°-45°-90°の比「1:1:√2」、30°-60°-90°の比「1:√3:2」を図で暗記して、定義に当てはめる。

📌 ルール3：三角比の相互関係 → 例4
\[ \tan\theta = \frac{\sin\theta}{\cos\theta}, \quad \sin^2\theta + \cos^2\theta = 1 \]

\(\sin^2\theta\)（「サイン二乗シータ」と読む）は \((\sin\theta)^2\) のこと。

→ だから結果は： sin と cos がわかれば tan も計算できる。また、片方がわかればもう片方も求まる。

💡 考え方を見る

tanの公式は定義から：\(\tan\theta = \dfrac{\text{対辺}}{\text{隣辺}} = \dfrac{\text{対辺}/\text{斜辺}}{\text{隣辺}/\text{斜辺}} = \dfrac{\sin\theta}{\cos\theta}\)

sin² + cos² = 1 は三平方の定理から：\(\left(\dfrac{\text{対辺}}{\text{斜辺}}\right)^2 + \left(\dfrac{\text{隣辺}}{\text{斜辺}}\right)^2 = \dfrac{\text{対辺}^2 + \text{隣辺}^2}{\text{斜辺}^2} = \dfrac{\text{斜辺}^2}{\text{斜辺}^2} = 1\)

📌 ルール4：90° - θ の関係 → 例5
\[ \sin(90° - \theta) = \cos\theta, \quad \cos(90° - \theta) = \sin\theta \]

→ だから結果は： sin 30° = cos 60°、cos 45° = sin 45° など、足して90°になる角のsinとcosは入れ替わる

✅ メリット： 覚える値が半分で済む！

3. 例題と解き方

三角比の定義と特別な角の値を使って、実際に計算してみましょう。

▶ 定義から三角比を求める

例1辺の長さから三角比を求める

直角三角形ABCで、∠C = 90°、AB = 5、BC = 3、CA = 4 のとき、sin A、cos A、tan A を求めよ。

解き方：

  1. ∠Aから見て、対辺 = BC = 3、隣辺 = CA = 4、斜辺 = AB = 5
  2. 定義に当てはめる
\[ \sin A = \frac{3}{5}, \quad \cos A = \frac{4}{5}, \quad \tan A = \frac{3}{4} \]

▶ 特別な角の三角比

例245°の三角比

\(\sin 45°\)、\(\cos 45°\)、\(\tan 45°\) の値を求めよ。

解き方：

  1. 45°-45°-90°の直角三角形の辺の比は 1 : 1 : √2
  2. 対辺 = 1、隣辺 = 1、斜辺 = √2 として定義に代入
\[ \sin 45° = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}, \quad \cos 45° = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}, \quad \tan 45° = \frac{1}{1} = 1 \]
例330°と60°の三角比

\(\sin 30°\)、\(\cos 60°\) の値を求めよ。

解き方：

  1. 30°-60°-90°の直角三角形の辺の比は 1 : √3 : 2（30°の向かいが1、60°の向かいが√3、斜辺が2）
  2. sin 30° → 対辺(30°の向かい) = 1、斜辺 = 2
  3. cos 60° → 隣辺(60°のとなり) = 1、斜辺 = 2
\[ \sin 30° = \frac{1}{2}, \quad \cos 60° = \frac{1}{2} \]

（sin 30° = cos 60° になっている！これは90° - θの関係）

▶ 相互関係を使う

例4sin から cos を求める

\(\sin\theta = \dfrac{3}{5}\) のとき、\(\cos\theta\) を求めよ（0° < θ < 90°）。

解き方：

  1. \(\sin^2\theta + \cos^2\theta = 1\) を使う
  2. \(\left(\dfrac{3}{5}\right)^2 + \cos^2\theta = 1\)
  3. \(\dfrac{9}{25} + \cos^2\theta = 1\)
  4. \(\cos^2\theta = 1 - \dfrac{9}{25} = \dfrac{16}{25}\)
  5. 0° < θ < 90° では cos θ > 0 なので、\(\cos\theta = \dfrac{4}{5}\)
\[ \cos\theta = \sqrt{1 - \sin^2\theta} = \sqrt{1 - \frac{9}{25}} = \sqrt{\frac{16}{25}} = \boxed{\frac{4}{5}} \]
例590° - θ の関係を使う

\(\sin 70°\) を cos を使って表せ。

解き方：

  1. 90° - 70° = 20° に注目
  2. \(\sin(90° - 20°) = \cos 20°\) より、\(\sin 70° = \cos 20°\)
\[ \sin 70° = \sin(90° - 20°) = \boxed{\cos 20°} \]

4. 簡単にするコツ

✏️ コツ1：まず「どの角から見るか」を決める
三角比は「角θから見た辺の比」なので、まず注目する角を確認。その角の向かい側が対辺、となりが隣辺、直角の向かいが斜辺。
✏️ コツ2：特別な角は「図」で覚える
30°-60°-90°と45°-45°-90°の三角形を図で暗記しておけば、定義に当てはめるだけ。表を丸暗記するより応用が利く。
✏️ コツ3：有理化は最後でOK
\(\dfrac{1}{\sqrt{2}}\) のままでも間違いではない。最終的な答えで有理化を求められたら \(\dfrac{\sqrt{2}}{2}\) に直す。計算途中は分母にルートがあっても気にしない。
✏️ コツ4：sin と cos の大小関係
0° < θ < 45° なら sin θ < cos θ、θ = 45° なら sin θ = cos θ、45° < θ < 90° なら sin θ > cos θ。検算に使える。

5. よくあるつまづき

⚠️ つまづき1：対辺と隣辺を間違える

ダメな例： 角θのとなりにある辺を「対辺」としてしまう

対策： 「対辺」は文字通り対面（向かい側）にある辺。θの頂点から向かいを見て、ぶつかる辺が対辺。

⚠️ つまづき2：sin と cos を逆にする

ダメな例： sin 30° = √3/2 としてしまう（正しくは 1/2）

対策： 「SOH-CAH-TOA」を3回唱えて覚える。迷ったら図を描いて定義に戻る。

⚠️ つまづき3：√2 と √3 を混同する

ダメな例： sin 60° = √2/2 としてしまう（正しくは √3/2）

対策： 45°のときだけ √2 が登場、30°と60°のときは √3 が登場。角度と√の数字を対応させて覚える。

⚠️ つまづき4：sin²θ の計算ミス

ダメな例： sin²30° = sin 60° としてしまう

対策： sin²θ は「sinθを2回かける」という意味。sin²30° = (1/2)² = 1/4 であり、sin 60° = √3/2 とは全く別物。

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

次の値を求めよ：\(\sin 60° + \cos 30°\)

答えを見る

\(\sin 60° = \dfrac{\sqrt{3}}{2}\)、\(\cos 30° = \dfrac{\sqrt{3}}{2}\)

\(\sin 60° + \cos 30° = \dfrac{\sqrt{3}}{2} + \dfrac{\sqrt{3}}{2} = \boxed{\sqrt{3}}\)

練習2

\(\cos\theta = \dfrac{5}{13}\) のとき、\(\sin\theta\) と \(\tan\theta\) を求めよ（0° < θ < 90°）。

答えを見る

\(\sin^2\theta + \cos^2\theta = 1\) より

\(\sin^2\theta = 1 - \dfrac{25}{169} = \dfrac{144}{169}\)

\(\sin\theta = \dfrac{12}{13}\)（θは鋭角なので正）

\(\tan\theta = \dfrac{\sin\theta}{\cos\theta} = \dfrac{12/13}{5/13} = \boxed{\dfrac{12}{5}}\)

📌 まとめ：sin = 対/斜、cos = 隣/斜、tan = 対/隣。「SOH-CAH-TOA」（ソー・カー・トア）で覚えよう！

おまけ：歴史ストーリー「ピラミッドの高さを測った男」

📖 タレスとピラミッド（紀元前600年頃・エジプト）

古代ギリシャの哲学者タレスがエジプトを訪れたとき、現地の人々からこう言われた。

「あのピラミッドの高さを知りたいのだが、登るわけにもいかないし、測る方法がない」

タレスは笑って答えた。「簡単なことさ」

彼は自分の影の長さを測った。身長と影の長さが同じになる瞬間——つまり太陽が45°の角度にある時間を待ったのだ。

「今だ！」

タレスはピラミッドの影の長さを測った。太陽が45°なら、物体の高さ = 影の長さ。つまり tan 45° = 1 を使ったのだ。

\[ \tan 45° = \frac{\text{高さ}}{\text{影の長さ}} = 1 \quad \therefore \text{高さ} = \text{影の長さ} \]

エジプト人たちは驚いた。「登らずに高さがわかるのか！」

タレスは肩をすくめた。「角度と比があれば、見えないものも測れる。それが数学だ」

...ちなみにこのタレス、「万物の根源は水である」と主張した人物でもある。三角比の先駆者が、同時に哲学の祖でもあったというのは面白い話だ。
