1. テーマの定義

「\(\sqrt{2}\) は分数で表せない」——この事実は、古代ギリシャの数学者たちを震撼させました。高校では、数をより厳密に分類し、絶対値や二重根号などの扱いを学びます。

このページでは、実数と絶対値を学びます：

  • 数の分類：有理数と無理数、実数の体系を理解する
  • 絶対値：\(|x|\) の定義と性質、方程式・不等式での扱い
  • 二重根号：\(\sqrt{a + \sqrt{b}}\) の形を外す技術

2. 覚えるルール

🎯 最重要：絶対値は「場合分け」で外す！ \(|x| = x\)（\(x \geq 0\)のとき）、\(|x| = -x\)（\(x < 0\)のとき）
数の分類：実数 = 有理数 + 無理数
📌 ルール1：数の分類

有理数：整数 \(a, b\)（\(b \neq 0\)）を使って \(\dfrac{a}{b}\) と表せる数

  • 整数（\(\cdots, -2, -1, 0, 1, 2, \cdots\)）
  • 有限小数（例：\(0.25 = \dfrac{1}{4}\)）
  • 循環小数（例：\(0.333\cdots = \dfrac{1}{3}\)）

無理数：分数で表せない数（循環しない無限小数）

  • 例：\(\sqrt{2}, \sqrt{3}, \sqrt{5}, \pi, e\)

実数：有理数と無理数をあわせたもの（数直線上のすべての点）

📌 ルール2：絶対値の定義 → 例1・2
\[ |x| = \begin{cases} x & (x \geq 0 \text{ のとき}) \\ -x & (x < 0 \text{ のとき}) \end{cases} \]

意味：「原点からの距離」。マイナスを取り除いて非負にする。

→ だから結果は： \(|x| \geq 0\) が常に成り立つ

💡 考え方を見る

\(x = -3\) のとき、\(|x| = |-3| = -(-3) = 3\)

「マイナスにマイナスをつけてプラスにする」という操作。

📌 ルール3：絶対値の性質 → 例3・4
\[ |a|^2 = a^2, \quad |ab| = |a||b|, \quad \left|\frac{a}{b}\right| = \frac{|a|}{|b|}\ (b \neq 0) \]

重要な関係式：

  • \(\sqrt{a^2} = |a|\)（平方根を外すと絶対値がつく！）
  • \(-|a| \leq a \leq |a|\)
📌 ルール4：二重根号の外し方 → 例5・6
\[ \sqrt{a + b + 2\sqrt{ab}} = \sqrt{a} + \sqrt{b} \quad (a, b > 0) \] \[ \sqrt{a + b - 2\sqrt{ab}} = |\sqrt{a} - \sqrt{b}| \quad (a, b > 0) \]

手順：

  1. \(\sqrt{\ }\) の中を「\((\sqrt{a} \pm \sqrt{b})^2\)」の形に変形
  2. 足して \(a+b\)、掛けて \(ab\) になる2数を探す
  3. 二重根号を外す（マイナスの場合は絶対値に注意）

✅ メリット： 複雑な根号を簡単な形に変形できる

3. 例題と解き方

ルールを使って実際に問題を解いてみましょう。

▶ 絶対値の計算

例1\(|3 - \pi|\) の値を求める

解き方：

  1. \(\pi \approx 3.14\) なので、\(3 - \pi < 0\)
  2. 中身が負なので、\(|3 - \pi| = -(3 - \pi)\)
\[ |3 - \pi| = -(3 - \pi) = \pi - 3 \]
例2\(\sqrt{(x-2)^2}\) を簡単にする

解き方：

  1. \(\sqrt{a^2} = |a|\) の公式を使う
  2. \(\sqrt{(x-2)^2} = |x-2|\)
  3. \(x\) の値で場合分けが必要な場合は分ける
\[ \sqrt{(x-2)^2} = |x-2| = \begin{cases} x-2 & (x \geq 2) \\ -(x-2) = 2-x & (x < 2) \end{cases} \]

▶ 絶対値を含む方程式・不等式

例3\(|x - 3| = 5\) を解く

解き方：

  1. 絶対値が5 → 「中身が5 または −5」
  2. \(x - 3 = 5\) または \(x - 3 = -5\)
\[ x - 3 = 5 \Rightarrow x = 8, \quad x - 3 = -5 \Rightarrow x = -2 \] \[ \therefore x = 8, -2 \]
例4\(|x - 1| < 3\) を解く

解き方：

  1. 「原点からの距離が3未満」→ はさみうちの形に変形
  2. \(-3 < x - 1 < 3\)
  3. 各辺に1を足す
\[ -3 < x - 1 < 3 \Rightarrow -2 < x < 4 \]

▶ 二重根号

例5\(\sqrt{5 + 2\sqrt{6}}\) を簡単にする

解き方：

  1. \(2\sqrt{6} = 2\sqrt{ab}\) より、\(ab = 6\) となる2数を探す
  2. 足して5、掛けて6 → \(a = 3, b = 2\)
  3. \(\sqrt{3} + \sqrt{2}\) の形になる
\[ \sqrt{5 + 2\sqrt{6}} = \sqrt{3 + 2 + 2\sqrt{3 \cdot 2}} = \sqrt{(\sqrt{3} + \sqrt{2})^2} = \sqrt{3} + \sqrt{2} \]
例6\(\sqrt{7 - 4\sqrt{3}}\) を簡単にする

解き方：

  1. \(4\sqrt{3} = 2\sqrt{12} = 2\sqrt{4 \cdot 3}\) と変形
  2. 足して7、掛けて12 → \(a = 4, b = 3\)
  3. マイナスなので \(|\sqrt{4} - \sqrt{3}| = |2 - \sqrt{3}|\)
  4. \(2 > \sqrt{3}\) なので絶対値を外せる
\[ \sqrt{7 - 4\sqrt{3}} = \sqrt{4 + 3 - 2\sqrt{4 \cdot 3}} = |2 - \sqrt{3}| = 2 - \sqrt{3} \]

▶ 有理数・無理数の判定

例7\(\sqrt{12}\) は有理数か無理数か

解き方：

  1. \(\sqrt{12} = \sqrt{4 \cdot 3} = 2\sqrt{3}\)
  2. \(\sqrt{3}\) は無理数
  3. 有理数 × 無理数 = 無理数
\[ \sqrt{12} = 2\sqrt{3} \quad \text{（無理数）} \]

4. 簡単にするコツ

✏️ コツ1：絶対値は「中身の符号」で判断
\(|x-a|\) を外すとき、\(x\) と \(a\) の大小関係を考える。\(x \geq a\) なら \(x-a\)、\(x < a\) なら \(a-x\)。
✏️ コツ2：二重根号は「足してa+b、掛けてab」
\(\sqrt{a+b \pm 2\sqrt{ab}}\) の形を見たら、「足して○、掛けて△」になる2数を探す。よく出るパターン：(3,2), (5,2), (4,3), (5,3) など。
✏️ コツ3：\(\sqrt{a^2} = |a|\) を忘れない
\(\sqrt{x^2} = x\) ではない！常に \(\sqrt{x^2} = |x|\)。特に \(x\) が負になりうる場合は重要。
✏️ コツ4：絶対値の方程式は「±」で分解
\(|A| = B\) は \(A = B\) または \(A = -B\) に分解。ただし \(B < 0\) なら解なし。

5. よくあるつまづき

⚠️ つまづき1：\(\sqrt{x^2} = x\) と思い込む

ダメな例： \(\sqrt{(-3)^2} = -3\) ← 間違い！

対策： \(\sqrt{x^2} = |x|\) が正しい。\(\sqrt{(-3)^2} = \sqrt{9} = 3 = |-3|\)

⚠️ つまづき2：二重根号のマイナスで絶対値を忘れる

ダメな例： \(\sqrt{5 - 2\sqrt{6}} = \sqrt{3} - \sqrt{2}\) ← 大小関係を確認してない！

対策： マイナスの場合は \(|\sqrt{a} - \sqrt{b}|\) になる。\(\sqrt{3} > \sqrt{2}\) なので OK だが、常に確認する。

⚠️ つまづき3：絶対値不等式の向きミス

ダメな例： \(|x| > 3\) を \(-3 < x < 3\) と解く

対策： \(|x| < 3\) は「挟む」→ \(-3 < x < 3\)。\(|x| > 3\) は「外側」→ \(x < -3\) または \(x > 3\)。

6. 仕上げチェックリスト

有理数と無理数の違いを説明できる
\(\sqrt{x^2} = |x|\) を使いこなせる
絶対値を含む方程式・不等式を場合分けで解ける
二重根号を「足して○、掛けて△」で外せる
\(\sqrt{2}, \sqrt{3}, \sqrt{5}\) などが無理数であることを知っている

7. 練習問題

学んだことを使って、以下の問題を解いてみよう。

練習1

\(|2 - \sqrt{5}|\) の値を求めよ。

答えを見る

\(\sqrt{5} \approx 2.236\) より \(2 - \sqrt{5} < 0\)

\(|2 - \sqrt{5}| = -(2 - \sqrt{5}) = \sqrt{5} - 2\)

練習2

\(|2x - 1| \leq 5\) を解け。

答えを見る

\(-5 \leq 2x - 1 \leq 5\)

\(-4 \leq 2x \leq 6\)

\(-2 \leq x \leq 3\)

練習3

\(\sqrt{6 + 2\sqrt{5}}\) を簡単にせよ。

答えを見る

足して6、掛けて5 → \(a = 5, b = 1\)

\(\sqrt{6 + 2\sqrt{5}} = \sqrt{5 + 1 + 2\sqrt{5 \cdot 1}} = \sqrt{(\sqrt{5} + 1)^2} = \sqrt{5} + 1\)

📌 まとめ：絶対値は「中身の符号で場合分け」！二重根号は「足して○、掛けて△」で外す！

おまけ：歴史ストーリー「√2を発見した男の悲劇」

📖 ヒッパソスとピタゴラス学派の秘密（紀元前5世紀 古代ギリシャ）

紀元前500年頃、古代ギリシャにピタゴラス学派という数学者集団がいた。彼らの信条は「万物は整数の比で表せる」。

世界の全ては、\(\dfrac{1}{2}\) とか \(\dfrac{3}{4}\) のような「きれいな分数」で表現できる——それが宇宙の真理だと信じていた。

ところがある日、学派のメンバーヒッパソスが、1辺が1の正方形の対角線の長さを計算していた。

\[ \text{対角線} = \sqrt{1^2 + 1^2} = \sqrt{2} \]

「この \(\sqrt{2}\) を分数で表すと...」

彼は何度計算しても、\(\sqrt{2}\) を分数で表すことができなかった。そして背理法で、「\(\sqrt{2}\) は絶対に分数で表せない」ことを証明してしまった。

ピタゴラス学派のリーダー：「は？ そんなわけないだろ。全ては整数比で表せるはずだ」

ヒッパソス：「でも、証明できちゃったんですけど...」

学派の信条が根底から崩れる大発見。しかし、これは学派にとって絶対に公開してはならない秘密だった。

伝説によると、ヒッパソスはその後、航海中に海に突き落とされたという。

「秘密を漏らした罰」なのか「事故」なのか、真相は今も分からない。ただ一つ確かなのは、\(\sqrt{2}\) が分数で表せないという事実は、数学の歴史を変える大発見だったということ。

教訓：上司の信念を覆す発見をしたときは、発表のタイミングに気をつけよう。
